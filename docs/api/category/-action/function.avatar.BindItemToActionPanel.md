# avatar.BindItemToActionPanel

Привязывает предмет к слоту панели действий.

## Описание

```lua
avatar.BindItemToActionPanel( itemId:, index )
```

Функция привязывает указанный предмет к заданному слоту панели действий. Привязываемый предмет должен относиться к одной из следующих категорий: используемые предметы (usable), оружие или одежда.

::: warning Замечание
Сторонним дополнениям запрещено вызывать данную функцию, пока персонаж игрока находится в бою.
:::

## Список параметров

- **`itemId`** (`ObjectId`)
Идентификатор предмета, привязываемого к слоту панели действий.

- **`index`** (`number`)
Индекс слота панели действий. Значение находится в диапазоне от `0` до максимального количества слотов.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Привязка предмета из инвентаря к нулевому слоту панели

```lua
local itemId = inventoryIds[0]
avatar.BindItemToActionPanel( itemId, 0 )
```

::: info Описание примера
В примере извлекается идентификатор предмета из таблицы `inventoryIds`. Затем вызывается функция `avatar.BindItemToActionPanel` для привязки этого предмета к нулевому слоту панели действий.
:::
