# avatar.BindMountSkinToActionPanel

Привязывает вызов маунта с определенным скином из конюшни главного игрока к слоту панели действий.

## Описание

```lua
avatar.BindMountSkinToActionPanel( skinId, index )
```

Функция привязывает использование конкретного скина маунта из конюшни к указанному слоту панели действий главного персонажа. После выполнения привязки игрок получает возможность вызывать данного маунта напрямую из заданного слота панели действий.

## Список параметров

- **`skinId`** (`ObjectId`)
Уникальный идентификатор скина маунта из конюшни, который привязывается к слоту.

- **`index`** (`number`)
Индекс слота панели действий, к которому осуществляется привязка. Нумерация слотов начинается с нуля `0`.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Привязка скина маунта к первому слоту панели действий

```lua
-- Идентификатор скина маунта из конюшни
local mountSkinId = 123456 

-- Привязка маунта к слоту с индексом 0
avatar.BindMountSkinToActionPanel( mountSkinId, 0 )
```
