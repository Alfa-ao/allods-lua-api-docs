# avatar.GetMaxActionCount

Возвращает общее количество слотов на всех панелях действий (ActionBar), включая пустые и заполненные.

## Описание

```lua
avatar.GetMaxActionCount()
```

Функция вычисляет и возвращает суммарное количество ячеек на всех панелях действий персонажа. В подсчет включаются все слоты, независимо от того, назначено ли в них конкретное действие или они остаются пустыми. Полученное значение отражает общую вместимость интерфейса панелей и не зависит от их текущего заполнения.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` — целое число, представляющее общее количество слотов на всех панелях действий.

## Примеры

### Получение общего количества слотов панелей действий

```lua
local maxActionCount = avatar.GetMaxActionCount()
```

::: info Описание примера
В переменную `maxActionCount` записывается целое число, равное суммарному количеству всех доступных слотов на панелях действий. Данное значение может быть использовано для организации циклов перебора ячеек, проверки лимитов интерфейса или инициализации массивов, связанных с действиями персонажа.
:::
