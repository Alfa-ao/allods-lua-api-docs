# avatar.GetPetAggroMode

Возвращает текущий тип поведения пета аватара.

## Описание

```lua
avatar.GetPetAggroMode()
```

Функция извлекает и возвращает числовое значение, представляющее текущий режим агрессии (тип поведения) питомца аватара. Возможные значения типа поведения соответствуют константам перечисления `PET_AGGRO_*`, которые также используются при установке режима через функцию `avatar.SetPetAggroMode`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` — текущий тип поведения питомца. Значение соответствует одной из констант перечисления `PET_AGGRO_*`.

## Примеры

### Получение текущего режима поведения питомка

```lua
local currentMode = avatar.GetPetAggroMode()
if currentMode == PET_AGGRO_PASSIVE then
    -- Пет находится в пассивном режиме
end
```

::: info Описание примера
В примере запрашивается текущий тип поведения питомца. Полученное значение сохраняется в переменную `currentMode` и сравнивается с константой `PET_AGGRO_PASSIVE`, что позволяет определить, находится ли питомец в пассивном режиме.
:::
