# avatar.PetAttack

Инициирует атаку питомца аватара на выбранную цель.

## Описание

```lua
avatar.PetAttack()
```

Функция отдает команду питомцу аватара начать атаку текущей выбранной цели. Механизм работает в связке с системой выбора цели интерфейса: питомец атакует объект, который в данный момент выбран главным персонажем. Если у персонажа отсутствует призванный питомец или цель недоступна, вызов функции игнорируется без генерации критических ошибок.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Инициирование атаки питомца по выбранной цели

```lua
avatar.PetAttack()
```

::: info Описание примера
Вызов функции отдает прямую команду питомцу атаковать текущую выбранную цель. Если питомец активен и цель валидна, начинается процесс сближения и нанесения урона.
:::
