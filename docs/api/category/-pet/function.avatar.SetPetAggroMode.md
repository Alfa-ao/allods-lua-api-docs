# avatar.SetPetAggroMode

Устанавливает тип поведения для пета аватара.

## Описание

```lua
avatar.SetPetAggroMode( mode )
```

Функция задает режим поведения питомца (пета) главного персонажа. Режим определяет алгоритм действий питомца в окружающем мире и во время боевых столкновений. Допустимые значения параметра `mode` соответствуют константам перечисления `PET_AGGRO_*`:
- `PET_AGGRO_AGGRESIVE` — агрессивный тип поведения. Питомец автоматически атакует противников в радиусе своей видимости.
- `PET_AGGRO_DEFENSIVE` — защитный тип поведения. Питомец вступает в бой только при получении урона или при атаке на владельца.
- `PET_AGGRO_PASSIVE` — пассивный тип поведения. Питомец не атакует противников по своей инициативе.

## Список параметров

- **`mode`** (`number`)
Тип поведения питомца. Значение должно соответствовать одной из констант перечисления `PET_AGGRO_*`.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Перевод питомца в защитный режим

```lua
avatar.SetPetAggroMode( PET_AGGRO_DEFENSIVE )
```

::: info Описание примера
Вызов функции устанавливает для питомца защитный режим (`PET_AGGRO_DEFENSIVE`). После выполнения операции питомец прекращает самостоятельные атаки и начинает реагировать только на угрозы, направленные непосредственно на него или на владельца.
:::
