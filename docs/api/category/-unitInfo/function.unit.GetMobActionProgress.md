# unit.GetMobActionProgress

Извлекает информацию о текущем прогрессе выполнения действия мобом.

## Описание

```lua
unit.GetMobActionProgress( unitId )
```

Функция запрашивает и возвращает данные о прогрессе произнесения заклинания или выполнения иного действия указанным мобом. Если у моба отсутствует активное действие или данные о прогрессе недоступны, возвращается `nil`.

Изменения состояния прогресса действия сопровождаются приходом событий `EVENT_MOB_ACTION_PROGRESS_START` (начало действия), `EVENT_MOB_ACTION_PROGRESS_FREEZE` (приостановка) и `EVENT_MOB_ACTION_PROGRESS_FINISH` (завершение действия).

## Список параметров

- **`unitId`** (`ObjectId`)
Уникальный идентификатор юнита (моба), прогресс действия которого необходимо получить.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если у юнита нет активного действия, возвращается `nil`. В противном случае возвращается таблица, содержащая следующие поля:

- **`duration`** (`number`)
Общая длительность действия в миллисекундах.

- **`progress`** (`number`)
Время, прошедшее с начала выполнения действия, в миллисекундах. Значение учитывает возможные сетевые задержки при передаче события.

- **`name`** (`WString`)
Локализованное название выполняемого действия.

- **`spellId`** (`SpellId` | `nil`)
Идентификатор заклинания, если выполняемое действие связано с известным заклинанием.

- **`isPrecast`** (`boolean` | `nil`)
Признак наличия времени предварительного применения (прекаста) у заклинания. Поле присутствует, если действие является заклинанием.

- **`isChannel`** (`boolean` | `nil`)
Признак наличия механики чаннелинга (непрерывного поддержания) у заклинания. Поле присутствует, если действие является заклинанием.

## Примеры

### Получение и анализ прогресса действия моба

```lua
local unitId = 12345
local progressInfo = unit.GetMobActionProgress( unitId )

if progressInfo then
    local actionName = userMods.FromWString( progressInfo.name )
    local percent = 0
    if progressInfo.duration > 0 then
        percent = math.floor( ( progressInfo.progress / progressInfo.duration ) * 100 )
    end
    
    LogInfo( "Моб выполняет действие: " .. actionName )
    LogInfo( "Прогресс: " .. percent .. "% (" .. progressInfo.progress .. " / " .. progressInfo.duration .. " мс)" )
    
    if progressInfo.spellId then
        LogInfo( "Идентификатор заклинания: " .. progressInfo.spellId )
    end
end
```

::: info Описание примера
В примере запрашивается информация о прогрессе действия для моба с заданным идентификатором. Если функция возвращает таблицу (прогресс присутствует), извлекается название действия и преобразуется в обычную строку с помощью `userMods.FromWString`. Далее вычисляется процент выполнения действия на основе соотношения прошедшего времени (`progress`) к общей длительности (`duration`). В лог выводится название действия, процент выполнения и, при наличии, идентификатор заклинания.
:::
