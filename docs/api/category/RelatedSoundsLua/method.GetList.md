# RelatedSoundsLua:GetList

Возвращает список всех звуков в звуковой группе.

## Описание

```lua
RelatedSoundsLua:GetList(): table
```

Функция возвращает список всех звуков в звуковой группе.

## Список параметров

- **`self`** (`RelatedSoundsLua`) - Группа звуков.

## Возвращаемые значения

Возвращает `table` - список имен всех звуков в группе. Индексация начинается с 1.
