# RelatedSoundsLua:GetSound

Возвращает один из звуков группы по его идентификатору.

## Описание

```lua
RelatedSoundsLua:GetSound( sysName: string ): Sound2DId | nil
```

Метод возвращает один из звуков группы по его системному идентификатору. В случае, если звук с указанным идентификатором не найден, возвращается `nil`.

## Список параметров

- **`sysName`** (`string`) - Идентификатор звука.

## Возвращаемые значения

Возвращает `Sound2DId` | `nil` - идентификатор звука, или `nil` в случае ошибки (звук не найден).

## Примеры

### Получение звука по идентификатору

```lua
local Sound2D = group:GetSound( "test01" )
```
