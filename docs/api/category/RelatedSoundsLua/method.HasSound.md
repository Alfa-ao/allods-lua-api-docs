# RelatedSoundsLua:HasSound
Проверяет наличие звука с указанным идентификатором в группе.

## Описание
```lua
RelatedSoundsLua:HasSound( sysName: string ): boolean
```
Функция анализирует группу связанных звуков и определяет, зарегистрирован ли в ней звук с указанным системным именем.

## Список параметров
- **`sysName`** (`string`) - Системный идентификатор звука.

## Возвращаемые значения
Возвращает `boolean` - `true`, если звук с указанным идентификатором существует в группе, и `false` в противном случае.

## Примеры
### Проверка наличия звука
```lua
if group:HasSound( "test01" ) then
    LogInfo( "Звук найден в группе." )
end
```
