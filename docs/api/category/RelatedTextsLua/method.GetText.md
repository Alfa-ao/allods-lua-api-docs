# RelatedTextsLua:GetText

Возвращает один из текстов группы по его идентификатору.

## Описание

```lua
RelatedTextsLua:GetText( sysName: string ): WString | nil
```

Извлекает текст из группы по указанному системному идентификатору.

## Список параметров

- **`sysName`** (`string`) - Идентификатор текста.

## Возвращаемые значения

Возвращает `WString` | `nil` - текст или `nil` в случае ошибки (текст не найден).

## Примеры

### Получение текста по идентификатору

```lua
local text = group:GetText( "test01" )
```
