# RelatedTextsLua:HasText

Проверяет наличие текста с указанным идентификатором в группе.

## Описание

```lua
RelatedTextsLua:HasText( sysName: string ): boolean
```

Функция проверяет наличие текста с указанным идентификатором в группе.

## Список параметров

- **`sysName`** (`string`) - Идентификатор текста.

## Возвращаемые значения

Возвращает `boolean` - `true`, если текст с указанным идентификатором существует, и `false` в противном случае.

## Примеры

### Проверка наличия текста

```lua
if group:HasText( "test01" ) then
end
```
