# RelatedTexturesLua:GetTexture

Возвращает одну из текстур группы по ее идентификатору.

## Описание

```lua
RelatedTexturesLua:GetTexture( sysName: string ): TextureId | nil
```

Функция возвращает идентификатор одной из текстур группы по ее системному имени.

## Список параметров

- **`sysName`** (`string`) - Идентификатор текстуры.

## Возвращаемые значения

Возвращает `TextureId` | `nil` - идентификатор найденной текстуры или `nil` в случае, если текстура не найдена.

## Примеры

### Получение текстуры по идентификатору

```lua
local texture = group:GetTexture( "testure01" )
```
