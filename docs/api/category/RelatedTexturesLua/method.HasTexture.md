# RelatedTexturesLua:HasTexture

Проверяет, существует ли текстура с данным идентификатором в группе.

## Описание

```lua
RelatedTexturesLua:HasTexture( sysName )
```

Проверка существования текстуры с данным идентификатором в группе.

## Список параметров

- **`sysName`** (`string`) - Идентификатор текстуры.

## Возвращаемые значения

Возвращает `boolean` - существует ли текстура с данным идентификатором.

## Примеры

### Проверка наличия текстуры

```lua
if group:HasTexture( "testure01" ) then
end
```
