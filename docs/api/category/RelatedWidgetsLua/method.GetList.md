# RelatedWidgetsLua:GetList

Возвращает список всех имен шаблонов виджетов в группе.

## Описание

```lua
RelatedWidgetsLua:GetList(): table
```

Функция возвращает список всех имен шаблонов виджетов в группе.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - список имен всех текстур в группе (индексация с 1).

## Примеры

### Получение списка имен шаблонов

```lua
LogInfo( widgetTemplates:GetList() )
```
