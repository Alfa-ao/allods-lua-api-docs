# RelatedWidgetsLua:GetWidget

Возвращает шаблон виджета по идентификатору.

## Описание

```lua
RelatedWidgetsLua:GetWidget( sysName: string ): WidgetDesc|nil
```

Функция возвращает шаблон виджета, соответствующий переданному системному имени. Если шаблон с указанным идентификатором не найден, возвращается `nil`.

## Список параметров

- **`sysName`** (`string`) - Идентификатор шаблона виджета

## Возвращаемые значения

Возвращает `WidgetDesc` | `nil` - Шаблон виджета или `nil` в случае ошибки (шаблон не найден)

## Примеры

### Получение дескриптора виджета

```lua
local widgetDesc = group:GetWidget( "testWidget01" )
```
