# RelatedWidgetsLua:HasWidget

Проверяет существование шаблона виджета в группе по его идентификатору.

## Описание

```lua
RelatedWidgetsLua:HasWidget( sysName ): boolean
```

Функция проверяет наличие шаблона виджета с указанным системным именем в текущей группе связанных виджетов.

## Список параметров

- **`sysName`** (`string`) - Идентификатор шаблона виджета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если шаблон виджета с данным идентификатором существует, иначе `false`.

## Примеры

### Проверка наличия виджета

```lua
if group:HasWidget( "testWidget01" ) then
    ...
end
```
