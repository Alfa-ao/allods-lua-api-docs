# WString:Compare

Комплексно сравнивает два объекта `WString`.

## Описание

```lua
WString:Compare( other, ignoreRegistry )
```

Комплексно сравнивает два объекта `WString`. Возвращает полный результат сравнения, имеет опцию игнорирования регистра.

::: tip Совет
Если полный результат сравнения и сравнение без учета регистра не требуется, следует использовать стандартные перегруженные операции сравнения (например, `==`, `~=`).
:::

## Список параметров

- **`other`** (`WString`)  
  Строка, с которой будет проведено сравнение. Не может быть `nil`.

- **`ignoreRegistry`** (`boolean` | `nil`)  
  Флаг игнорирования регистра при сравнении. По умолчанию `false`.

## Возвращаемые значения

Возвращает `number` (целое число) или `nil` в зависимости от результата сравнения:

- `nil` — сравнить не удалось (ошибка типа).
- Отрицательное число — первая строка лексикографически меньше второй.
- `0` — строки полностью совпадают.
- Положительное число — первая строка лексикографически больше второй.

## Примеры

### Пример #1 Сравнение с учетом и без учета регистра

```lua
local text = userMods.ToWString( "Test string" )

-- Сравнение с учетом регистра (по умолчанию)
local result1 = text:Compare( userMods.ToWString( "test string" ) )
-- Результат: -32 (строки различаются регистром)

-- Сравнение без учета регистра
local result2 = text:Compare( userMods.ToWString( "test string" ), true )
-- Результат: 0 (строки совпадают)
```
