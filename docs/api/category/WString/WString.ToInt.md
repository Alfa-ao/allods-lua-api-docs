# WString:ToInt

Конвертирует строку в целое число.

## Описание

```lua
WString:ToInt()
```

Конвертирует строку в целое число, если парсинг не удался возвращает `nil`.

## Список параметров

- **`self`** (`WString` | `not nil`)  
  Объект `WString`, который необходимо конвертировать.

## Возвращаемые значения

Возвращает `number` (int) или `nil` — результат конвертации (`nil` при невозможности конвертации).

## Примеры

### Пример #1 Успешная конвертация

```lua
userMods.ToWString( "1234" ):ToInt() -- 1234
```

### Пример #2 Возврат nil при невозможности конвертации

```lua
userMods.ToWString( "text" ):ToInt() -- nil
```
