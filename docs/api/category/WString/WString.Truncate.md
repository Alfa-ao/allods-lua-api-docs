# WString:Truncate

Обрезает строку, добавляя многоточие.

## Описание

```lua
WString:Truncate( maxOverallLen, trailingDotsCount, needTrim )
```

Обрезает строку, добавляя многоточие в конце результирующей строки, если её длина превышает заданный лимит.

## Список параметров

- **`maxOverallLen`** (`Int`)  
Максимальная длина результирующей строки (включая добавляемые точки).

- **`trailingDotsCount`** (`Int` | `nil`)  
Количество точек, добавляемых в конец результирующей строки в случае, если длина строки превышает `maxOverallLen`. По умолчанию `3`.

- **`needTrim`** (`Bool` | `nil`)  
Произвести операцию trim (удалить пробелы и табуляции с обоих концов строки) после обрезания строки и до добавления точек. По умолчанию `true`.

## Возвращаемые значения

Возвращает `WString` — результирующую строку.

## Примеры

### Пример использования `Truncate`

```lua
local text = userMods.ToWString( "123456789" )
local result = text:Truncate( 8 ) -- 12345...
```
