# Global
Объявляет глобальную переменную и присваивает ей указанное начальное значение.

## Описание
```lua
Global( name: string, initialValue: any )
```
Функция объявляет глобальную переменную и присваивает ей указанное начальное значение. Все глобальные переменные, за исключением функций, перед использованием должны быть явно объявлены.

## Список параметров
- **`name`** (`string`)
Имя регистрируемой глобальной переменной.
- **`initialValue`** (`any`)
Начальное значение глобальной переменной. Допускается использование любого значения любого типа.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Объявление глобальных переменных различных типов
```lua
Global( "someVar", nil )
Global( "someVar2", 1 )
Global( "someVar3", {} )
```
