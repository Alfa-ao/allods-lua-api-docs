# GlobalConst
Объявляет общую глобальную константу и присваивает ей указанное начальное значение.

## Описание
```lua
GlobalConst( name: string, initialValue: number )
```
Функция объявляет общую глобальную константу и присваивает ей указанное начальное значение. Объявленная константа становится доступной всем аддонам. Механически функция добавляет константу в список Lua-констант, объявленных в нативном коде.

::: warning Замечание
Имя константы обязано быть уникальным. Перед объявлением рекомендуется проверять занятость имени с помощью функции `HasGlobalConst`.
:::

## Список параметров
- **`name`** (`string`)
Имя регистрируемой общей глобальной константы.
- **`initialValue`** (`number`)
Значение общей глобальной константы. Должно быть целым числом.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Объявление глобальных констант с проверкой уникальности
```lua
if not HasGlobalConst( "MY_ADDON_VERSION" ) then
    GlobalConst( "MY_ADDON_VERSION", 1 )
end
```
::: info Описание примера
В примере сначала проверяется, не объявлена ли уже константа `MY_ADDON_VERSION` с помощью функции `HasGlobalConst`. Если константа отсутствует, вызывается функция `GlobalConst` для её объявления и присвоения начального значения `1`.
:::
