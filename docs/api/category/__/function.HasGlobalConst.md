# HasGlobalConst
Проверяет существование общей глобальной константы с указанным именем.

## Описание
```lua
HasGlobalConst( name: string ): boolean
```
Функция проверяет, зарегистрирована ли в глобальной области видимости константа с заданным именем. Используется для предотвращения повторной инициализации или перезаписи уже существующих глобальных констант.

## Список параметров
- **`name`** (`string`)
Имя проверяемой глобальной константы.

## Возвращаемые значения
Возвращает `boolean` — `true`, если глобальная константа с указанным именем существует, и `false` в противном случае.

## Примеры
### Безопасное создание глобальной константы
```lua
if not HasGlobalConst( "someVar" ) then
    GlobalConst( "someVar", 0 )
end
```
::: info Описание примера
В примере выполняется проверка наличия глобальной константы `someVar` с помощью функции `HasGlobalConst`. Если константа не существует (функция возвращает `false`), инициируется её создание и инициализация нулевым значением через функцию `GlobalConst`.
:::
