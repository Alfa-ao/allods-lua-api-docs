# apitype
Возвращает строку с именем типа переданного в качестве аргумента объекта.

## Описание
```lua
apitype( value: any ): string
```
Функция определяет тип переданного объекта и возвращает его строковое представление. Если объект имеет стандартный встроенный тип Lua, результат полностью аналогичен вызову стандартной функции `type(value)`. Если объект принадлежит к одному из специфических типов, предоставляемых Lua API игры, результатом будет строковое имя этого пользовательского типа.

## Список параметров
- **`value`** (`any`)
Любое значение, тип которого необходимо определить.

## Возвращаемые значения
Возвращает `string` — строку с именем типа, к которому принадлежит переданный аргумент.

## Примеры
### Определение типа объекта Lua API
```lua
local unitId = avatar.GetId()
local valueType = apitype( unitId )
```
::: info Описание примера
В примере сначала получается идентификатор главного персонажа с помощью функции `avatar.GetId`. Затем для полученного значения вызывается функция `apitype`, которая возвращает строковое имя типа данного объекта.
:::
