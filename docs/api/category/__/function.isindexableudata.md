# isindexableudata
Проверяет, является ли переданное значение индексируемой userdata.

## Описание
```lua
isindexableudata( value: any ): boolean
```
Функция анализирует переданный аргумент и определяет, относится ли он к типу индексируемой userdata. Индексируемая userdata отличается от стандартной userdata тем, что при сравнении (операторы `==`, `~=`) и при использовании в качестве ключа таблицы сравнение и построение индекса происходят по значению, а не по адресу ссылки. На текущий момент большинство объектов userdata являются индексируемыми, включая всех наследников `ResourceId` и все типы виджетов. Не являются индексируемыми `WString`, `ValuedText`, `ValuedObject` и подобные типы.

::: warning Замечание
Для всех ресурсов одного типа статус индексируемости является постоянным и может измениться только при обновлении версии клиента. В связи с этим проверять каждый объект в рантайме не требуется.
:::

## Список параметров
- **`value`** (`any`)
Любое значение, которое необходимо проверить на принадлежность к типу индексируемой userdata.

## Возвращаемые значения
Возвращает `boolean` — `true`, если переданный аргумент является индексируемой userdata, и `false` в противном случае.

## Примеры
### Проверка значения на индексируемость
```lua
local isIndexable = isindexableudata( value )
```
