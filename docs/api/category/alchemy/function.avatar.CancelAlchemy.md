# avatar.CancelAlchemy

Отменяет текущее алхимическое действие.

## Описание

```lua
avatar.CancelAlchemy()
```

Функция инициирует принудительную остановку текущего алхимического действия, выполняемого главным персонажем. Если в момент вызова алхимическое действие не активно, вызов функции игнорируется. После успешной отмены или прерывания процесса клиенту передается событие `EVENT_ALCHEMY_CANCELED`.

::: warning Замечание
После прерывания процесса, параметр `isSuccess` события `EVENT_ALCHEMY_CANCELED` возвращает `false`, а окно алхимии закрывается.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Прерывание текущего алхимического действия

```lua
avatar.CancelAlchemy()
```
