# avatar.GetAlchemyInfo

Возвращает базовую информацию об умении алхимии главного персонажа.

## Описание

```lua
avatar.GetAlchemyInfo()
```

Функция извлекает и возвращает базовую информацию об умении алхимии главного персонажа. Если у персонажа отсутствует умение алхимии, возвращается `nil`. В противном случае возвращается таблица, содержащая идентификатор умения, список доступных рецептов, параметры бонусов и текущее состояние алхимического процесса.

::: warning Замечание
Вызов функции вне открытия окна алхимии приведёт к ошибке 
`GMUIHelper::IMainPlayer not found`
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если умение алхимии отсутствует, возвращается `nil`. При наличии умения возвращается таблица со следующими полями:

- **`id`** (`SkillId`) — идентификатор ресурса умения алхимии.

- **`recipes`** (`table`) — список идентификаторов ресурсов (`RecipeId`) доступных рецептов. Для получения подробной информации о конкретном рецепте используется функция `avatar.GetRecipeInfo(id)`.

- **`correctionCount`** (`number`) — доступное количество коррекций (сдвигов барабанов) для персонажа на текущем уровне умения (`score`).

- **`drumsCount`** (`number`) — доступное количество слотов (барабанов) для размещения ингредиентов. Принимает значение `0`, если алхимический процесс не активен (`active == false`).

- **`drumSize`** (`number`) — количество компонентов в одном барабане.

- **`perfectBonus`** (`number`) — дополнительное количество результирующих предметов в случае полного совпадения расположения компонентов в рецепте и инструменте.

- **`perComponentBonus`** (`number`) — количество предметов, добавляемых за каждый корректно размещенный компонент.

- **`unusedRollsBonus`** (`number`) — бонус (дробное число) за неиспользованные довороты (коррекции) при идеальном совпадении компонентов.

- **`defaultResultCount`** (`number`) — базовое количество результирующих предметов.

- **`active`** (`boolean`) — `true`, если действие алхимии начато (был использован алхимический инструмент).
(HELP)
- **`reactionInited`** (`boolean`) — `true`, если была произведена первичная алхимическая реакция (вызов `avatar.DoAlchemyReaction()`).
(HELP)
- **`finished`** (`boolean`) — `true`, если коррекция и выбор линии реакции были произведены (вызов `avatar.FinishAlchemyReaction(line)`). На этом этапе осталось только забрать результат.

## Примеры

### Получение доступного количества коррекций

```lua
local alchemyInfo = avatar.GetAlchemyInfo()
if alchemyInfo then
    local maximalCorrections = alchemyInfo.correctionCount
    LogInfo( "Доступно коррекций: ", maximalCorrections )
end
```

::: info Описание примера
В примере запрашивается информация об умении алхимии. Если функция возвращает таблицу (то есть умение существует), из неё извлекается значение поля `correctionCount`, представляющее максимальное количество доступных коррекций (сдвигов барабанов) на текущем уровне умения. Полученное значение выводится в лог.
:::
