# avatar.GetAlchemyMatchedRecipe

Возвращает информацию о рецепте, по которому создастся предмет в выбранной строке после проведения алхимической реакции с учетом текущих корректировок.

## Описание

```lua
avatar.GetAlchemyMatchedRecipe( line )
```

Функция извлекает и возвращает данные о рецепте, соответствующем текущему расположению компонентов в указанной строке алхимического инструмента. Учитываются текущие корректировки (сдвиги) барабанов. Если подходящий рецепт не найден, возвращается `nil`. В противном случае возвращается таблица, содержащая идентификатор ресурса рецепта и флаг, указывающий на идеальное совпадение расположения компонентов. Идеальное совпадение означает, что расположение компонентов в инструменте полностью идентично рецепту, что приводит к увеличению количества зелья на бонусную величину (см. `avatar.GetAlchemyInfo()`).

## Список параметров

- **`line`** (`number`)
Строка в которой необходимо узнать результат (`-1`, `0`, `1`).

## Возвращаемые значения

Возвращает `table` или `nil`.

Если рецепт для указанной строки не найден, возвращается `nil`. При успешном определении рецепта возвращается таблица со следующими полями:

- **`recipeId`** (`RecipeId`) — уникальный идентификатор ресурса рецепта, по которому будет произведен результат.

- **`perfectRecipe`** (`boolean`) — флаг идеального совпадения. Значение `true` указывает на то, что расположение компонентов в инструменте полностью соответствует рецепту.

## Примеры

### Расчет бонусного количества продукта

```lua
local recipe = avatar.GetAlchemyMatchedRecipe( 0 )
if recipe then
    local recipeInfo = avatar.GetRecipeInfo( recipe.recipeId )
    local count = componentsCount
    if recipe.perfectRecipe then
        count = count + alchemyInfo.perfectBonus
    end
end
```
