# avatar.IsAlchemyComponentsReady

::: danger (DEPRECATED)
Функция логически устарела.
:::

Проверяет наличие достаточного количества реагентов в алхимическом приборе для начала реакции.

## Описание

```lua
avatar.IsAlchemyComponentsReady()
```

Функция анализирует текущее состояние алхимической ступки и определяет, размещены ли в слоты ступки все необходимые компоненты в требуемом объеме для успешного запуска реакции варки. 

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` — `true`, если компонентов достаточно для начала реакции, и `false` в противном случае.

## Примеры

### Проверка готовности и запуск реакции варки

```lua
if avatar.IsAlchemyComponentsReady() then
    avatar.DoAlchemyReaction()
end
```
