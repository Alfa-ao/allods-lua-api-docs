# avatar.IsAlchemyLineAvailable

Проверяет доступность указанной линии результата алхимии для главного персонажа.

## Описание

```lua
avatar.IsAlchemyLineAvailable( line )
```

Функция определяет, разблокирована ли конкретная линия (строка) результата в интерфейсе алхимии у главного персонажа. Линии результата соответствуют возможным сдвигам компонентов при создании зелий.

## Список параметров

- **`line`** (`number`)
Индекс линии результата алхимии. Принимает значения `-1`, `0` или `1`, где `0` — центральная линия (без сдвига), `-1` и `1` — боковые линии.

## Возвращаемые значения

Возвращает `boolean` — `true`, если указанная линия доступна (разблокирована) для использования, и `false` в противном случае.

## Примеры

### Проверка доступности боковых линий алхимии

```lua
for line = -1, 1 do
    if avatar.IsAlchemyLineAvailable( line ) then
        LogInfo( "Доступна линия результата: ", line )
    else
        LogInfo( "Линия результата недоступна: ", line )
    end
end
```
