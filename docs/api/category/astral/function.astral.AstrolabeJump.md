# astral.AstrolabeJump
Выполняет прыжок в заданный сектор астрала до выбранной POI за марки (быстрое путешествие).

## Описание
```lua
astral.AstrolabeJump( poiId: ObjectId, spellId: SpellId )
```
Функция инициирует процесс быстрого перемещения (прыжка) в заданный сектор астрала до выбранной точки интереса (POI) за внутриигровую валюту (марки). Для успешного выполнения путешествия требуется указать идентификатор заклинания, обеспечивающего данное перемещение. Актуальные идентификаторы заклинаний для быстрого перемещения в астрале извлекаются посредством вызова функции `astral.GetAstrolabeInfo`.

## Список параметров
- **`poiId`** (`ObjectId`)
Идентификатор астральной точки интереса (POI), в которую осуществляется перемещение.
- **`spellId`** (`SpellId`)
Идентификатор заклинания, при помощи которого инициируется перемещение.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Инициирование прыжка в астрал с предварительным получением данных
```lua
local astrolabeInfo = astral.GetAstrolabeInfo()
if astrolabeInfo then
    local spellId = astrolabeInfo.jumpSpellId
    local poiId = 12345 
    astral.AstrolabeJump( poiId, spellId )
end
```
::: info Описание примера
В примере сначала извлекается информация об астрале посредством вызова функции `astral.GetAstrolabeInfo`. При успешном получении данных извлекается идентификатор заклинания для перемещения, который вместе с идентификатором целевой точки интереса передается в функцию `astral.AstrolabeJump` для инициирования прыжка.
:::
