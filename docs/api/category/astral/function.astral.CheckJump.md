# astral.CheckJump
Проверяет возможность совершения прыжка группы в заданный сектор астрала до выбранной POI за марки.

## Описание
```lua
astral.CheckJump( poiId: ObjectId, direct: boolean )
```
Функция инициирует проверку возможности совершения прыжка в астрале. Результат проверки передается клиенту через событие `EVENT_ASTRAL_SECTOR_CHECK_JUMP_RESULT`. Параметр `direct` определяет тип проверяемого прыжка: с использованием ускорителей (из ангара на остров) или обычный прыжок (из астрала на остров).

::: warning Замечание
Результат выполнения операции приходит в событии `EVENT_ASTRAL_SECTOR_CHECK_JUMP_RESULT`.
:::

## Список параметров
- **`poiId`** (`ObjectId`)
Идентификатор астральной POI, до которой планируется совершить прыжок.
- **`direct`** (`boolean`)
Флаг, определяющий тип проверяемого прыжка. Если передано значение `true`, проверяется возможность совершения прыжка с использованием ускорителей (прыжок из ангара на остров). Если передано значение `false`, проверяется возможность совершения обычного прыжка на остров из астрала.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Проверка возможности прыжка с предварительным получением информации о POI
```lua
local poiId = 12345 -- Идентификатор целевой POI
local poiInfo = astral.GetPOIInfo( poiId )
if poiInfo then
    astral.CheckJump( poiId, true )
end
```
::: info Описание примера
В примере сначала извлекается информация о POI с помощью `astral.GetPOIInfo`. Если POI существует, инициируется проверка возможности совершения прыжка до этой точки с использованием ускорителей (параметр `direct` равен `true`). Результат проверки будет передан в событии `EVENT_ASTRAL_SECTOR_CHECK_JUMP_RESULT`.
:::
