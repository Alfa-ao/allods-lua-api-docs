# astral.GetAddonInfo
Извлекает и возвращает информацию об астральном аддоне по его уникальному идентификатору.

## Описание
```lua
astral.GetAddonInfo( addonId: ObjectId ): table | nil
```
Функция запрашивает и формирует таблицу с детальными характеристиками указанного астрального аддона. Возвращаемые данные включают идентификатор, название, описание, координатные границы секторов, смещение уровня и параметры отображения на карте. Если информация по заданному идентификатору отсутствует или недоступна, функция возвращает `nil`.

## Список параметров
- **`addonId`** (`ObjectId`)
Уникальный идентификатор астрального аддона, информацию о котором необходимо получить.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если информация доступна, возвращается таблица со следующими полями:
- **`id`** (`ObjectId`)
Уникальный идентификатор аддона.
- **`name`** (`WString`)
Название аддона.
- **`description`** (`WString`)
Описание аддона.
- **`rect`** (`table`)
Таблица с минимальными и максимальными координатами секторов, входящих в аддон. Содержит следующие поля:
  - **`x1`**, **`y1`** (`number`) — минимальные координаты.
  - **`x2`**, **`y2`** (`number`) — максимальные координаты.
- **`levelOffset`** (`number`)
Числовое значение смещения уровня аддона.
- **`isSphere`** (`boolean`)
Логический флаг, определяющий формат отображения аддона на карте. Значение `true` указывает, что аддон необходимо отображать в виде сферы или круга (координаты секторов преобразуются и ужимаются в круг).

Возвращает `nil`, если информация об аддоне недоступна или он не найден.

## Примеры
### Получение и вывод названия астрального аддона
```lua
local addonInfo = astral.GetAddonInfo( addonId )
if addonInfo then
    LogInfo( addonInfo.name )
end
```
