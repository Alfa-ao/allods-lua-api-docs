# astral.GetAddonSectors
Возвращает список идентификаторов всех секторов, находящихся в заданном астральном аддоне.

## Описание
```lua
astral.GetAddonSectors( addonId: ObjectId ): table
```
Функция извлекает и возвращает таблицу с идентификаторами всех секторов, которые расположены внутри указанного астрального аддона. Элементы возвращаемой таблицы индексированы с единицы.

## Список параметров
- **`addonId`** (`ObjectId`)
Идентификатор астрального аддона, для которого запрашивается список секторов.

## Возвращаемые значения
Возвращает `table` — список (индексирован с 1) идентификаторов (`ObjectId`) секторов, находящихся в аддоне.

## Примеры
### Получение и перебор секторов аддона
```lua
local edges = astral.GetAddonSectors( addonId )
for i, objectId in ipairs( edges ) do
    local sectorInfo = astral.GetSectorInfo( objectId )
    if sectorInfo then
        local name = sectorInfo.name
    end
end
```
::: info Описание примера
В примере запрашивается список идентификаторов секторов для указанного аддона. Затем с помощью цикла `ipairs` происходит перебор полученной таблицы. Для каждого идентификатора вызывается функция `astral.GetSectorInfo` для получения детальной информации о секторе, из которой извлекается его имя.
:::
