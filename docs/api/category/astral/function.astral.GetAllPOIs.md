# astral.GetAllPOIs
Возвращает список идентификаторов всех точек астральных событий.

## Описание
```lua
astral.GetAllPOIs(): table
```
Функция возвращает индексированный с единицы список идентификаторов (`ObjectId`) точек астральных событий, присутствующих в текущем секторе.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированный с единицы список идентификаторов (`ObjectId`) точек астральных событий.

## Примеры
### Получение информации о первой точке астрального события
```lua
local pois = astral.GetAllPOIs()
if pois[ 1 ] then
    local poiInfo = astral.GetPOIInfo( pois[ 1 ] )
    if poiInfo then
        LogInfo( poiInfo.name )
    end
end
```
::: info Описание примера
В примере запрашивается список всех точек астральных событий. Если список не пуст, извлекается идентификатор первой точки, который передается в функцию `astral.GetPOIInfo` для получения подробной информации. При успешном получении данных в лог выводится название точки.
:::
