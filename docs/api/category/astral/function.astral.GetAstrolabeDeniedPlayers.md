# astral.GetAstrolabeDeniedPlayers
Возвращает списки имён членов команды корабля, препятствующих использованию астролябии из-за отсутствия альтернативной валюты или благосклонности покровителя.

## Описание
```lua
astral.GetAstrolabeDeniedPlayers(): table
```
Функция возвращает таблицу с именами игроков, входящих в команду корабля, которые не могут участвовать в прыжке через астролябию. Причины блокировки разделяются на две категории: отсутствие необходимой альтернативной валюты и недостаточная благосклонность покровителя.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую следующие поля:
- **`withoutCoins`** (`table of WString`)
Индексированный с единицы список имён игроков команды, у которых отсутствует необходимая альтернативная валюта для совершения прыжка.
- **`withoutFairies`** (`table of WString`)
Индексированный с единицы список имён игроков команды, у которых покровитель не проявляет благосклонность.

## Примеры
### Получение и вывод списков игроков с ограничениями
```lua
local deniedPlayers = astral.GetAstrolabeDeniedPlayers()
LogInfo( "Без валюты: ", deniedPlayers.withoutCoins )
LogInfo( "Без благосклонности: ", deniedPlayers.withoutFairies )
```
