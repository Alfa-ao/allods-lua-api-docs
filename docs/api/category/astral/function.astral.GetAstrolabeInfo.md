# astral.GetAstrolabeInfo
Извлекает информацию об астролябии для главного персонажа.

## Описание
```lua
astral.GetAstrolabeInfo(): table | nil
```
Функция возвращает таблицу с информацией об астролябии или `nil`, если данные недоступны. Данные становятся доступны, если главный игрок находится на своем корабле, является капитаном и астролябия может быть использована.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если информация доступна, возвращается таблица со следующими полями:
- **`jumpSpellId`** (`SpellId`) — идентификатор заклинания для стандартного прыжка в сектор.
- **`chargedJumpSpellId`** (`SpellId`) — идентификатор заклинания для прыжка в сектор с использованием заряда ускорения.

## Примеры
### Получение и проверка состояния заклинания прыжка
```lua
local astrolabeInfo = astral.GetAstrolabeInfo()
if astrolabeInfo and astrolabeInfo.jumpSpellId then
    LogInfo( avatar.GetSpellState( astrolabeInfo.jumpSpellId ) )
end
```
::: info Описание примера
В примере извлекается информация об астролябии. Если данные успешно получены и идентификатор заклинания стандартного прыжка существует, выводится текущее состояние данного заклинания с помощью функции `avatar.GetSpellState`.
:::
