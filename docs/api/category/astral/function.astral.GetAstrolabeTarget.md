# astral.GetAstrolabeTarget
Возвращает идентификатор сектора для астрального прыжка, находящегося в фазе разгона.

## Описание
```lua
astral.GetAstrolabeTarget(): ObjectId | nil
```
Функция возвращает идентификатор сектора, являющегося целью текущего астрального прыжка (быстрого путешествия), если он находится в фазе разгона. Если прыжок не выполняется или информация о целевом секторе недоступна, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`ObjectId` | `nil` — идентификатор сектора, если идет прыжок и информация о секторе доступна; `nil` в противном случае.

## Примеры
### Получение информации о целевом секторе астрального прыжка
```lua
local sectorId = astral.GetAstrolabeTarget()
if sectorId then
  local sectorInfo = astral.GetSectorInfo( sectorId )
  if sectorInfo then
    LogInfo( "Идет разгон" )
  end
end
```
::: info Описание примера
В примере извлекается идентификатор целевого сектора. Если идентификатор успешно получен, запрашивается подробная информация о секторе с помощью `astral.GetSectorInfo`. При успешном получении информации выводится сообщение о том, что идет разгон.
:::
