# astral.GetHubCenter
Извлекает координаты центра астрального хаба, в котором находится игрок.

## Описание
```lua
astral.GetHubCenter(): GamePosition | nil
```
Функция возвращает структуру `GamePosition`, содержащую координаты центра астрального хаба, если главный персонаж в данный момент находится в хабе. В случае, если игрок не находится в астральном хабе, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `GamePosition` или `nil`.
- **`GamePosition`** — координаты центра астрального хаба, если игрок находится в хабе.
- **`nil`** — если игрок не находится в астральном хабе.

## Примеры
### Получение координат центра астрального хаба
```lua
local pos = astral.GetHubCenter()
```
