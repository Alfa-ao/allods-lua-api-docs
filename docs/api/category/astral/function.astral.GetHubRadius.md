# astral.GetHubRadius
Возвращает радиус стабильной зоны астрального хаба, в котором находится игрок.

## Описание
```lua
astral.GetHubRadius(): number
```
Функция возвращает числовое значение, представляющее собой радиус стабильной зоны астрального хаба в метрах, в котором в данный момент находится главный персонаж.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — радиус стабильной зоны астрального хаба в метрах.

## Примеры
### Получение радиуса стабильной зоны
```lua
local radius = astral.GetHubRadius()
```
