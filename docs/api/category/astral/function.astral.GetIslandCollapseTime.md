# astral.GetIslandCollapseTime
Извлекает и возвращает информацию о времени сворачивания астрального острова, на котором находится главный игрок.

## Описание
```lua
astral.GetIslandCollapseTime(): table | nil
```
Функция возвращает таблицу с информацией о времени сворачивания астрального острова, на котором находится главный игрок, или `nil`, если игрок не находится на астральном острове.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если игрок находится на астральном острове, возвращается таблица (`table`) со следующими полями:
- **`durationMs`** (`number`)
Полное время в миллисекундах до сворачивания острова.
- **`remainingMs`** (`number`)
Время в миллисекундах, оставшееся до сворачивания острова на данный момент.

Возвращает `nil`, если игрок не находится на астральном острове.

## Примеры
### Вычисление пропорции оставшегося времени жизни острова
```lua
local collapseTime = astral.GetIslandCollapseTime()
if collapseTime then
    local astralIslandLife = collapseTime.remainingMs / collapseTime.durationMs
end
```
