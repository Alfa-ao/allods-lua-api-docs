# astral.GetIslandDeathLimit
Извлекает информацию о текущем количестве смертей и предельно допустимом лимите для острова.

## Описание
```lua
astral.GetIslandDeathLimit(): table
```
Функция возвращает таблицу, содержащую информацию о лимите смертей и текущем счетчике смертей на острове. Полученные данные применяются для отслеживания прогресса и проверки соблюдения ограничений, установленных для конкретной локации.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`):
- **`limit`** (`number`) — предельно допустимое количество смертей на острове.
- **`deathCount`** (`number`) — текущее количество смертей, зарегистрированных на острове.

## Примеры
### Получение и вывод информации о смертях на острове
```lua
local info = astral.GetIslandDeathLimit()
LogInfo( "count: ", info.deathCount, ", limit: ", info.limit )
```
