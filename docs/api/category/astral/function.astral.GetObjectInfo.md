# astral.GetObjectInfo
Возвращает описание астрального объекта в астральном хабе игрока.

## Описание
```lua
astral.GetObjectInfo( objectId: ObjectId ): table | nil
```
Функция извлекает и возвращает детальное описание астрального объекта, находящегося в астральном хабе игрока. Базовая информация предоставляется системой `GameState`, а расширенная — системой `UIState`. Для определения позиции объекта используется структура `GamePosition`.

::: warning Замечание
Поля `image` и `collisionIcon` возвращаются исключительно при запросе расширенной информации через систему `UIState`.
:::

## Список параметров
- **`objectId`** (`ObjectId`)
Уникальный идентификатор юнита (астрального объекта).

## Возвращаемые значения
Возвращает `table` или `nil`. Если объект найден, возвращается таблица со следующими полями:
- **`name`** (`WString`)
Имя астрального объекта.
- **`description`** (`WString`)
Текстовое описание астрального объекта.
- **`position`** (`GamePosition`)
Координаты объекта в хабе.
- **`isEnabled`** (`boolean`)
Индикатор доступности объекта.
- **`sectorId`** (`ObjectId` | `nil`)
Идентификатор сектора, в который ведёт червоточина, если объект является червоточиной.
- **`collisionRadius`** (`number` | `nil`)
Радиус коллизии объекта астрала в виде целого числа. Если значение не задано, возвращается `nil`.

Часть, доступная только в UI:
- **`image`** (`TextureId`)
Идентификатор текстуры с иконкой объекта.
- **`collisionIcon`** (`TextureId`)
Идентификатор масштабируемой текстуры для проекции коллизии объекта астрала на карте.

## Примеры
### Получение информации об астральном объекте
```lua
local object = astral.GetObjectInfo( objectId )
if object then
  local name = object.name
end
```
