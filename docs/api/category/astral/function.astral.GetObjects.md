# astral.GetObjects
Возвращает список идентификаторов астральных объектов в хабе главного игрока.

## Описание
```lua
astral.GetObjects(): table
```
Функция возвращает таблицу с идентификаторами астральных объектов, находящихся в хабе главного игрока.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — список идентификаторов объектов (`ObjectId`). Таблица индексирована с 1.

## Примеры
### Перебор и получение информации об астральных объектах
```lua
local objects = astral.GetObjects()

for _, objectId in ipairs(objects) do
    local object = astral.GetObjectInfo(objectId)
    if object then
        LogInfo("Object:", object.name)
    end
end
```
::: info Описание примера
В примере сначала извлекается список идентификаторов астральных объектов. Затем в цикле перебирается каждый идентификатор, для которого вызывается функция получения детальной информации. При успешном получении данных имя объекта выводится в информационный лог.
:::
