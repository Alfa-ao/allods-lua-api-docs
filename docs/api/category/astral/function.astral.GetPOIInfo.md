# astral.GetPOIInfo
Возвращает информацию о точке астрального эвента по её идентификатору.
## Описание
```lua
astral.GetPOIInfo( poiId: ObjectId ): table | nil
```
Функция извлекает и возвращает детальную информацию о конкретной точке астрального события. Если информация о точке с указанным идентификатором недоступна, функция возвращает `nil`.

## Список параметров
- **`poiId`** (`ObjectId`)
Идентификатор точки астрального события, информацию о которой необходимо получить.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если информация о точке доступна, возвращается таблица со следующими полями:
- **`id`** (`ObjectId`) — уникальный идентификатор точки.
- **`sectorId`** (`ObjectId`) — уникальный идентификатор сектора.
- **`name`** (`WString` | `nil`) — название астрального события.
- **`description`** (`WString` | `nil`) — описание астрального события.
- **`visibleOnMap`** (`boolean`) — `false`, если точка отображается только в списке POI для прыжка; `true`, если точка также отображается на астральной карте.
- **`canJump`** (`boolean`) — `false`, если к данной точке нельзя совершить прыжок с помощью астролябии (в таком случае точка не отображается в списке для прыжка).
- **`texture`** (`TextureId`) — идентификатор текстуры для иконки точки. Поле реализовано исключительно для нужд пользовательского интерфейса.

## Примеры
### Получение информации о первой доступной точке
```lua
local pois = astral.GetAllPOIs()
if pois[ 0 ] then
    local poiInfo = astral.GetPOIInfo( pois[ 0 ] )
    if poiInfo then
        LogInfo( poiInfo.name )
    end
end
```
::: info Описание примера
В примере сначала извлекается список всех доступных точек астральных событий с помощью функции `astral.GetAllPOIs()`. Затем проверяется наличие хотя бы одной точки в списке. Если точка существует, вызывается функция `astral.GetPOIInfo()` для получения детальной информации о ней. При успешном получении данных в информационный лог выводится название астрального события.
:::
