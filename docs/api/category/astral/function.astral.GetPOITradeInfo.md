# astral.GetPOITradeInfo
Извлекает и возвращает информацию о торговле в точке астрального события по её идентификатору.

## Описание
```lua
astral.GetPOITradeInfo( poiId: ObjectId ): table | nil
```
Функция извлекает и возвращает детальную информацию о текущем состоянии торговли в указанной точке астрального события. Возвращаемые данные включают актуальные цены на продукцию и сырье, а также информацию о заполненности соответствующих складов. Если информация по заданному идентификатору недоступна или точка не существует, функция возвращает `nil`.

## Список параметров
- **`poiId`** (`ObjectId`)
Уникальный идентификатор точки астрального события, информацию о торговле в которой необходимо получить.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если информация доступна, возвращается таблица со следующими полями:
- **`prodPrice`** (`number`) — текущая цена продукции.
- **`rawPrice`** (`number`) — текущая цена сырья.
- **`prodStockpile`** (`number`) — текущая заполненность складов продукцией.
- **`rawStockpile`** (`number`) — текущая заполненность складов сырьем.
- **`maxProdStockpile`** (`number`) — максимальный объем складов для хранения продукции.
- **`maxRawStockpile`** (`number`) — максимальный объем складов для хранения сырья.
или `nil`, если информация недоступна.

## Примеры
### Получение информации о торговле в точке
```lua
local poiInfo = astral.GetPOITradeInfo( poiId )
```
