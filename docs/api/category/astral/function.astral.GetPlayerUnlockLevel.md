# astral.GetPlayerUnlockLevel
Возвращает уровень астрального анлока указанного игрока.

## Описание
```lua
astral.GetPlayerUnlockLevel( playerId: ObjectId ): number
```
Функция извлекает и возвращает числовое значение, представляющее уровень астрального анлока для игрока с заданным идентификатором. Корректное значение возвращается исключительно для дружественных игроков; для всех остальных персонажей функция возвращает `0`.

::: warning Замечание
Корректное значение уровня анлока возвращается только для дружественных игроков. Для остальных игроков функция всегда возвращает `0`.
:::

## Список параметров
- **`playerId`** (`ObjectId`)
Идентификатор игрока, уровень астрального анлока которого необходимо получить.

## Возвращаемые значения
Возвращает `number` — уровень астрального анлока указанного игрока.

## Примеры
### Получение уровня астрального анлока
```lua
local level = astral.GetPlayerUnlockLevel( playerId )
```
