# astral.GetSectorAstrolabeInfo
Извлекает информацию о возможности прыжка в астральный сектор с помощью астролябии.

## Описание
```lua
astral.GetSectorAstrolabeInfo( sectorId: ObjectId ): table
```
Функция извлекает и возвращает информацию о возможности совершения прыжка в указанный астральный сектор с использованием астролябии. Возвращаемые данные включают требуемый уровень разблокировки и текущий статус доступности сектора для прыжка.

## Список параметров
- **`sectorId`** (`ObjectId`)
Уникальный идентификатор астрального сектора, информацию о котором необходимо получить.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую информацию о запрошенном секторе. Таблица включает следующие поля:
- **`unlockLevel`** (`number`)
Уровень разблокировки, требующийся для совершения прыжка в данный сектор.
- **`isAvailable`** (`boolean`)
Флаг, указывающий на возможность осуществления прыжка в сектор.

## Примеры
### Проверка доступности сектора для прыжка
```lua
local info = astral.GetSectorAstrolabeInfo( sectorId )
if not info.isAvailable then
    LogInfo( "Cannot jump" )
end
```
