# astral.GetSectorDirectPOI
Возвращает идентификатор точки астрального события в заданном секторе для совершения прямого прыжка.

## Описание
```lua
astral.GetSectorDirectPOI( sectorId: ObjectId ): ObjectId | nil
```
Функция извлекает и возвращает идентификатор точки астрального события (POI) в указанном секторе, к которому планируется совершить прямой прыжок без выбора промежуточных вариантов. Если точка не найдена или сектор не содержит подходящего события, функция возвращает `nil`.

## Список параметров
- **`sectorId`** (`ObjectId`)
Уникальный идентификатор сектора, в котором производится поиск точки астрального события.

## Возвращаемые значения
Возвращает `ObjectId` — идентификатор точки астрального события в заданном секторе, или `nil`, если точка не найдена.

## Примеры
### Получение и проверка информации о точке прямого прыжка
```lua
local directPoi = astral.GetSectorDirectPOI( sectorId )
if directPoi then
    local poiInfo = astral.GetPOIInfo( directPoi )
    if poiInfo then
        LogInfo( poiInfo.name )
    end
end
```
::: info Описание примера
В примере выполняется запрос идентификатора точки прямого прыжка для заданного сектора. При успешном получении идентификатора извлекается подробная информация о точке с помощью функции `astral.GetPOIInfo`. Если информация успешно загружена, в лог выводится имя точки.
:::
