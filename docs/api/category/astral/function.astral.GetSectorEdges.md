# astral.GetSectorEdges
Возвращает список идентификаторов всех секторов, связанных с заданным сектором.

## Описание
```lua
astral.GetSectorEdges( sectorId: ObjectId|AstralSectorId ): table
```
Функция извлекает и возвращает таблицу с идентификаторами всех соседних секторов, имеющих прямую связь (ребра) с указанным сектором в астрале. Возвращаемый список представляет собой таблицу, индексация элементов в которой начинается с единицы.

## Список параметров
- **`sectorId`** (`ObjectId` | `AstralSectorId`)
Уникальный идентификатор сектора, для которого необходимо получить список связанных с ним соседних секторов.

## Возвращаемые значения
Возвращает `table` — список идентификаторов (`ObjectId`) секторов, связанных с данным. Индексация элементов в таблице начинается с единицы.

## Примеры
### Получение и перебор списка связанных секторов
```lua
local edges = astral.GetSectorEdges( sectorId )
for i, objectId in ipairs( edges ) do
    local sectorInfo = astral.GetSectorInfo( objectId )
    if sectorInfo then
        local name = sectorInfo.name
    end
end
```
::: info Описание примера
В примере запрашивается список всех связанных секторов для заданного `sectorId`. Затем с помощью цикла `ipairs` происходит перебор полученной таблицы. Для каждого найденного идентификатора соседнего сектора вызывается функция `astral.GetSectorInfo` для получения подробной информации, из которой в дальнейшем извлекается имя сектора.
:::
