# astral.GetSectorInfo
Возвращает информацию об астральном секторе по его идентификатору.

## Описание
```lua
astral.GetSectorInfo( sectorId: ObjectId | AstralSectorId ): table | nil
```
Функция извлекает и возвращает детальную информацию об астральном секторе по заданному идентификатору. Возвращаемые данные включают координаты, параметры видимости, требования для посещения и характеристики качества. Если информация о секторе недоступна, возвращается `nil`.

## Список параметров
- **`sectorId`** (`ObjectId` | `AstralSectorId`)
Уникальный идентификатор астрального сектора.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если информация доступна, возвращается таблица со следующими полями:
- **`id`** (`ObjectId`) — уникальный идентификатор сектора.
- **`addonId`** (`ObjectId` | `nil`) — идентификатор астрального аддона, если такая информация имеется.
- **`name`** (`WString` | `nil`) — название сектора.
- **`description`** (`ValuedText` | `nil`) — описание сектора.
- **`techLevel`** (`number`) — технологический уровень сектора.
- **`quality`** (`number`) — уровень качества сектора. Значения соответствуют перечислению `ITEM_QUALITY_*`.
- **`requiredUnlock`** (`number`) — требуемый уровень разблокировки астрала.
- **`ticketsCount`** (`number`) — количество необходимых ключей для посещения острова.
- **`maxServerProgress`** (`number`) — значение серверного прогресса, выше которого ключи для посещения перестают требоваться.
- **`number`** (`number`) — порядковый номер сектора в астральном аддоне.
- **`x`** (`number`) — координата X расположения сектора в аддоне.
- **`y`** (`number`) — координата Y расположения сектора в аддоне.
- **`astrolabeVisible`** (`boolean`) — `true`, если сектор виден в астролябии.
- **`mapVisible`** (`boolean`) — `true`, если сектор виден на карте.
- **`allowChoiceLoot`** (`boolean`) — `true`, если в секторе можно выбирать лут.

## Примеры
### Получение и вывод названия текущего сектора
```lua
local sectorId = astral.GetCurrentSector()
if sectorId then
    local sectorInfo = astral.GetSectorInfo( sectorId )
    if sectorInfo then
        LogInfo( sectorInfo.name )
    end
end
```
::: info Описание примера
В примере сначала извлекается идентификатор текущего астрального сектора с помощью `astral.GetCurrentSector`. Если идентификатор успешно получен, вызывается `astral.GetSectorInfo` для загрузки информации. При успешном получении таблицы в информационный лог выводится название сектора.
:::
