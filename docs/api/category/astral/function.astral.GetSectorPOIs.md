# astral.GetSectorPOIs
Возвращает список идентификаторов точек астральных эвентов в заданном секторе.

## Описание
```lua
astral.GetSectorPOIs( sectorId: ObjectId ): table
```
Функция извлекает и возвращает индексированный с единицы список идентификаторов точек астральных эвентов (POI), расположенных в указанном секторе. Полученные идентификаторы могут быть использованы для получения детальной информации о каждой точке посредством функции `astral.GetPOIInfo`.

## Список параметров
- **`sectorId`** (`ObjectId`)
Уникальный идентификатор сектора астрала, для которого необходимо получить список точек интереса.

## Возвращаемые значения
Возвращает `table` — индексированный с единицы список идентификаторов (`ObjectId`) точек астральных эвентов, находящихся в заданном секторе.

## Примеры
### Получение и обработка информации о первой точке интереса в секторе
```lua
local pois = astral.GetSectorPOIs( sectorId )
if pois[ 1 ] then
    local poiInfo = astral.GetPOIInfo( pois[ 1 ] )
    if poiInfo then
        LogInfo( poiInfo.name )
    end
end
```
::: info Описание примера
В примере выполняется запрос списка точек астральных эвентов для заданного сектора. Если список не пуст и содержит хотя бы один элемент, извлекается идентификатор первой точки. Далее с помощью функции `astral.GetPOIInfo` получается подробная информация об этой точке, и если данные успешно загружены, в лог выводится её имя.
:::
