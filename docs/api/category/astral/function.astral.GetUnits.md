# astral.GetUnits
Возвращает список идентификаторов астральных юнитов в хабе главного игрока.

## Описание
```lua
astral.GetUnits(): table
```
Функция извлекает и возвращает таблицу, содержащую уникальные идентификаторы всех астральных юнитов, находящихся в текущем хабе главного игрока. Индексация элементов в возвращаемой таблице начинается с единицы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую идентификаторы (`ObjectId`) астральных юнитов. Индексация элементов в таблице начинается с `1`.

## Примеры
### Перебор астральных юнитов и получение их уровня
```lua
local units = astral.GetUnits()
for i, unitId in ipairs( units ) do
    local level = astralUnit.GetLevel( unitId )
    -- Дальнейшая обработка уровня юнита
end
```
::: info Описание примера
В примере сначала извлекается список идентификаторов астральных юнитов с помощью функции `astral.GetUnits`. Затем с использованием цикла `ipairs` происходит перебор всех элементов полученной таблицы. На каждой итерации вызывается функция `astralUnit.GetLevel` для получения уровня конкретного юнита по его идентификатору.
:::
