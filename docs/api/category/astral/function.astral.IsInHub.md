# astral.IsInHub
Проверяет, находится ли главный персонаж в астральном хабе.

## Описание
```lua
astral.IsInHub(): boolean
```
Функция проверяет текущее местоположение главного персонажа и определяет, пребывает ли он в зоне астрального хаба. Возвращает логическое значение, указывающее на факт нахождения персонажа в указанной локации.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если главный персонаж находится в астральном хабе, и `false` в противном случае.

## Примеры
### Проверка нахождения в астральном хабе
```lua
local inHub = astral.IsInHub()
```
