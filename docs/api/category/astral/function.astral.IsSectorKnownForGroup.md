# astral.IsSectorKnownForGroup
Определяет доступность астрального сектора для игрока с учетом группового анлока.

## Описание
```lua
astral.IsSectorKnownForGroup( playerId: ObjectId, sectorId: ObjectId ): boolean
```
Функция возвращает логическое значение, указывающее на то, исследован ли указанный астральный сектор у заданного игрока. Проверка доступности сектора производится с учетом группы, в которой состоит игрок: сектор признается доступным, если он открыт непосредственно у самого игрока или доступен по групповому анлоку.

Условие доступности сектора: `max(sectorAvatarLevel, sectorAstralLevel) <= max(playerAstralUnlock, groupAstralUnlock)`.

## Список параметров
- **`playerId`** (`ObjectId`)
Уникальный идентификатор игрока, для которого производится проверка.
- **`sectorId`** (`ObjectId`)
Уникальный идентификатор астрального сектора, доступность которого необходимо проверить.

## Возвращаемые значения
Возвращает `boolean` — `true`, если астральный сектор исследован и доступен для игрока с учетом группового анлока, и `false` в противном случае.

## Примеры
### Проверка доступности астрального сектора
```lua
local sectorVisible = astral.IsSectorKnownForGroup( playerId, sectorId )
```
