# astral.IsSectorKnownForPlayer
Проверяет, исследован ли указанный астральный сектор игроком без учета членства в группе.
## Описание
```lua
astral.IsSectorKnownForPlayer( playerId: ObjectId, sectorId: ObjectId ): boolean
```
Функция определяет доступность астрального сектора для конкретного игрока на основе его астрального анлока. Сектор считается исследованным, если максимальное значение из требуемого уровня аватара (`sectorAvatarLevel`) и астрального уровня (`sectorAstralLevel`) сектора меньше или равно астральному анлоку игрока (`playerAstralUnlock`). При проверке игнорируется факт нахождения игрока в какой-либо группе.
## Список параметров
- **`playerId`** (`ObjectId`)
Идентификатор игрока, для которого выполняется проверка.
- **`sectorId`** (`ObjectId`)
Идентификатор астрального сектора, доступность которого необходимо проверить.
## Возвращаемые значения
Возвращает `boolean` — `true`, если сектор исследован и доступен игроку, и `false` в противном случае.
## Примеры
### Проверка доступности астрального сектора
```lua
local sectorVisible = astral.IsSectorKnownForPlayer( playerId, sectorId )
if sectorVisible then
    -- Сектор исследован и доступен для игрока
end
```
