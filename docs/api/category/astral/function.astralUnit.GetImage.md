# astralUnit.GetImage
Возвращает иконку астрального моба, отображаемую на карте.

## Описание
```lua
astralUnit.GetImage( unitId: ObjectId ): TextureId
```
Функция возвращает идентификатор ресурса иконки, обозначающей астрального моба на карте. Полученное значение используется для визуализации юнита в интерфейсе карты.

## Список параметров
- **`unitId`** (`ObjectId`)
Уникальный идентификатор юнита, для которого запрашивается иконка.

## Возвращаемые значения
Возвращает `TextureId` — идентификатор ресурса иконки, обозначающей моба.

## Примеры
### Получение иконки астрального моба
```lua
local textureId = astralUnit.GetImage( unitId )
```
