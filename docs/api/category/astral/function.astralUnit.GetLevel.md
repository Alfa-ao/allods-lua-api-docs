# astralUnit.GetLevel
Возвращает уровень астрального моба.

## Описание
```lua
astralUnit.GetLevel( unitId: ObjectId ): number
```
Функция извлекает и возвращает числовое значение, представляющее собой уровень астрального моба с заданным идентификатором.

## Список параметров
- **`unitId`** (`ObjectId`)
Уникальный идентификатор юнита (астрального моба), уровень которого необходимо получить.

## Возвращаемые значения
Возвращает `number` — целое число, представляющее собой уровень астрального моба.

## Примеры
### Получение уровня астрального моба
```lua
local level = astralUnit.GetLevel( unitId )
```
