# astralUnit.HasAggro
Возвращает признак наличия агрессии астрального юнита к кораблю главного игрока.

## Описание
```lua
astralUnit.HasAggro( unitId: ObjectId ): boolean
```
Функция проверяет состояние астрального юнита и возвращает логическое значение, указывающее на то, проявляет ли указанный юнит агрессию к кораблю, на котором в данный момент находится главный игрок. Отслеживание изменений состояния агрессии производится через событие `EVENT_ASTRAL_UNIT_AGGRO_CHANGED`, которое передается при модификации соответствующих данных на стороне сервера.

## Список параметров
- **`unitId`** (`ObjectId`)
Уникальный идентификатор астрального юнита, агрессивность которого необходимо проверить.

## Возвращаемые значения
Возвращает `boolean` — `true`, если астральный юнит агрессивен к кораблю главного игрока, и `false` в противном случае.

## Примеры
### Проверка агрессивности астрального юнита
```lua
local isAstralUnitInCombatWithMainPlayer = astralUnit.HasAggro( unitId )
```
