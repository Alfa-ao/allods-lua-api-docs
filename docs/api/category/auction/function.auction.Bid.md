# auction.Bid

Отправляет ставку на указанный лот аукциона.

## Описание

```lua
auction.Bid( id, price )
```

Функция отправляет процесс размещения ставки на аукционе для лота с заданным идентификатором. Итог выполнения (успех, ошибка или отклонение ставки) передается клиенту через событие `EVENT_AUCTION_BID_RESULT`.

## Список параметров

- **`id`** (`ObjectId`)
Уникальный идентификатор лота аукциона.

- **`price`** (`number`)
Сумма предлагаемой ставки в валюте аукциона.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Проверка состояния и размещение ставки

```lua
local state = auction.GetAuctionState( id )
if state then
    if not state.bidInProgress then
        auction.Bid( id, 1000 )
    end
end
```

::: info Описание примера
В примере сначала запрашивается текущее состояние лота аукциона. Если состояние успешно получено и на данный момент не идет процесс обработки другой ставки (флаг `bidInProgress` имеет значение `false`), отправляется новая ставка в размере 1000 единиц валюты.
:::
