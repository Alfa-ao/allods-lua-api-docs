# auction.Buyout

Выкупает предмет на аукционе по назначенной владельцем окончательной цене.

## Описание

```lua
auction.Buyout( id )
```

Функция отправляет запрос на выкуп аукционного лота по цене мгновенного выкупа, установленной продавцом. После отправки запроса сервер обрабатывает и возвращает итоговый статус выполнения через событие `EVENT_AUCTION_BID_RESULT`.

::: warning Замечание
Перед отправкой запроса на выкуп необходимо убедиться, что для данного лота не идет процесс обработки торгов (флаг `bidInProgress` должен быть равен `false`). Результат операции приходит в событии `EVENT_AUCTION_BID_RESULT`.
:::

Подробнее:
- [События (EVENT_AUCTION_*)](events.auction#event-auction-bid-result)

## Список параметров

- **`id`** (`ObjectId`)
Уникальный идентификатор лота на аукционе, который необходимо выкупить.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Безопасный выкуп лота с предварительной проверкой состояния

```lua
local state = auction.GetAuctionState( id )
if state then
    if not state.bidInProgress then
        auction.Buyout( id )
    end
end
```

::: info Описание примера
В примере сначала запрашивается текущее состояние аукциона для указанного идентификатора. Если состояние успешно получено и торги по лоту не находятся в процессе обработки (`bidInProgress` равен `false`), отправляется запрос на мгновенный выкуп.
:::
