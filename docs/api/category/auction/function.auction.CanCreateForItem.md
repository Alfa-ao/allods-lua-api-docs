# auction.CanCreateForItem

Определяет возможность создания аукциона для предмета с заданным идентификатором.

## Описание

```lua
auction.CanCreateForItem( itemId: ObjectId ): boolean
```

Функция проверяет, разрешено ли выставление на аукцион предмета с указанным идентификатором. Для успешного создания аукциона предмет должен находиться в инвентаре. Возвращает логическое значение, указывающее на доступность операции создания лота.

## Список параметров

- **`itemId`** (`ObjectId`)
Идентификатор предмета, который планируется выставить на аукцион. Предмет обязательно должен находиться в инвентаре.

## Возвращаемые значения

`boolean` — `true`, если для указанного предмета разрешено создание аукциона, и `false` в противном случае.

## Примеры

### Проверка возможности и создание аукциона

```lua
local itemId = 12345 -- Идентификатор предмета из инвентаря

if auction.CanCreateForItem( itemId ) then
    -- Создание аукциона: стартовая цена 10, цена выкупа 100, срок 24 часа
    auction.CreateForItem( itemId, 10, 100, AUCTION_CREATETIME_HOURS24 )
else
    LogInfo( "Невозможно создать аукцион для данного предмета." )
end
```

::: info Описание примера
В примере сначала выполняется проверка возможности создания лота. Если функция возвращает `true`, вызывается функция `auction.CreateForItem` для фактического выставления предмета на аукцион с заданными параметрами (стартовая цена, цена выкупа и длительность). Если возвращается `false`, создание аукциона пропускается, что предотвращает возможные ошибки выполнения.
:::
