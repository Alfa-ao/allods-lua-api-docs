# auction.GetAuctionInfo

Возвращает информацию об аукционе по его идентификатору.

## Описание

```lua
auction.GetAuctionInfo( id: ObjectId ): table|nil
```

Функция извлекает и возвращает детальную информацию о конкретном аукционе. Возвращаемые данные включают идентификаторы предмета и аукциона, информацию о текущей ставке, продавце и покупателе, ценовые параметры, а также оставшееся время до завершения торгов. Если аукцион с указанным идентификатором не найден, функция возвращает `nil`.

::: warning Замечание
Значение поля `participationStatus` в возвращаемой таблице представляет собой строку, соответствующую одному из предопределенных статусов участия `"ENUM_AuctionDescriptorParticipationStatus_..."`.
:::

Коды результата см. в [ENUM](enums.auction#enum_auctiondescriptorparticipationstatus).

## Список параметров

- **`id`** (`ObjectId`)
Уникальный идентификатор аукциона, информацию о котором необходимо получить.

## Возвращаемые значения

Возвращает таблицу (`table`):

- **`id`** (`ObjectId`) — уникальный идентификатор аукциона.
- **`itemId`** (`ObjectId`) — уникальный идентификатор предмета, выставленного на торги.
- **`bidderName`** (`WString`) — имя игрока, сделавшего последнюю ставку.
- **`currentBid`** (`number`) — размер последней ставки.
- **`sellerName`** (`WString`) — имя владельца аукциона (продавца).
- **`startPrice`** (`number`) — стартовая цена лота.
- **`buyoutPrice`** (`number`) — цена окончательного выкупа.
- **`participationStatus`** (`string`) — статус участия главного персонажа в данном аукционе.
- **`timeLeftSeconds`** (`number`) — общее оставшееся время до завершения торгов в секундах.
- **`timeLeft`** (`table`) — таблица с детализированным оставшимся временем, содержащая следующие поля:
  - **`d`** (`number`) — количество оставшихся дней.
  - **`h`** (`number`) — количество оставшихся часов.
  - **`m`** (`number`) — количество оставшихся минут.
  - **`s`** (`number`) — количество оставшихся секунд.

или `nil`, если аукцион не существует.

## Примеры

### Получение информации об активных аукционах

```lua
local auctions = auction.GetAuctions()
for i = 0, GetTableSize( auctions ) - 1 do
    local auctionInfo = auction.GetAuctionInfo( auctions[ i ] )
    if auctionInfo then
        local itemInfo = avatar.GetItemInfo( auctionInfo.itemId )
        -- Дальнейшая обработка информации о предмете и аукционе
    end
end
```

::: info Описание примера
В данном примере сначала получается список всех активных аукционов с помощью `auction.GetAuctions()`. Затем в цикле перебирается каждый идентификатор аукциона. Для каждого идентификатора вызывается `auction.GetAuctionInfo()`. 

Если информация успешно получена (аукцион существует), извлекается идентификатор предмета (`itemId`), который используется для получения полной информации о предмете через `avatar.GetItemInfo()`.
:::
