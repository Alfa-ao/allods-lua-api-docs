# auction.GetAuctions

Возвращает список идентификаторов аукционов на текущей странице, найденных с помощью запроса `auction.Search`.

## Описание

```lua
auction.GetAuctions()
```

Функция извлекает результаты предыдущего поиска аукционов, выполненного функцией `auction.Search`. Возвращаемый список привязан к текущей странице результатов. Если до вызова данной функции поиск не производился, возвращается пустая таблица.

::: warning Замечание
Функция не выполняет самостоятельный поиск. Она лишь возвращает результаты уже выполненного запроса `auction.Search`.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает таблицу (`table`), содержащую идентификаторы (`ObjectId`) аукционов. Индексация элементов в таблице начинается с нуля.

## Примеры

### Получение и перебор списка найденных аукционов

```lua
-- Получаем таблицу с идентификаторами аукционов
local auctions = auction.GetAuctions()

-- Перебираем полученную таблицу (индексация с 0)
for i = 0, GetTableSize( auctions ) - 1 do
    -- Получаем подробную информацию о каждом аукционе по его идентификатору
    local auctionInfo = auction.GetAuctionInfo( auctions[ i ] )
end
```

::: info Описание примера
В примере сначала извлекается таблица идентификаторов аукционов. Затем с помощью цикла `for` происходит перебор всех элементов таблицы. Поскольку индексация начинается с нуля, цикл выполняется от `0` до `GetTableSize(auctions) - 1`. На каждой итерации идентификатор передается в функцию `auction.GetAuctionInfo` для получения детальной информации о конкретном аукционе.
:::
