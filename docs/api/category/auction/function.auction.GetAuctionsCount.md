# auction.GetAuctionsCount

Возвращает количество аукционных лотов, найденных последним запросом `auction.Search`.

## Описание

```lua
auction.GetAuctionsCount(): number
```

Функция возвращает целое число, представляющее количество лотов аукциона, которые были найдены в результате выполнения метода `auction.Search`. Если поиск не производился или не завершился успешно, функция вернёт `0`.

## Список параметров

Функция не принимает параметров.

## Возвращаемые значения

Возвращает `number` — количество найденных аукционных лотов. Если поиск не был произведён, возвращает `0`.

## Примеры

### Получение и вывод количества найденных лотов

```lua
local auctionsCount = auction.GetAuctionsCount()
LogInfo( "Всего найдено лотов: ", auctionsCount )
```

::: info Описание примера
В переменную `auctionsCount` записывается количество лотов, найденных последним вызовом `auction.Search`. Затем это значение выводится в информационный лог. Если до этого `auction.Search` не вызывался, в логе будет выведено `0`.
:::
