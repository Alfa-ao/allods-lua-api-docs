# auction.GetAuctionsPage

Возвращает номер текущей страницы результатов поиска аукционов.

## Описание

```lua
auction.GetAuctionsPage()
```

Функция возвращает номер текущей страницы списка аукционов, сформированного в результате выполнения поискового запроса `auction.Search(...)`. Нумерация страниц начинается с единицы. Если поисковый запрос ранее не выполнялся, функция возвращает `0`.

## Список параметров

Функция не принимает параметров.

## Возвращаемые значения

Возвращает `number` — номер текущей страницы результатов поиска. Если поисковый запрос не был произведён, возвращается `0`.

## Примеры

### Получение номера текущей страницы аукционов

```lua
local auctionsPage = auction.GetAuctionsPage()

if auctionsPage > 0 then
    -- Поиск был выполнен, можно обрабатывать результаты текущей страницы
else
    -- Поисковый запрос ранее не выполнялся
end
```

::: info Описание примера
В примере демонстрируется безопасное получение номера страницы. Переменная `auctionsPage` будет содержать целое число, начиная с `1`, если до этого был успешно выполнен поиск через `auction.Search(...)`. В противном случае значение будет равно `0`, что позволяет избежать ошибок при попытке отобразить несуществующие данные.
:::
