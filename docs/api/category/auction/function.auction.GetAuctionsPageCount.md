# auction.GetAuctionsPageCount

Возвращает количество страниц в списке всех аукционов, найденных запросом `auction.Search(...)`.

## Описание

```lua
auction.GetAuctionsPageCount(): number
```

Функция возвращает целое число, представляющее общее количество страниц в результатах поиска аукционов. Количество страниц формируется на основе параметров, переданных в функцию поиска. Если предварительный запрос через `auction.Search(...)` не производился или не дал результатов, функция вернёт `0`.

::: warning Замечание
Для получения актуального количества страниц необходимо предварительно выполнить поиск с помощью функции `auction.Search(...)`. Текущая активная страница при этом получается через `auction.GetAuctionsPage()`.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` — целое число, обозначающее общее количество страниц в результатах поиска.

## Примеры

### Получение и вывод информации о страницах аукциона

```lua
local auctionsPage = auction.GetAuctionsPage()
local auctionsPageCount = auction.GetAuctionsPageCount()
LogInfo( "Страница", auctionsPage, " из ", auctionsPageCount )
```

::: info Описание примера
В лог информации будет выведено сообщение с текущим номером страницы и общим количеством страниц, например: `Страница 1 из 5`. Если поиск не был выполнен, будет выведено: `Страница 0 из 0`.
:::
