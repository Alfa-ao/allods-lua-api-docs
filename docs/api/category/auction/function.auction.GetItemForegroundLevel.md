# auction.GetItemForegroundLevel

Возвращает приоритетный уровень предмета при его продаже на аукционе.

## Описание

```lua
auction.GetItemForegroundLevel( id )
```

Приоритетный уровень предмета представляет собой значение, на которое удобнее всего ориентироваться игроку при покупке лота. В зависимости от различных условий и характеристик, приоритетным уровнем может являться либо фактический уровень самого предмета, либо минимальный требуемый уровень для его экипировки или использования.

## Список параметров

- **`id`** (`ObjectId`)
Уникальный идентификатор предмета, выставленного на аукционе.

## Возвращаемые значения

Возвращает целое число (`number`), представляющее собой приоритетный уровень предмета. В случае возникновения ошибки или отсутствия предмета возвращается `nil`.

## Примеры

### Получение приоритетного уровня предмета

```lua
local foregroundLevel = auction.GetItemForegroundLevel( id )
```
