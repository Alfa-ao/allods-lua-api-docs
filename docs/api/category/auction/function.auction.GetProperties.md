# auction.GetProperties

Возвращает информацию о свойствах аукциона.

## Описание

```lua
auction.GetProperties()
```

Функция возвращает таблицу с информацией о свойствах аукциона. Если данные ещё не были получены с сервера, функция возвращает `nil` и посылает запрос к серверу. Ответ от сервера приходит в виде события `EVENT_AUCTION_PROPERTIES`.

При возникновении ошибки на стороне сервера повторный запрос возможен через определённый промежуток времени (в текущей реализации — 10 секунд). После успешного получения данных повторные вызовы функции не приводят к отправке новых запросов на сервер, а возвращают ранее сохранённый (кэшированный) результат.

::: warning Замечание
Если данные ещё не кэшированы, возвращается `nil`, а фактические свойства станут доступны только после получения и обработки события `EVENT_AUCTION_PROPERTIES`.
:::

Подробнее:
- [События (EVENT_AUCTION_*)](events.auction#event-auction-properties)

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если данные успешно получены, возвращается таблица со следующими полями:
- **`pawnFactorPercent`** (`number`) — размер залога в процентах (целое число).
- **`bidStepInPercent`** (`number`) — шаг ставки в процентах (целое число).
- **`discardingFineInPercent`** (`number`) — стоимость отмены аукциона в процентах (целое число).
- **`taxInPercent`** (`number`) — размер комиссии (таксы) за выставление на аукцион в процентах (целое число).
- **`searchPageSize`** (`number`) — количество элементов на одной странице поиска (целое число).
- **`bidIncreaseTimeSec`** (`number`) — интервал времени в секундах, на который увеличивается время аукциона при совершении ставки (целое число).

Если данные ещё не получены с сервера или находится в процессе загрузки, возвращается `nil`.

## Примеры

### Получение и использование свойств аукциона

```lua
local properties = auction.GetProperties()

if properties then
    -- Вычисление суммы залога на основе цены предмета
    local price = 1000
    local pawnPercent = properties.pawnFactorPercent / 100
    local pawnAmount = price * pawnPercent
    
    print("Размер залога: " .. pawnAmount)
else
    print("Данные ещё не получены с сервера. Ожидание события EVENT_AUCTION_PROPERTIES.")
end
```

::: info Описание примера
В примере выполняется попытка получить свойства аукциона. Если таблица `properties` не равна `nil`, вычисляется сумма залога (`pawnAmount`) путём умножения цены предмета на процент залога (`pawnFactorPercent`), переведенный в десятичную дробь. Если данные ещё не загружены, выводится сообщение о необходимости ожидания события `EVENT_AUCTION_PROPERTIES`.
:::
