# auction.IsCreationInProgress

Проверяет, находится ли процесс создания аукциона в активном состоянии.

## Описание

```lua
auction.IsCreationInProgress()
```

Функция возвращает логическое значение, указывающее на текущий статус асинхронной операции создания аукциона. Результат принимает значение `true`, если создание аукциона было инициировано (например, с помощью `auction.CreateForItem`), но ответное событие `EVENT_AUCTION_CREATION_RESULT` от сервера ещё не получено. Если процесс не активен или уже завершён, возвращается `false`.

::: warning Замечание
Функция предназначена для предотвращения дублирования запросов на создание аукциона до получения ответа от сервера.
:::

Подробнее:
- [События (EVENT_AUCTION_*)](events.auction#event-auction-creation-result)

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` — `true`, если идёт процесс создания аукциона, иначе `false`.

## Примеры

### Проверка перед созданием аукциона

```lua
if not auction.IsCreationInProgress() then
  auction.Create( slot, startPrice, buyoutPrice, time )
end
```

::: info Разбор результата
Перед вызовом функции создания аукциона проверяется, не выполняется ли уже аналогичный запрос. Это позволяет избежать отправки дублирующих пакетов на сервер и возможных ошибок до получения результата предыдущей операции.
:::
