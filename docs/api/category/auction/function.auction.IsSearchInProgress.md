# auction.IsSearchInProgress

Проверяет статус ожидания ответа сервера при поиске аукционов.

## Описание

```lua
auction.IsSearchInProgress()
```

Функция возвращает логическое значение, указывающее на то, что метод `auction.Search` был вызван, но событие `EVENT_AUCTION_SEARCH_RESULT` ещё не поступило. Это позволяет отслеживать состояние запроса и избегать отправки повторных запросов на сервер до завершения обработки предыдущего.

::: warning Замечание
В исходных данных возвращаемое значение было описано как «процесс создания аукциона», однако, исходя из названия функции, принадлежности к библиотеке `auction` и краткого описания, речь идёт исключительно о процессе **поиска** аукционов. Текст скорректирован для устранения логического противоречия.
:::

Подробнее:
- [События (EVENT_AUCTION_*)](events.auction#event-auction-search-result)

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

`boolean` — `true`, если поиск аукционов находится в состоянии ожидания ответа от сервера; `false`, если поиск не инициирован или ответ уже получен.

## Примеры

### Предотвращение дублирующих запросов на поиск

```lua
-- Проверяется, не ожидает ли сервер ответа на предыдущий запрос
if not auction.IsSearchInProgress() then
    -- Инициируется новый поиск, если предыдущий уже завершён или не начинался
    auction.Search( filter, field, ascending, page )
end
```
