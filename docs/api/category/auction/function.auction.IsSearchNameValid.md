# auction.IsSearchNameValid

Проверяет допустимость маски для поиска по имени в фильтре аукциона.

## Описание

```lua
auction.IsSearchNameValid( name )
```

Функция анализирует переданную строку и определяет, является ли она допустимой маской для поиска по имени в фильтре поиска на аукционе. Результат проверки предназначен для валидации пользовательского ввода перед выполнением поиска.

::: warning Замечание
Для выполнения непосредственного поиска по аукциону используется функция `auction.Search`.
:::

## Список параметров

- **`name`** (`WString` | `nil`)
Маска для поиска по имени. Передача значения `nil` интерпретируется как отсутствие фильтра по имени (пустая маска).

## Возвращаемые значения

Возвращает `boolean` — `true`, если маска допустима для поиска, и `false`, если маска содержит недопустимые символы или некорректный формат.

## Примеры

### Валидация ввода и управление кнопкой поиска

```lua
local isValid = auction.IsSearchNameValid( filter.name )
wtSearchBtn:Enable( isValid )
```

::: info Разбор результата
В примере проверяется корректность имени, заданного в фильтре. Если маска допустима, кнопка поиска (`wtSearchBtn`) переводится в активное состояние. В противном случае кнопка остается заблокированной, что предотвращает выполнение некорректного запроса к аукциону.
:::
