# avatar.AcceptShareQuest
Отправляет согласие на принятие разделяемого задания от другого игрока.

## Описание

```lua
avatar.AcceptShareQuest( shareId: ObjectId )
```

Функция отправляет серверу запрос на согласие принять разделяемое задание, предложенное другим игроком.

## Список параметров

- **`shareId`** (`ObjectId`)
Уникальный идентификатор предложения о разделении задания.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры

### Отправка согласия на принятие задания

```lua
local shareId = 12345 -- Уникальный идентификатор предложения
avatar.AcceptShareQuest( shareId )
```
