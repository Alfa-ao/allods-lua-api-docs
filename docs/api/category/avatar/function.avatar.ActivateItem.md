# avatar.ActivateItem
Активирует неактивированный предмет с использованием специального инструмента-активатора.

## Описание
```lua
avatar.ActivateItem( itemId: ObjectId, instrumentId: ObjectId )
```
Функция инициирует процесс активации предмета, находящегося в инвентаре. Неактивированные предметы часто появляются в результате крафта и требуют применения специальных предметов-активаторов для получения своих базовых свойств и возможности дальнейшего использования. Для успешного выполнения операции инструмент-активатор также должен находиться в инвентаре персонажа.

::: warning Замечание
Перед активацией рекомендуется проверить возможность проведения операции с помощью функции `avatar.CanActivateItem`, чтобы избежать ошибок выполнения. Дополнительную информацию о требованиях к активации конкретного предмета можно получить через `itemLib.GetActivationInfo`.
:::

::: tip Совет
Список всех доступных активаторов для заданного предмета возвращается функцией `itemLib.GetActivators`.
:::

## Список параметров
- **`itemId`** (`ObjectId`)
Уникальный идентификатор неактивированного предмета, который необходимо активировать. Предмет обязательно должен находиться в инвентаре.

- **`instrumentId`** (`ObjectId`)
Уникальный идентификатор предмета-активатора, используемого для проведения операции активации.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Активация предмета с предварительной проверкой
```lua
local itemId = 12345 -- Идентификатор неактивированного предмета
local instrumentId = 67890 -- Идентификатор инструмента-активатора

if avatar.CanActivateItem( itemId, instrumentId ) then
    avatar.ActivateItem( itemId, instrumentId )
else
    LogInfo( "Невозможно активировать предмет данным активатором." )
end
```
::: info Описание примера
В примере сначала выполняется проверка возможности активации предмета с помощью функции `avatar.CanActivateItem`. Если проверка успешна, инициируется процесс активации через `avatar.ActivateItem`. В противном случае выводится информационное сообщение, что предотвращает выполнение некорректного запроса и возможные ошибки.
:::
