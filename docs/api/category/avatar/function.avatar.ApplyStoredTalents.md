# avatar.ApplyStoredTalents
Выучивает помеченные таланты и вехи.

## Описание
```lua
avatar.ApplyStoredTalents()
```
Функция инициирует процесс изучения талантов и вех, которые были предварительно помечены для сохранения. Пометка конкретных талантов осуществляется с помощью функции `avatar.StoreFieldTalent`. После вызова функции все накопленные пометки применяются к персонажу.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Пометка и применение талантов
```lua
-- Пометка таланта в определенной ячейке
local field = 1
local row = 2
local column = 3
avatar.StoreFieldTalent( field, row, column )

-- Применение всех помеченных талантов и вех
avatar.ApplyStoredTalents()
```
::: info Описание примера
В примере сначала выполняется пометка конкретного таланта с указанием координат ячейки через функцию `avatar.StoreFieldTalent`. Затем вызывается `avatar.ApplyStoredTalents`, которая применяет все накопленные пометки, фактически изучая помеченные таланты и вехи.
:::
