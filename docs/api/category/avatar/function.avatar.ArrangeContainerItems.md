# avatar.ArrangeContainerItems
Перемещает предметы внутри контейнера в заданном порядке, располагая их в его начале.

## Описание
```lua
avatar.ArrangeContainerItems( moves: table, compact: boolean, containerType: number, pocket: ?number )
```
Функция инициирует процесс перемещения нескольких предметов внутри указанного контейнера. Предметы извлекаются из исходных слотов и помещаются в целевые слоты в соответствии с переданной таблицей перемещений. Дополнительно может производиться компактизация стеков и пустых слотов.

::: warning Замечание
Перемещение предметов в банковской ячейке невозможно, если она не открыта.
:::

## Список параметров
- **`moves`** (`table`)
Таблица, задающая порядок перемещения предметов. Ключом таблицы выступает номер целевого слота, в который необходимо поместить предмет, а значением — номер исходного слота (считая с начала кармана), из которого предмет извлекается. Индексация ключей начинается с нуля.
- **`compact`** (`boolean`)
Флаг, определяющий необходимость выполнения компактизации. При значении `true` производится автоматическое объединение стекуемых предметов и сжатие пустых слотов.
- **`containerType`** (`number`)
Идентификатор контейнера, в котором выполняется перемещение. Задается константами `ITEM_CONT_...`. Поддерживается перемещение в сумке и банковской ячейке.
- **`pocket`** (`number` | `nil`)
Индекс кармана внутри контейнера. Базовый карман имеет индекс `-1`, остальные нумеруются с нуля. Если параметр не указан или передан `nil`, используется базовый карман (`-1`).

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Обмен предметами между первыми двумя слотами сумки
```lua
-- Меняем местами предметы из слота 0 и слота 1 в базовом кармане сумки
avatar.ArrangeContainerItems( { [0] = 1, [1] = 0 }, false, ITEM_CONT_INVENTORY )
```
::: info Описание примера
В примере формируется таблица перемещений, где предмет из слота `1` перемещается в слот `0`, а предмет из слота `0` перемещается в слот `1`. Компактизация отключена (`false`), операция выполняется в базовом кармане инвентаря (`ITEM_CONT_INVENTORY`).
:::
