# avatar.BindMountMetamorphToActionPanel

Привязывает вызов маунта-метаморфа из конюшни главного персонажа к слоту экшен панели действий.

## Описание

```lua
avatar.BindMountMetamorphToActionPanel( index )
```

Функция инициирует привязку использования маунта-метаморфа, полученного из конюшни главного персонажа, к конкретному слоту экшен панели действий. После выполнения привязки игрок получает возможность активировать трансформацию в маунта-метаморфа напрямую из указанного слота панели.

::: tip Совет
Перед привязкой рекомендуется проверять актуальное количество доступных слотов панели действий, чтобы избежать выхода за границы допустимых индексов.
:::

## Список параметров

- **`index`** (`number`)
Индекс слота панели действий, к которому привязывается вызов маунта-метаморфа. Значение должно находиться в диапазоне от `0` до максимального количества слотов.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Привязка маунта-метаморфа к первому слоту панели действий

```lua
avatar.BindMountMetamorphToActionPanel( 0 )
```

::: info Описание примера
Выполняется привязка вызова маунта-метаморфа из конюшни к слоту панели действий с индексом `0`. После успешного выполнения операции соответствующее действие становится доступным для использования в первом слоте экшен панели.
:::
