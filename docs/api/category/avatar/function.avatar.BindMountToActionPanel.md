# avatar.BindMountToActionPanel

Привязывает вызов маунта из конюшни главного игрока к слоту панели действий.

## Описание

```lua
avatar.BindMountToActionPanel( mountId, index )
```

Функция инициирует процесс привязки конкретного маунта из конюшни персонажа к указанному слоту панели действий. После выполнения операции маунт становится доступен для быстрого вызова непосредственно из интерфейса экшен-панели.

## Список параметров

- **`mountId`** (`ObjectId`)
Уникальный идентификатор маунта из конюшни главного игрока, который необходимо привязать к панели действий.

- **`index`** (`number`)
Индекс слота панели действий, к которому привязывается маунт. Значение должно находиться в диапазоне от 0 до максимального количества доступных слотов.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Привязка первого доступного маунта к нулевому слоту

```lua
local mounts = mount.GetMounts()
if mounts and mounts[0] then
    avatar.BindMountToActionPanel( mounts[0], 0 )
end
```

::: info Описание примера
В примере извлекается список доступных маунтов. При наличии маунта с индексом `0` вызывается функция привязки, которая закрепляет его за нулевым слотом панели действий.
:::
