# avatar.BindSpellToActionPanel

Привязывает умение из книги умений к указанному слоту панели действий.

## Описание

```lua
avatar.BindSpellToActionPanel( spellId, index )
```

Функция привязывает умение из книги умений к заданному слоту панели действий. При получении новых умений той же цепочки более высокого уровня, если новое умение вытесняет старое, все привязки старого умения в панели заменяются на новые. Если умение нового уровня добавляется без вытеснения, заменяется привязка к первому слоту умения наибольшего имеющегося в панели уровня.

Предусмотрен механизм автоматической привязки. Если у ресурса умения указано поле `DefaultAction`, то при получении умения первого уровня оно автоматически привязывается к заданному слоту, при условии, что слот пуст.

(HELP) Уточнить, что это такое "...:load_action_panel"

Состояние панели сохраняется в файл `Personal/user.cfg` и восстанавливается при входе в игру. Восстановление отключается глобальной переменной `[:DebugGame#head-0b2a7b310dbb423db644652c70a9638e03d13671:load_action_panel]`. При отключенном восстановлении умения привязываются автоматически (все умения, а не только первого уровня).

::: warning Замечание
Сторонним аддонам запрещено использовать данную функцию, пока персонаж игрока находится в бою.
:::

## Список параметров

- **`spellId`** (`SpellId`)
Идентификатор умения, привязываемого к слоту.

- **`index`** (`number`)
Индекс слота панели действий. Значение находится в диапазоне от `0` до максимального количества слотов.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Привязка первого умения из книги умений к первому слоту

```lua
local spellbook = avatar.GetSpellBook()
local spellId = spellbook[0]

avatar.BindSpellToActionPanel( spellId, 0 )
```
