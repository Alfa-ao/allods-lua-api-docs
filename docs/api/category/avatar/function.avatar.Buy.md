# avatar.Buy
Отправляет запрос на покупку предмета у NPC-торговца.

## Описание
```lua
avatar.Buy( objectId: ObjectId, quantity: number )
```
Функция инициирует процесс приобретения указанного количества предметов у NPC-торговца. Запрос отправляется на сервер, который проверяет наличие предмета в ассортименте торговца, а также достаточность средств у персонажа для совершения операции.

## Список параметров
- **`objectId`** (`ObjectId`)
Уникальный идентификатор предмета в списке товаров NPC-торговца, который необходимо приобрести.
- **`quantity`** (`number`)
Целочисленное значение, определяющее количество предметов для покупки.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Покупка предмета из списка торговца
```lua
local list = avatar.GetVendorList()
if list[0] then
  local itemId = list[0].id
  avatar.Buy( itemId, 1 )
end
```
::: info Описание примера
В примере сначала извлекается список товаров текущего NPC-торговца с помощью функции `avatar.GetVendorList()`. Далее проверяется наличие хотя бы одного предмета в списке (индекс `0`). Если предмет существует, извлекается его идентификатор и отправляется запрос на покупку одной единицы данного товара.
:::
