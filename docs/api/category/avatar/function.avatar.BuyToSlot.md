# avatar.BuyToSlot
Покупает предмет у NPC-торговца и помещает его в указанный слот инвентаря.

## Описание
```lua
avatar.BuyToSlot( objectId: ObjectId, quantity: number, slot: number ): nil
```
Функция инициирует процесс покупки предмета у текущего NPC-торговца. Приобретаемый предмет автоматически помещается в заданный слот инвентаря главного персонажа. Количество покупаемых единиц задается соответствующим параметром. Для успешного выполнения операции предмет должен присутствовать в списке товаров торговца, а у персонажа должно быть достаточно средств.

## Список параметров
- **`objectId`** (`ObjectId`)
  Уникальный идентификатор предмета, который необходимо приобрести у NPC-торговца.
- **`quantity`** (`number`)
  Количество единиц предмета для покупки. Значение должно быть целым положительным числом.
- **`slot`** (`number`)
  Номер слота инвентаря, в который будет помещен купленный предмет.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Покупка первого предмета из списка торговца
```lua
local targetSlot = 12
local vendorList = avatar.GetVendorList()

if vendorList[0] ~= nil then
    local itemId = vendorList[0].id
    avatar.BuyToSlot( itemId, 1, targetSlot )
end
```
::: info Описание примера
В примере извлекается список товаров текущего NPC-торговца. Если список не пуст и содержит хотя бы один элемент (индекс `0`), извлекается идентификатор первого предмета. Затем инициируется покупка одной единицы этого предмета с автоматическим помещением в слот инвентаря с номером `12`.
:::
