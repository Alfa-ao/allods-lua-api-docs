# avatar.CalcCurrentGearScore
Высчитывает и возвращает текущий рейтинг экипировки аватара.

## Описание
```lua
avatar.CalcCurrentGearScore(): number
```
Функция анализирует надетые на главного персонажа предметы и на основе их характеристик вычисляет суммарный рейтинг экипировки (Gear Score). Вычисленное числовое значение возвращается в качестве результата. Полученный рейтинг позволяет быстро оценить общий уровень снаряжения аватара без необходимости вручную суммировать параметры отдельных предметов.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — текущее значение рейтинга экипировки.

## Примеры
### Получение и вывод текущего рейтинга экипировки
```lua
local avatarCurrentGearScore = avatar.CalcCurrentGearScore()
LogInfo( "Текущий рейтинг экипировки: ", avatarCurrentGearScore )
```
