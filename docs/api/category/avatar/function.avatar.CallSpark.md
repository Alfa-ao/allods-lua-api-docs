# avatar.CallSpark
Инициирует преждевременный вызов искры главного игрока с кладбища.

## Описание
```lua
avatar.CallSpark(): nil
```
Функция отправляет запрос на преждевременный вызов искры главного игрока, находящегося на кладбище. Операция вызова искры требует оплаты внутриигровой валютой (мирро). Если на счету персонажа недостаточно средств для совершения операции, сервер отклоняет запрос, и клиенту передается событие `EVENT_CALL_SPARK_FAILED`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Преждевременный вызов искры
```lua
avatar.CallSpark()
```
