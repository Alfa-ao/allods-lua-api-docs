# avatar.CanPlaceMetaUpgradeAgent
Проверяет возможность помещения предмета в основу агента мета-предметов.

## Описание
```lua
avatar.CanPlaceMetaUpgradeAgent( agentId: ObjectId ): boolean
```
Функция проверяет, разрешено ли помещать указанный предмет в основу агента мета-предметов, и возвращает логическое значение, указывающее на результат данной проверки.

## Список параметров
- **`agentId`** (`ObjectId`)
Идентификатор улучшаемого предмета.

## Возвращаемые значения
Возвращает `boolean` — `true`, если предмет можно поместить в основу агента мета-предметов, и `false` в противном случае.

## Примеры
### Проверка возможности помещения предмета
```lua
local canPlace = avatar.CanPlaceMetaUpgradeAgent( agentId )
```
