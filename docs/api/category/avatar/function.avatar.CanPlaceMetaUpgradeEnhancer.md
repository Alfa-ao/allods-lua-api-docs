# avatar.CanPlaceMetaUpgradeEnhancer
Проверяет возможность размещения предмета-улучшителя в слот улучшаемого мета-предмета.

## Описание
```lua
avatar.CanPlaceMetaUpgradeEnhancer( sourceId: ObjectId | nil, enhancerId: ObjectId ): boolean
```
Функция проверяет, разрешено ли размещение предмета-улучшителя с заданным идентификатором в слот улучшаемого мета-предмета. Результат проверки возвращается в виде логического значения, что позволяет определить совместимость предметов и текущее состояние слотов до фактического выполнения операции улучшения.

## Список параметров
- **`sourceId`** (`ObjectId` | `nil`)
Уникальный идентификатор улучшаемого предмета. Передача значения `nil` допустима, если для конкретного типа улучшения источник не требуется.
- **`enhancerId`** (`ObjectId`)
Уникальный идентификатор предмета-улучшителя, который планируется поместить в слот.

## Возвращаемые значения
Возвращает `boolean` — `true`, если предмет-улучшитель может быть размещен в слот улучшаемого предмета, и `false` в противном случае.

## Примеры
### Проверка возможности размещения улучшителя
```lua
local sourceItem = avatar.GetItemInfo( sourceId )
if sourceItem then
    local canPlace = avatar.CanPlaceMetaUpgradeEnhancer( sourceId, enhancerId )
    if canPlace then
        -- Предмет-улучшитель совместим и может быть размещен
    end
end
```
::: info Описание примера
В примере сначала запрашивается информация об улучшаемом предмете с помощью `avatar.GetItemInfo`. При успешном получении данных вызывается функция `avatar.CanPlaceMetaUpgradeEnhancer` для проверки возможности размещения предмета-улучшителя. Если функция возвращает `true`, предмет совместим и готов к использованию в операции улучшения.
:::
