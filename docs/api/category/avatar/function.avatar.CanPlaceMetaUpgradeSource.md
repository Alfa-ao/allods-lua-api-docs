# avatar.CanPlaceMetaUpgradeSource
Проверяет возможность размещения предмета в слот основы улучшателя мета-предметов.

## Описание
```lua
avatar.CanPlaceMetaUpgradeSource( sourceId: ObjectId ): boolean
```
Функция анализирует переданный идентификатор предмета и определяет, допускается ли его использование в качестве основы для процедуры улучшения мета-предметов. Возвращает логическое значение, подтверждающее или отрицающее возможность выполнения данной операции.

## Список параметров
- **`sourceId`** (`ObjectId`)
Уникальный идентификатор предмета, проверяемого на возможность размещения в слот основы улучшателя.

## Возвращаемые значения
Возвращает `boolean` — `true`, если предмет допускается к использованию в качестве основы, и `false`, если размещение невозможно или предмет не соответствует требованиям.

## Примеры
### Проверка возможности размещения предмета
```lua
local sourceId = 12345
local canPlace = avatar.CanPlaceMetaUpgradeSource( sourceId )
```
