# avatar.CanUpdateBaseTalent
Проверяет возможность обновления или изучения таланта из базовой таблицы талантов.

## Описание
```lua
avatar.CanUpdateBaseTalent( layer: number, line: number ): boolean
```
Функция анализирует состояние базовой таблицы талантов и определяет, доступно ли для изучения или повышения уровня талант, расположенный на пересечении заданной строки и столбца. Проверка учитывает наличие свободных очков таланта, выполнение предварительных требований и достижение необходимого уровня персонажа. Возвращает логическое значение, указывающее на возможность совершения операции без фактического применения изменений.

## Список параметров
- **`layer`** (`number`)
Номер строки в базовой таблице талантов, в которой расположен проверяемый талант.
- **`line`** (`number`)
Номер столбца в базовой таблице талантов, в котором расположен проверяемый талант.

## Возвращаемые значения
Возвращает `boolean` — `true`, если талант можно обновить или выучить, и `false` в противном случае.

## Примеры
### Проверка возможности изучения таланта
```lua
local layer = 1
local line = 2

local canUpdate = avatar.CanUpdateBaseTalent( layer, line )

if canUpdate then
    -- Талант доступен для изучения или обновления
end
```
