# avatar.CanUpdateFieldTalent
Проверяет возможность обновления или изучения таланта в указанной ячейке поля талантов.

## Описание
```lua
avatar.CanUpdateFieldTalent( field: number, row: number, column: number ): boolean
```
Функция анализирует указанные координаты ячейки в поле талантов и определяет, доступно ли для нее действие по обновлению (повышению уровня) или первичному изучению таланта. Возвращает логическое значение, указывающее на доступность операции с учетом текущих характеристик персонажа, имеющихся очков талантов и внутренних требований самого таланта.

## Список параметров
- **`field`** (`number`)
Номер поля талантов.

- **`row`** (`number`)
Номер строки, в которой расположена проверяемая ячейка таланта.

- **`column`** (`number`)
Номер столбца, в котором расположена проверяемая ячейка таланта.

## Возвращаемые значения
Возвращает `boolean` — `true`, если талант в указанной ячейке можно обновить или изучить, и `false`, если действие недоступно.

## Примеры
### Проверка возможности изучения или обновления таланта
```lua
local field = 1
local row = 2
local column = 3

local canUpdate = avatar.CanUpdateFieldTalent( field, row, column )
if canUpdate then
    LogInfo( "Талант доступен для изучения или обновления." )
end
```
