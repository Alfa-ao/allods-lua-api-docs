# avatar.CancelClassChanger
Отправляет на сервер запрос на отказ от продолжения процесса смены класса.

## Описание
```lua
avatar.CancelClassChanger(): nil
```
Функция инициирует отправку на сервер запроса на прерывание и отмену текущего процесса смены класса. Данная операция применима на стадии выбора замен для предметов экипировки. После вызова функции процесс смены класса прекращается.

::: warning Замечание
Функция применяется исключительно на стадии выбора замен для предметов. Для получения подробной информации о текущем состоянии процесса и необходимых заменах используется функция `avatar.GetClassChangerInfo()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Отмена процесса смены класса
```lua
avatar.CancelClassChanger()
```
