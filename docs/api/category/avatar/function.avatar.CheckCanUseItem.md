# avatar.CheckCanUseItem
Проверяет возможность использования предмета из контейнера игрока.

## Описание
```lua
avatar.CheckCanUseItem( itemId: ObjectId, isSendEvent: boolean ): boolean
```
Функция анализирует предмет с заданным идентификатором и определяет, разрешено ли его текущее использование. Проверка учитывает внутренние ограничения предмета, такие как требуемый уровень, класс персонажа, состояние экипировки или наличие специфических условий.

## Список параметров
- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета, находящегося в контейнере игрока, который необходимо проверить на возможность использования.
- **`isSendEvent`** (`boolean`)
Флаг, управляющий генерацией системного уведомления. Если передано значение `true`, при невозможности использования предмета клиенту отправляется и отображается соответствующее сообщение с причиной отказа. Если передано `false`, проверка выполняется без вывода пользовательских уведомлений.

## Возвращаемые значения
Возвращает `boolean` — `true`, если использование предмета разрешено, и `false`, если предмет не может быть использован.

## Примеры
### Скрытая проверка возможности использования
```lua
if not avatar.CheckCanUseItem( itemId, false ) then
    return
end
```
::: info Описание примера
Выполняется скрытая проверка предмета. Параметр `isSendEvent` установлен в `false`, что предотвращает вывод стандартных системных сообщений об ошибке. Если предмет использовать нельзя, дальнейшее выполнение прерывается.
:::
