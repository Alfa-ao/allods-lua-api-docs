# avatar.CheckCanUseItemOnPoint
Проверяет возможность использования предмета из контейнера игрока на определенную точку ландшафта.

## Описание
```lua
avatar.CheckCanUseItemOnPoint( itemId: ObjectId, position: GamePosition, isSendEvent: boolean ): boolean
```
Функция выполняет проверку, разрешено ли использование предмета с заданным идентификатором на указанную точку ландшафта. Предмет обязательно должен находиться в контейнере (инвентаре) игрока. Результат проверки возвращается в виде логического значения, указывающего на возможность выполнения операции воздействия на ландшафт.

## Список параметров
- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета, который планируется использовать. Предмет должен физически присутствовать в контейнере игрока.
- **`position`** (`GamePosition`)
Таблица, описывающая координаты точки на ландшафте, на которую предполагается воздействовать предметом.
- **`isSendEvent`** (`boolean`)
Флаг, определяющий необходимость вывода системного сообщения в случае неудачной проверки. Если передано значение `true`, при невозможности использования предмета будет автоматически сгенерировано и выведено соответствующее сообщение об ошибке.

## Возвращаемые значения
Возвращает `boolean` — `true`, если использование предмета на указанную точку разрешено, и `false` в противном случае.

## Примеры
### Проверка и использование предмета на точке ландшафта
```lua
local itemId = 12345 -- Идентификатор предмета из инвентаря
local position = { x = 100.0, y = 200.0, z = 300.0 } -- Координаты точки на ландшафте

if avatar.CheckCanUseItemOnPoint( itemId, position, false ) then
    avatar.UseItemOnPoint( itemId, position )
else
    LogInfo( "Использование предмета на данной точке невозможно." )
end
```
::: info Описание примера
В примере формируются идентификатор предмета и координаты точки на ландшафте. Затем вызывается функция проверки с параметром `isSendEvent`, равным `false`, чтобы избежать автоматического вывода системных сообщений об ошибке на экран. Если проверка возвращает `true`, инициируется фактическое использование предмета через `avatar.UseItemOnPoint`. В противном случае выводится информационное сообщение в лог.
:::
