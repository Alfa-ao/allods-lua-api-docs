# avatar.ClearActionPanel
Полностью очищает текущую раскладку панели действий.

## Описание
```lua
avatar.ClearActionPanel()
```
Функция освобождает все ячейки и полностью обнуляет конфигурацию текущей панели действий. Изменения применяются исключительно к активной раскладке и не затрагивают другие варианты панелей, такие как раскладки для разных классов или граней талантов.

::: warning Замечание
В пользовательских аддонах действует ограничение: метод срабатывает исключительно в обработчиках реакции на действия мыши (например, нажатие кнопки), но не на события перемещения курсора.
:::

::: warning Замечание
В пользовательских аддонах запрещено вызывать данную функцию, пока главный персонаж игрока находится в состоянии боя.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Очистка текущей панели действий
```lua
avatar.ClearActionPanel()
```
