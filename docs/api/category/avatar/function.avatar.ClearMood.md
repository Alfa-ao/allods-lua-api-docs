# avatar.ClearMood
Сбрасывает эмоцию у главного игрока.

## Описание
```lua
avatar.ClearMood(): nil
```
Функция сбрасывает текущую эмоцию (настроение) главного игрока, возвращая его визуальное состояние к нейтральному. После вызова функции проигрываемая анимация эмоции немедленно прекращается.

::: warning Замечание
Функция выполняется мгновенно и не требует предварительной проверки наличия активной эмоции. Если эмоция не была установлена, вызов не приводит к ошибкам.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Сброс эмоции персонажа
```lua
-- Сброс текущей эмоции главного игрока
avatar.ClearMood()
```
