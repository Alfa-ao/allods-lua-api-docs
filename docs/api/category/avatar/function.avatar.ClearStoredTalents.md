# avatar.ClearStoredTalents
Сбрасывает помеченные таланты и вехи.

## Описание
```lua
avatar.ClearStoredTalents()
```
Функция очищает внутренний список ранее помеченных талантов и вех, которые были сохранены с помощью функций распределения (например, `avatar.StoreFieldTalent`). Применяется для полного сброса накопленных отметок перед началом нового процесса анализа билда или перераспределения очков.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Сохранение и последующий сброс отметок талантов
```lua
avatar.StoreFieldTalent( field, row, column )
avatar.ClearStoredTalents()
```
::: info Описание примера
В примере сначала выполняется сохранение отметки таланта в определенной ячейке поля с помощью `avatar.StoreFieldTalent`. Затем вызывается `avatar.ClearStoredTalents` для полного сброса всех накопленных отметок.
:::
