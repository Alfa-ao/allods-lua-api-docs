# avatar.DeactivateUsableDevice
Завершает использование активного устройства.

## Описание
```lua
avatar.DeactivateUsableDevice()
```
Функция инициирует процесс отключения и прекращения использования активного устройства. Возвращаемое значение отсутствует.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Завершение использования устройства
```lua
avatar.DeactivateUsableDevice()
```
