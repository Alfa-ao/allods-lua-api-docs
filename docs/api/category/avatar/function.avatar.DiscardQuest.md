# avatar.DiscardQuest
Инициирует удаление задания из журнала заданий аватара.

## Описание
```lua
avatar.DiscardQuest( questId: QuestId )
```
Функция отправляет запрос на сервер для исключения указанного задания из активного журнала заданий аватара. Задание с заданным идентификатором удаляется из списка текущих заданий.

::: warning Замечание
Удаление задания из журнала является необратимым действием. Для повторного принятия задания может потребоваться выполнение начальных условий или повторное обращение к соответствующему NPC.
:::

## Список параметров
- **`questId`** (`QuestId`)
Уникальный идентификатор задания, которое необходимо удалить из журнала заданий.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Удаление задания из журнала
```lua
local questId = 12345

avatar.DiscardQuest( questId )
```
