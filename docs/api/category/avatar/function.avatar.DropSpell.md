# avatar.DropSpell
Сбрасывает подготовленное заклинание.

## Описание
```lua
avatar.DropSpell( spellId: SpellId )
```
Функция сбрасывает подготовленное заклинание, удаляя его из слота подготовки.

::: warning Замечание
Для сторонних аддонов существуют ограничения на использование данного метода:
1. Метод недоступен на PvP-активностях.
2. Метод работает исключительно в обработчиках реакции на действия мыши (кроме перемещения), то есть по нажатию кнопки мыши.
:::

## Список параметров
- **`spellId`** (`SpellId`)
Уникальный идентификатор заклинания, которое необходимо сбросить.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Сброс первого заклинания из книги заклинаний
```lua
local spellbook = avatar.GetSpellBook()
avatar.DropSpell( spellbook[ 1 ] )
```
::: info Описание примера
В примере с помощью функции `avatar.GetSpellBook()` извлекается таблица с доступными заклинаниями. Затем вызывается функция `avatar.DropSpell()` для сброса первого заклинания из полученного списка.
:::
