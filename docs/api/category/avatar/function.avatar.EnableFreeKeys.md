# avatar.EnableFreeKeys
Включает или отключает бескорыстную помощь.

## Описание
```lua
avatar.EnableFreeKeys( enable: boolean )
```
Функция управляет состоянием режима бескорыстной помощи. При передаче значения `true` режим активируется, при передаче `false` — деактивируется. Возвращаемое значение отсутствует.

## Список параметров
- **`enable`** (`boolean`)
Флаг состояния режима. Значение `true` включает бескорыстную помощь, значение `false` — выключает.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Включение бескорыстной помощи
```lua
avatar.EnableFreeKeys( true )
```
::: info Описание примера
Вызов функции с параметром `true` приводит к активации режима бескорыстной помощи.
:::
