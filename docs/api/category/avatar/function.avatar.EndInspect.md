# avatar.EndInspect
Завершает инспектирование цели главного игрока.

## Описание
```lua
avatar.EndInspect()
```
Завершает процесс инспектирования цели главного игрока. Возвращаемое значение отсутствует.

::: warning Замечание
Для сторонних аддонов существуют ограничения. Метод недоступен на PvP-активностях, в которых предусмотрена возможность голосовать за изгнание игрока. Для проверки доступности используется функция `avatar.IsInspectAllowed()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Завершение инспектирования цели
```lua
avatar.EndInspect()
```
