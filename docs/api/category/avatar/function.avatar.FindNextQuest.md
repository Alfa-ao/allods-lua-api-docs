# avatar.FindNextQuest
Запрашивает выдачу следующего доступного задания при пустом списке текущих заданий, отправляя аватар на встречу с соответствующим NPC.

## Описание
```lua
avatar.FindNextQuest( mode: number ): boolean
```
Проверяет наличие активных заданий и при их отсутствии инициирует запрос к серверу на поиск следующего доступного квеста. Возвращает логическое значение, указывающее на статус отправки запроса. Фактический результат запроса передается через событие `EVENT_QUEST_FIND_NEXT_RESULT`.

## Список параметров
- **`mode`** (`number`)
Определяет область поиска следующего задания. Принимает значения из перечисления `QUEST_FIND_NEXT_QUEST_MODE_...`, указывающие, где искать квест: в текущей зоне аватара или во всем игровом мире.

## Возвращаемые значения
**`boolean`**
Возвращает `false`, если клиент уже ожидает ответа от сервера на аналогичный запрос. Возвращает `true`, если запрос успешно отправлен.

## Примеры
### Поиск следующего квеста в текущей зоне
```lua
avatar.FindNextQuest( QUEST_FIND_NEXT_QUEST_MODE_ZONE )
```
