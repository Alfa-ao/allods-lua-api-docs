# avatar.GetAbilityInfo
Возвращает информацию об умении по идентификатору его ресурса.

## Описание
```lua
avatar.GetAbilityInfo( id: AbilityId ): table | nil
```
Возвращает таблицу с подробной информацией об умении по указанному идентификатору ресурса. Если умение с заданным идентификатором не найдено, возвращает `nil`. Основная реализация функции находится в логике игры, расширенная реализация (с дополнительными полями, такими как `texture`) предоставляется в UI.

::: warning Замечание
Поле `texture` в возвращаемой таблице доступно только при вызове функции в контексте UI.
:::

## Список параметров
- **`id`** (`AbilityId`)
Идентификатор ресурса умения.

## Возвращаемые значения
- `table` - Таблица с информацией об умении, если умение найдено.
- `nil` - Если умение с указанным идентификатором отсутствует.

Поля возвращаемой таблицы:
- `id` (`AbilityId`) - Идентификатор умения.
- `name` (`WString`) - Название умения.
- `description` (`ValuedText` | `nil`) - Описание с подставленными текущими значениями параметров.
- `level` (`number`) - Уровень, начиная с которого доступно изучение умения.
- `rank` (`number`) - Ранг умения (нумерация начинается с 1).
- `sysInfo` (`string`) - Системная информация для отладки (путь к файлу).
- `hasReplacementSpell` (`boolean`) - Признак наличия подменяющего заклинания (`true`, если подменяющее заклинание есть).
- `texture` (`TextureId`) - Идентификатор текстуры для иконки умения (реализовано только для UI).

## Примеры
### Получение названия умения по идентификатору
```lua
local abilities = avatar.GetAbilities()
local abilityInfo = avatar.GetAbilityInfo( abilities[0] )
if abilityInfo then
  local name = abilityInfo.name
end
```
