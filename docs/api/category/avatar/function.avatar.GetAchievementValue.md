# avatar.GetAchievementValue
Возвращает количество очков, заработанных в событии.

## Описание
```lua
avatar.GetAchievementValue( ruleId: number ): number
```
Функция извлекает и возвращает количество очков, заработанных в событии. Возвращает целое число, представляющее собой итоговое количество очков.

## Список параметров
- **`ruleId`** (`number`)
Идентификатор события.

## Возвращаемые значения
`number` - количество очков, заработанных в событии.

## Примеры
### Получение значения очков за событие
```lua
local ruleId = 12345
local achievementValue = avatar.GetAchievementValue(ruleId)
```
