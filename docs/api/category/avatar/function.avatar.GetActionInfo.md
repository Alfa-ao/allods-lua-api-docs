# avatar.GetActionInfo

Возвращает информацию о действии, назначенном в указанный слот экшен-панели, и его базовые параметры.

## Описание

```lua
avatar.GetActionInfo( index )
```

Функция извлекает данные о действии, находящемся в слоте экшен-панели (панели быстрого доступа) по заданному индексу. Если слот пуст, возвращается `nil`. Если в слоте назначено действие, возвращается таблица с информацией о типе действия и идентификаторе объекта.

Индексация слотов начинается с нуля. Тип действия в возвращаемой таблице соответствует одной из предопределенных констант `ACTION_TYPE_*`:
- `ACTION_TYPE_SPELL` — заклинание или умение.
- `ACTION_TYPE_ITEM` — предмет из инвентаря.
- `ACTION_TYPE_MOUNT` — средство передвижения (маунт).
- `ACTION_TYPE_EMOTE` — эмоция.
- `ACTION_TYPE_MOUNT_METAMORPH` — метаморфоза средства передвижения.

## Список параметров

- **`index`** (`number`)
Индекс слота экшен-панели. Значение должно находиться в диапазоне от `0` до максимального количества слотов.

## Возвращаемые значения

Возвращает `table`:

- **`type`** (`number`)
Тип объекта в слоте. Значение соответствует одной из констант `ACTION_TYPE_*`.

- **`id`** (`ObjectId`)
Уникальный идентификатор объекта в слоте (например, идентификатор заклинания, предмета или эмоции).

Если слот пуст, возвращается `nil`.

## Примеры

### Получение информации о заклинании в слоте

```lua
local i = 0 -- Индекс первого слота экшен-панели
local actionInfo = avatar.GetActionInfo( i )

if actionInfo then
    if actionInfo.type == ACTION_TYPE_SPELL then
        local spellInfo = avatar.GetSpellObjectInfo( actionInfo.id )
        
        if spellInfo then
            common.LogInfo( "Заклинание в слоте " .. ( i + 1 ) .. ": " .. spellInfo.debugName )
        end
    end
end
```

::: info Описание примера
В примере запрашивается информация о действии в первом слоте экшен-панели (индекс `0`). Если слот не пуст (`actionInfo` не равен `nil`), проверяется тип действия. Если тип соответствует заклинанию (`ACTION_TYPE_SPELL`), извлекается дополнительная информация о заклинании по его идентификатору (`actionInfo.id`) с помощью функции `avatar.GetSpellObjectInfo`. При успешном получении данных в лог выводится отладочное имя заклинания.
:::
