# avatar.GetActiveBuild
Возвращает номер активной специализации аватара.

## Описание
```lua
avatar.GetActiveBuild(): number | nil
```
Функция извлекает и возвращает номер текущей активной специализации главного персонажа. Нумерация специализаций начинается с нуля. В случае возникновения ошибки или отсутствия данных функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — номер активной специализации аватара (нумерация начинается с 0). В случае ошибки или отсутствия данных возвращается `nil`.

## Примеры
### Получение и вывод номера активной специализации
```lua
local buildIndex = avatar.GetActiveBuild()
LogInfo( "Активная специализация: ", buildIndex )
```
