# avatar.GetActiveCraftingSkill
Возвращает идентификатор крафтового навыка, процесс создания предметов по которому в данный момент активен.

## Описание
```lua
avatar.GetActiveCraftingSkill(): SkillId | nil
```
Извлекает и возвращает идентификатор крафтового навыка, крафт по которому в настоящее время выполняется. Если в данный момент процесс крафта не происходит, функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
- `SkillId` — идентификатор ресурса крафтового навыка, если процесс крафта активен.
- `nil` — если в данный момент крафт не выполняется.

## Примеры
### Получение идентификатора активного крафта
```lua
local skillId = avatar.GetActiveCraftingSkill()
if skillId then
    LogInfo( "Активный крафт: ", skillId )
else
    LogInfo( "Крафт в данный момент не выполняется." )
end
```
