# avatar.GetAttackResult

Возвращает текстовое описание ошибки, возникшей при попытке совершения немагической атаки.

## Описание

```lua
avatar.GetAttackResult( code )
```

(HELP) где список ATTACK_FAILED_* ??

Функция извлекает и возвращает локализованное текстовое описание ошибки на основе переданного числового кода. Применяется в системе пользовательского интерфейса (UIState) для отображения причин неудачной попытки ближнего или физического боя. Если передан неизвестный код, функция возвращает стандартное описание, соответствующее `ATTACK_RESULT_UNKNOWN`.

::: warning Замечание
Функция обрабатывает исключительно коды ошибок немагических атак. Для магических атак и других типов взаимодействий применяются иные механизмы и перечисления.
:::

## Список параметров

- **`code`** (`number`)
Код ошибки атаки. Принимает значения из перечислений `ATTACK_FAILED_*` или `ATTACK_RESULT_UNKNOWN`.

## Возвращаемые значения

Возвращает `WString` — локализованную строку с описанием ошибки, соответствующей указанному коду.

## Примеры

### Получение и вывод текста ошибки атаки

```lua
local errorCode = ATTACK_FAILED_TARGET_TOO_FAR
local errorText = avatar.GetAttackResult( errorCode )

if errorText and not errorText:IsEmpty() then
    local displayText = userMods.FromWString( errorText )
    LogInfo( "Ошибка атаки: ", displayText )
end
```

::: info Описание примера
В примере передается код ошибки `ATTACK_FAILED_TARGET_TOO_FAR`. Функция `avatar.GetAttackResult` возвращает объект `WString` с локализованным текстом ошибки. Для вывода в стандартный лог объект `WString` предварительно преобразуется в обычную строку с помощью `userMods.FromWString`. Проверка текста `not errorText:IsEmpty()` не является пустым.
:::
