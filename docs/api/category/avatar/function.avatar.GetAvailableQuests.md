# avatar.GetAvailableQuests
Извлекает от собеседника список заданий, которые он может выдать персонажу.

## Описание
```lua
avatar.GetAvailableQuests(): table
```
Функция возвращает таблицу с идентификаторами квестов, которые текущий собеседник может выдать персонажу. Возвращаемые данные формируются на основе текущего состояния диалога и доступны только после предварительного запроса взаимодействий.

::: warning Замечание
Функция вызывается строго после отправки запроса на взаимодействие через `avatar.RequestInteractions()`. Обращение к функции до завершения запроса взаимодействий может привести к получению некорректных или пустых данных.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую идентификаторы (`QuestId`) доступных для выдачи квестов.

## Примеры
### Получение и проверка списка доступных квестов
```lua
local availableQuestList = avatar.GetAvailableQuests()
if availableQuestList then
    -- Обработка списка доступных квестов
end
```
