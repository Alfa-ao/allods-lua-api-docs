# avatar.GetBarriersDelay
Возвращает начальную длительность действия барьеров класса «паладин» в миллисекундах.

## Описание
```lua
avatar.GetBarriersDelay(): number
```
Функция возвращает значение начальной длительности действия барьеров, специфичных для класса «паладин». Возвращаемое значение представляет собой временной интервал в миллисекундах, определяющий базовое время до момента разрешения эффекта барьера.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — начальную длительность действия барьеров в миллисекундах.

## Примеры
### Получение длительности барьеров
```lua
local delayMs = avatar.GetBarriersDelay()
```
