# avatar.GetBarriersInfo
Возвращает информацию об активных барьерах главного персонажа (паладина).

## Описание
```lua
avatar.GetBarriersInfo(): table
```
Функция извлекает и возвращает таблицу с данными о текущих барьерах, установленных главным персонажем (классом паладин). Возвращаемая таблица индексирована с нуля до количества активных барьеров минус один. Каждый элемент представляет собой вложенную таблицу, содержащую числовые параметры конкретного барьера: величину накопленного урона и оставшееся время действия в миллисекундах.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), индексированную от `0` до `BarriersCount - 1`. Значениями элементов являются таблицы со следующими полями:
- **`damage`** (`number`) — величина текущего накопленного урона в барьере.
- **`remainingTimeMs`** (`number`) — оставшееся время до разрешения барьера в миллисекундах.

## Примеры
### Получение и вывод информации о барьерах
```lua
local barriers = avatar.GetBarriersInfo()
for i = 0, GetTableSize( barriers ) - 1 do
    local barrier = barriers[i]
    common.LogInfo( "common", "damage: " .. barrier.damage .. ", time (ms): " .. barrier.remainingTimeMs )
end
```
::: info Описание примера
В примере извлекается таблица с информацией о всех активных барьерах. Затем в цикле происходит перебор всех элементов таблицы от `0` до размера таблицы минус один. Для каждого барьера выводится информация о накопленном уроне и оставшемся времени действия в миллисекундах в информационный лог.
:::
