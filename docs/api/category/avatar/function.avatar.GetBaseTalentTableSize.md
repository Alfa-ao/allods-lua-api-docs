# avatar.GetBaseTalentTableSize
Возвращает размер базовой таблицы талантов.

## Описание
```lua
avatar.GetBaseTalentTableSize(): table
```
Функция извлекает и возвращает размеры базовой таблицы талантов. Возвращаемые данные включают количество строк и столбцов, представленные в виде числовых значений внутри таблицы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`):
- **`layersCount`** (`number`)
Количество строк в базовой таблице талантов.
- **`linesCount`** (`number`)
Количество столбцов в базовой таблице талантов.

## Примеры
### Получение размера базовой таблицы талантов
```lua
local size = avatar.GetBaseTalentTableSize()
```
