# avatar.GetBonusPools
Возвращает список бонусов развития главного игрока.

## Описание
```lua
avatar.GetBonusPools(): table
```
Функция извлекает и возвращает индексированный с нуля список таблиц, описывающих бонусы развития. В возвращаемом списке может присутствовать только один элемент, описывающий бонус конкретного типа. Строковые представления типов бонусов соответствуют значениям перечисления `ENUM_DevelopmentTrack_...`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированный с нуля список таблиц, описывающих определённый бонус. Каждая вложенная таблица содержит следующие поля:
- **`type`** (`number`) — тип бонуса. Значение соответствует перечислению `ENUM_DevelopmentTrack_...`.
- **`sysType`** (`string`) — строковое представление типа бонуса (например, `"ENUM_DevelopmentTrack_..."`).
- **`total`** (`number`) — общее количество бонусов данного типа.
- **`left`** (`number`) — количество оставшихся бонусов данного типа.

## Примеры
### Вычисление процента оставшихся бонусов
```lua
local bonuses = avatar.GetBonusPools()
for i = 0, GetTableSize( bonuses ) - 1 do
    local bonusInfo = bonuses[ i ]
    local percent = bonusInfo.left / bonusInfo.total * 100
end
```
::: info Описание примера
В примере извлекается список бонусов развития с помощью `avatar.GetBonusPools`. Затем в цикле перебирается каждый элемент списка с использованием `GetTableSize`, и вычисляется процент оставшихся бонусов от их общего количества.
:::
