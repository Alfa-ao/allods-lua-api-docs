# avatar.GetBuffValuedObject
Возвращает новый экземпляр ValuedObject для указанного бафа.

## Описание
```lua
avatar.GetBuffValuedObject( buffId: BuffId ): ValuedObject
```
Функция создает и возвращает новый экземпляр `ValuedObject`, связанный с заданным бафом. 

Возвращаемый объект содержит следующие характеристики:
- Тип объекта устанавливается в значение `VAL_OBJ_TYPE_BUFF`.
- Текстовое представление содержит имя бафа.
- Изображение содержит иконку бафа.
- Идентификатор (`Id`) соответствует `BuffId` ресурса бафа.

::: warning Замечание
Поскольку `ValuedObject` реализован как `userdata`, среда Lua взаимодействует с такими объектами по ссылке, а не по значению. Это означает, что передача объекта в функции или присваивание его другой переменной не создает копию, а оперирует оригинальным экземпляром в памяти.
:::

## Список параметров
- **`buffId`** (`BuffId`)
Уникальный идентификатор ресурса бафа.

## Возвращаемые значения
Возвращает `ValuedObject` — экземпляр объекта, описывающего баф.

## Примеры
### Получение объекта бафа
```lua
local valuedObject = avatar.GetBuffValuedObject( buffId )
```
