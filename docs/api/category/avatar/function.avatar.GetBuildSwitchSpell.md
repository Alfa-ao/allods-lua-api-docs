# avatar.GetBuildSwitchSpell
Возвращает идентификатор заклинания, переключающего специализацию (билд) аватара.

## Описание
```lua
avatar.GetBuildSwitchSpell(): number
```
Функция извлекает и возвращает числовой идентификатор заклинания (спелла), которое используется для смены специализации (билда) персонажа. Полученный идентификатор применяется для запроса детальной информации об условиях и требованиях переключения билда через функции библиотеки заклинаний.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — числовой идентификатор заклинания (спелла), переключающего специализацию аватара.

## Примеры
### Получение и вывод требований для переключения билда
```lua
local spellId = avatar.GetBuildSwitchSpell()
if spellId then
    local requirements = spellLib.GetRequirements( spellId )
    LogInfo( "Требования для переключения билда: ", requirements )
end
```
::: info Описание примера
В примере сначала извлекается идентификатор заклинания переключения билда. Если идентификатор успешно получен, он передается в функцию `spellLib.GetRequirements` для извлечения списка требований к переключению специализации. Полученные требования выводятся в информационный лог.
:::
