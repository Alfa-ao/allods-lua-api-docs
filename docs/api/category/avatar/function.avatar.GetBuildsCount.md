# avatar.GetBuildsCount
Возвращает количество специализаций аватара.

## Описание
```lua
avatar.GetBuildsCount(): number
```
Функция извлекает общее количество сохраненных специализаций аватара. Возвращает целое число (`number`), представляющее количество специализаций. Значение может превышать текущее число доступных специализаций в случае, если их лимит когда-либо был временно увеличен, а затем вновь уменьшен. Сохраненные данные таких специализаций остаются доступными, но активировать их будет нельзя.

::: warning Замечание
Если количество сохраненных специализаций превышает текущий доступный лимит, активация лишних специализаций будет заблокирована, несмотря на наличие их сохраненных данных.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — общее количество сохраненных специализаций аватара.

## Примеры
### Вывод количества специализаций в лог
```lua
local buildsCount = avatar.GetBuildsCount()
LogInfo( buildsCount )
```
