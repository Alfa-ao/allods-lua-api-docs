# avatar.GetBuildsUnlockedCount
Возвращает количество доступных специализаций аватара.

## Описание
```lua
avatar.GetBuildsUnlockedCount(): number
```
Функция возвращает количество открытых специализаций для текущего аватара. Возвращает целое число.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — количество доступных специализаций аватара.

## Примеры
### Вывод количества доступных специализаций в лог
```lua
LogInfo( avatar.GetBuildsUnlockedCount() )
```
