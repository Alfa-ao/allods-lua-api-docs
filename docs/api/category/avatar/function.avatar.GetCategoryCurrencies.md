# avatar.GetCategoryCurrencies
Возвращает список идентификаторов валют главного игрока, относящихся к заданной категории.

## Описание
```lua
avatar.GetCategoryCurrencies( categoryId: CurrencyCategoryId ): table
```
Функция извлекает и возвращает информацию о валютах главного игрока, содержащихся в указанной категории. Возвращаемый список идентификаторов отсортирован по названию валюты в алфавитном порядке. Индексация элементов в возвращаемой таблице начинается с нуля.

## Список параметров
- **`categoryId`** (`CurrencyCategoryId`)
Идентификатор категории валюты, для которой запрашивается список доступных ресурсов.

## Возвращаемые значения
Возвращает таблицу (`table`), индексированную с нуля, содержащую идентификаторы (`CurrencyId`) имеющихся у игрока валют в указанной категории.

## Примеры
### Получение информации о первой валюте в категории
```lua
local categoryId = 1 -- Идентификатор категории валюты
local ids = avatar.GetCategoryCurrencies( categoryId )
if ids and ids[0] then
  local info = avatar.GetCurrencyInfo( ids[0] )
  if info then
    local currentValue = info.value
  end
end
```
::: info Описание примера
Запрашивается список идентификаторов валют для заданной категории. Если таблица не пуста и содержит хотя бы один элемент с индексом `0`, извлекается подробная информация о первой валюте через `avatar.GetCurrencyInfo`. Далее из полученной таблицы извлекается текущее количество ресурса.
:::
