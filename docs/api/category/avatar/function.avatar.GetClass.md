# avatar.GetClass
Возвращает системное имя класса аватара.
## Описание
```lua
avatar.GetClass(): string
```
Функция извлекает и возвращает системное имя класса, к которому принадлежит текущий аватар. Возвращает строковое значение, представляющее собой внутренний идентификатор класса персонажа.
## Список параметров
Параметры отсутствуют.
## Возвращаемые значения
Возвращает `string` — системное имя класса аватара.
## Примеры
### Получение и вывод имени класса аватара
```lua
local sysClassName = avatar.GetClass()
LogInfo( "sysClassName: ", sysClassName )
```
