# avatar.GetClassChangeData
Возвращает информацию, связанную со сменой класса персонажа.

## Описание
```lua
avatar.GetClassChangeData(): table
```
Функция возвращает таблицу с данными, касающимися смены класса. Возвращаемая таблица содержит идентификатор заклинания, позволяющего бесплатно сменить класс, что актуально при работе с окном талантов.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую информацию о смене класса:
- **`freeClassChangeSpell`** (`resourceId`) — идентификатор заклинания для смены класса (для окна талантов).

## Примеры
### Получение и вывод идентификатора заклинания смены класса
```lua
local classChangeData = avatar.GetClassChangeData()
LogInfo( "Change class spell:", classChangeData.freeClassChangeSpell )
```
