# avatar.GetClassChangerInfo
Возвращает информацию для смены класса аватара, включая варианты замены экипировки.
## Описание
```lua
avatar.GetClassChangerInfo(): table | nil
```
Функция извлекает и возвращает данные, необходимые для завершения процедуры смены класса аватара. Имеет смысл вызывать, только если аватар находится в соответствующем состоянии, которое определяется по приходящему событию `EVENT_CHARACTER_CLASS_CHANGER_CHANGED`, когда класс для замены уже выбран (для выбора используется `ClassSelectQuestion`), и требуется уточнить детали замены экипировки. Возвращает таблицу с данными о замене класса или `nil`, если аватар не находится в фазе смены класса.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` | `nil`:
- `nil` — если аватар не находится в соответствующей фазе смены класса.
- `table` — таблица с данными о замене класса, содержащая следующие поля:
  - **`charClassId`** (`ResourceId`) — идентификатор ресурса CharacterClass, указывающий на класс, на который производится замена.
  - **`substs`** (`table` | `nil`) — список возможных замен для имеющихся у аватара вещей. `nil`, если таких вещей нет. Иначе список таблиц с полями:
    - `itemId` (`ObjectId`) — идентификатор имеющейся у аватара вещи, которая будет заменена.
    - `itemSubstitutions` (`table`) — список описаний возможных замен этой вещи (для последующего выбора одной из них через `avatar.SetClassChangerInfo`). Список таблиц с полями:
      - `itemSubstituteParts` (`table`) — список замен, которые могут быть выданы взамен имеющейся вещи (или отобраны вместе с ней). Каждый элемент списка является таблицей-списком, из которого выбирается одна замена. Каждый элемент вложенного списка является таблицей с полями:
        - `substItemId` (`ObjectId`) — идентификатор вещи, которая будет выдана или отобрана. При выборе этой замены идентификатор передается в `avatar.SetClassChangerInfo` как элемент списка `changeList`.
        - `isRemoved` (`boolean`) — если `false`, вещь выдается аватару; если `true`, вещь отбирается у аватара.
        - `enchantsSubstitutions` (`table`) — список замен самоцветов для всех самоцветов, вставленных в заменяемую вещь и подлежащих замене. Список таблиц с полями:
          - `sourceEnchantItemId` (`ObjectId`) — идентификатор самоцвета, который будет заменён.
          - `oneOfEnchantSubstitution` (`table`) — список идентификаторов (`ObjectId`) самоцветов, один из которых необходимо выбрать на замену (передается в `avatar.SetClassChangerInfo`).

## Примеры
### Получение информации о замене класса
```lua
local info = avatar.GetClassChangerInfo()
```
