# avatar.GetClassId
Возвращает идентификатор класса главного персонажа.

## Описание
```lua
avatar.GetClassId(): CharacterClassId
```
Функция запрашивает и возвращает уникальный числовой идентификатор класса, к которому принадлежит главный персонаж. Возвращаемое значение соответствует типу `CharacterClassId` и используется для определения игровой роли персонажа, а также для применения специфичной для данного класса логики в аддоне.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `CharacterClassId` (`number`) — числовой идентификатор класса главного персонажа.

## Примеры
### Получение идентификатора класса
```lua
local characterClassId = avatar.GetClassId()
```
