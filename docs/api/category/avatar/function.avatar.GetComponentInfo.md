# avatar.GetComponentInfo
Возвращает описание крафтового компонента по идентификатору ресурса.

## Описание
```lua
avatar.GetComponentInfo( id: ComponentPropertyId ): table | nil
```
Функция извлекает и возвращает детальную информацию о крафтовом компоненте на основе переданного идентификатора ресурса. Возвращает таблицу с данными компонента, если он найден, или `nil`, если компонент с указанным идентификатором отсутствует.

::: warning Замечание
Поле `image` в возвращаемой таблице предназначено исключительно для нужд пользовательского интерфейса и содержит идентификатор текстуры иконки компонента.
:::

## Список параметров
- **`id`** (`ComponentPropertyId`)
Уникальный идентификатор ресурса компонента.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если компонент найден, возвращается таблица со следующими полями:
- **`id`** (`ComponentPropertyId`) — идентификатор ресурса компонента.
- **`name`** (`WString`) — название компонента.
- **`description`** (`WString`) — описание компонента.
- **`image`** (`TextureId`) — идентификатор текстуры для иконки компонента (используется в UI).

Если компонент не найден по идентификатору, возвращается `nil`.

## Примеры
### Получение и вывод названий компонентов алхимического барабана
```lua
local drumInfo = avatar.GetAlchemyDrumInfo( drum )
local components = drumInfo.components
for i = 0, GetTableSize( components ) - 1 do
    local componentInfo = avatar.GetComponentInfo( components[ i ] )
    if componentInfo then
        LogInfo( userMods.FromWString( componentInfo.name ) )
    end
end
```
::: info Описание примера
В примере сначала извлекается информация об алхимическом барабане. Затем в цикле перебирается список идентификаторов компонентов, входящих в его состав. Для каждого идентификатора вызывается функция `avatar.GetComponentInfo`. Если компонент успешно найден, его название выводится в информационный лог.
:::
