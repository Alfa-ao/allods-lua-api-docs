# avatar.GetContextActionShortInfo
Возвращает краткую информацию о контекстном действии.

## Описание
```lua
avatar.GetContextActionShortInfo( id: ObjectId ): table | nil
```
Функция извлекает и возвращает краткую информацию о контекстном действии по его идентификатору. Контекстное действие может присутствовать в списке действий аватара, но при этом быть недоступным (например, если цель не находится в зоне действия или вне полусферы игрока). Возвращает таблицу с данными о действии или `nil`, если действие не найдено. В случае изменения доступности действия, на клиент приходит событие `EVENT_CONTEXT_ACTIONS_CHANGED`.

::: warning Замечание
Для получения более полной информации о контекстном действии используется функция `avatar.GetContextActionInfo`.
:::

::: tip Совет
Реализация функции разделена: основная часть находится в логике игры, расширенная — в пользовательском интерфейсе.
:::

## Список параметров
- **`id`** (`ObjectId`)
Идентификатор контекстного действия.

## Возвращаемые значения
Возвращает таблицу (`table`):
- **`image`** (`TextureId`) — идентификатор текстуры для иконки (реализовано только для UI).
- **`enabled`** (`boolean`) — значение `true`, если действие доступно, и `false`, если недоступно.
- **`sysType`** (`string`) — название типа контекстного действия (строковый enum `ENUM_CONTEXT_ACTION_TYPE_XXX`).
- **`objectId`** (`ObjectId` | `nil`) — идентификатор объекта, к которому применяется контекстное действие, если он существует.
- **`cooldown`** (`table` | `nil`) — информация о кулдауне, если она доступна. Таблица содержит следующие поля:
  - `durationMs` (`number`) — длительность кулдауна в миллисекундах.
  - `remainingMs` (`number`) — оставшееся время кулдауна в миллисекундах.

или `nil`, если действие не найдено.

## Примеры
### Получение информации о доступности действия
```lua
local actionInfo = avatar.GetContextActionShortInfo( id )
if actionInfo then
  local enabled = actionInfo.enabled
end
```
