# avatar.GetCurrencies
Возвращает список идентификаторов альтернативных валют, имеющихся у игрока.

## Описание
```lua
avatar.GetCurrencies()
```
Функция извлекает и возвращает информацию об альтернативных валютах и их количестве у главного персонажа. Основная реализация метода находится в логике игры (UIState/GameState), а расширенная — в пользовательском интерфейсе. Возвращает таблицу, индексированную с нуля, содержащую идентификаторы имеющихся валют.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированную с 0 таблицу, содержащую идентификаторы имеющихся у игрока валют. Элементы таблицы поддерживают вызов метода `GetInfo()` для получения детальной информации о конкретной валюте.

## Примеры
### Получение информации о первой доступной валюте
```lua
local ids = avatar.GetCurrencies()
if ids and ids[0] then
  local info = ids[0]:GetInfo()
  if info then
    local currentValue = info.value
  end
end
```
::: info Описание примера
В примере запрашивается список всех доступных альтернативных валют. Если таблица не пуста и содержит хотя бы один элемент (с индексом `0`), извлекается подробная информация о первой валюте через метод `GetInfo()`. При успешном получении информации считывается её текущее количество (`value`).
:::
