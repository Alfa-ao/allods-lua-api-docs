# avatar.GetCurrencyCategories
Возвращает список идентификаторов категорий валют, имеющихся у главного персонажа.

## Описание
```lua
avatar.GetCurrencyCategories(): table
```
Функция извлекает и возвращает таблицу с идентификаторами категорий валют, находящихся в распоряжении главного персонажа. Возвращаемый список отсортирован по системному названию категории валюты. Для работы с полученными идентификаторами применяется тип `CurrencyCategoryId`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированную с нуля таблицу, содержащую идентификаторы (`CurrencyCategoryId`) ресурсов категорий имеющихся валют.

## Примеры
### Получение и обработка списка категорий валют
```lua
local ids = avatar.GetCurrencyCategories()
if ids and ids[0] then
  local name = ids[0]:GetInfo().name
end
```
::: info Описание примера
Запрашивается список идентификаторов категорий валют. При наличии хотя бы одной категории извлекается системное имя первой категории через вызов метода `GetInfo()`.
:::
