# avatar.GetCurrencyDescription
Возвращает текстовое описание альтернативной валюты с подставленными актуальными значениями параметров.

## Описание
```lua
avatar.GetCurrencyDescription( currencyId: CurrencyId ): ValuedText | nil
```
Функция извлекает шаблон описания альтернативной валюты и формирует итоговый текст, подставляя в него текущие значения параметров. Возвращает объект `ValuedText` или `nil`, если валюта с указанным идентификатором не существует.

## Список параметров
- **`currencyId`** (`CurrencyId`)
Идентификатор ресурса альтернативной валюты.

## Возвращаемые значения
Возвращает `ValuedText` или `nil`. Описание валюты в виде объекта `ValuedText` с подставленными текущими значениями параметров, либо `nil`, если указанный идентификатор некорректен или ресурс не найден.

## Примеры
### Получение описания первой валюты из категории
```lua
local ids = avatar.GetCategoryCurrencies( categoryId )
if ids and ids[0] then
  local description = avatar.GetCurrencyDescription( ids[0] )
end
```
::: info Описание примера
Сначала извлекается таблица идентификаторов валют для заданной категории. При условии, что таблица существует и содержит хотя бы один элемент с индексом `0`, вызывается функция получения описания для этого идентификатора.
:::
