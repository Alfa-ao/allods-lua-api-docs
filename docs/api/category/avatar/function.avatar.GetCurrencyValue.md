# avatar.GetCurrencyValue
Возвращает количество указанной альтернативной валюты у главного персонажа.

## Описание
```lua
avatar.GetCurrencyValue( currencyId: CurrencyId | nil ): table | nil
```
Функция извлекает и возвращает информацию о количестве заданной альтернативной валюты у главного персонажа. Возвращает таблицу с текущим и максимальным значением ресурса или `nil` в случае ошибки.

::: tip Совет
Перед использованием возвращаемой таблицы необходимо проверять её на `nil`, чтобы избежать ошибок выполнения при отсутствии валюты или возникновении ошибок на стороне клиента.
:::

## Список параметров
- **`currencyId`** (`CurrencyId` | `nil`)
Идентификатор ресурса альтернативной валюты.

## Возвращаемые значения
- **`table`** | **`nil`** — таблица с информацией о валюте или `nil` в случае ошибки.

Поля возвращаемой таблицы:
- **`value`** (`number`) — текущее количество валюты у игрока.
- **`maxValue`** (`number` | `nil`) — максимально возможное количество валюты, если такой лимит задан системой.

## Примеры
### Получение и анализ количества альтернативной валюты
```lua
local myrrhId = avatar.GetCurrencyId( "myrrh" )
if myrrhId then
  local currencyInfo = avatar.GetCurrencyValue( myrrhId )
  if currencyInfo then
    local currentValue = currencyInfo.value
    local maxValue = currencyInfo.maxValue
    LogInfo( "Текущее количество: ", currentValue, " / Максимум: ", maxValue )
  end
end
```
::: info Описание примера
В примере сначала получается идентификатор валюты "мирра" через `avatar.GetCurrencyId`. Если идентификатор успешно получен, вызывается `avatar.GetCurrencyValue` для извлечения данных. Из полученной таблицы извлекаются текущее (`value`) и максимальное (`maxValue`) значения ресурса, после чего выводятся в информационный лог.
:::
