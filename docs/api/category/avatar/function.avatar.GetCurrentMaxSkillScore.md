# avatar.GetCurrentMaxSkillScore
Извлекает максимально возможный уровень навыка для текущего уровня главного персонажа.

## Описание
```lua
avatar.GetCurrentMaxSkillScore( skillId: ObjectId ): number
```
Функция запрашивает и возвращает предельное значение уровня навыка, которое может быть достигнуто главным персонажем на его текущем уровне развития. Полученное значение применяется для ограничения прокачки или отображения доступного потенциала навыка в интерфейсе.

## Список параметров
- **`skillId`** (`ObjectId`)
Уникальный идентификатор навыка, для которого определяется максимально возможный уровень.

## Возвращаемые значения
Возвращает `number` — целое число, представляющее максимально возможный уровень навыка для текущего уровня главного персонажа.

## Примеры
### Получение предельного уровня навыка
```lua
local skillId = 123 -- Идентификатор проверяемого навыка
local maxAvatarScore = avatar.GetCurrentMaxSkillScore( skillId )

if maxAvatarScore then
    LogInfo( "Максимально возможный уровень навыка: ", maxAvatarScore )
end
```
