# avatar.GetDestinyPoints
Возвращает информацию об очках судьбы персонажа.
## Описание
```lua
avatar.GetDestinyPoints(): table
```
Функция возвращает таблицу с информацией об очках судьбы персонажа. Возвращаемые данные включают общее количество накопленных очков и состояние бонусного пула.
::: warning Замечание
Если значение поля `bonus` больше `0`, имеется запас неактивированных очков судьбы. Если значение меньше `0`, присутствует бонусный запас, который автоматически переводит новые неактивированные очки судьбы в накопленные.
:::
## Список параметров
Параметры отсутствуют.
## Возвращаемые значения
Возвращает таблицу (`table`):
- **`total`** (`number`) — накопленные очки судьбы персонажа.
- **`bonus`** (`number`) — бонусный пул очков судьбы.
## Примеры
### Получение информации об очках судьбы
```lua
local destinyPoints = avatar.GetDestinyPoints()
```
