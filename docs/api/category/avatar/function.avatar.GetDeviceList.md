# avatar.GetDeviceList
Извлекает список всех устройств, находящихся в зоне действия аватара.

## Описание
```lua
avatar.GetDeviceList(): table
```
Функция возвращает таблицу с идентификаторами всех видимых устройств, расположенных в зоне действия аватара. Индексация элементов в возвращаемой таблице начинается с единицы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую идентификаторы (`ObjectId`) видимых устройств. Индексация элементов в таблице начинается с единицы.

## Примеры
### Получение и перебор списка видимых устройств
```lua
local devices = avatar.GetDeviceList()
for _, value in ipairs( devices ) do
    LogInfo( "id: " .. tostring( value ) )
end
```
