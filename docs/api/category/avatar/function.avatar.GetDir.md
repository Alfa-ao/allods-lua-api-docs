# avatar.GetDir
Возвращает текущий угол направления аватара в радианах.

## Описание
```lua
avatar.GetDir(): number
```
Функция извлекает и возвращает числовое значение, представляющее собой текущий угол направления аватара в радианах. Функция не принимает параметров и возвращает угол поворота модели персонажа относительно мировых координат.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — угол направления аватара в радианах.

## Примеры
### Получение направления аватара
```lua
local dir = avatar.GetDir()
```
