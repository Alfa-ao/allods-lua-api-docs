# avatar.GetDressSlotInfo
Возвращает стандартную информацию о заданном слоте экипировки.

## Описание
```lua
avatar.GetDressSlotInfo( dressSlot: number ): table | nil
```
Извлекает и возвращает таблицу с базовыми характеристиками указанного слота экипировки. Если информация о слоте недоступна или передан некорректный идентификатор, возвращается `nil`. Список доступных слотов задается перечислением `DRESS_SLOT_...`.

## Список параметров
- **`dressSlot`** (`number`)
Номер слота экипировки. Значение задается с помощью констант перечисления `DRESS_SLOT_...`.

## Возвращаемые значения
Возвращает таблицу (`table`) с информацией о слоте экипировки или `nil`, если данные недоступны.
Таблица содержит следующие поля:
- **`minLevel`** (`number`) — минимальный уровень, с которого данный слот экипировки становится доступен персонажу.

## Примеры
### Получение и вывод информации о слоте шлема
```lua
local slotInfo = avatar.GetDressSlotInfo( DRESS_SLOT_HELM )
LogInfo( "Информация о слоте: ", slotInfo )
```
