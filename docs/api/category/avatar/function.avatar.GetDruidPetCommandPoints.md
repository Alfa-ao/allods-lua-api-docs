# avatar.GetDruidPetCommandPoints
Возвращает текущее значение и допустимые границы ресурса CommandPoints питомца друида.

## Описание
```lua
avatar.GetDruidPetCommandPoints(): table | nil
```
Функция извлекает информацию о ресурсе CommandPoints для активного питомца класса друид. Если у персонажа класса друид активен питомец, функция возвращает таблицу с текущим, минимальным и максимальным значениями ресурса. Если питомец отсутствует или класс аватара отличается от друида, функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Функция возвращает таблицу с информацией о ресурсе CommandPoints питомца или `nil`, если питомец отсутствует.

Если питомец друида активен, возвращается таблица (`table`) со следующими полями:
- **`value`** (`number`) — текущее значение ресурса CommandPoints.
- **`minValue`** (`number`) — минимальное значение (нижняя граница допустимого диапазона).
- **`maxValue`** (`number`) — максимальное значение (верхняя граница допустимого диапазона).

## Примеры
### Получение и проверка значения CommandPoints
```lua
local commandPoints = avatar.GetDruidPetCommandPoints()

if commandPoints ~= nil then
  LogInfo( "commandPoints = ", commandPoints.value )
end
```
