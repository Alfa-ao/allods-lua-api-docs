# avatar.GetEmoteInfo
Возвращает информацию об эмоции.

## Описание
```lua
avatar.GetEmoteInfo( emoteId: number ): table | nil
```
Функция извлекает и возвращает детальную информацию об эмоции по её уникальному идентификатору. Данные включают текстовые описания, системные имена, команды для чата, а также информацию о состоянии (возможность проигрывания, влияние на настроение). Часть возвращаемых данных (иконки) формируется на стороне UI.

::: warning Замечание
Если эмоция с указанным идентификатором не существует или не загружена, функция возвращает `nil`. Поля `image` и `image2` заполняются только при наличии соответствующей реализации в UI.
:::

## Список параметров
- **`emoteId`** (`number` | `ObjectId`)
Уникальный идентификатор эмоции, информацию о которой необходимо получить.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если эмоция найдена, возвращается таблица со следующими полями:
- **`id`** (`ObjectId`) — уникальный идентификатор объекта эмоции.
- **`canRun`** (`boolean`) — индикатор возможности проигрывания эмоции в текущий момент.
- **`hasMood`** (`boolean`) — `true`, если проигрывание эмоции изменяет настроение игрока.
- **`name`** (`WString`) — локализованное название эмоции.
- **`description`** (`WString`) — локализованное описание эмоции.
- **`aliases`** (`table` of `WString`) — таблица слэш-команд для активации эмоции через чат. Индексация элементов начинается с `1`.
- **`sysName`** (`string`) — системное название эмоции, совпадающее с идентификатором при запуске через чат.
- **`textTarget`** (`WString`) — локализованный текст, отображаемый в случае, когда эмоция направлена на конкретную цель.
- **`textNoTarget`** (`WString`) — локализованный текст, отображаемый в случае, когда эмоция не имеет цели.
- **`image`** (`TextureId` | `nil`) — идентификатор текстуры основной иконки эмоции (доступно только в UI).
- **`image2`** (`TextureId` | `nil`) — идентификатор текстуры дополнительной иконки эмоции (доступно только в UI).

## Примеры
### Получение и вывод системного имени эмоции
```lua
local emoteId = 12345
local emoteInfo = avatar.GetEmoteInfo( emoteId )
if emoteInfo then
  common.LogInfo( "Emote system name: " .. emoteInfo.sysName )
end
```
