# avatar.GetExpToLevel
Возвращает количество опыта, необходимое персонажу для достижения указанного уровня.

## Описание
```lua
avatar.GetExpToLevel( level: number ): number
```
Функция вычисляет и возвращает суммарное количество очков опыта, которое требуется персонажу для перехода на заданный уровень. Значение используется для оценки прогресса, отображения индикаторов опыта или проверки условий повышения уровня.

## Список параметров
- **`level`** (`number`)
Целочисленный идентификатор уровня, для которого необходимо получить требуемое количество опыта.

## Возвращаемые значения
Возвращает `number` — целое число, представляющее количество опыта, необходимое для достижения указанного уровня.

## Примеры
### Получение опыта для перехода на следующий уровень
```lua
local avatarId = avatar.GetId()
local currentLevel = unit.GetLevel( avatarId )
local expToNextLevel = avatar.GetExpToLevel( currentLevel + 1 )
```
