# avatar.GetExperience
Возвращает информацию об опыте персонажа.

## Описание
```lua
avatar.GetExperience(): table
```
Извлекает и возвращает таблицу с детальной информацией об опыте, уровнях и бонусах главного персонажа.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую следующие поля:
- **`currentExp`** (`number`) — текущий опыт персонажа.
- **`currentLevelExp`** (`number`) — опыт, необходимый для получения текущего уровня персонажа.
- **`nextLevelExp`** (`number`) — опыт, необходимый для получения следующего уровня персонажа.
- **`extraExp`** (`number`) — накопленный бонусный опыт, который может быть переведен в текущий (`currentExp`) при помощи «Фолианта знаний».
- **`extraExpCap`** (`number`) — максимально возможное значение накопленного бонусного опыта (`extraExp`).
- **`bonus`** (`number`) — множитель получаемого опыта (1.0 — опыт по умолчанию).
- **`pvpExp`** (`number`) — опыт, полученный за убийства других игроков.

## Примеры
### Получение информации об опыте персонажа
```lua
local theXp = avatar.GetExperience()
```
