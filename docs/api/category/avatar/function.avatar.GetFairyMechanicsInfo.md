# avatar.GetFairyMechanicsInfo
Возвращает информацию о механике покровителей.

## Описание
```lua
avatar.GetFairyMechanicsInfo(): table | nil
```
Функция возвращает детальную информацию о текущей механике покровителей, включая кастомное описание, затронутые группы заклинаний и модификаторы влияния. Если информация о механике отсутствует или не найдена, функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Функция возвращает `table` или `nil`. Если информация о механике покровителей найдена, возвращается таблица со следующей структурой:
- **`description`** (`WString`)
Кастомное описание механики покровителей. Может быть пустым.
- **`groups`** (`table`)
Индексированный с 0 список описаний групп заклинаний, на которые влияет покровитель. Каждая группа представлена таблицей со следующими полями:
- **`spells`** (`table`)
Индексированный с 0 список идентификаторов ресурсов заклинаний (`SpellId`) данной группы.
- **`entries`** (`table`)
Индексированный с 0 список описаний влияния покровителя на заклинания этой группы. Каждое описание представлено таблицей со следующими полями:
- **`fairyRank`** (`number`)
Уровень покровительства (целое число).
- **`immunityTimeMs`** (`number`)
Время, на которое игрок получает иммунитет, в миллисекундах (целое число).
- **`durationMod`** (`number`)
Модификатор длительности (число с плавающей точкой).

## Примеры
### Получение информации о заклинаниях первой группы
```lua
local info = avatar.GetFairyMechanicsInfo()
if info then
  local spellsOfFirstGroup = info.groups[0].spells
  if spellsOfFirstGroup then
    local spellId = spellsOfFirstGroup[0]
  end
end
```
::: info Описание примера
В примере запрашивается информация о механике покровителей. При успешном получении данных извлекается список идентификаторов заклинаний из первой группы. Если список заклинаний существует, извлекается идентификатор первого заклинания для дальнейшей обработки.
:::
