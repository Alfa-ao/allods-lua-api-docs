# avatar.GetFieldTalentTableSize
Возвращает размерность поля талантов.

## Описание
```lua
avatar.GetFieldTalentTableSize(): table
```
Функция возвращает размерность поля талантов, предоставляя информацию о количестве полей, строк и столбцов. Возвращаемые данные используются для определения структуры и габаритов сетки талантов.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`):
- **`fieldsCount`** (`number`)
Количество полей талантов.
- **`rowsCount`** (`number`)
Количество строк в поле талантов.
- **`columnsCount`** (`number`)
Количество столбцов в поле талантов.

## Примеры
### Получение и использование размерности поля талантов
```lua
local size = avatar.GetFieldTalentTableSize()
if size then
    LogInfo( "Полей: ", size.fieldsCount, ", Строк: ", size.rowsCount, ", Столбцов: ", size.columnsCount )
end
```
