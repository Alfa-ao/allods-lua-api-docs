# avatar.GetFreeStatPointsToDistribute
Возвращает количество доступных для распределения пунктов характеристик.

## Описание
```lua
avatar.GetFreeStatPointsToDistribute(): integer | nil
```
Функция возвращает количество доступных для распределения пунктов характеристик, которые еще не были намечены к распределению. Корректная работа возможна только при условии, что функция `avatar.CanImproveInnateStats()` возвращает `true`.

::: warning Замечание
Функция возвращает актуальные данные только при активном состоянии возможности улучшения врожденных характеристик (когда `avatar.CanImproveInnateStats()` возвращает `true`).
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`integer` — количество доступных для распределения пунктов характеристик. Если информация отсутствует, возвращает `nil`.

## Примеры
### Получение количества свободных очков характеристик
```lua
local points = avatar.GetFreeStatPointsToDistribute()
```
