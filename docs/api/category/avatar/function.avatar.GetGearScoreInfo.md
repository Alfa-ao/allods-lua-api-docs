# avatar.GetGearScoreInfo
Возвращает информацию о текущем рейтинге экипировки аватара, пороговых уровнях и связанных наградах.

## Описание
```lua
avatar.GetGearScoreInfo(): table | nil
```
Функция возвращает таблицу с данными о текущем рейтинге экипировки персонажа, а также информацию о пороговых значениях для получения пропусков в следующие секторы и открытия новых рецептов. Если данные недоступны, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`) или `nil`:
- **`currentValue`** (`number`) — текущее значение рейтинга экипировки.
- **`levelsPass`** (`table`) — таблица пороговых значений рейтинга, при достижении которых аватар получает пропуск в следующий сектор. Индексация таблицы начинается с 1.
- **`levelsRecipe`** (`table`) — таблица пороговых значений рейтинга, при достижении которых аватару открываются новые рецепты. Индексация таблицы начинается с 1.

## Примеры
### Получение и проверка рейтинга экипировки
```lua
local avatarGearScore = avatar.GetGearScoreInfo()
if avatarGearScore then
    local currentScore = avatarGearScore.currentValue
    -- Дальнейшая обработка текущего значения рейтинга
end
```
