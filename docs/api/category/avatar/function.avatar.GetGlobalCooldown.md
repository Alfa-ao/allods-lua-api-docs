# avatar.GetGlobalCooldown
Возвращает информацию о глобальном времени восстановления умений аватара.

## Описание
```lua
avatar.GetGlobalCooldown(): table
```
Функция возвращает данные о текущем глобальном времени восстановления (Global Cooldown) способностей аватара. Если в данный момент глобальное восстановление не активно, возвращается прогнозируемая длительность следующего восстановления и нулевое оставшееся время.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`) со следующими полями:
- **`duration`** (`int64`) — полная длительность текущего глобального времени восстановления в миллисекундах. Если текущий GCD отсутствует, содержит прогнозируемую длительность следующего.
- **`remaining`** (`int64`) — оставшееся время текущего глобального времени восстановления в миллисекундах. Если текущий GCD отсутствует, значение равно `0`.

## Примеры
### Получение данных о глобальном времени восстановления
```lua
local info = avatar.GetGlobalCooldown()
if info then
    LogInfo( "Длительность GCD: ", info.duration, " мс" )
    LogInfo( "Осталось времени: ", info.remaining, " мс" )
end
```
::: info Описание примера
В примере запрашивается таблица с информацией о глобальном времени восстановления. Если данные успешно получены, в информационный лог выводятся значения полной длительности и оставшегося времени в миллисекундах.
:::
