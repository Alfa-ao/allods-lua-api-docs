# avatar.GetGlossaryDescription
Возвращает описание ресурса глоссария с подставленными значениями.

## Описание
```lua
avatar.GetGlossaryDescription( glossaryId: GlossaryId ): ValuedText
```
Функция извлекает и возвращает текстовое описание ресурса глоссария (GlossaryResource) по его уникальному идентификатору. В возвращаемом тексте автоматически подставляются значения переменных (descVars), если они предусмотрены в ресурсе.

## Список параметров
- **`glossaryId`** (`GlossaryId`)
Уникальный идентификатор ресурса глоссария, описание которого необходимо получить.

## Возвращаемые значения
Возвращает `ValuedText` — отформатированное описание ресурса глоссария с подставленными значениями переменных.

## Примеры
### Получение описания ресурса глоссария
```lua
local glossaryId = 12345 -- Идентификатор ресурса глоссария
local valuedText = avatar.GetGlossaryDescription( glossaryId )
```
