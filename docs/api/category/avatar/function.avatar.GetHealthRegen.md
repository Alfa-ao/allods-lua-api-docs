# avatar.GetHealthRegen
Возвращает значения регенерации здоровья главного персонажа.

## Описание
```lua
avatar.GetHealthRegen(): table
```
Функция возвращает детальную информацию о регенерации здоровья главного персонажа, разбитую по источникам получения (базовое значение, таланты, экипировка, баффы), а также общее эффективное значение и длительность тика. Структура возвращаемой таблицы аналогична результатам функции `avatar.GetInnateStats()`, за исключением поля `sysName`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую следующие поля:
- **`base`** (`number`) — базовое значение регенерации за тик.
- **`talents`** (`number`) — величина регенерации за тик, получаемая от талантов.
- **`equipment`** (`number`) — величина регенерации за тик, получаемая от надетых на персонажа предметов экипировки.
- **`buffs`** (`number`) — величина регенерации за тик, получаемая от активных бафов.
- **`effective`** (`number`) — эффективное (текущее суммарное) значение регенерации за тик.
- **`tickMs`** (`number`) — длительность одного тика регенерации в миллисекундах.

## Примеры
### Получение и вывод значений регенерации здоровья
```lua
local healthRegen = avatar.GetHealthRegen()
LogInfo( "healthRegen base: ", healthRegen.base, " healthRegen effective: ", healthRegen.effective )
```
