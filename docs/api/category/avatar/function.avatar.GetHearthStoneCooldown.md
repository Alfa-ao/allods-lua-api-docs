# avatar.GetHearthStoneCooldown
Возвращает информацию о текущем времени восстановления (кулдауне) камня возвращения (HearthStone).

## Описание
```lua
avatar.GetHearthStoneCooldown(): table
```
Функция возвращает данные о времени восстановления камня возвращения, включая оставшееся время и общую длительность кулдауна в миллисекундах.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`):
- **`remainingMs`** (`number`)
Оставшееся время кулдауна в миллисекундах.
- **`durationMs`** (`number`)
Общая длительность кулдауна в миллисекундах.

## Примеры
### Получение информации о кулдауне камня возвращения
```lua
local cooldown = avatar.GetHearthStoneCooldown()
if cooldown.remainingMs > 0 then
    LogInfo( "До окончания кулдауна осталось: ", cooldown.remainingMs, " мс." )
else
    LogInfo( "Камень возвращения готов к использованию." )
end
```
