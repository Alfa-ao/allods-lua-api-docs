# avatar.GetHearthStoneSpell
Возвращает идентификатор заклинания, реализующего механику камня возвращения (HearthStone).

## Описание
```lua
avatar.GetHearthStoneSpell(): SpellId | nil
```
Функция возвращает идентификатор заклинания, посредством которого реализовано перемещение персонажа с использованием камня возвращения. Если заклинание не найдено или недоступно для текущего персонажа, функция возвращает `nil`.

::: warning Замечание
Возвращаемое заклинание не содержит информации о кулдауне (времени восстановления) камня возвращения. Для получения актуальных данных о кулдауне необходимо использовать функцию `avatar.GetHearthStoneCooldown()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `SpellId` — идентификатор заклинания камня возвращения, или `nil`, если заклинание не определено.

## Примеры
### Получение идентификатора заклинания камня возвращения
```lua
local spellId = avatar.GetHearthStoneSpell()
if spellId then
    -- Использование полученного идентификатора заклинания
end
```
