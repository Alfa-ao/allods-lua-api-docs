# avatar.GetId
Возвращает уникальный идентификатор аватара.

## Описание
```lua
avatar.GetId(): ObjectId | nil
```
Функция возвращает уникальный идентификатор текущего аватара игрока. Данный идентификатор используется для обращения к аватару и его данным в различных функциях API.

::: warning Замечание
В настоящее время при старте скриптов функция может возвращать `nil`. Рекомендуется проверять возвращаемое значение на `nil` перед его использованием.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `ObjectId` — уникальный идентификатор аватара.

## Примеры
### Получение идентификатора аватара
```lua
local id = avatar.GetId()
if id then
    LogInfo( "Идентификатор аватара: ", id )
else
    LogInfo( "Идентификатор аватара еще не инициализирован." )
end
```
