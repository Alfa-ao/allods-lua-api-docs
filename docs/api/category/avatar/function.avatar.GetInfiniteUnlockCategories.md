# avatar.GetInfiniteUnlockCategories
Возвращает список категорий постоянных возможностей, имеющихся у аватара.

## Описание
```lua
avatar.GetInfiniteUnlockCategories(): table
```
Функция возвращает таблицу с идентификаторами категорий постоянных возможностей, которые разблокированы у текущего аватара. Индексация элементов в возвращаемой таблице начинается с единицы. Если у аватара отсутствуют постоянные возможности, функция возвращает пустую таблицу.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Функция возвращает `table` — таблицу, индексированную с 1, содержащую идентификаторы (`UnlockCategoryId`) постоянных возможностей, имеющихся у аватара. Если возможности отсутствуют, возвращается пустая таблица.

## Примеры
### Перебор и вывод названий доступных постоянных возможностей
```lua
for _, categoryId in ipairs( avatar.GetInfiniteUnlockCategories() ) do
    local info = categoryId:GetInfo()
    LogInfo( info.name )
end
```
