# avatar.GetInfiniteUnlocksInCategory
Возвращает список идентификаторов постоянных возможностей (анлоков) главного игрока в указанной категории.

## Описание
```lua
avatar.GetInfiniteUnlocksInCategory( unlockCategoryId: UnlockCategoryId ): table<number, ObjectId>
```
Функция извлекает и возвращает индексированную с единицы таблицу, содержащую уникальные идентификаторы всех постоянных возможностей (анлоков), которые были разблокированы главным игроком в рамках заданной категории. Полученные данные позволяют анализировать прогресс игрока и определять доступный функционал в рамках конкретной группы возможностей.

## Список параметров
- **`unlockCategoryId`** (`UnlockCategoryId`)
Идентификатор типа категории возможностей (анлоков), для которой выполняется запрос списка разблокированных элементов.

## Возвращаемые значения
Возвращает таблицу (`table<number, ObjectId>`), индексированную с единицы, содержащую идентификаторы (`ObjectId`) постоянных возможностей (анлоков). Если в заданной категории у игрока отсутствуют разблокированные возможности, возвращается пустая таблица.

## Примеры
### Перебор всех категорий и вывод названий разблокированных возможностей
```lua
for _, unlockCategoryId in ipairs( avatar.GetInfiniteUnlockCategories() ) do
    LogInfo( unlockCategoryId:GetInfo().name )
    for _, unlockId in ipairs( avatar.GetInfiniteUnlocksInCategory( unlockCategoryId ) ) do
        local info = avatar.GetUnlockInfo( unlockId )
        LogInfo( info.name )
    end
end
```
::: info Описание примера
В примере выполняется обход всех доступных категорий постоянных возможностей с помощью `avatar.GetInfiniteUnlockCategories()`. Для каждой категории выводится её название, после чего извлекается список разблокированных возможностей через `avatar.GetInfiniteUnlocksInCategory()`. Вложенный цикл перебирает полученные идентификаторы, извлекает полную информацию о каждой возможности через `avatar.GetUnlockInfo()` и выводит её название в информационный лог.
:::
