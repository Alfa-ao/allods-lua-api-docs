# avatar.GetInnateStatDescription
Возвращает описание InnateStat характеристики аватара.

## Описание
```lua
avatar.GetInnateStatDescription( sysName: string ): ValuedText | nil
```
Функция извлекает и возвращает текстовое описание внутренней (InnateStat) характеристики аватара с подставленными в него актуальными числовыми значениями параметров. Для идентификации требуемой характеристики используется её служебное системное имя. Если характеристика не найдена или не имеет описания, функция возвращает `nil`.

## Список параметров
- **`sysName`** (`string`)
Служебное системное имя характеристики. Значение должно соответствовать одному из идентификаторов перечисления `ENUM_InnateStats` (например, `"ENUM_InnateStats_Will"`).

## Возвращаемые значения
Возвращает `ValuedText` — форматированное описание характеристики с подставленными текущими значениями параметров, или `nil`, если описание отсутствует или характеристика не найдена.

## Примеры
### Получение описания характеристики "Воля"
```lua
local willDesc = avatar.GetInnateStatDescription( "ENUM_InnateStats_Will" )
```
