# avatar.GetInnateStats
Возвращает основные характеристики аватара.

## Описание
```lua
avatar.GetInnateStats(): table
```
Функция возвращает таблицу, индексированную с нуля, содержащую основные (первичные) характеристики аватара. Индексы таблицы соответствуют значениям множества `ENUM_InnateStats`. Для каждой характеристики предоставляется подробная информация, включая базовые и эффективные значения, а также модификаторы от различных источников. Дополнительно рассчитываются производные (вторичные) характеристики (`InnateStatSecondary`). Если для первичной характеристики не предусмотрена производная, соответствующее значение вторичной характеристики равно нулю.

::: warning Замечание
Индексация возвращаемой таблицы начинается с нуля. Индексы элементов соответствуют значениям `ENUM_InnateStats`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), индексированную с нуля (`[0..]`). Элементы таблицы представляют собой вложенные таблицы со следующими полями:
- **`sysName`** (`string`)
Текстовый вариант перечисления `ENUM_InnateStats`.
- **`base`** (`number`)
Базовое значение характеристики.
- **`effective`** (`number`)
Эффективное (текущее) значение характеристики.
- **`talents`** (`number`)
Величина изменения характеристики от талантов.
- **`equipment`** (`number`)
Величина изменения характеристики от надетых на персонажа предметов.
- **`enchants`** (`number`)
Величина изменения характеристики от энчантов, вставленных в надетые предметы.
- **`isCursed`** (`boolean`)
Индикатор влияния проклятого предмета на характеристику.
- **`buffs`** (`number`)
Величина изменения характеристики от активных баффов.
- **`fairyBonus`** (`number`)
Бонус к характеристике от покровителя.
- **`secondary`** (`table`)
Таблица с описанием вторичных характеристик, рассчитанных на основе текущей первичной. Содержит поля:
  - **`current`** (`table`) — вторичные характеристики, рассчитанные на основе текущего эффективного значения.
  - **`next`** (`table`) — вторичные характеристики, рассчитанные на основе следующего значения (увеличенного на единицу от текущего эффективного).

## Примеры
### Получение и перебор основных характеристик аватара
```lua
local stats = avatar.GetInnateStats()
for i = 0, GetTableSize( stats ) - 1 do
    local stat = stats[i]

    local base = stat.base
    local effective = stat.effective
    local sysName = stat.sysName

    local secondaryN = stat.secondary.current.N
end
```
::: info Описание примера
В примере запрашивается таблица основных характеристик аватара. Затем в цикле происходит перебор всех элементов таблицы. Для каждой характеристики извлекаются базовое и эффективное значения, а также системное имя. Дополнительно демонстрируется доступ к вторичным характеристикам через поле `secondary.current`.
:::
