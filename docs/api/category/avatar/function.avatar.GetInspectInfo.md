# avatar.GetInspectInfo
Возвращает информацию о навыке игрока по осмотру чужих вещей и текущих просматриваемых объектах.

## Описание
```lua
avatar.GetInspectInfo(): table
```
Функция возвращает таблицу, содержащую данные о ранге навыка осмотра чужих вещей, а также идентификаторы текущего инспектируемого игрока или его маунта.

::: warning Замечание
Для сторонних аддонов существуют ограничения: метод недоступен на PvP-активностях, где предусмотрена возможность голосования за изгнание игрока. Перед использованием рекомендуется проверять доступность с помощью `avatar.IsInspectAllowed()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` со следующими полями:
- **`rank`** (`number` | `nil`)
Ранг навыка игрока по осмотру чужих вещей, если он доступен.
- **`playerId`** (`ObjectId` | `nil`)
Уникальный идентификатор инспектируемого игрока.
- **`mountId`** (`ObjectId` | `nil`)
Уникальный идентификатор маунта инспектируемого игрока.

## Примеры
### Получение информации об инспектируемом объекте
```lua
local info = avatar.GetInspectInfo()
if info then
    local rank = info.rank
    if info.mountId then
        local mountInfo = mount.GetInfo( info.mountId )
    end
end
```
::: info Описание примера
В примере извлекается таблица с информацией об осмотре. Если таблица успешно получена, считывается ранг навыка. При наличии идентификатора маунта (`mountId`) извлекается дополнительная информация о нем с помощью `mount.GetInfo()`.
:::
