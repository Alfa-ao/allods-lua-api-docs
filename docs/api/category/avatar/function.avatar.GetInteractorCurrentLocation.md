# avatar.GetInteractorCurrentLocation
Извлекает идентификатор точки телепортации, в которой находится мастер телепорта, с которым взаимодействует главный персонаж.

## Описание
```lua
avatar.GetInteractorCurrentLocation(): ObjectId | nil
```
Функция возвращает уникальный идентификатор точки телепортации, в которой в данный момент находится мастер телепорта, ведущий диалог с главным персонажем. Если главный персонаж не разговаривает с мастером телепорта или информация о его текущем местоположении недоступна, функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `ObjectId` — уникальный идентификатор точки телепортации, или `nil`, если информация о местоположении мастера телепорта отсутствует.

## Примеры
### Получение информации о текущей точке телепортации
```lua
local locationId = avatar.GetInteractorCurrentLocation()
if locationId then
    local locationInfo = avatar.GetTeleportLocationInfo( locationId )
    if locationInfo then
        local name = locationInfo.name
        -- Дальнейшая обработка имени точки телепортации
    end
end
```
::: info Описание примера
В примере сначала запрашивается идентификатор текущей точки телепортации мастера, с которым ведется диалог. Если идентификатор успешно получен, извлекается полная информация о локации с помощью функции `avatar.GetTeleportLocationInfo`. При успешном получении информации извлекается название точки телепортации для дальнейшего использования.
:::
