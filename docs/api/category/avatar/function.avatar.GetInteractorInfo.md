# avatar.GetInteractorInfo
Возвращает подробную информацию о текущем собеседнике при наличии активного диалога.

## Описание
```lua
avatar.GetInteractorInfo(): table | nil
```
Функция возвращает таблицу с актуальными данными о сущности, с которой главный персонаж ведет диалог или взаимодействие, либо `nil`, если собеседник отсутствует. Структура и набор полей возвращаемой таблицы полностью идентичны данным, предоставляемым функцией `object.GetInteractorInfo( id )`.

Для получения корректных и точных данных вызов функции необходимо осуществлять строго после того, как придет событие `EVENT_INTERACTION_STARTED`. В процессе активного разговора смена цели взаимодействия не поддерживается.

До начала диалога предварительный сбор сведений возможен через обращение к `object.GetInteractorInfo( id )`, однако полученные таким образом данные могут оказаться неточными или устаревшими до момента фактического начала взаимодействия.

::: warning Замечание
Функция возвращает гарантированно точные данные только после того, как придет событие `EVENT_INTERACTION_STARTED`. Попытка получения информации до инициации диалога может привести к получению некорректных сведений.
:::

## Список параметров
Функция не принимает параметров.

## Возвращаемые значения
Возвращает `table` с полями, описанными в `object.GetInteractorInfo( id )`, если собеседник присутствует, или `nil`, если активный диалог отсутствует.

## Примеры
### Получение информации о собеседнике и проверка типа
```lua
local info = avatar.GetInteractorInfo()
if info then
    local isMailBox = info.isMailBox
    -- Дальнейшая обработка в зависимости от типа собеседника
end
```
::: info Описание примера
В примере извлекается таблица с информацией о текущем собеседнике. При успешном получении данных (отсутствие `nil`) происходит чтение поля `isMailBox`, что позволяет определить, является ли собеседник почтовым ящиком, и выполнить соответствующую логику.
:::
