# avatar.GetInteractorNextCues
Извлекает список возможных вариантов ответа на текущую реплику NPC-собеседника.

## Описание
```lua
avatar.GetInteractorNextCues(): table
```
Функция извлекает и возвращает массив доступных реплик-ответов, которые игрок может выбрать в диалоге с NPC. Каждая возвращаемая реплика содержит полную информацию о следующем сообщении от NPC, которое будет получено в случае выбора данного варианта ответа. Структура возвращаемых таблиц идентична описанию в функции `avatar.GetInteractorCue()`.

::: warning Замечание
Вызов функции допустим только после того, как придет событие `EVENT_INTERACTION_STARTED`. Во время активного диалога запрещается изменять цель взаимодействия.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую список реплик (`Cue`). Индексация элементов в таблице начинается с нуля. Каждая реплика представлена таблицей, структура которой описана в функции `avatar.GetInteractorCue()`.

## Примеры
### Получение и обработка списка доступных ответов
```lua
local nextCues = avatar.GetInteractorNextCues()
if nextCues[0] then
  local answer0 = nextCues[0].name
end
```
