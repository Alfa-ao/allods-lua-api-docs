# avatar.GetInterlocutor
Возвращает идентификатор текущего собеседника аватара.

## Описание
```lua
avatar.GetInterlocutor(): ObjectId | nil
```
Функция возвращает идентификатор текущего собеседника аватара. Собеседником может являться устройство или NPC (см. CategoryObject). Если взаимодействие с собеседником не установлено, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`ObjectId` — уникальный идентификатор собеседника или `nil`, если собеседник отсутствует.

## Примеры
### Получение идентификатора собеседника
```lua
local unitId = avatar.GetInterlocutor()
```
