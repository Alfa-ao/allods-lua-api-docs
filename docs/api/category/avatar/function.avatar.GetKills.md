# avatar.GetKills
Возвращает количество PvP-убийств, совершённых аватаром за всю карьеру.

## Описание
```lua
avatar.GetKills(): number | nil
```
Функция возвращает целое число, представляющее количество PvP-убийств, совершённых аватаром за всю карьеру. Если информация на данный момент отсутствует, возвращается `nil`. При изменении количества убийств приходит событие `EVENT_VICTIM_TRACKER_CHANGED`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
- `number` — количество PvP-убийств, совершённых аватаром за всю карьеру.
- `nil` — если информация на данный момент отсутствует.

## Примеры
### Получение количества PvP-убийств
```lua
local kills = avatar.GetKills()
if kills then
    LogInfo( "Количество PvP-убийств: ", kills )
else
    LogInfo( "Информация о PvP-убийствах отсутствует." )
end
```
