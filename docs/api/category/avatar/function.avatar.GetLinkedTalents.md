# avatar.GetLinkedTalents
Извлекает связанные заклинания и способности для указанного идентификатора.

## Описание
```lua
avatar.GetLinkedTalents( object: SpellId|AbilityId ): table
```
Функция возвращает таблицу идентификаторов связанных заклинаний и способностей для переданного идентификатора. Предназначена для использования в интерфейсе окна талантов. Возвращаемая таблица индексируется с единицы.

## Список параметров
- **`object`** (`SpellId` | `AbilityId`)
Идентификатор заклинания или способности, для которого необходимо получить связанные элементы.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую идентификаторы (`SpellId` | `AbilityId`) связанных заклинаний и способностей. Индексация элементов в таблице начинается с единицы.

## Примеры
### Получение и перебор связанных талантов
```lua
local spellId = 12345 -- Идентификатор проверяемого заклинания
local linkedTalents = avatar.GetLinkedTalents( spellId )

if linkedTalents then
    for i = 1, #linkedTalents do
        local linkedId = linkedTalents[i]
        LogInfo( "Найден связанный талант с идентификатором: ", linkedId )
    end
else
    LogInfo( "Связанные таланты не найдены." )
end
```
::: info Описание примера
В примере запрашивается список связанных талантов для заклинания с идентификатором `12345`. Если таблица успешно получена, происходит её перебор с помощью цикла `for`, начиная с индекса `1`. На каждой итерации извлекается идентификатор связанного таланта и выводится в информационный лог.
:::
