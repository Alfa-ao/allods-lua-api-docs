# avatar.GetLootBagSlotCount
Возвращает общее количество слотов в открытой сумке с лутом, включая пустые.

## Описание
```lua
avatar.GetLootBagSlotCount(): number
```
Функция возвращает общее количество слотов в открытой сумке с лутом, включая пустые. В подсчет включаются все ячейки контейнера. Игровая валюта (деньги) не учитывается как слот и является отдельной сущностью.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — целое число, представляющее собой общее количество слотов в сумке.

## Примеры
### Получение количества слотов в сумке с лутом
```lua
local slotCount = avatar.GetLootBagSlotCount()
LogInfo( "Общее число слотов в сумке с лутом: ", slotCount )
```
