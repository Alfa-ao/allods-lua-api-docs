# avatar.GetLootBagSlots
Возвращает содержимое сумки с лутом, включая количество денег и список идентификаторов вещей.

## Описание
```lua
avatar.GetLootBagSlots(): table
```
Функция возвращает таблицу с количеством денег и списком идентификаторов вещей, находящихся в сумке с лутом. Список вещей индексируется от `0` до `avatar.GetLootBagSlotCount() - 1`. Для пустых слотов в списке вещей возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую следующие поля:
- **`money`** (`number`) — количество денег в сумке с лутом.
- **`items`** (`table`) — список идентификаторов вещей в сумке. Индексируется от `0` до `avatar.GetLootBagSlotCount() - 1`. Элементы списка имеют тип `ObjectId`, для пустых слотов возвращается `nil`.

## Примеры
### Получение содержимого сумки с лутом
```lua
local loot = avatar.GetLootBagSlots()
local lootMoney = loot.money
local lootTable = loot.items
```
::: info Описание примера
В примере запрашивается содержимое сумки с лутом. Из полученной таблицы извлекается количество денег (`loot.money`) и таблица идентификаторов предметов (`loot.items`).
:::
