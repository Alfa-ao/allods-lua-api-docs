# avatar.GetMainhandSpeed
Возвращает базовое и эффективное значения скорости атаки основным оружием аватара.

## Описание
```lua
avatar.GetMainhandSpeed(): table
```
Функция возвращает таблицу, содержащую базовое и эффективное (текущее) значения скорости атаки основным оружием аватара. Базовое значение представляет собой исходную скорость без учета модификаторов, а эффективное — итоговую скорость с учетом всех наложенных эффектов и экипировки.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table`:
- **`base`** (`number`) — базовое значение скорости атаки.
- **`effective`** (`number`) — эффективное (текущее) значение скорости атаки с учетом всех модификаторов.

## Примеры
### Получение и вывод значений скорости атаки
```lua
local speed = avatar.GetMainhandSpeed()
LogInfo( "speed base:" .. speed.base .. " speed effective:" .. speed.effective )
```
