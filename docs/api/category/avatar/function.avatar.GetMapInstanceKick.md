# avatar.GetMapInstanceKick
Возвращает информацию о причине и времени до выброса главного игрока из инстанса.

## Описание
```lua
avatar.GetMapInstanceKick(): table | nil
```
Функция возвращает данные о текущем счетчике времени до принудительного выброса главного персонажа из инстанса. Если счетчик отсутствует или не активен, функция возвращает `nil`.

::: warning Замечание
Значение поля `sysCause` в возвращаемой таблице представляет собой строку, соответствующую одному из предопределенных перечислений `ENUM_AvatarKickCause_*`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table`.
Если счетчик выброса активен, возвращается таблица со следующими полями:
- **`durationMs`** (`number`)
Общее время на выброс из инстанса в миллисекундах.
- **`remainingMs`** (`number`)
Оставшееся время до выброса из инстанса в миллисекундах.
- **`sysCause`** (`string`)
Причина выброса. Значение соответствует одному из перечислений `ENUM_AvatarKickCause_*`.

Возвращает `nil`, если счетчик выброса отсутствует.

## Примеры
### Получение и вывод оставшегося времени до выброса
```lua
local timeInfo = avatar.GetMapInstanceKick()
if timeInfo then
    LogInfo( "Оставшееся время до выброса из инстанса: ", timeInfo.remainingMs / 1000 )
end
```
