# avatar.GetMaxSkillScore
Возвращает максимально возможный уровень скилла в игре.

## Описание
```lua
avatar.GetMaxSkillScore( skillId: ObjectId ): number
```
Функция возвращает максимально возможный уровень развития конкретного скилла, доступный в игре. Полученное значение представляет собой системное ограничение для указанного навыка и используется для проверки предельных границ при прокачке или отображении характеристик персонажа.

## Список параметров
- **`skillId`** (`ObjectId`)
Уникальный идентификатор скилла, для которого необходимо получить предельное значение уровня.

## Возвращаемые значения
Возвращает `number` — целое число, представляющее собой максимально возможный уровень указанного скилла.

## Примеры
### Получение предельного уровня скилла
```lua
local skillId = 123 -- Идентификатор нужного скилла
local maxScore = avatar.GetMaxSkillScore( skillId )
```
