# avatar.GetMoney
Возвращает информацию о количестве денег у главного персонажа.

## Описание
```lua
avatar.GetMoney(): number
```
Функция возвращает текущее количество игровой валюты в инвентаре главного персонажа. Возвращаемое значение имеет тип `int64`, однако в текущей реализации игры фактическое ограничение составляет около 2 миллиардов (ограничение типа `int32`).

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` (`int64`) — количество денег у главного персонажа.

## Примеры
### Получение и вывод количества денег
```lua
local money = avatar.GetMoney()
LogInfo( "Количество денег: ", money )
```
