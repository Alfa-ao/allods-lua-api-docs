# avatar.GetNecromancerBloodPool
Извлекает и возвращает данные о пуле крови (BloodPool) некроманта.

## Описание
```lua
avatar.GetNecromancerBloodPool(): table
```
Функция возвращает таблицу, содержащую текущее значение пула крови, а также его минимальную и максимальную границы для персонажа-некроманта.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table`:
- **`value`** (`number`) — текущее значение пула крови.
- **`minValue`** (`number`) — минимальное значение (нижняя граница допустимого диапазона).
- **`maxValue`** (`number`) — максимальное значение (верхняя граница допустимого диапазона).

## Примеры
### Получение и вывод текущего значения пула крови
```lua
local bloodPool = avatar.GetNecromancerBloodPool()
LogInfo( "bloodPool = ", bloodPool.value )
```
