# avatar.GetNextLevelUnlockInfo
Возвращает описание возможности, необходимой для получения следующего уровня главного персонажа.

## Описание
```lua
avatar.GetNextLevelUnlockInfo(): table | nil
```
Функция возвращает таблицу с описанием возможности (анлока), необходимой для получения следующего уровня главного персонажа, или `nil`, если следующий уровень не заблокирован. Данные используются для определения причин блокировки перехода на новый уровень и информирования игрока о требуемых действиях.

::: warning Замечание
Функция возвращает осмысленные данные только в случае, если следующий уровень действительно заблокирован. Для проверки факта блокировки рекомендуется предварительно использовать функцию `avatar.IsNextLevelLocked()`.
:::

## Список параметров
Функция не принимает параметров.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если следующий уровень заблокирован из-за отсутствия возможности, возвращается таблица со следующими полями:
- **`unlockName`** (`WString`) — название требуемой возможности.
- **`unlockDescription`** (`WString`) — описание действий, необходимых для получения данной возможности.

Если следующий уровень не заблокирован или информация отсутствует, возвращается `nil`.

## Примеры
### Проверка блокировки следующего уровня и получение информации о требуемой возможности
```lua
if avatar.IsNextLevelLocked() then
    local info = avatar.GetNextLevelUnlockInfo()
    if info then
        LogInfo( "Следующий уровень закрыт, так как отсутствует возможность: ", info.unlockName )
        LogInfo( "Описание: ", info.unlockDescription )
    end
end
```
::: info Описание примера
В примере сначала проверяется, заблокирован ли следующий уровень главного персонажа с помощью функции `avatar.IsNextLevelLocked()`. Если уровень заблокирован, вызывается `avatar.GetNextLevelUnlockInfo()` для получения информации о требуемой возможности. При успешном получении таблицы в лог выводятся название возможности и описание действий для её открытия.
:::
