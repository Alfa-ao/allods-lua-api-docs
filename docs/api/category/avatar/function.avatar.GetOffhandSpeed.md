# avatar.GetOffhandSpeed
Возвращает значение скорости атаки дополнительным оружием аватара.

## Описание
```lua
avatar.GetOffhandSpeed(): table
```
Функция возвращает таблицу с параметрами скорости атаки дополнительным оружием аватара. Возвращаемые данные содержат базовое значение скорости, а также эффективное (текущее) значение, которое учитывает все активные модификаторы и характеристики персонажа.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`):
- **`base`** (`number`)
Базовое значение скорости атаки.
- **`effective`** (`number`)
Эффективное (текущее) значение скорости атаки с учетом всех модификаторов.

## Примеры
### Получение и вывод скорости атаки дополнительным оружием
```lua
local speed = avatar.GetOffhandSpeed()
common.LogInfo( "speed base:" .. speed.base .. " speed effective:" .. speed.effective )
```
