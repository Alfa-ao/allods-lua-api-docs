# avatar.GetPetActiveSpell
Возвращает индекс активного заклинания пета аватара.

## Описание
```lua
avatar.GetPetActiveSpell(): number
```
Функция возвращает целочисленный индекс активного заклинания пета аватара. Если активное заклинание не задано или произошла ошибка, возвращается значение `-1`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — индекс активного заклинания. В случае ошибки или отсутствия активного заклинания возвращается `-1`.

## Примеры
### Проверка и смена активного заклинания
```lua
local spells = avatar.GetPetSpells()
if spells[ 0 ] and ( 0 ~= avatar.GetPetActiveSpell() ) then
  avatar.SetPetActiveSpell( 0 )
end
```
::: info Описание примера
В примере извлекается список заклинаний пета с помощью `avatar.GetPetSpells`. Если заклинание с индексом `0` существует и текущее активное заклинание отличается от `0`, вызывается `avatar.SetPetActiveSpell` для установки нулевого заклинания в качестве активного.
:::
