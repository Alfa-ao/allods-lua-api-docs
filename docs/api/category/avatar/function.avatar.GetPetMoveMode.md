# avatar.GetPetMoveMode
Возвращает текущий тип движения пета аватара.
## Описание
```lua
avatar.GetPetMoveMode(): number
```
Функция извлекает и возвращает числовое значение, представляющее текущий режим передвижения пета аватара. Возвращаемое значение соответствует одному из типов поведения, определенных в перечислении `PET_MOVE_XXX`. Для установки нового режима движения применяется функция `avatar.SetPetMoveMode`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — тип поведения (режим движения) пета аватара. Значение соответствует одной из констант перечисления `PET_MOVE_XXX`.

## Примеры
### Получение текущего режима движения пета
```lua
local mode = avatar.GetPetMoveMode()
```
::: info Описание примера
В переменную `mode` записывается числовое значение текущего режима движения пета. Полученный результат допускается использовать для проверки текущего состояния или отображения информации в пользовательском интерфейсе.
:::
