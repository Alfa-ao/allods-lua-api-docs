# avatar.GetPetSpells
Возвращает список идентификаторов ресурсов переключаемых заклинаний пета аватара.

## Описание
```lua
avatar.GetPetSpells(): table
```
Функция возвращает таблицу (`table`), содержащую идентификаторы ресурсов переключаемых заклинаний пета аватара. Индексация элементов в возвращаемой таблице начинается с нуля.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` (`SpellId`), содержащую идентификаторы ресурсов переключаемых заклинаний. Индексация элементов в таблице начинается с нуля.

## Примеры
### Получение информации об активном заклинании пета
```lua
local spells = avatar.GetPetSpells()
local spellIndex = avatar.GetPetActiveSpell()
if spells[ spellIndex ] then
  local spellInfo = avatar.GetSpellInfo( spells[ spellIndex ] )
end
```
::: info Описание примера
В примере сначала получается список всех доступных переключаемых заклинаний пета. Затем извлекается индекс активного заклинания. Если по данному индексу в таблице заклинаний существует элемент, извлекается полная информация об этом заклинании с помощью функции `avatar.GetSpellInfo`.
:::
