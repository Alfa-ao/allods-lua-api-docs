# avatar.GetPos
Возвращает информацию о текущих координатах аватара.

## Описание
```lua
avatar.GetPos(): GamePosition
```
Функция возвращает структуру `GamePosition`, содержащую актуальные пространственные координаты аватара в игровом мире.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `GamePosition` — таблицу, содержащую координаты аватара.

## Примеры
### Получение и вывод координаты X аватара
```lua
local pos = avatar.GetPos()
LogInfo( "x: ", pos.posX )
```
