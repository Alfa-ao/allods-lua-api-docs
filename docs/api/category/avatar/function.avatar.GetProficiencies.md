# avatar.GetProficiencies
Возвращает список идентификаторов профессий главного игрока.
## Описание
```lua
avatar.GetProficiencies(): table
```
Функция возвращает таблицу с идентификаторами профессий главного игрока. Список профессий зависит от класса персонажа и его уровня и представляет собой умения персонажа носить определенный класс предметов.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — список идентификаторов профессий (`ObjectId`), индексированный с нуля.

## Примеры
### Получение и перебор списка профессий
```lua
local proficiencies = avatar.GetProficiencies()
for i, id in pairs( proficiencies ) do
    local proficiencyInfo = avatar.GetProficiencyInfo( id )
    local name = proficiencyInfo.name
end
```
::: info Описание примера
В примере получается список идентификаторов профессий главного игрока. Затем в цикле перебираются все элементы списка, и для каждого идентификатора извлекается подробная информация о профессии с помощью функции `avatar.GetProficiencyInfo`.
:::
