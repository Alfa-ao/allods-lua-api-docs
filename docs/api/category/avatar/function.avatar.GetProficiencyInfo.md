# avatar.GetProficiencyInfo
Возвращает детальную информацию о конкретной профессии главного персонажа по её уникальному идентификатору.

## Описание
```lua
avatar.GetProficiencyInfo( id: ObjectId ): table | nil
```
Функция извлекает и возвращает структурированные данные о профессии главного персонажа, идентифицируемой переданным уникальным номером. В текущей реализации игры под профессией понимается умение персонажа использовать (носить) вещи определённого класса экипировки. Доступность и характеристики конкретной профессии напрямую зависят от текущего уровня и выбранного класса главного персонажа.

Реализация функции разделена на две части: базовая логика находится на стороне ядра игры (GameState), а расширенные данные, необходимые исключительно для отображения в интерфейсе (UIState), формируются на стороне клиентского UI.

::: warning Замечание
Поле `image` в возвращаемой таблице формируется и доступно только в контексте UI. При вызове функции из базовой игровой логики (GameState) данное поле может отсутствовать или быть неинициализированным.
:::

## Список параметров
- **`id`** (`ObjectId`)
Уникальный идентификатор объекта профессии, информацию о котором необходимо получить.

## Возвращаемые значения
Возвращает `table` с детальной информацией о профессии или `nil`, если профессия с указанным идентификатором не существует или недоступна для текущего персонажа.

Структура возвращаемой таблицы:
- **`id`** (`ObjectId`) — уникальный идентификатор профессии.
- **`name`** (`WString`) — локализованное название профессии.
- **`description`** (`WString`) — локализованное текстовое описание профессии.
- **`sysInfo`** (`string`) — служебная системная информация, предназначенная для отладки (содержит путь к внутреннему файлу конфигурации профессии).
- **`image`** (`TextureId`) — идентификатор текстуры, используемой для отображения иконки профессии в пользовательском интерфейсе.

## Примеры
### Получение информации о первой доступной профессии
```lua
local proficiences = avatar.GetProficiences()
if GetTableSize( proficiences ) > 0 then
    local proficiencyInfo = avatar.GetProficiencyInfo( proficiences[0] )
    if proficiencyInfo then
        local name = proficiencyInfo.name
        LogInfo( "Название профессии: ", name )
    end
end
```
::: info Описание примера
В примере сначала запрашивается общий список доступных профессий главного персонажа с помощью `avatar.GetProficiences()`. Затем проверяется, что список не пуст, и извлекается идентификатор первой профессии (`proficiences[0]`). Этот идентификатор передается в `avatar.GetProficiencyInfo()` для получения полной информации. При успешном получении данных из таблицы извлекается и выводится в лог название профессии.
:::
