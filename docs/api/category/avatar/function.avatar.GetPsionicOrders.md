# avatar.GetPsionicOrders
Возвращает список идентификаторов специальных заклинаний для класса мистика.

## Описание
```lua
avatar.GetPsionicOrders(): table
```
Функция возвращает индексированный с единицы список идентификаторов ресурсов специальных заклинаний (спеллов), доступных для класса мистика.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированный с единицы список идентификаторов (`SpellId`) ресурсов заклинаний.

## Примеры
### Получение и вывод названий специальных заклинаний
```lua
local spells = avatar.GetPsionicOrders()
for index, spellId in ipairs(spells) do
    LogInfo( avatar.GetSpellInfo( spellId ).name )
end
```
::: info Описание примера
В примере извлекается список идентификаторов специальных заклинаний мистика. Далее в цикле перебираются полученные идентификаторы, и для каждого из них запрашивается подробная информация о заклинании с помощью `avatar.GetSpellInfo`, после чего в информационный лог выводится название заклинания.
:::
