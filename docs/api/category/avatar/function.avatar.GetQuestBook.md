# avatar.GetQuestBook
Возвращает список идентификаторов заданий, находящихся в журнале заданий аватара.

## Описание
```lua
avatar.GetQuestBook(): table
```
Функция извлекает и возвращает таблицу с идентификаторами всех заданий, которые в данный момент находятся в журнале заданий аватара. Индексация элементов в возвращаемой таблице начинается с единицы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую идентификаторы (`QuestId`) заданий. Индексация элементов в таблице начинается с единицы.

## Примеры
### Перебор и вывод идентификаторов активных заданий
```lua
local book = avatar.GetQuestBook()
for k, v in ipairs( book ) do
    LogInfo( "  book.k: "..k )
    LogInfo( "  book.v: "..v )
end
```
