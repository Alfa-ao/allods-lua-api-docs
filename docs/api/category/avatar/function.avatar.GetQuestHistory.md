# avatar.GetQuestHistory
Возвращает список заданий, выполненных аватаром.

## Описание
```lua
avatar.GetQuestHistory(): table
```
Функция возвращает таблицу, содержащую идентификаторы (`QuestId`) выполненных квестов. Индексация элементов в возвращаемом списке начинается с единицы, что позволяет использовать стандартные механизмы перебора массивов.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — список идентификаторов (`QuestId`) выполненных квестов. Индексация элементов начинается с `1`.

## Примеры
### Перебор и вывод идентификаторов выполненных квестов
```lua
local book = avatar.GetQuestHistory()
for index, questId in ipairs( book ) do
  common.LogInfo( "  id: " .. tostring( questId ) )
end
```
