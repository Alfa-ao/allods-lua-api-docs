# avatar.GetQuestImage
Возвращает идентификатор текстуры изображения, относящегося к тексту описания задания.

## Описание
```lua
avatar.GetQuestImage( questId: QuestId ): TextureId | nil
```
Функция извлекает и возвращает идентификатор текстуры изображения, которое ассоциировано с текстовым описанием конкретного задания. Реализация функции доступна исключительно в системе UIState. Если для указанного задания изображение отсутствует, функция возвращает `nil`.

## Список параметров
- **`questId`** (`QuestId`)
Уникальный идентификатор ресурса задания, для которого необходимо получить изображение.

## Возвращаемые значения
Возвращает `TextureId` — идентификатор текстуры изображения задания, или `nil`, если изображение для данного задания отсутствует.

## Примеры
### Получение изображения описания задания
```lua
local image = avatar.GetQuestImage( questId )
```
