# avatar.GetQuestInfo
Возвращает подробное описание задания по его идентификатору.

## Описание
```lua
avatar.GetQuestInfo( questId: QuestId ): table | nil
```
Функция извлекает и возвращает таблицу с полной информацией о задании, включая его текстовые описания, параметры, цели и местоположения. Для определения типа квеста используются константы `QUEST_TYPE_*`. Для получения геоданных карты по полям `goalLocation` и `returnLocation` применяется функция `cartographer.GetQuestGeodata`.

## Список параметров
- **`questId`** (`QuestId`)
Уникальный идентификатор задания, информацию о котором необходимо получить.

## Возвращаемые значения
Возвращает таблицу (`table`) с детальным описанием задания или `nil`, если информация не получена.

Поля возвращаемой таблицы:
- **`id`** (`QuestId`) — идентификатор ресурса задания.
- **`name`** (`ValuedText` | `nil`) — локализованное название задания.
- **`sysName`** (`string`) — внутреннее уникальное системное имя квеста.
- **`debugName`** (`string` | `nil`) — отладочное имя квеста (имя файла) или `nil`. Предназначено для отладочной версии.
- **`level`** (`number`) — уровень квеста.
- **`isHideLevel`** (`boolean`) — если `true`, уровень квеста не отображается в пользовательском интерфейсе.
- **`requiredLevel`** (`number`) — необходимый уровень главного игрока для принятия квеста.
- **`goal`** (`ValuedText` | `nil`) — текст цели задания.
- **`startText`** (`ValuedText` | `nil`) — текст, отображаемый при выдаче задания.
- **`checkText`** (`ValuedText` | `nil`) — текст, отображаемый при проверке готовности задания к сдаче.
- **`finishText`** (`ValuedText` | `nil`) — текст, отображаемый при успешном завершении задания.
- **`kickText`** (`ValuedText` | `nil`) — текст, отображаемый при провале задания.
- **`plotLine`** (`string`) — имя квестовой цепочки (нелокализованная строка для внутреннего использования).
- **`shared`** (`boolean`) — если `true`, данным заданием можно поделиться с другими игроками вручную.
- **`canCancel`** (`boolean`) — если `true`, задание можно отменить.
- **`canBeSkipped`** (`boolean`) — если `true`, задание можно пропустить (мгновенно завершить за Очки Судьбы).
- **`type`** (`number`) — тип квеста. Значения соответствуют перечислению `QUEST_TYPE_*`.
- **`isAutomatic`** (`boolean`) — если `true`, задание принимается и сдается автоматически (без диалога с NPC, например, ежедневное задание).
- **`isForInstance`** (`boolean`) — если `true`, задание предназначено для прохождения в подземелье (инстансе).
- **`isPvP`** (`boolean`) — если `true`, задание предназначено для PvP.
- **`isInSecretSequence`** (`boolean`) — если `true`, задание входит в линейку, открывающую тайны мира.
- **`isTutorial`** (`boolean`) — если `true`, задание предназначено для обучения.
- **`isItemMall`** (`boolean`) — если `true`, задание обучает особенностям Лавки Редкостей.
- **`isLowPriority`** (`boolean`) — если `true`, задание является второстепенным.
- **`isNextQuestFinder`** (`boolean`) — если `true`, задание автоматически сгенерировано для помощи в поиске следующего задания.
- **`uiPriority`** (`number`) — приоритет для сортировки квестов в аддоне NPCTalk (соответствует полю "uiPriority" в базе данных).
- **`isRepeatable`** (`boolean`) — если `true`, задание можно выполнить повторно через некоторое время.
- **`canRepeat`** (`boolean`) — если `true`, повторно выполняемое задание доступно для выполнения прямо сейчас.
- **`repeatPeriod`** (`number`) — период в днях, через который можно повторно выполнить задание. `0`, если можно выполнить сразу или повторное выполнение невозможно (см. `isRepeatable`).
- **`isSecret`** (`boolean`) — если `true`, это главное задание тайны мира.
- **`zoneName`** (`WString`) — имя зоны, к которой привязан квест.
- **`zonesMapId`** (`ObjectId` | `nil`) — идентификатор интерфейсной карты задания, если не произошло ошибки.
- **`returnLocation`** (`table` | `nil`) — точка сдачи квеста, если она задана.
  - `zonesMapId` (`ObjectId`) — идентификатор интерфейсной карты зоны точки.
  - `position` (`GamePosition`) — координаты точки.
  - `isIndoor` (`boolean`) — если `true`, точка цели или сдачи квеста находится в помещении.

## Примеры
### Получение и вывод базовой информации о задании
```lua
local info = avatar.GetQuestInfo( questId )
if info then
    LogInfo( "Quest level: " .. info.level )
    LogInfo( "Quest name: " .. info.name )
    LogInfo( "Quest goal: " .. info.goal )
end
```
