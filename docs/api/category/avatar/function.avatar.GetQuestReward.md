# avatar.GetQuestReward
Получает описание награды за выполнение задания.

## Описание
```lua
avatar.GetQuestReward( questId: QuestId ): table | nil
```
Функция извлекает и возвращает таблицу с детальным описанием награды за выполнение задания. Возвращаемые данные включают информацию о денежном вознаграждении, опыте, верности и влиянии в гильдии, списках обязательных и альтернативных предметов, репутационных бонусах для фракций, альтернативной валюте и открываемых возможностях. Если информация о награде отсутствует или идентификатор некорректен, функция возвращает `nil`.

## Список параметров
- **`questId`** (`QuestId`)
Уникальный идентификатор задания, информацию о награде за которое необходимо получить.

## Возвращаемые значения
Возвращает таблицу (`table`) с описанием награды или `nil`, если информация не получена.
Структура возвращаемой таблицы:
- **`money`** (`number`) — сумма денежного вознаграждения.
- **`experience`** (`number`) — количество опыта в награду.
- **`loyalty`** (`number`) — получаемая верность гильдии.
- **`authority`** (`number`) — получаемое влияние в гильдии.
- **`mandatoryItems`** (`table`) — список идентификаторов (`ObjectId`) обязательных наградных предметов, которые известны.
- **`mandatoryItemsCount`** (`number`) — общее количество наградных предметов. Значение может быть больше длины таблицы `mandatoryItems`, если некоторые награды неизвестны.
- **`alternativeItems`** (`table`) — список идентификаторов (`ObjectId`) альтернативных наградных предметов.
- **`reputations`** (`table`) — индексированный с 0 список таблиц со значениями репутационных наград для разных фракций. Поля каждого пункта:
  - `faction` (`WString`) — имя фракции.
  - `value` (`number`) — значение, на которое повысится репутация.
- **`currencies`** (`table`) — индексированный с 0 список таблиц с наградами альтернативной валютой. Поля каждого пункта:
  - `currencyId` (`CurrencyId`) — идентификатор альтернативной валюты.
  - `value` (`number`) — величина награды.
- **`unlocks`** (`table`) — индексированный с 1 список идентификаторов (`UnlockId`) возможностей (анлоков), выдаваемых в награду.

## Примеры
### Получение и обработка информации о награде за задание
```lua
local questReward = avatar.GetQuestReward( questId )
if questReward then
    LogInfo( "Quest reward: ", questReward.money )

    for i = 0, GetTableSize( questReward.currencies ) - 1 do
        local currencyId = questReward.currencies[ i ].currencyId
        local currencyInfo = avatar.GetCurrencyInfo( currencyId )
    end
end
```
::: info Описание примера
В примере запрашивается описание награды за задание с указанным идентификатором. Если информация успешно получена, выводится сумма денежного вознаграждения. Затем выполняется перебор списка наград альтернативной валютой для извлечения идентификаторов валют и получения их подробной информации через функцию `avatar.GetCurrencyInfo`.
:::
