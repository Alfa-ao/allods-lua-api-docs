# avatar.GetQuestShareInvitations
Возвращает список приглашений от игроков, делящихся своими заданиями.
## Описание
```lua
avatar.GetQuestShareInvitations(): table
```
Функция возвращает таблицу, содержащую список приглашений от игроков, делящихся своими заданиями. Индексация элементов в таблице начинается с единицы. Каждый элемент представляет собой таблицу с информацией о конкретном приглашении.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), индексированную с единицы. Элементы таблицы представляют собой таблицы со следующими полями:
- **`shareId`** (`ObjectId`)
Уникальный идентификатор приглашения.
- **`questId`** (`QuestId`)
Идентификатор задания.
- **`sharerName`** (`WString`)
Имя игрока, который делится заданием.

## Примеры
### Получение и обработка списка приглашений
```lua
local invitations = avatar.GetQuestShareInvitations()
for _, info in ipairs( invitations ) do
    LogInfo( "  sharerName: ", userMods.FromWString( info.sharerName ) )
    local quest = avatar.GetQuestInfo( info.questId )
    
    if quest then
        LogInfo( "  quest name: ", userMods.FromWString( quest.name ) )
    end
end
```
::: info Описание примера
В примере извлекается список приглашений на совместное выполнение заданий. Затем происходит перебор полученной таблицы. Для каждого приглашения выводится имя игрока, отправившего приглашение, и, при наличии информации о задании, его название.
:::
