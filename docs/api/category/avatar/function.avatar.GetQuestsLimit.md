# avatar.GetQuestsLimit
Возвращает текущее и максимально допустимое количество заданий в журнале заданий.

## Описание
```lua
avatar.GetQuestsLimit(): number, number
```
Функция возвращает текущее и максимально допустимое количество заданий в журнале заданий.

::: warning Замечание
При подсчете текущего числа заданий некоторые специальные задания (например, compendium) не учитываются и не влияют на лимиты.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает два значения типа `number`:
- Текущее количество заданий.
- Максимально допустимое количество заданий.

## Примеры
### Вывод информации о лимитах заданий в лог
```lua
local current, max = avatar.GetQuestsLimit()
LogInfo( "Limit: ", current, " / ", max )
```
