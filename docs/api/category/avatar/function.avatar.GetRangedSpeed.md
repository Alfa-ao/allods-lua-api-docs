# avatar.GetRangedSpeed
Возвращает значение скорости атаки дистанционным оружием аватара.

## Описание
```lua
avatar.GetRangedSpeed(): table
```
Функция возвращает таблицу, содержащую базовое и эффективное (текущее) значения скорости атаки дистанционным оружием аватара.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`):
- **`base`** (`number`) — базовое значение скорости атаки.
- **`effective`** (`number`) — эффективное (текущее) значение скорости атаки.

## Примеры
### Получение и вывод скорости атаки
```lua
local speed = avatar.GetRangedSpeed()
common.LogInfo( "speed base:" .. speed.base .. " speed effective:" .. speed.effective )
```
