# avatar.GetRecipeAfflatusItem
Возвращает идентификатор предмета, выдающего озарение, для указанного рецепта.

## Описание
```lua
avatar.GetRecipeAfflatusItem( id: RecipeId ): ItemId | nil
```
Функция извлекает и возвращает идентификатор предмета, который предоставляет озарение для заданного рецепта. Идентификатор ресурса рецепта передается в качестве параметра. Если рецепт не найден по указанному идентификатору или для него не прописан предмет озарения, функция возвращает `nil`.

## Список параметров
- **`id`** (`RecipeId`)
Идентификатор ресурса рецепта.

## Возвращаемые значения
Возвращает `ItemId` или `nil`:
- **`ItemId`** — идентификатор предмета, выдающего озарение.
- **`nil`** — если рецепт не найден по идентификатору или предмет озарения не прописан в рецепте.

## Примеры
### Получение предмета озарения для рецепта
```lua
local recipeId = 1001
local itemId = avatar.GetRecipeAfflatusItem( recipeId )
```
