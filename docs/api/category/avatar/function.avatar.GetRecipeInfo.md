# avatar.GetRecipeInfo
Извлекает и возвращает детальную информацию о рецепте по заданному идентификатору ресурса.

## Описание
```lua
avatar.GetRecipeInfo( id: RecipeId ): table | nil
```
Функция формирует и возвращает структуру данных, содержащую исчерпывающую информацию о рецепте. Возвращаемые данные включают название, описание, необходимые параметры для изучения, компоненты, создаваемые предметы и информацию о следующих рецептах. Часть возвращаемых полей (например, `image`) реализована исключительно для интерфейса (UI).

## Список параметров
- **`id`** (`RecipeId`)
Идентификатор ресурса рецепта.

## Возвращаемые значения
Возвращает таблицу (`table`) с информацией о рецепте или `nil`, если рецепт не найден по указанному идентификатору.

Структура возвращаемой таблицы:
- **`id`** (`RecipeId`) — идентификатор ресурса рецепта.
- **`name`** (`WString`) — название рецепта.
- **`description`** (`ValuedText` | `nil`) — описание с подставленными текущими значениями параметров.
- **`skillId`** (`SkillId` | `nil`) — идентификатор ресурса умения, которому принадлежит рецепт (если умение выучено игроком).
- **`score`** (`number`) — необходимый уровень умения для изучения рецепта.
- **`image`** (`TextureId`) — идентификатор текстуры для иконки рецепта (реализовано только для UI).
- **`components`** (`table`) — массив идентификаторов компонентов рецепта. Тип передаваемых идентификаторов зависит от умения: для алхимии передаются `ComponentPropertyId` (идентификаторы ресурсов алхимических компонентов), для создания доспехов (`armorCraft` или `DiceCraft`) — `ObjectId` (идентификаторы объектов предметов).
- **`resultQuantity`** (`number`) — количество предметов, получаемых из рецепта по умолчанию.
- **`defaultItem`** (`ObjectId` | `nil`) — идентификатор предмета, получаемого из рецепта по умолчанию.
- **`nextRecipe`** (`UnlockId` | `nil`) — идентификатор разблокировки следующего ресурса (если разблокировка ещё не выполнена).
- **`nextRecipeId`** (`RecipeId` | `nil`) — идентификатор следующего рецепта.
- **`nextRecipePoints`** (`number`) — количество очков просветления, необходимых для разблокировки следующего рецепта.
- **`qualificationPoints`** (`CurrencyId`) — идентификатор альтернативной валюты.
- **`resultItems`** (`table`) — индексированный с нуля массив идентификаторов предметов, создаваемых по рецепту. Элементы отсортированы по качеству: от менее качественного (индекс 0) к более качественному.
- **`bindResult`** (`boolean`) — флаг, указывающий, будет ли результат создания привязан к аватару.

## Примеры
### Перебор рецептов и получение информации о создаваемых предметах
```lua
local alchemy = avatar.GetAlchemyInfo()
local recipes = alchemy.recipes
for i = 0, GetTableSize( recipes ) - 1 do
    local recipeInfo = avatar.GetRecipeInfo( recipes[ i ] )
    if recipeInfo then
        LogInfo( userMods.FromWString( recipeInfo.name ) )

        local firstItemInfo = avatar.GetItemInfo( recipeInfo.resultItems[ 0 ] )
        if firstItemInfo then
            LogInfo( "Предмет, создаваемый рецептом - ", userMods.FromWString( firstItemInfo.name ) )
        end
    end
end
```
::: info Описание примера
В примере запрашивается информация об алхимии и извлекается список идентификаторов рецептов. В цикле перебирается каждый идентификатор, для которого вызывается функция `avatar.GetRecipeInfo`. При успешном получении информации выводится название рецепта. Затем извлекается идентификатор первого создаваемого предмета из массива `resultItems`, и с помощью функции `avatar.GetItemInfo` получается и выводится его название.
:::
