# avatar.GetRecommendedStats
возвращает текущие рекомендованные характеристики для аватара.

## Описание
```lua
avatar.GetRecommendedStats(): table
```
Функция возвращает таблицу с текущими рекомендованными характеристиками для аватара. Набор характеристик зависит от класса персонажа и активного аспекта.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`) со следующими полями:
- **`commonGoldStats`** (`table` | `nil`)
Список обще рекомендованных характеристик золотого тира. Каждая запись содержит следующие поля:
- `resourceId` (`SpecialStatId`) — ресурс характеристики.
- `innateType` (`string` | `nil`) — тип характеристики (только для `InnateStats`). Значения соответствуют перечислению `ENUM_InnateStats_...`.
- **`commonSilverStats`** (`table` | `nil`)
Список обще рекомендованных характеристик серебряного тира.
- **`slots`** (`table` | `nil`)
Таблица рекомендованных характеристик для конкретных слотов экипировки. Ключом является значение перечисления `DRESS_SLOT_*`. Каждая ячейка содержит следующие поля:
- `goldStats` (`table` | `nil`) — список характеристик золотого тира.
- `silverStats` (`table` | `nil`) — список характеристик серебряного тира.

## Примеры
### Получение и вывод рекомендованных характеристик
```lua
local recommendedStats = avatar.GetRecommendedStats()
LogInfo( "Current recommended stats: ", recommendedStats )
```
