# avatar.GetReputationInfo
Возвращает информацию о репутации аватара с указанной фракцией.

## Описание
```lua
avatar.GetReputationInfo( factionId: FactionId ): table | nil
```
Функция извлекает и возвращает детальную информацию о текущем состоянии репутации главного персонажа с заданной фракцией. Если репутация с указанной фракцией еще не открыта или данные отсутствуют, функция возвращает `nil`. Возвращаемая таблица содержит числовые значения репутации, текущий и следующий уровни, а также пороговые значения для перехода между уровнями.

::: warning Замечание
Если репутация с указанной фракцией не открыта у персонажа, функция возвращает `nil`. Перед обращением к полям возвращаемой таблицы необходимо проверять результат на `nil`.
:::

## Список параметров
- **`factionId`** (`FactionId`)
Уникальный идентификатор фракции, информацию о репутации с которой необходимо получить.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если репутация открыта, возвращается таблица со следующими полями:
- **`factionId`** (`FactionId`) — идентификатор фракции.
- **`rep`** (`number`) — текущее числовое значение репутации.
- **`level`** (`number`) — текущий уровень репутации. Значение соответствует перечислению `REPUTATION_LEVEL_*`.
- **`nextLevel`** (`number`) — следующий уровень репутации. Если достигнут максимальный уровень, возвращается значение максимального уровня. Значение соответствует перечислению `REPUTATION_LEVEL_*`.
- **`repCurrLevel`** (`number` | `nil`) — пороговое значение репутации для получения текущего уровня. Возвращает `nil`, если для текущего уровня порог не задан.
- **`repNextLevel`** (`number` | `nil`) — пороговое значение репутации для перехода на следующий уровень. Возвращает `nil`, если достигнут максимальный уровень репутации.

Если репутация с данной фракцией не открыта, возвращается `nil`.

## Примеры
### Получение и анализ информации о репутации
```lua
local factionId = 1 -- Идентификатор целевой фракции
local repInfo = avatar.GetReputationInfo( factionId )

if repInfo then
    LogInfo( "Текущая репутация: ", repInfo.rep )
    LogInfo( "Текущий уровень: ", repInfo.level )

    if repInfo.repNextLevel then
        local progress = repInfo.rep - (repInfo.repCurrLevel or 0)
        local needed = repInfo.repNextLevel - (repInfo.repCurrLevel or 0)
        LogInfo( "Прогресс до следующего уровня: ", progress, " / ", needed )
    else
        LogInfo( "Достигнут максимальный уровень репутации." )
    end
else
    LogInfo( "Репутация с данной фракцией не открыта." )
end
```
::: info Описание примера
В примере запрашивается информация о репутации с фракцией. Если данные успешно получены (результат не `nil`), в лог выводятся текущее значение репутации и её уровень. Дополнительно рассчитывается и выводится прогресс до следующего уровня, если он еще не достигнут (проверяется наличие поля `repNextLevel`). Если функция вернула `nil`, выводится сообщение о том, что репутация не открыта.
:::
