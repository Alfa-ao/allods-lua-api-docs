# avatar.GetReputationValue
Возвращает числовое значение репутации, необходимое для достижения заданного уровня.

## Описание
```lua
avatar.GetReputationValue( reputationLevel: number ): number | nil
```
Функция извлекает и возвращает числовое значение репутации, которое требуется для достижения указанного уровня. Если передан некорректный идентификатор уровня, функция возвращает `nil`. Значения уровней репутации задаются перечислением `REPUTATION_LEVEL_*`.

## Список параметров
- **`reputationLevel`** (`number`)
Уровень репутации. Значение задается с помощью констант перечисления `REPUTATION_LEVEL_*`.

## Возвращаемые значения
Возвращает `number` — числовое значение репутации, необходимое для достижения заданного уровня, или `nil`, если уровень репутации указан некорректно.

## Примеры
### Получение значения репутации для заданного уровня
```lua
local reputationValue = avatar.GetReputationValue( REPUTATION_LEVEL_FRIENDLY )
if reputationValue then
    LogInfo( "Для достижения уровня требуется ", reputationValue, " очков репутации." )
end
```
