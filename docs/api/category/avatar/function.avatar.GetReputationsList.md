# avatar.GetReputationsList
Возвращает список открытых репутаций аватара.

## Описание
```lua
avatar.GetReputationsList(): table
```
Функция возвращает таблицу с информацией об открытых репутациях аватара. Каждый элемент структуры содержит идентификатор фракции, текущее значение и уровень репутации, а также пороговые значения для текущего и следующего уровней.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), где ключом выступает числовой индекс (`number`), а значением — таблица с информацией о конкретной репутации, содержащая следующие поля:
- **`factionId`** (`FactionId`)
Идентификатор фракции.
- **`rep`** (`number`)
Текущее значение репутации.
- **`level`** (`number`)
Текущий уровень репутации. Значения соответствуют перечислению `REPUTATION_LEVEL_...`.
- **`nextLevel`** (`number`)
Следующий либо максимальный уровень репутации. Значения соответствуют перечислению `REPUTATION_LEVEL_...`.
- **`repCurrLevel`** (`number` | `nil`)
Пороговое значение репутации для получения текущего уровня. Возвращает `nil`, если значение отсутствует.
- **`repNextLevel`** (`number` | `nil`)
Пороговое значение репутации для получения следующего уровня. Возвращает `nil`, если значение отсутствует.

## Примеры
### Получение и перебор списка открытых репутаций
```lua
local reputationsList = avatar.GetReputationsList()
if reputationsList then
    for i = 0, GetTableSize( reputationsList ) - 1 do
        local repInfo = reputationsList[ i ]
        LogInfo( "Фракция: ", repInfo.factionId, ", Уровень: ", repInfo.level )
    end
end
```
