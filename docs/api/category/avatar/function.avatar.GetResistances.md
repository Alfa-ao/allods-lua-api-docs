# avatar.GetResistances
Возвращает таблицу с информацией о сопротивляемостях аватара.
## Описание
```lua
avatar.GetResistances(): table
```
Функция возвращает таблицу, индексированную с нуля, содержащую информацию о сопротивляемостях аватара. Индексы таблицы соответствуют значениям перечисления `RESISTANCE_...` (или `ENUM_Element_...`). Поля каждого элемента представляют собой таблицы со структурой, аналогичной возвращаемой функции `avatar.GetInnateStats()`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, индексированную с нуля (`[0..]`). Индексы элементов соответствуют значениям перечисления `RESISTANCE_...` (или `ENUM_Element_...`). Значения элементов представляют собой таблицы с полями, структура которых аналогична возвращаемой функции `avatar.GetInnateStats()` (например, `base`, `effective`, `sysName`).

## Примеры
### Перебор и вывод значений сопротивляемостей
```lua
local stats = avatar.GetResistances()
for i = 0, GetTableSize( stats ) - 1 do
    local stat = stats[i]
    LogInfo( "base: ", stat.base, ", effective: ", stat.effective, ", sysName: ", stat.sysName )
end
```
::: info Описание примера
В примере извлекается таблица сопротивляемостей аватара. Затем в цикле перебираются все элементы таблицы, и в информационный лог выводятся базовое, эффективное значения, а также системное имя каждой сопротивляемости.
:::
