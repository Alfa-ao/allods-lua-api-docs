# avatar.GetReturnableQuests
Возвращает список заданий, которые текущий NPC-собеседник может принять от игрока.

## Описание
```lua
avatar.GetReturnableQuests(): table
```
Функция возвращает список идентификаторов заданий, которые NPC-собеседник готов принять. Список формируется на основе текущего диалога с NPC и содержит только те задания, которые игрок может сдать.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — список идентификаторов заданий (`QuestId`). Индексация элементов в таблице начинается с единицы.

## Примеры
### Получение списка заданий для сдачи
```lua
local retQuestList = avatar.GetReturnableQuests()
```
