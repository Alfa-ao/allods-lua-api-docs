# avatar.GetRubyStartLevel
Возвращает уровень главного игрока, с которого доступно распределение вех.

## Описание
```lua
avatar.GetRubyStartLevel(): number
```
Функция возвращает целочисленное значение уровня главного игрока, начиная с которого становится доступным распределение вех развития.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — целочисленное значение уровня главного игрока, с которого доступно распределение вех.

## Примеры
### Получение уровня начала распределения вех
```lua
local level = avatar.GetRubyStartLevel()
```
