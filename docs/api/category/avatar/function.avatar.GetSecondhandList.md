# avatar.GetSecondhandList
Возвращает список вещей вторичного рынка на продажу у NPC-торговца.

## Описание
```lua
avatar.GetSecondhandList(): table
```
Функция возвращает таблицу с описаниями объектов, доступных для покупки на вторичном рынке у NPC-торговца.

::: warning Замечание
Функция должна быть вызвана после выполнения запроса `avatar.RequestVendor()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую описания объектов на продажу. Каждый элемент таблицы представляет собой структуру со следующими полями:
- **`id`** (`ObjectId`) — уникальный идентификатор объекта.
- **`quantity`** (`number`) — количество предметов, имеющихся у торговца. Если значение равно `-1`, количество не ограничено.
- **`counter`** (`number`) — значение предметного счётчика (заряды). Значение `-1` означает, что счётчик отсутствует.
- **`isBound`** (`boolean`) — признак привязки предмета.
- **`isSecondhand`** (`boolean`) — признак товара с вторичного рынка (всегда принимает значение `true`).
- **`price`** (`number`) — цена одного предмета в основной внутриигровой валюте.
- **`requiredUnlocks`** (`table` | `nil`) — таблица требуемых анлоков для покупки (индексация с 1) или `nil`, если анлоки не требуются.
- **`requiredAchievements`** (`table` | `nil`) — таблица требуемых очков событий для покупки (индексация с 1) или `nil`, если достижения не требуются. Каждый элемент таблицы содержит поля:
  - **`ruleId`** (`RuleId`) — идентификатор события.
  - **`achievement`** (`number`) — требуемое количество очков события.
- **`alternativePrices`** (`table` | `nil`) — список цен в альтернативных валютах (индексируется от 0) или `nil`, если информация отсутствует. Каждый элемент списка содержит поля:
  - **`currencyId`** (`CurrencyId`) — идентификатор альтернативной валюты.
  - **`value`** (`number`) — цена в данной альтернативной валюте.

## Примеры
### Перебор списка предметов вторичного рынка
```lua
local list = avatar.GetSecondhandList()
for i = 0, GetTableSize( list ) - 1 do
    local itemId = list[i].id
    local object = avatar.GetItemInfo( itemId )
    local name = object.name
end
```
::: info Описание примера
В примере получается список предметов вторичного рынка с помощью `avatar.GetSecondhandList()`. Затем в цикле перебираются все элементы списка. Для каждого предмета извлекается его идентификатор (`id`), который используется для получения полной информации о предмете через `avatar.GetItemInfo()`, после чего извлекается имя предмета.
:::
