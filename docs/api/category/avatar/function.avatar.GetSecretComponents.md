# avatar.GetSecretComponents
Возвращает список описаний компонент тайны мира.

## Описание
```lua
avatar.GetSecretComponents( secretId: ObjectId ): table
```
Функция извлекает и возвращает таблицу с описаниями компонент заданной тайны мира. Если компонента не раскрыта (не сдан завершающий квест), текст компоненты не возвращается (поле `text` принимает значение `nil`).

## Список параметров
- **`secretId`** (`ObjectId`)
Уникальный идентификатор тайны мира.

## Возвращаемые значения
Возвращает таблицу (`table`), индексированную с единицы. Каждый элемент таблицы представляет собой структуру со следующими полями:
- **`opened`** (`boolean`)
`true`, если компонент открыт (взят начальный квест).
- **`closed`** (`boolean`)
`true`, если компонент закрыт (сдан завершающий квест).
- **`text`** (`WString` | `nil`)
Текст компоненты, если она раскрыта.
- **`level`** (`number`)
Уровень персонажа, с которого компонент будет доступен для выполнения.
- **`index`** (`number`)
Порядковый номер компоненты (нумерация начинается с нуля).

## Примеры
### Получение и проверка статуса компонент тайны мира
```lua
local secrets = avatar.GetSecrets()
for i, secretId in pairs( secrets ) do
    local components = avatar.GetSecretComponents( secretId )
    if components then
        local component = components[1]
        if component and component.closed then
            local text = component.text
            LogInfo( "Текст компоненты: ", text )
        end
    end
end
```
::: info Описание примера
В примере выполняется получение списка всех доступных тайн мира. Для каждой тайны извлекается список её компонент. Проверяется первая компонента (индекс `1`), и если она закрыта (завершающий квест сдан), выводится её текст в информационный лог.
:::
