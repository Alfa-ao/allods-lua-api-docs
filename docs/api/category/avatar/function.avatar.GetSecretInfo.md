# avatar.GetSecretInfo
Возвращает описание тайны мира.

## Описание
```lua
avatar.GetSecretInfo( secretId: ObjectId ): table | nil
```
Функция извлекает и возвращает детальную информацию о тайне мира по её идентификатору. Основная реализация логики находится в системной части (UIState/GameState), при этом часть данных, таких как изображение книги, формируется исключительно на стороне пользовательского интерфейса. Если тайна мира с указанным идентификатором не найдена, функция возвращает `nil`.

## Список параметров
- **`secretId`** (`ObjectId`)
Уникальный идентификатор тайны мира.

## Возвращаемые значения
Возвращает таблицу (`table`) с описанием тайны мира или `nil`, если тайна не найдена.

Структура возвращаемой таблицы:
- **`id`** (`ObjectId`)
Идентификатор тайны мира.
- **`questId`** (`QuestId`)
Идентификатор ресурса главного задания тайны мира.
- **`closed`** (`boolean`)
`true`, если все компоненты тайны мира завершены.
- **`finished`** (`boolean`)
`true`, если тайна мира была полностью завершена и получена главная награда.
- **`level`** (`number`)
Уровень персонажа, с которого доступна тайна мира.
- **`image`** (`TextureId`)
Идентификатор изображения книги тайны мира. Поле формируется и доступно только при вызове функции из пользовательского интерфейса.

## Примеры
### Получение и перебор информации о всех доступных тайнах мира
```lua
local secrets = avatar.GetSecrets()
for i, secretId in pairs( secrets ) do
  local secretInfo = avatar.GetSecretInfo( secretId )
  if secretInfo then
    -- Обработка информации о тайне мира
  end
end
```
::: info Описание примера
В примере сначала извлекается список всех идентификаторов тайн мира с помощью `avatar.GetSecrets()`. Затем в цикле перебирается каждый идентификатор и передается в функцию `avatar.GetSecretInfo()` для получения детального описания конкретной тайны.
:::
