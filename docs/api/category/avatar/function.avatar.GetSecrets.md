# avatar.GetSecrets
Возвращает список идентификаторов всех тайн мира, которые уже доступны или будут доступны аватару.

## Описание
```lua
avatar.GetSecrets(): table
```
Функция возвращает таблицу (`table`), содержащую список идентификаторов (`ObjectId`) всех тайн мира. В список включаются как уже открытые игроком тайны, так и те, которые станут доступны в будущем. Индексация элементов в возвращаемой таблице начинается с единицы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), индексированную с единицы, содержащую идентификаторы (`ObjectId`) тайн мира.

## Примеры
### Получение и перебор списка тайн мира
```lua
local secrets = avatar.GetSecrets()
for i, secretId in ipairs( secrets ) do
  local secretInfo = avatar.GetSecretInfo( secretId )
end
```
::: info Описание примера
В примере извлекается таблица идентификаторов всех тайн мира. Затем с помощью цикла `for` и функции `ipairs` происходит перебор всех элементов списка. На каждой итерации идентификатор тайны передается в функцию `avatar.GetSecretInfo` для получения детальной информации о конкретной тайне.
:::
