# avatar.GetServerId
Возвращает постоянный (серверный) идентификатор аватара.

## Описание
```lua
avatar.GetServerId(): PersistentId
```
Функция возвращает постоянный серверный идентификатор аватара. Данный идентификатор является уникальным и неизменным на протяжении всего времени существования персонажа, что позволяет использовать его для однозначной идентификации аватара на стороне сервера и в межсерверных взаимодействиях.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `PersistentId` — постоянный серверный идентификатор аватара. Значение гарантированно не равно `nil`.

## Примеры
### Получение серверного идентификатора аватара
```lua
local id = avatar.GetServerId()
```
