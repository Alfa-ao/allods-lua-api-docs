# avatar.GetShardLevelCap
Возвращает максимальный доступный уровень персонажа на текущем шарде.

## Описание
```lua
avatar.GetShardLevelCap(): number
```
Функция возвращает целое число, представляющее максимальный доступный уровень персонажа на текущем игровом шарде. Данное значение определяет предельный порог прокачки, установленный сервером для всех персонажей.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`number` — целое число, обозначающее максимальный доступный уровень персонажа. В случае отсутствия данных или ошибки возвращается `1`.

## Примеры
### Получение максимального уровня шарда
```lua
local levelCap = avatar.GetShardLevelCap()
```
