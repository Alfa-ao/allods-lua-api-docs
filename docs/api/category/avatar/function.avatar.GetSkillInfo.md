# avatar.GetSkillInfo
Возвращает информацию об умении по его идентификатору.

## Описание
```lua
avatar.GetSkillInfo( id: SkillId ): table | nil
```
Функция извлекает детальную информацию об умении, включая данные о любом скилле, даже если он еще не изучен главным персонажем. В случае, если умение не изучено, поля уровня и опыта в возвращаемой таблице принимают значение `nil`.

::: warning Замечание
В игре существуют две системы прокачки умений. В старой системе уровень изменяется плавно от 1 до максимально возможной величины. В новой системе прокачка происходит ступенчатыми уровнями: уровни накапливают опыт, выраженный в процентах, и при достижении 100% уровень повышается на 1. Поле `exp` в возвращаемой таблице заполняется только для умений, использующих новую ступенчатую систему.
:::

## Список параметров
- **`id`** (`SkillId`)
Уникальный идентификатор умения (скилла), информацию о котором необходимо получить.

## Возвращаемые значения
Возвращает таблицу (`table`) с информацией об умении или `nil`, если умение с указанным идентификатором не существует.

Структура возвращаемой таблицы:
- **`id`** (`SkillId`)
Уникальный идентификатор ресурса умения.
- **`score`** (`number` | `nil`)
Текущий уровень умения. Принимает значение `nil`, если умение не изучено главным персонажем.
- **`exp`** (`number` | `nil`)
Накопленный опыт текущего уровня умения в процентах. Принимает значение `nil`, если умение не изучено или использует старую систему плавной прокачки.

## Примеры
### Получение и вывод опыта изученных умений
```lua
local skills = avatar.GetSkills()
for i = 0, GetTableSize( skills ) - 1 do
    local skillInfo = avatar.GetSkillInfo( skills[ i ] )
    if skillInfo then
        LogInfo( skillInfo.exp )
    end
end
```
::: info Описание примера
В примере извлекается список всех доступных умений с помощью `avatar.GetSkills()`. Затем в цикле перебирается каждый идентификатор, и для каждого вызывается `avatar.GetSkillInfo()`. Если информация об умении успешно получена, в информационный лог выводится значение накопленного опыта (`exp`).
:::
