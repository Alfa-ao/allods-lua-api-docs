# avatar.GetSkills
Возвращает список скилов главного героя.

## Описание
```lua
avatar.GetSkills(): table
```
Функция извлекает и возвращает таблицу, содержащую идентификаторы ресурсов скилов главного героя.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу идентификаторов (`SkillId`) ресурсов скилов главного героя. Индексация элементов в таблице начинается с нуля.

## Примеры
### Получение и перебор списка доступных скилов
```lua
local skills = avatar.GetSkills()
for i = 0, GetTableSize( skills ) - 1 do
  local skillInfo = avatar.GetSkillInfo( skills[ i ] )
end
```
::: info Описание примера
В примере извлекается таблица идентификаторов скилов. Затем с помощью цикла `for` происходит перебор всех элементов таблицы. Поскольку индексация начинается с нуля, цикл выполняется от `0` до `GetTableSize(skills) - 1`. На каждой итерации идентификатор передается в функцию `avatar.GetSkillInfo` для получения детальной информации о конкретном скиле.
:::
