# avatar.GetSparkCost
Возвращает стоимость в (мирро) для преждевременного вызова искры главного игрока, находящегося на кладбище.

## Описание
```lua
avatar.GetSparkCost(): number | nil
```
Функция возвращает стоимость вызова искры в мирро (`number`) или `nil`, если игрок не находится в состоянии воскрешения. Стоимость постоянно меняется из-за различных условий. При изменении стоимости в момент нахождения на кладбище передается событие `EVENT_CALL_SPARK_COST_CHANGED`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
- `number` — стоимость вызова искры в мирро.
- `nil` — если игрок не находится в состоянии воскрешения.

## Примеры
### Получение стоимости вызова искры
```lua
local price = avatar.GetSparkCost()
if price then
    LogInfo( "Стоимость вызова искры: ", price )
end
```
