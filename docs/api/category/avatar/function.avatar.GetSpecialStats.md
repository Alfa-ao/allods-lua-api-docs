# avatar.GetSpecialStats
Возвращает специальные характеристики аватара.

## Описание
```lua
avatar.GetSpecialStats(): table
```
Функция извлекает и возвращает таблицу, содержащую детализированную информацию о специальных характеристиках аватара. Возвращаемая таблица представляет собой массив, индексированный начиная с единицы, элементы которого являются таблицами с полями, описывающими конкретную характеристику.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — массив таблиц, индексированный начиная с `1`. Каждая вложенная таблица содержит следующие поля:
- **`id`** (`SpecialStatId`)
Идентификатор специальной характеристики.
- **`name`** (`WString`)
Название характеристики.
- **`tooltipName`** (`WString`)
Название характеристики для всплывающей подсказки.
- **`sysName`** (`string`)
Служебное название характеристики.
- **`base`** (`number`)
Базовое значение характеристики.
- **`effective`** (`number`)
Эффективное (текущее) значение характеристики.
- **`talents`** (`number`)
Величина изменения от талантов.
- **`equipment`** (`number`)
Величина изменения от надетых на персонажа предметов.
- **`enchants`** (`number`)
Величина изменения от чар (энчантов), вставленных в надетые на персонажа предметы.
- **`isCursed`** (`boolean`)
Флаг, указывающий на влияние проклятого предмета на характеристику.
- **`buffs`** (`number`)
Величина изменения от активных эффектов (баффов).
- **`fairyBonus`** (`number`)
Бонус от покровителя.
- **`type`** (`number`)
Класс характеристики, соответствует значениям из `ENUM_SpecialStatType`.
- **`description`** (`ValuedText` | `nil`)
Описание с подставленными текущими значениями параметров.

## Примеры
### Получение и вывод списка специальных характеристик
```lua
LogInfo( "Avatar special stats:" )
local stats = avatar.GetSpecialStats()
for index, stat in ipairs( stats ) do
    LogInfo( index, ". ", stat.name, " ", stat.description )
end
```
