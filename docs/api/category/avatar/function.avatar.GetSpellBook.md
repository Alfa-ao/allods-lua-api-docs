# avatar.GetSpellBook
Возвращает список идентификаторов активных умений и заклинаний аватара.

## Описание
```lua
avatar.GetSpellBook(): table
```
Функция извлекает и возвращает таблицу, содержащую уникальные идентификаторы (`SpellId`) всех активных умений и заклинаний, доступных аватару. Пассивные умения в данный список не включаются; для их получения применяется функция `avatar.GetAbilities()`. Индексация элементов в возвращаемой таблице начинается с единицы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую идентификаторы (`SpellId`) умений и заклинаний. Индексация элементов в таблице начинается с `1`.

## Примеры
### Получение и вывод названий всех заклинаний аватара
```lua
local spellbook = avatar.GetSpellBook()
for i, id in ipairs( spellbook ) do
    local spellInfo = spellLib.GetDescription( id )
    common.LogInfo( tostring( spellInfo.name ) )
end
```
::: info Описание примера
В примере извлекается список идентификаторов заклинаний аватара с помощью `avatar.GetSpellBook()`. Затем в цикле `for` происходит перебор полученной таблицы. Для каждого идентификатора вызывается `spellLib.GetDescription()` для получения подробной информации о заклинании, после чего название заклинания выводится в информационный лог через `common.LogInfo()`.
:::
