# avatar.GetSpentTalentPoints
Извлекает количество вложенных очков талантов.

## Описание
```lua
avatar.GetSpentTalentPoints(): number
```
Функция возвращает количество очков талантов, которые были вложены главным персонажем.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — целое число, представляющее количество вложенных очков талантов.

## Примеры
### Получение количества вложенных очков талантов
```lua
local spentTalentPoints = avatar.GetSpentTalentPoints()
```
