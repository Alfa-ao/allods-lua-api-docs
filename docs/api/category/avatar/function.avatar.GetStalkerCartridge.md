# avatar.GetStalkerCartridge
Возвращает список стрел в колчане сталкера.

## Описание
```lua
avatar.GetStalkerCartridge(): table
```
Функция возвращает таблицу, содержащую информацию о типах стрел, находящихся в патронташе персонажа-сталкера. Каждый элемент таблицы описывает конкретный тип стрел, включая идентификаторы заклинаний для зарядки и выстрела, а также текущее и максимальное количество стрел данного типа.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), индексация элементов в которой начинается с нуля. Каждый элемент таблицы представляет собой структуру со следующими полями:
- **`enchantSpellId`** (`SpellId`)
Идентификатор ресурса заклинания, заряжающего стрелу данного типа.
- **`shotSpellId`** (`SpellId`)
Идентификатор ресурса заклинания, выстреливающего стрелу данного типа.
- **`arrowCount`** (`number`)
Количество уже заряженных стрел данного типа.
- **`maxArrowCount`** (`number`)
Максимальное количество заряженных стрел данного типа.

## Примеры
### Получение и перебор списка стрел в колчане
```lua
local arrows = avatar.GetStalkerCartridge()
for i = 0, GetTableSize( arrows ) - 1 do
    local arrow = arrows[ i ]
    local spellInfo = avatar.GetSpellInfo( arrow.enchantSpellId )
    if spellInfo then
        LogInfo( userMods.FromWString( spellInfo.name ), ": ", arrow.arrowCount )
    end
end
```
::: info Описание примера
В примере выполняется извлечение таблицы стрел из колчана. Затем в цикле перебирается каждый элемент таблицы. Для каждой стрелы запрашивается информация о заклинании зарядки с помощью `avatar.GetSpellInfo`. При успешном получении информации в лог выводится название заклинания и текущее количество стрел данного типа.
:::
