# avatar.GetStalkerDamagePoolInfo
Возвращает информацию о текущих значениях активных барьеров лучника.

## Описание
```lua
avatar.GetStalkerDamagePoolInfo(): table | nil
```
Функция возвращает список таблиц с информацией о текущих значениях барьеров лучника, если они присутствуют и активны. В случае отсутствия барьеров возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — список таблиц, индексированный с единицы, или `nil`, если барьеры отсутствуют.

Структура возвращаемой таблицы:
- **`damage`** (`number`)
Текущее значение урона в i-том барьере.
- **`resolveTime`** (`number`)
Время в миллисекундах, оставшееся до исчезновения i-го барьера.
- **`internalId`** (`number`)
Внутренний уникальный идентификатор i-го барьера.

## Примеры
### Получение и вывод информации о барьерах
```lua
local stalkerPools = avatar.GetStalkerDamagePoolInfo()
if stalkerPools then
    for i = 1, GetTableSize( stalkerPools ) do
        local pool = stalkerPools[ i ]
        common.LogInfo( "Барьер ID: ", pool.internalId, " Урон: ", pool.damage, " Время: ", pool.resolveTime )
    end
end
```
::: info Описание примера
В примере запрашивается информация о барьерах лучника. Если данные успешно получены (таблица не равна `nil`), происходит перебор всех элементов списка. Для каждого барьера в информационный лог выводятся его уникальный идентификатор, текущее значение урона и оставшееся время до исчезновения.
:::
