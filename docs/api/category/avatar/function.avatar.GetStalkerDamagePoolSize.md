# avatar.GetStalkerDamagePoolSize
Возвращает размер (максимальное время жизни) барьеров лучника.

## Описание
```lua
avatar.GetStalkerDamagePoolSize(): number
```
Функция возвращает целое число, представляющее лимит времени жизни урона в барьере лучника, измеряемый в миллисекундах.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — лимит времени жизни урона в барьере в миллисекундах.

## Примеры
### Вывод размера пула урона барьера в лог
```lua
common.LogInfo( avatar.GetStalkerDamagePoolSize() )
```
