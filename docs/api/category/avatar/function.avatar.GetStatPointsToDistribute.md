# avatar.GetStatPointsToDistribute
Возвращает информацию о пунктах характеристик, намеченных для распределения.

## Описание
```lua
avatar.GetStatPointsToDistribute(): table | nil
```
Функция возвращает таблицу с ключами, соответствующими значениям перечисления `ENUM_InnateStats`, и целочисленными значениями, равными количеству пунктов, намеченных для добавления к соответствующим характеристикам. Если информация отсутствует, возвращается `nil`. Получение данных возможно только при условии, что функция `avatar.CanImproveInnateStats()` возвращает `true`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если информация отсутствует, возвращается `nil`. В противном случае возвращается таблица, где ключи соответствуют значениям перечисления `ENUM_InnateStats`, а значения (`integer`) равны количеству пунктов, намеченных для добавления к соответствующим характеристикам.

## Примеры
### Получение пунктов для распределения с предварительной проверкой
```lua
if avatar.CanImproveInnateStats() then
    local points = avatar.GetStatPointsToDistribute()
    if points then
        -- Обработка таблицы с пунктами характеристик
    end
end
```
::: info Описание примера
В примере сначала проверяется возможность улучшения врожденных характеристик с помощью `avatar.CanImproveInnateStats()`. Если проверка успешна, вызывается `avatar.GetStatPointsToDistribute()` для извлечения таблицы с намеченными пунктами. При успешном получении данных предполагается их дальнейшая обработка.
:::
