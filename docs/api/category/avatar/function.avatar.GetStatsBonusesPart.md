# avatar.GetStatsBonusesPart
Возвращает информацию о бонусах к характеристикам аватара.

## Описание
```lua
avatar.GetStatsBonusesPart(): table | nil
```
Функция возвращает таблицу с детализированной информацией о бонусах к характеристикам аватара. Если информация отсутствует, возвращается `nil`. При изменении данных о бонусе от покровителя (`fairyBonus`) передается событие `EVENT_FAIRY_BONUS_CHANGED`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`. Значение `nil` возвращается, если информация отсутствует.
Иначе возвращается таблица со следующими полями:
- **`isBonusesEmpty`** (`boolean`) — не используется.
- **`fairyBonus`** (`table` | `nil`) — информация о бонусе от покровителя. Таблица содержит следующие поля:
  - `statPointsCount` (`number`) — величина бонуса.
  - `currentStat` (`number`) — индекс характеристики из таблицы `allowedStats`, к которой предоставляется бонус.
  - `stats` (`table`) — список характеристик, индексированный от `1`. Каждый элемент представляет собой таблицу со следующими полями:
    - `id` (`SpecialStatId`) — идентификатор характеристики.
    - `isAllowed` (`boolean`) — `true`, если характеристика доступна для выбора.
    - `name` (`WString`) — название характеристики.
    - `tooltipName` (`WString`) — название характеристики для всплывающей подсказки.
    - `sysName` (`string`) — служебное название характеристики.
    - `base` (`number`) — базовое значение характеристики.
    - `effective` (`number`) — эффективное (текущее) значение характеристики.
    - `talents` (`number`) — величина изменения от талантов.
    - `equipment` (`number`) — величина изменения от надетых на персонажа предметов.
    - `enchants` (`number`) — величина изменения от чар (энчантов), вставленных в надетые на персонажа предметы.
    - `isCursed` (`boolean`) — `true`, если на характеристику влияет проклятый предмет.
    - `buffs` (`number`) — величина изменения от активных бафов.
    - `fairyBonus` (`number`) — бонус от покровителя.
    - `description` (`ValuedText` | `nil`) — описание с подставленными текущими значениями параметров.

## Примеры
### Получение величины бонуса от покровителя
```lua
local bonuses = avatar.GetStatsBonusesPart()
if bonuses then
  local fairyBonusValue = bonuses.fairyBonus.statPointsCount
end
```
::: info Описание примера
В примере запрашивается информация о бонусах к характеристикам аватара. Если данные успешно получены, извлекается величина бонуса от покровителя (`statPointsCount`) из вложенной таблицы `fairyBonus`.
:::
