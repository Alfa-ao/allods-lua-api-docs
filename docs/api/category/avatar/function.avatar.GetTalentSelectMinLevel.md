# avatar.GetTalentSelectMinLevel
Возвращает минимальный уровень аватара, начиная с которого доступно распределение очков таланта.

## Описание
```lua
avatar.GetTalentSelectMinLevel(): number
```
Функция возвращает минимальный уровень аватара, начиная с которого доступно распределение очков таланта. Полученное значение позволяет определить, достиг ли персонаж необходимого порога для открытия веток талантов и использования соответствующих игровых механик.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — минимальный уровень, начиная с которого доступно распределение очков таланта.

## Примеры
### Получение минимального уровня для распределения талантов
```lua
local talentSelectMinLevel = avatar.GetTalentSelectMinLevel()
```
