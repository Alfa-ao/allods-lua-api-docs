# avatar.GetTarget
Возвращает идентификатор текущей цели аватара, включая дополнительную цель.

## Описание
```lua
avatar.GetTarget(): ObjectId | nil
```
Функция возвращает уникальный идентификатор цели аватара. Если для аватара установлена дополнительная цель, извлекается её идентификатор. В случае отсутствия цели возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`ObjectId` | `nil` — уникальный идентификатор цели аватара или `nil`, если цель отсутствует.

## Примеры
### Получение идентификатора текущей цели
```lua
local unitId = avatar.GetTarget()
```
