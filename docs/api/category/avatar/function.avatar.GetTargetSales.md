# avatar.GetTargetSales
Возвращает информацию о доступных в данный момент таргетных продажах.

## Описание

```lua
avatar.GetTargetSales(): table | nil
```

Функция запрашивает и формирует список актуальных таргетных предложений. Если на момент вызова активные продажи отсутствуют, возвращается `nil`. При наличии предложений возвращается таблица, индексация элементов которой начинается с `1`. Каждый элемент таблицы содержит детализированную информацию о конкретном предложении, включая идентификаторы предметов, текстовые сообщения, временные метки и параметры количества.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если активные предложения отсутствуют, возвращается `nil`. В противном случае возвращается таблица, индексированная с `1`, содержащая таблицы предложений со следующими полями:
- **`id`** (`ObjectId`) — уникальный идентификатор предложения.
- **`item`** (`ItemId` | `nil`) — идентификатор предлагаемого предмета.
- **`alternativeItem`** (`ItemId` | `nil`) — идентификатор идентичного предмета из лавки редкостей.
- **`text`** (`WString`) — локализованное текстовое сообщение, сопровождающее предложение.
- **`tag`** (`string`) — тип предложения, соответствующий значениям перечисления `ENUM_SaleTag`.
- **`endTime`** (`LuaFullDateTime`) — дата и время окончания действия предложения.
- **`amount`** (`table` | `nil`) — информация о предлагаемом количестве предметов. Если значения диапазона меньше `1`, пользователь самостоятельно выбирает необходимое количество. Содержит следующие поля:
  - **`min`** (`number`) — минимальное количество предметов.
  - **`max`** (`number`) — максимальное количество предметов.

## Примеры

### Получение списка доступных таргетных продаж

```lua
local availableOffers = avatar.GetTargetSales()

if availableOffers then
    for i = 1, #availableOffers do
        local offer = availableOffers[i]
        local offerText = userMods.FromWString(offer.text)
        LogInfo("Предложение " .. i .. ": " .. offerText)
    end
end
```

::: info Описание примера
В примере извлекается список доступных таргетных предложений. Если список не пуст (не равен `nil`), выполняется цикл перебора всех элементов. Для каждого предложения текст преобразуется из `WString` в обычную строку и выводится в информационный лог.
:::
