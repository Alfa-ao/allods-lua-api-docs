# avatar.GetTeleportLocationInfo
Возвращает информацию о точке телепортации.

## Описание
```lua
avatar.GetTeleportLocationInfo( locationId: ObjectId | TeleportMasterId ): table | nil
```
Функция извлекает и возвращает детальную информацию о точке телепортации. Получение информации доступно для имеющихся точек телепортации главного игрока, а также для точек телепортации мастера телепорта, с которым ведет диалог главный игрок, включая точку местоположения самого мастера.

## Список параметров
- **`locationId`** (`ObjectId` | `TeleportMasterId`)
Идентификатор точки телепорта или идентификатор мастера телепорта.

## Возвращаемые значения
Возвращает таблицу (`table`) с информацией о точке телепортации или `nil`, если точка не найдена.
Структура возвращаемой таблицы:
- **`id`** (`ObjectId`) — уникальный идентификатор точки.
- **`name`** (`WString`) — название точки.
- **`isHighPriority`** (`boolean`) — флаг, указывающий, что данная точка должна отображаться перед остальными точками телепортации.
- **`minLevel`** (`number`) — минимальный комфортный уровень игры. Возвращает `0`, если карта не прописана.
- **`maxLevel`** (`number`) — максимальный комфортный уровень игры. Возвращает `0`, если карта не прописана.

## Примеры
### Получение названия точки телепортации
```lua
local info = avatar.GetTeleportLocationInfo( locationId )
if info then
    local name = info.name
end
```
