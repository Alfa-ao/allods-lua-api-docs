# avatar.GetTeleportLocations
Возвращает список известных главному персонажу точек телепортации от мастеров телепортов.

## Описание
```lua
avatar.GetTeleportLocations(): table
```
Функция возвращает индексированный с единицы список идентификаторов точек телепортации, открытых главному персонажу.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированный с единицы список уникальных идентификаторов (`ObjectId`) точек телепортации.

## Примеры
### Получение информации о первой доступной точке телепортации
```lua
local list = avatar.GetTeleportLocations()
if list[1] then
  local locationInfo = avatar.GetTeleportLocationInfo( list[1] )
  if locationInfo then
    local name = locationInfo.name
  end
end
```
::: info Описание примера
В примере запрашивается полный список известных точек телепортации. Выполняется проверка наличия первой точки в списке (так как индексация начинается с 1). При наличии элемента извлекается детальная информация о точке телепортации посредством `avatar.GetTeleportLocationInfo`, после чего считывается её название.
:::
