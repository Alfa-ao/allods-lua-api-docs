# avatar.GetTemporaryUnlockCategories
Возвращает список категорий временных возможностей, имеющихся у аватара.

## Описание
```lua
avatar.GetTemporaryUnlockCategories(): table
```
Функция извлекает и возвращает таблицу с идентификаторами категорий временных возможностей, которые в данный момент активны для аватара. Если у аватара отсутствуют временные возможности, возвращается пустая таблица.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), индексированную с единицы, содержащую идентификаторы (`UnlockCategoryId`) временных возможностей аватара. Если временные возможности отсутствуют, возвращается пустая таблица.

## Примеры
### Перебор и вывод названий активных временных возможностей
```lua
for _, categoryId in ipairs( avatar.GetTemporaryUnlockCategories() ) do
    local info = categoryId:GetInfo()
    LogInfo( info.name )
end
```
::: info Описание примера
В примере выполняется получение списка идентификаторов категорий временных возможностей. С помощью цикла `for` происходит перебор полученной таблицы. Для каждого идентификатора вызывается метод `GetInfo()`, извлекающий подробную информацию о категории, после чего в лог выводится её название.
:::
