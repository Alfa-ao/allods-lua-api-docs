# avatar.GetTransportList
Извлекает список идентификаторов всех транспортов, находящихся в зоне действия аватара.

## Описание
```lua
avatar.GetTransportList(): table
```
Функция возвращает таблицу (`table`), содержащую уникальные идентификаторы (`TransportId`) всех транспортных средств, которые в данный момент видимы и находятся в зоне действия аватара. Индексация элементов в возвращаемой таблице начинается с единицы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — список идентификаторов (`TransportId`) видимых транспортов. Индексация элементов начинается с `1`.

## Примеры
### Получение и перебор списка видимых транспортов
```lua
local transports = avatar.GetTransportList()
for _key, transportId in ipairs( transports ) do
    common.LogInfo( "id: " .. tostring( transportId ) )
end
```
