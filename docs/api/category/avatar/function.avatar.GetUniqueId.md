# avatar.GetUniqueId
Возвращает уникальный постоянный (кросс-серверный) идентификатор аватара.

## Описание
```lua
avatar.GetUniqueId(): UniqueId
```
Функция возвращает уникальный постоянный (кросс-серверный) идентификатор аватара. Данный идентификатор не привязан к конкретному серверу и сохраняется за персонажем на постоянной основе, оставаясь неизменным при переходе между игровыми мирами.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `UniqueId` — уникальный постоянный (кросс-серверный) идентификатор аватара. Возвращаемое значение гарантированно не равно `nil`.

## Примеры
### Получение уникального идентификатора аватара
```lua
local id = avatar.GetUniqueId()
```
