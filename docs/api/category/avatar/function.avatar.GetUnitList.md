# avatar.GetUnitList
Возвращает список всех видимых юнитов в зоне действия аватара, исключая сам аватар.

## Описание
```lua
avatar.GetUnitList(): table
```
Функция возвращает индексированный с единицы список идентификаторов всех видимых юнитов в зоне действия аватара. Сам аватар в данный список не включается.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированный с единицы список идентификаторов (`ObjectId`) видимых юнитов.

## Примеры
### Получение и перебор списка видимых юнитов
```lua
local units = avatar.GetUnitList()
for _, value in ipairs( units ) do
  common.LogInfo( "id: " .. tostring( value ) )
end
```
