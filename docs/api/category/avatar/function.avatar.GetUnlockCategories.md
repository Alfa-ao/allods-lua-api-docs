# avatar.GetUnlockCategories
Возвращает список категорий возможностей (анлоков), имеющихся у аватара.

## Описание
```lua
avatar.GetUnlockCategories(): table
```
Функция возвращает таблицу, содержащую идентификаторы категорий возможностей (анлоков), имеющихся у аватара. Таблица индексируется с единицы. Если у аватара отсутствуют какие-либо возможности, возвращается пустая таблица.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую идентификаторы (`UnlockCategoryId`) категорий возможностей аватара. Индексация элементов начинается с `1`. Если возможности отсутствуют, возвращается пустая таблица.

## Примеры
### Перебор и вывод названий доступных категорий
```lua
for _, categoryId in ipairs( avatar.GetUnlockCategories() ) do
    local info = categoryId:GetInfo()
    LogInfo( info.name )
end
```
