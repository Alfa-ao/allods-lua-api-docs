# avatar.GetUnlockInfo
Возвращает описание возможности (анлока), имеющейся у аватара.

## Описание
```lua
avatar.GetUnlockInfo( unlockId: ObjectId | UnlockId ): table | nil
```
Функция извлекает и возвращает детальную информацию о возможности (анлоке) аватара по заданному идентификатору. Основная реализация логики находится в системе UIState/GameState, расширенная реализация — в интерфейсе (UI). При возникновении ошибки (некорректный идентификатор, аватар недоступен и т.п.) возвращается `nil`.

## Список параметров
- **`unlockId`** (`ObjectId` | `UnlockId`)
Идентификатор возможности (анлока).

## Возвращаемые значения
Возвращает таблицу (`table`) с описанием возможности или `nil` в случае ошибки.

Структура возвращаемой таблицы:
- **`id`** (`ObjectId` | `nil`) — идентификатор возможности.
- **`unlockId`** (`UnlockId` | `nil`) — идентификатор ресурса возможности.
- **`name`** (`WString`) — название возможности.
- **`description`** (`WString`) — описание действий для получения возможности.
- **`sourceDescription`** (`GlossaryId` | `nil`) — идентификатор ресурса универсального описателя для источника анлока (при наличии).
- **`sysName`** (`string`) — внутреннее имя анлока, соответствующее полю "sysUIScriptName" в базе данных (может использоваться для специальной обработки).
- **`sysDebugName`** (`string` | `nil`) — внутреннее дебажное имя ресурса анлока.
- **`image`** (`TextureId`) — идентификатор текстуры для иконки возможности (реализовано только для UI).
- **`isBoundToRemort`** (`boolean`) — `true`, если возможность будет доступна у Перерождения (реморта).
- **`endTime`** (`LuaFullDateTime` | `nil`) — дата окончания действия анлока, если анлок временный, иначе `nil`.
- **`category`** (`table` | `nil`) — таблица с описанием категории, если возможность ей принадлежит.
  - **`name`** (`WString`) — название категории.
  - **`sysName`** (`string`) — системное название категории для реализации уникальных свойств.
  - **`categoryId`** (`UnlockCategoryId`) — идентификатор категории анлока.

## Примеры
### Получение и вывод названий всех доступных возможностей
```lua
local unlocks = avatar.GetUnlocks()
for _, unlockId in ipairs( unlocks ) do
    local info = avatar.GetUnlockInfo( unlockId )
    if info then
        LogInfo( info.name )
    end
end
```
::: info Описание примера
В примере извлекается список всех доступных возможностей аватара с помощью функции `avatar.GetUnlocks()`. Затем в цикле перебирается каждый идентификатор. Для каждого идентификатора вызывается функция `avatar.GetUnlockInfo()`. При успешном получении информации в лог выводится название возможности.
:::
