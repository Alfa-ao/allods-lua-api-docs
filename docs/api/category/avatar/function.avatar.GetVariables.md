# avatar.GetVariables
Возвращает список идентификаторов гейммеханических переменных главного персонажа.
## Описание
```lua
avatar.GetVariables(): table
```
Функция возвращает индексированную с нуля таблицу с идентификаторами (`VariableId`) ресурсов имеющихся гейммеханических переменных игрока, таких как количество Сил Льда у мага. Перечень доступных для просмотра переменных задается в файле `\Client\GMUIHelper\VariablesAliases.(VariablesAliases).xdb`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированную с нуля таблицу, содержащую идентификаторы (`VariableId`) ресурсов имеющихся переменных.

## Примеры
### Получение значения первой доступной переменной
```lua
local ids = avatar.GetVariables()
if ids and ids[0] then
    local info = avatar.GetVariableInfo( ids[0] )
    if info then
        local currentValue = info.value
    end
end
```
::: info Описание примера
В примере запрашивается список идентификаторов всех доступных гейммеханических переменных. Если таблица не пустая и содержит хотя бы один элемент, извлекается подробная информация о первой переменной с помощью функции `avatar.GetVariableInfo`. При успешном получении информации считывается ее текущее значение.
:::
