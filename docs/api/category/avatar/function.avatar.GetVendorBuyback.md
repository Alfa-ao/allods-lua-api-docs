# avatar.GetVendorBuyback
Возвращает список идентификаторов предметов, проданных NPC-торговцу, для их последующего выкупа.
## Описание
```lua
avatar.GetVendorBuyback(): table
```
Функция формирует и возвращает таблицу с описаниями объектов, которые были проданы NPC-торговцу и доступны для обратного выкупа.

::: warning Замечание
Функция возвращает актуальные данные только после успешного выполнения запроса через функцию `avatar.RequestVendor()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую описания доступных для выкупа предметов:
- **`id`** (`ObjectId`) — уникальный идентификатор объекта.
- **`quantity`** (`number`) — количество предметов, имеющихся у торговца. Если передано значение `-1`, количество не ограничено.
- **`price`** (`number`) — цена одного предмета.
- **`counter`** (`number`) — значение предметного счётчика (заряды). Если передано значение `-1`, счётчик отсутствует.
- **`isBound`** (`boolean`) — флаг, указывающий на привязку предмета к персонажу.

## Примеры
### Получение и обработка списка предметов на выкуп
```lua
local list = avatar.GetVendorBuyback()
for i = 0, GetTableSize( list ) - 1 do
    local itemId = list[i].id
    local object = avatar.GetItemInfo( itemId )
    local name = object.name
end
```
::: info Описание примера
В примере сначала извлекается таблица с описаниями предметов, доступных для выкупа, с помощью функции `avatar.GetVendorBuyback()`. Затем в цикле перебирается каждый элемент таблицы. Для каждого идентификатора предмета вызывается функция `avatar.GetItemInfo()` для получения полной информации об объекте, после чего извлекается его название.
:::
