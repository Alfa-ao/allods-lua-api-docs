# avatar.GetVendorList
Возвращает список идентификаторов предметов, доступных для продажи у NPC-торговца, исключая товары вторичного рынка.

## Описание
```lua
avatar.GetVendorList(): table
```
Функция возвращает таблицу (`table`), содержащую описания объектов (`VendorItem`), доступных для продажи у NPC-торговца, исключая товары вторичного рынка. Извлекаемые данные включают идентификаторы предметов, их количество, цену, требуемые разблокировки и достижения, а также информацию об альтернативных валютах.

::: warning Замечание
Функция возвращает данные, полученные после вызова `avatar.RequestVendor()`. Если запрос не был выполнен или данные еще не обновлены, список может быть пустым или содержать устаревшую информацию.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую элементы типа `VendorItem`:
- **`id`** (`ObjectId`) — уникальный идентификатор объекта.
- **`quantity`** (`number`) — количество предметов, имеющихся у торговца. Если значение равно `-1`, количество не ограничено.
- **`counter`** (`number`) — значение счетчика предмета (заряды). Если значение равно `-1`, счетчик отсутствует.
- **`isBound`** (`boolean`) — флаг, указывающий на привязку предмета к персонажу.
- **`isSecondhand`** (`boolean`) — флаг товара с вторичного рынка. В данном списке всегда принимает значение `false`.
- **`price`** (`number`) — цена одного предмета в основной внутриигровой валюте.
- **`requiredUnlocks`** (`table` | `nil`) — таблица требуемых разблокировок (`UnlockId`) для покупки. Индексация начинается с `1`. Если разблокировки не требуются, возвращает `nil`.
- **`requiredAchievements`** (`table` | `nil`) — таблица требуемых очков событий для покупки. Индексация начинается с `1`. Каждый элемент представляет собой таблицу с полями:
  - `ruleId` (`RuleId`) — идентификатор события.
  - `achievement` (`number`) — требуемое количество очков события.
  Если достижения не требуются, возвращает `nil`.
- **`alternativePrices`** (`table` | `nil`) — список цен в альтернативных валютах. Индексация начинается с `0`. Каждый элемент представляет собой таблицу с полями:
  - `currencyId` (`CurrencyId`) — идентификатор альтернативной валюты.
  - `value` (`number`) — цена в данной альтернативной валюте.
  Если информация об альтернативных валютах отсутствует, возвращает `nil`.

## Примеры
### Получение и перебор списка товаров торговца
```lua
local list = avatar.GetVendorList()
for i = 0, GetTableSize( list ) - 1 do
    local itemId = list[ i ].id
    local object = avatar.GetItemInfo( itemId )
    local name = object.name
    -- Дальнейшая обработка информации о предмете
end
```
::: info Описание примера
В примере извлекается список товаров текущего торговца с помощью `avatar.GetVendorList()`. Затем в цикле перебирается каждый элемент списка. Для каждого идентификатора предмета вызывается `avatar.GetItemInfo()` для получения дополнительной информации, такой как название предмета.
:::
