# avatar.GetViewedBuild
Возвращает номер специализации, данные о которой доступны для просмотра.

## Описание
```lua
avatar.GetViewedBuild(): number | nil
```
Функция возвращает номер показываемой специализации аватара, данные о которой доступны для просмотра. Нумерация специализаций начинается с нуля. В случае возникновения ошибки возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — номер показываемой специализации аватара (нумерация начинается с 0).
Возвращает `nil`, если произошла ошибка.

## Примеры
### Вывод номера специализации в лог
```lua
LogInfo( avatar.GetViewedBuild() )
```
