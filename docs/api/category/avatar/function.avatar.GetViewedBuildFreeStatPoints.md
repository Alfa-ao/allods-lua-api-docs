# avatar.GetViewedBuildFreeStatPoints
Возвращает количество пунктов характеристик, доступных для распределения в просматриваемой специализации.

## Описание
```lua
avatar.GetViewedBuildFreeStatPoints(): number
```
Функция возвращает количество пунктов характеристик, доступных для распределения в просматриваемой специализации. Текущая просматриваемая специализация задается с помощью функции `avatar.SetViewedBuild`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — количество пунктов характеристик, доступных для распределения в просматриваемой специализации.

## Примеры
### Получение количества свободных пунктов характеристик
```lua
local stats = avatar.GetViewedBuildFreeStatPoints()
```
