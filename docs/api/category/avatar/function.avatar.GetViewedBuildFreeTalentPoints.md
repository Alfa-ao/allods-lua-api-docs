# avatar.GetViewedBuildFreeTalentPoints
Возвращает количество свободных очков умений, доступных для распределения в просматриваемой специализации.

## Описание
```lua
avatar.GetViewedBuildFreeTalentPoints(): number
```
Функция возвращает количество свободных очков умений, доступных для распределения в просматриваемой специализации. Просматриваемая специализация предварительно задается с помощью функции `avatar.SetViewedBuild`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — количество очков умений, доступных для распределения в просматриваемой специализации.

## Примеры
### Получение количества свободных очков умений
```lua
local talents = avatar.GetViewedBuildFreeTalentPoints()
```
