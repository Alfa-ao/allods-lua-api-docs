# avatar.GetWarriorCombatAdvantage
Возвращает величину боевого духа воина.

## Описание
```lua
avatar.GetWarriorCombatAdvantage(): number
```
Функция возвращает числовое значение, представляющее собой текущую величину боевого духа для класса персонажа «воин», извлеченное из внутреннего состояния аватара.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`number` — целое число, обозначающее текущую величину боевого духа воина.

## Примеры
### Получение величины боевого духа
```lua
local combatAdvantage = avatar.GetWarriorCombatAdvantage()
```
