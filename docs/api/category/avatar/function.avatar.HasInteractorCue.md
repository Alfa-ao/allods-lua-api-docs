# avatar.HasInteractorCue
Проверяет наличие реплики для разговора у текущего собеседника.

## Описание
```lua
avatar.HasInteractorCue(): boolean
```
Функция проверяет наличие активной реплики у текущего собеседника. Возвращает `true`, если у собеседника имеется реплика для разговора, например, для выдачи или принятия активных квестов. Дополнительную информацию о самой реплике предоставляет функция `avatar.GetInteractorCue()`.

::: warning Замечание
Функцию необходимо вызывать после того, как придет событие `EVENT_INTERACTION_STARTED`. В процессе разговора запрещается менять цель взаимодействия.
:::

::: tip Совет
Для получения подробной информации о реплике рекомендуется использовать функцию `avatar.GetInteractorCue()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если собеседник имеет активную реплику для разговора (например, для выдачи или принятия квестов), и `false` в противном случае.

## Примеры
### Проверка наличия реплики у собеседника
```lua
local hasCue = avatar.HasInteractorCue()
```
