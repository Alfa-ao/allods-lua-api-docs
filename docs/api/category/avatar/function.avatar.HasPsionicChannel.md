# avatar.HasPsionicChannel
Определяет наличие доступа главного персонажа к классовому каналу чата псиоников.

## Описание
```lua
avatar.HasPsionicChannel(): boolean
```
Функция проверяет, обладает ли главный персонаж доступом к классовому каналу чата псиоников. Псионикам разрешено отправлять сообщения и просматривать данный канал. Метод возвращает логическое значение, указывающее на наличие или отсутствие соответствующих прав доступа.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если главный персонаж имеет доступ к классовому каналу чата псиоников, и `false` в противном случае.

## Примеры
### Проверка доступа к каналу чата
```lua
local canViewPsionicChannel = avatar.HasPsionicChannel()
```
