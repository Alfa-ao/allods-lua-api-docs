# avatar.HasSpell
Проверяет наличие заклинания в книге заклинаний аватара.
## Описание
```lua
avatar.HasSpell( spellId: SpellId ): boolean
```
Функция проверяет наличие заклинания в книге заклинаний аватара по заданному идентификатору ресурса. Если заклинание является специальным и не выдается в книгу заклинаний, функция возвращает `false`. Основное применение функции заключается в проверке наличия игрушек в гардеробе персонажа.
## Список параметров
- **`spellId`** (`SpellId`)
Идентификатор ресурса заклинания, наличие которого необходимо проверить.
## Возвращаемые значения
Возвращает `boolean` — `true`, если заклинание присутствует в книге заклинаний аватара, и `false` в противном случае.
## Примеры
### Проверка наличия игрушки в гардеробе
```lua
local toySpellId = 12345 -- Идентификатор заклинания игрушки
if avatar.HasSpell( toySpellId ) then
    LogInfo( "Игрушка доступна в гардеробе." )
else
    LogInfo( "Игрушка отсутствует в гардеробе." )
end
```
