# avatar.ImproveInnateStat
Отводит пункты характеристик для будущего распределения.

## Описание
```lua
avatar.ImproveInnateStat( stat: number, amount: number )
```
Функция отводит указанное количество пунктов характеристик для их последующего распределения. Операция выполняется успешно только при условии, что функция `avatar.CanImproveInnateStats()` возвращает `true`. Отрицательное значение параметра `amount` уменьшает ранее запланированное количество пунктов для данной характеристики, но не позволяет опустить общее запланированное значение ниже нуля.

## Список параметров
- **`stat`** (`number`)
Значение из перечисления `ENUM_InnateStats`, соответствующее характеристике, которую планируется изменить.
- **`amount`** (`number`)
Количество пунктов характеристик, на которое планируется выполнить изменение. Значение не может превышать текущее количество свободных пунктов. Допускается передача отрицательного значения: в этом случае уменьшается ранее заданное этой же функцией количество пунктов для указанной характеристики, однако общее запланированное количество не может быть уменьшено ниже нуля.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Отвод пунктов для характеристики с предварительной проверкой
```lua
local stat = ENUM_InnateStats.STRENGTH
local amount = 5

if avatar.CanImproveInnateStats() then
    avatar.ImproveInnateStat( stat, amount )
end
```
::: info Описание примера
В примере сначала определяются целевая характеристика и количество пунктов. Затем проверяется возможность отвода пунктов с помощью `avatar.CanImproveInnateStats()`. Если проверка успешна, вызывается `avatar.ImproveInnateStat` для отвода 5 пунктов в выбранную характеристику.
:::
