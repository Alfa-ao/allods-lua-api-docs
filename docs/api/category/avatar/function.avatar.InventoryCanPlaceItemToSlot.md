# avatar.InventoryCanPlaceItemToSlot
Проверяет возможность размещения предмета в указанный слот инвентаря по его типу.

## Описание
```lua
avatar.InventoryCanPlaceItemToSlot( itemId: ObjectId, slot: number ): boolean
```
Функция проверяет, соответствует ли тип предмета с заданным идентификатором требованиям указанного слота инвентаря. Инвентарь состоит из базовой сумки и дополнительных карманов, в которые можно помещать только специфические типы предметов (такие предметы не размещаются в базовой сумке). Слоты индексируются начиная с `0` по всему объему инвентаря, включая карманы. Функция возвращает логическое значение, указывающее на допустимость размещения предмета в слоте, при этом не проверяя фактическую занятость слота или другие условия.

::: warning Замечание
Метод не проверяет, свободен ли указанный слот, а лишь валидирует соответствие типа предмета типу слота.
:::

## Список параметров
- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета, который планируется разместить.
- **`slot`** (`number`)
Индекс слота инвентаря (включая базовую сумку и карманы), начиная с `0`.

## Возвращаемые значения
`boolean` — `true`, если предмет подходит по типу для указанного слота, и `false` в противном случае.

## Примеры
### Проверка возможности размещения предмета в слот
```lua
local itemId = 12345 -- Идентификатор предмета
local slot = 5       -- Индекс целевого слота инвентаря

local canPlace = avatar.InventoryCanPlaceItemToSlot( itemId, slot )
if canPlace then
    -- Предмет подходит для данного слота по типу
else
    -- Предмет не предназначен для этого слота
end
```
