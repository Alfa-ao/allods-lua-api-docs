# avatar.InventoryGetBaseBagSlotCount
Возвращает базовое количество слотов в основной сумке инвентаря главного персонажа.

## Описание
```lua
avatar.InventoryGetBaseBagSlotCount(): number
```
Функция возвращает базовое число слотов в инвентаре главного персонажа, исключая дополнительные карманы (учитывается только основная сумка).

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — базовое число слотов в инвентаре.

## Примеры
### Получение базового количества слотов инвентаря
```lua
local baseCount = avatar.InventoryGetBaseBagSlotCount()
```
