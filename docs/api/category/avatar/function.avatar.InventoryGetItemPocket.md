# avatar.InventoryGetItemPocket
Возвращает индекс кармана инвентаря, предназначенного для конкретного предмета.

## Описание
```lua
avatar.InventoryGetItemPocket( itemId: ObjectId ): number
```
Функция определяет целевой карман инвентаря для предмета с заданным идентификатором. Инвентарь включает базовую сумку и специализированные карманы, в которые помещаются только определенные типы предметов. Метод возвращает числовой индекс кармана, соответствующий правилам сортировки. Если предмет не имеет привязки к специализированному карману, функция возвращает `-1`, что указывает на возможность помещения предмета в базовую сумку.

## Список параметров
- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета, для которого определяется целевой карман инвентаря.

## Возвращаемые значения
Возвращает `number` — целочисленный индекс кармана, начиная с `0`. Если предмет предназначен для базовой сумки, возвращается `-1`.

## Примеры
### Определение кармана и получение информации о нем
```lua
local itemId = 12345 -- Идентификатор предмета
local pocketIndex = avatar.InventoryGetItemPocket( itemId )

if pocketIndex >= 0 then
    local pocketInfo = avatar.InventoryGetPocketInfo( pocketIndex )
    -- Обработка информации о специализированном кармане
else
    -- Предмет предназначен для базовой сумки
end
```
::: info Описание примера
В примере запрашивается индекс кармана для предмета с идентификатором `12345`. Если возвращаемое значение больше или равно нулю, это означает, что предмет относится к специализированному карману, и извлекается дополнительная информация о нем с помощью функции `avatar.InventoryGetPocketInfo`. Если возвращается `-1`, предмет помещается в базовую сумку.
:::
