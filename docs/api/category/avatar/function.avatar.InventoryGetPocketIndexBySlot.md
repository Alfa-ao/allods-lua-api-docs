# avatar.InventoryGetPocketIndexBySlot
Возвращает индекс кармана инвентаря, в котором расположен указанный слот.

## Описание
```lua
avatar.InventoryGetPocketIndexBySlot( slot: number ): number
```
Инвентарь персонажа состоит из базовой сумки и дополнительных карманов, предназначенных для размещения определенных типов предметов. Слоты индексируются начиная с `0` по всему объему инвентаря, включая все карманы. Функция определяет и возвращает индекс кармана для заданного слота. Если указанный слот относится к базовой сумке, функция возвращает `-1`.

## Список параметров
- **`slot`** (`number`)
Индекс слота инвентаря, для которого необходимо определить принадлежность к карману. Индексация слотов начинается с `0`.

## Возвращаемые значения
Возвращает `number` — индекс кармана, начиная с `0`. Если указанный слот находится в базовой сумке, возвращает `-1`.

## Примеры
### Определение кармана для слота инвентаря
```lua
local slotIndex = 15
local pocketIndex = avatar.InventoryGetPocketIndexBySlot( slotIndex )

if pocketIndex == -1 then
    LogInfo( "Слот находится в базовой сумке." )
else
    LogInfo( "Слот находится в кармане с индексом: ", pocketIndex )
end
```
