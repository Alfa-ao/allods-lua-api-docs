# avatar.InventoryGetPocketsCount
Возвращает число карманов в инвентаре главного персонажа.

## Описание
```lua
avatar.InventoryGetPocketsCount(): number
```
Функция возвращает целое число, представляющее количество карманов в инвентаре главного персонажа.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`number` — целое число, обозначающее количество карманов в инвентаре.

## Примеры
### Получение количества карманов в инвентаре
```lua
local pocketsCount = avatar.InventoryGetPocketsCount()
```
