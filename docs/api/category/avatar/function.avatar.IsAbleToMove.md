# avatar.IsAbleToMove
Возвращает статус доступности клиентского управления перемещением аватара.

## Описание
```lua
avatar.IsAbleToMove(): boolean
```
Функция возвращает логическое значение, указывающее на доступность клиентского управления перемещением аватара. Значение `false` означает, что управление заблокировано (например, при срабатывании механики `charge`).

::: warning Замечание
Управление перемещением может быть принудительно заблокировано сервером или игровыми механиками. В период блокировки клиент не может инициировать перемещение аватара.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если клиентское управление перемещением аватара доступно; `false`, если управление заблокировано.

## Примеры
### Проверка доступности управления перемещением
```lua
local canMove = avatar.IsAbleToMove()
```
