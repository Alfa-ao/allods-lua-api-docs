# avatar.IsClientDetectorEnabled
Возвращает состояние детектора интерактивных объектов на клиенте.

## Описание
```lua
avatar.IsClientDetectorEnabled( type: number ): boolean
```
Функция возвращает логическое значение, указывающее на текущее состояние детектора интерактивных объектов на клиенте. К таким объектам относятся детекторы квестодателей, торговцев и других подобных сущностей. Состояние детектора определяет, будет ли клиент отслеживать появление соответствующих объектов в зоне видимости. Для изменения состояния детектора используется функция `avatar.EnableClientDetector`.

## Список параметров
- **`type`** (`number`)
Тип детектора. Задается значением из перечисления `CLIENT_DETECTOR_*` (например, `CLIENT_DETECTOR_QUEST_NPC`).

## Возвращаемые значения
Возвращает `boolean`: `true`, если указанный детектор включен, и `false`, если выключен.

## Примеры
### Проверка состояния детектора квестодателей
```lua
if avatar.IsClientDetectorEnabled( CLIENT_DETECTOR_QUEST_NPC ) then
    LogInfo( "детектор квестодателей включен" )
end
```
