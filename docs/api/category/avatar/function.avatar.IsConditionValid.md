# avatar.IsConditionValid
Проверяет выполнение условия, заданного дизайнером.

## Описание
```lua
avatar.IsConditionValid( alias: string ): boolean
```
Функция проверяет, выполняется ли некоторое условие, заданное дизайнером, по его строковому псевдониму. Результат проверки возвращается в виде логического значения, указывающего на истинность условия.

## Список параметров
- **`alias`** (`string`)
Строковое название (псевдоним) условия, которое необходимо проверить.

## Возвращаемые значения
`boolean` — `true`, если условие выполняется, и `false` в противном случае.

## Примеры
### Проверка выполнения условия
```lua
local isValid = avatar.IsConditionValid( "PRESENT_IN_BAG" )
```
