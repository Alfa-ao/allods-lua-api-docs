# avatar.IsExist
Определяет, существует ли главный игрок в игровом мире.

## Описание
```lua
avatar.IsExist(): boolean
```
Функция проверяет факт существования главного игрока. Возвращает логическое значение, указывающее на текущий статус присутствия персонажа в мире.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если главный игрок уже существует, и `false` в противном случае.

## Примеры
### Получение идентификатора существующего игрока
```lua
if avatar.IsExist() then
    local id = avatar.GetId()
end
```
::: info Описание примера
В примере проверяется существование главного игрока. Если игрок существует, извлекается и сохраняется его уникальный идентификатор с помощью функции `avatar.GetId`.
:::
