# avatar.IsInRespawn
Проверяет, находится ли главный персонаж в состоянии воскрешения.

## Описание
```lua
avatar.IsInRespawn(): boolean
```
Функция возвращает `true`, если главный персонаж находится в состоянии воскрешения, и `false` во всех остальных случаях. Состояние воскрешения начинается после переноса персонажа на кладбище функцией `avatar.Respawn()` и длится до момента возвращения искры к персонажу.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если персонаж находится в состоянии воскрешения, иначе `false`.

## Примеры
### Проверка состояния и призыв искры
```lua
if avatar.IsInRespawn() then
    avatar.CallSpark()
end
```
::: info Описание примера
В примере проверяется, находится ли персонаж в состоянии воскрешения. Если условие выполняется, вызывается функция `avatar.CallSpark()`.
:::
