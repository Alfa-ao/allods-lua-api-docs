# avatar.IsInspectAllowed
Проверяет возможность инспектирования другого игрока.

## Описание
```lua
avatar.IsInspectAllowed(): boolean
```
Функция возвращает логическое значение, определяющее доступность инспектирования игрока. `true` — если инспектирование разрешено, `false` — если операция в данный момент недоступна.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если инспектирование доступно, иначе `false`.

## Примеры
### Инициация инспектирования с предварительной проверкой
```lua
if avatar.IsInspectAllowed() then
    avatar.StartInspect( unitId )
end
```
::: info Описание примера
В примере выполняется проверка доступности инспектирования. Если функция возвращает `true`, инициируется процесс инспектирования объекта с указанным идентификатором `unitId` с помощью функции `avatar.StartInspect`.
:::
