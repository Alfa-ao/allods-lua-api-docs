# avatar.IsInteractorRootCue
Определяет наличие начальной реплики у собеседника для старта диалога.

## Описание
```lua
avatar.IsInteractorRootCue(): boolean
```
Функция возвращает логическое значение, указывающее на наличие у собеседника (NPC) начальной реплики для разговора. Данная реплика отображается в самом начале диалога и служит точкой входа, начиная с которой игрок может продолжить взаимодействие с персонажем.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если в текущий момент диалога с собеседником активна начальная реплика, и `false` в противном случае.

## Примеры
### Проверка наличия начальной реплики
```lua
local inRootCue = avatar.IsInteractorRootCue()
if inRootCue then
    -- Диалог находится в самом начале, доступна стартовая реплика NPC
end
```
::: info Описание примера
В примере запрашивается статус начальной реплики. Если функция возвращает `true`, это означает, что взаимодействие с собеседником только начато и доступна стартовая реплика для продолжения разговора.
:::
