# avatar.IsInteractorVendor
Определяет, является ли текущий собеседник торговцем.

## Описание
```lua
avatar.IsInteractorVendor(): boolean
```
Функция возвращает логическое значение, указывающее на то, является ли текущий собеседник торговцем.

::: warning Замечание
Функция вызывается после того, как приходит событие `EVENT_INTERACTION_STARTED`. Во время активного диалога смена цели взаимодействия не допускается.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если собеседник является торговцем, и `false` в противном случае.

## Примеры
### Проверка статуса собеседника
```lua
local isVendor = avatar.IsInteractorVendor()
```
