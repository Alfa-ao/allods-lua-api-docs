# avatar.IsItemsStackable
Проверяет возможность объединения предметов при их перемещении между слотами контейнеров.

## Описание
```lua
avatar.IsItemsStackable( itemFromId: ObjectId, itemToId: ObjectId ): boolean
```
Функция определяет, произойдет ли объединение предмета из одного слота контейнера с предметом в целевом слоте при перемещении. Используется для выявления необходимости проверки изменения связанности и времени существования предметов в процессе перемещения.

## Список параметров
- **`itemFromId`** (`ObjectId`)
Перемещаемый предмет, который должен находиться в одном из контейнеров.
- **`itemToId`** (`ObjectId`)
Предмет, находящийся в целевом слоте, куда предполагается переместить `itemFromId`.

## Возвращаемые значения
`boolean` — `true`, если указанные предметы при перемещении `itemFromId` будут объединены (хотя бы частично), иначе `false`.

## Примеры
### Проверка возможности объединения предметов перед перемещением
```lua
if avatar.IsItemsStackable( itemFromId, itemToId ) then
    -- Вывод предупреждения о возможном изменении предмета
end
```
