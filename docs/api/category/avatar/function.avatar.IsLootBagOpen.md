# avatar.IsLootBagOpen
Возвращает состояние сумки с лутом на сервере.

## Описание
```lua
avatar.IsLootBagOpen(): boolean
```
Функция возвращает логическое значение, указывающее на текущее состояние сумки с лутом на стороне сервера. Результат показывает, открыта ли сумка для получения добычи в данный момент.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если сумка с лутом открыта для получения добычи, и `false` в противном случае.

## Примеры
### Проверка состояния сумки с лутом
```lua
local isOpen = avatar.IsLootBagOpen()
if isOpen then
    LogInfo( "Сумка с лутом открыта." )
else
    LogInfo( "Сумка с лутом закрыта." )
end
```
