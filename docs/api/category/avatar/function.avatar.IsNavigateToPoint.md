# avatar.IsNavigateToPoint
Определяет, перемещается ли персонаж к точке на карте по найденному пути.

## Описание
```lua
avatar.IsNavigateToPoint(): boolean
```
Функция возвращает логическое значение, указывающее, выполняется ли в данный момент перемещение персонажа к заданной точке на карте по найденному пути.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если в данный момент выполняется перемещение персонажа к точке на карте по найденному пути, и `false` в противном случае.

## Примеры
### Проверка статуса перемещения персонажа
```lua
local isNavigate = avatar.IsNavigateToPoint()
if isNavigate then
    LogInfo( "Персонаж перемещается к заданной точке." )
end
```
