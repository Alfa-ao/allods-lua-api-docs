# avatar.IsNextLevelLocked
Проверяет, заблокировано ли получение следующего уровня для главного игрока.

## Описание
```lua
avatar.IsNextLevelLocked(): boolean
```
Функция проверяет текущий статус ограничений на повышение уровня для главного персонажа. Возвращает логическое значение, указывающее на наличие или отсутствие блокировки получения следующего уровня.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если получение следующего уровня невозможно для главного игрока, и `false` в противном случае.

## Примеры
### Проверка блокировки следующего уровня
```lua
if avatar.IsNextLevelLocked() then
    LogInfo( "следующий уровень закрыт" )
end
```
