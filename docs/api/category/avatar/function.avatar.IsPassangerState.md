# avatar.IsPassangerState
Определяет, находится ли персонаж в состоянии пассажира.

## Описание
```lua
avatar.IsPassangerState(): boolean
```
Функция возвращает логическое значение, указывающее на то, является ли главный персонаж пассажиром. Состояние пассажира активируется, например, при путешествии на двухместном средстве передвижения, когда персонаж не управляет транспортом, а следует за водителем. Полученные данные позволяют адаптировать пользовательский интерфейс и ограничить выполнение определенных действий, которые недоступны для пассажиров.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если персонаж находится в состоянии пассажира, и `false` в противном случае.

## Примеры
### Проверка состояния пассажира
```lua
local isPassangerState = avatar.IsPassangerState()
```
