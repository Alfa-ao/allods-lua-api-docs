# avatar.IsPetInCombat

Возвращает признак нахождения активного питомца главного персонажа в состоянии боя.

## Описание

```lua
avatar.IsPetInCombat()
```

Функция проверяет текущий боевой статус активного питомца главного персонажа и возвращает соответствующее логическое значение. Если питомец в данный момент участвует в боевом столкновении, возвращается `true`. Если питомец находится вне боя, находится в режиме ожидания или у главного персонажа отсутствует активный питомец, функция возвращает `false`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` — `true`, если активный питомец главного персонажа находится в состоянии боя, иначе `false`.

## Примеры

### Проверка статуса питомца и вывод сообщения в лог

```lua
if avatar.IsPetInCombat() then
    LogInfo( "Питомец находится в бою!" )
end
```

::: info Описание примера
В примере вызывается функция `avatar.IsPetInCombat()`. Если активный питомец главного персонажа участвует в бою, функция возвращает `true`, и в информационный лог выводится соответствующее сообщение. Если питомец не в бою или отсутствует, условие не выполняется и запись в лог не производится.
:::
