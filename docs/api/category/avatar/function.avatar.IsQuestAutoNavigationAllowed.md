# avatar.IsQuestAutoNavigationAllowed
Проверяет доступность автонавигации для указанного задания.

## Описание
```lua
avatar.IsQuestAutoNavigationAllowed( questId: QuestId ): boolean
```
Функция проверяет, разрешена ли автонавигация для задания с заданным идентификатором. Возвращает логическое значение, указывающее на возможность использования автоматической навигации по маршруту задания.

## Список параметров
- **`questId`** (`QuestId`)
Уникальный идентификатор задания, для которого выполняется проверка доступности автонавигации.

## Возвращаемые значения
Возвращает `boolean` — `true`, если автонавигация для указанного задания доступна, и `false` в противном случае.

## Примеры
### Проверка доступности автонавигации
```lua
local questId = 12345 -- Идентификатор задания
local autoNavigationAllowed = avatar.IsQuestAutoNavigationAllowed( questId )

if autoNavigationAllowed then
    -- Автонавигация доступна, можно инициировать процесс перемещения
end
```
