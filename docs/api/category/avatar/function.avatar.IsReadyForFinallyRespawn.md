# avatar.IsReadyForFinallyRespawn
Возвращает логическое значение, указывающее на возможность окончательного воскрешения главного игрока из чистилища.

## Описание
```lua
avatar.IsReadyForFinallyRespawn(): boolean
```
Функция возвращает `boolean` — `true`, если искра главного игрока уже вернулась после переноса на кладбище и он может окончательно воскреснуть в обычном мире с помощью `avatar.FinallyRespawn()`, и `false`, если процесс возвращения искры еще не завершен.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если игрок может окончательно воскреснуть из чистилища, и `false` в противном случае.

## Примеры
### Проверка готовности и окончательное воскрешение
```lua
if avatar.IsReadyForFinallyRespawn() then
  avatar.FinallyRespawn()
end
```
::: info Описание примера
В примере сначала запрашивается текущий статус возвращения искры с помощью `avatar.IsReadyForFinallyRespawn()`. Если функция возвращает `true`, инициируется окончательное воскрешение игрока в обычном мире через вызов `avatar.FinallyRespawn()`.
:::
