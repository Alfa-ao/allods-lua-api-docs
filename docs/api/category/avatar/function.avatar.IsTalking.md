# avatar.IsTalking
Определяет, находится ли аватар в состоянии разговора с NPC.
## Описание
```lua
avatar.IsTalking(): boolean
```
Функция возвращает логическое значение, указывающее на то, ведется ли диалог аватара с неигровым персонажем (NPC).
## Список параметров
Параметры отсутствуют.
## Возвращаемые значения
Возвращает `boolean` — `true`, если аватар находится в состоянии разговора с NPC, и `false` в противном случае.
## Примеры
### Проверка состояния диалога
```lua
local isTalking = avatar.IsTalking()
if isTalking then
    LogInfo( "Аватар ведет диалог с NPC." )
end
```
