# avatar.IsTargetInMeleeRange
Проверяет, находится ли выбранный юнит в радиусе ближнего боя.

## Описание
```lua
avatar.IsTargetInMeleeRange(): boolean
```
Функция возвращает логическое значение, указывающее на нахождение выбранного юнита в радиусе ближнего боя (`meleeRange`). Если цель не выбрана или находится за пределами допустимой дистанции, возвращается `false`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`boolean` — `true`, если выбранный юнит находится в радиусе ближнего боя, и `false` в противном случае (включая случаи, когда цель не выбрана).

## Примеры
### Проверка дистанции до выбранной цели
```lua
local inMeleeRange = avatar.IsTargetInMeleeRange()
if inMeleeRange then
    LogInfo( "Цель находится в радиусе ближнего боя." )
else
    LogInfo( "Цель не выбрана или находится слишком далеко." )
end
```
