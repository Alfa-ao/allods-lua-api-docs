# avatar.IsTargetInspected
Определяет, инспектируется ли цель главного игрока.

## Описание
```lua
avatar.IsTargetInspected(): boolean
```
Функция возвращает логическое значение, указывающее на то, инспектируется ли в данный момент цель главного игрока.

::: warning Замечание
Для сторонних аддонов предусмотрены ограничения: метод недоступен на PvP-активностях, в которых реализована возможность голосования за изгнание игрока. Проверить доступность метода в текущем контексте можно с помощью функции `avatar.IsInspectAllowed()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если текущая цель инспектируется, и `false` в противном случае.

## Примеры
### Завершение инспекции при активном состоянии
```lua
if avatar.IsTargetInspected() then
    avatar.EndInspect()
end
```
::: info Описание примера
Проверяется состояние цели главного игрока. Если цель в данный момент инспектируется, вызывается функция `avatar.EndInspect()` для завершения процесса инспекции.
:::
