# avatar.IsTeleportLocationValid
Проверяет валидность и доступность идентификатора точки телепорта.

## Описание
```lua
avatar.IsTeleportLocationValid( locationId: ObjectId | TeleportMasterId ): boolean
```
Функция анализирует переданный идентификатор и определяет, является ли он валидной точкой телепорта. Валидным признается существующий локатор, который доступен аватару с учетом разблокировок и предикатов в текущий момент времени.

## Список параметров
- **`locationId`** (`ObjectId` | `TeleportMasterId`)
Идентификатор точки телепорта или идентификатор мастера телепорта.

## Возвращаемые значения
Возвращает `boolean` — `true`, если идентификатор является валидной и доступной точкой телепорта, и `false` в противном случае.

## Примеры
### Проверка доступности точки телепорта
```lua
if avatar.IsTeleportLocationValid( locationId ) then
    LogInfo( "Точка телепорта валидна" )
end
```
