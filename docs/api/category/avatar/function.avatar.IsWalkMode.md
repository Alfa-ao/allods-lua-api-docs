# avatar.IsWalkMode
Определяет, находится ли персонаж игрока в режиме ходьбы.
## Описание
```lua
avatar.IsWalkMode(): boolean
```
Функция возвращает логическое значение, указывающее, находится ли персонаж игрока в режиме ходьбы.
## Список параметров
Параметры отсутствуют.
## Возвращаемые значения
`boolean` — `true`, если персонаж игрока находится в режиме ходьбы, и `false` в противном случае.
## Примеры
### Проверка режима ходьбы персонажа
```lua
local isWalkMode = avatar.IsWalkMode()
```
