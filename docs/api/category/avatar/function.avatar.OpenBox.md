# avatar.OpenBox
Открывает сундуки из контейнеров персонажа.

## Описание
```lua
avatar.OpenBox( boxId: ObjectId, quantity: number | nil )
```
Функция инициирует процесс открытия сундуков. Предмет-сундук обязательно должен находиться в одном из контейнеров персонажа, например в инвентаре или переполненной сумке. Результат выполнения операции передается через событие `EVENT_BOX_OPERATION_RESULT`.

## Список параметров
- **`boxId`** (`ObjectId`)
Уникальный идентификатор предмета-сундука. Предмет обязательно должен находиться в одном из контейнеров персонажа (например, в инвентаре или переполненной сумке).
- **`quantity`** (`number` | `nil`)
Количество открываемых сундуков. Параметр является необязательным. Если значение не передано или равно `nil`, по умолчанию открывается один сундук.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Открытие одного сундука
```lua
local boxId = 12345 -- Идентификатор сундука из инвентаря
avatar.OpenBox( boxId )
```
::: info Описание примера
В примере инициируется открытие одного сундука по его идентификатору. Поскольку параметр `quantity` не передан, используется значение по умолчанию (1 сундук). Результат операции передается через событие `EVENT_BOX_OPERATION_RESULT`.
:::

### Открытие нескольких сундуков
```lua
local boxId = 12345 -- Идентификатор сундука из инвентаря
local quantity = 5
avatar.OpenBox( boxId, quantity )
```
::: info Описание примера
В примере инициируется открытие 5 сундуков с одинаковым идентификатором. Результат выполнения операции передается через событие `EVENT_BOX_OPERATION_RESULT`.
:::
