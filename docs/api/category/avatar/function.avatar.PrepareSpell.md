# avatar.PrepareSpell
Инициирует подготовку заклинания.
## Описание
```lua
avatar.PrepareSpell( spellId: SpellId )
```
Функция инициирует процесс подготовки заклинания с заданным идентификатором.
::: warning Замечание
Для сторонних аддонов предусмотрены следующие ограничения:
1. Метод недоступен на PvP-активностях.
2. Метод функционирует исключительно в обработчиках реакции на действия (не перемещение) мыши, то есть при нажатии кнопки.
:::
## Список параметров
- **`spellId`** (`SpellId`)
Уникальный идентификатор подготавливаемого заклинания.
## Возвращаемые значения
Возвращаемое значение отсутствует.
## Примеры
### Подготовка заклинания из книги заклинаний
```lua
local spellbook = avatar.GetSpellBook()
avatar.PrepareSpell( spellbook[1] )
```
::: info Описание примера
В примере сначала извлекается список доступных заклинаний с помощью функции `avatar.GetSpellBook()`. Затем инициируется подготовка заклинания, находящегося в полученном списке под индексом 1, посредством вызова `avatar.PrepareSpell()`.
:::
