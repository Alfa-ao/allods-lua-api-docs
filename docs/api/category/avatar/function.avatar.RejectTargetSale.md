# avatar.RejectTargetSale
Отклоняет предложение таргетной продажи.

## Описание

```lua
avatar.RejectTargetSale( saleId: ObjectId )
```

Функция инициирует отклонение предложения таргетной продажи с заданным идентификатором. После выполнения операции предложение аннулируется, а предмет остается в инвентаре продавца.

## Список параметров

- **`saleId`** (`ObjectId`)
Уникальный идентификатор предложения таргетной продажи, которое необходимо отклонить.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры

### Отклонение первого доступного предложения таргетной продажи

```lua
local availableOffers = avatar.GetTargetSales()
if availableOffers and availableOffers[0] then
    avatar.RejectTargetSale( availableOffers[0] )
end
```

::: info Описание примера
В примере извлекается список доступных предложений таргетной продажи. При наличии хотя бы одного предложения (с индексом `0`) его идентификатор передается в функцию `avatar.RejectTargetSale` для отклонения.
:::
