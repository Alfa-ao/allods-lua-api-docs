# avatar.RequestInteractions
Инициирует запрос списка доступных взаимодействий с собеседником.

## Описание
```lua
avatar.RequestInteractions()
```
Функция возвращает nil, инициируя запрос доступных взаимодействий с собеседником для получения списка заданий или проверки возможности торговли. Если собеседник готов к торговле или выдаче заданий, передается событие `EVENT_INTERACTION_STARTED`.

::: warning Замечание
Использование метода допускается только при нахождении в состоянии разговора, которое устанавливается вызовом `avatar.StartInteract()`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Запрос взаимодействий с собеседником
```lua
avatar.RequestInteractions()
```
