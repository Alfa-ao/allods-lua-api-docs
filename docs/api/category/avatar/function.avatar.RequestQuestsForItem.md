# avatar.RequestQuestsForItem
Запрашивает список заданий для предмета, находящегося в слоте контейнера.
## Описание
```lua
avatar.RequestQuestsForItem( itemId: ObjectId )
```
Функция отправляет запрос на получение заданий, связанных с указанным предметом. Предмет обязательно должен располагаться в слоте одного из контейнеров: сумки, переполненной сумки или банковской ячейки. Итог выполнения операции передается через событие `EVENT_ITEM_QUESTS_RECEIVED`.
::: warning Замечание
Если предмет не находится в слоте контейнера (сумка, переполненная сумка, банковская ячейка), запрос не будет выполнен.
:::
## Список параметров
- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета. Предмет должен находиться в слоте какого-либо контейнера (сумка, переполненная сумка, банковская ячейка).
## Возвращаемые значения
Возвращаемое значение отсутствует.
## Примеры
### Запрос заданий для предмета
```lua
local itemId = 12345 -- Идентификатор предмета из инвентаря
avatar.RequestQuestsForItem( itemId )
```
