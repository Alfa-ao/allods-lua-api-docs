# avatar.RequestVendor
Запрашивает у торговца список имеющихся товаров.

## Описание
```lua
avatar.RequestVendor()
```
Функция возвращает отсутствие значений. Инициирует запрос к собеседнику для получения списка имеющихся товаров. Если собеседник готов торговать, на клиент передается событие `EVENT_VENDOR_LIST_UPDATED`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Запрос списка товаров у торговца
```lua
avatar.RequestVendor()
```
