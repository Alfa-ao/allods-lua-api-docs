# avatar.Respawn
Инициирует оживление аватара на кладбище.

## Описание
```lua
avatar.Respawn()
```
Функция инициирует процесс оживления аватара на кладбище. После отправки команды аватар не воскресает немедленно, а ожидает прибытия своей искры, находясь в чистилище.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Инициирование оживления аватара
```lua
avatar.Respawn()
```
