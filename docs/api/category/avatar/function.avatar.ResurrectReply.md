# avatar.ResurrectReply
Отправляет ответ с согласием или отказом на оживление главного персонажа.

## Описание
```lua
avatar.ResurrectReply( accept: boolean )
```
Функция отправляет ответ с согласием или отказом на оживление главного персонажа. Запрос на оживление приходит через событие `EVENT_RESSURECT_REQUESTED`.

## Список параметров
- **`accept`** (`boolean`)
Определяет решение игрока. Значение `true` подтверждает оживление, значение `false` — отклоняет запрос.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Согласие на оживление
```lua
avatar.ResurrectReply( true )
```
