# avatar.ReturnQuest
Инициирует сдачу задания юнита, который является текущей целью аватара.

## Описание
```lua
avatar.ReturnQuest( questId: QuestId, rewardId: ObjectId | nil )
```
Функция инициирует процесс сдачи задания юнита, являющегося текущей целью аватара. Параметр `rewardId` позволяет выбрать конкретную награду за выполнение задания.

::: warning Замечание
Сдача задания юнита, не являющегося текущей целью аватара, невозможна.
:::

## Список параметров
- **`questId`** (`QuestId`)
Идентификатор квеста, который необходимо сдать.
- **`rewardId`** (`ObjectId` | `nil`)
Идентификатор выбираемой награды за выполнение задания. Если выбор конкретной награды не требуется, передается значение `nil`.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Сдача задания без выбора конкретной награды
```lua
local questId = 12345 -- Идентификатор текущего квеста
avatar.ReturnQuest( questId, nil )
```
