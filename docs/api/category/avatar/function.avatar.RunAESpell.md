# avatar.RunAESpell
Начинает сотворение заклинания в указанную точку на карте.
## Описание
```lua
avatar.RunAESpell( spellId: SpellId, position: table )
```
Функция инициирует процесс сотворения заклинания (Area of Effect) в указанную точку на карте. Для сторонних аддонов предусмотрены ограничения: функция недоступна на PvP-активностях; функция работает исключительно в обработчиках реакций на действия мыши (например, на нажатие кнопки), но не на перемещение.
::: warning Замечание
Функция работает только в обработчиках реакции на действия мыши (нажатие кнопки) и недоступна на PvP-активностях для сторонних аддонов.
:::
## Список параметров
- **`spellId`** (`SpellId`)
Уникальный идентификатор заклинания, которое необходимо сотворить.
- **`position`** (`table`)
Таблица, содержащая координаты точки на карте (структура `GamePosition`), в которую будет направлено заклинание.
## Возвращаемые значения
Возвращаемое значение отсутствует.
## Примеры
### Обработка клика по точке на карте
```lua
function OnEventAEMarkPointClicked( params )
    avatar.RunAESpell( aeSpellId, params )
end
```
::: info Описание примера
В примере обрабатывается событие `EVENT_AEMARK_POINT_CLICKED`, которое приходит при клике по точке на карте. В обработчике вызывается функция `avatar.RunAESpell`, которой передаются идентификатор заклинания и параметры точки.
:::
