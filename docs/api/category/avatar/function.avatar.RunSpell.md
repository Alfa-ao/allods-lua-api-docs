# avatar.RunSpell
Начинает сотворение заклинания на цель по умолчанию.

## Описание
```lua
avatar.RunSpell( spellId: SpellId )
```
Функция инициирует процесс сотворения заклинания с заданным идентификатором на автоматически определенную цель. Выбор цели зависит от типа заклинания: для заклинаний «в цель» используется текущая цель (или сам персонаж, если текущая цель не дружественная, а заклинание дружественное); для заклинаний «на себя» — сам персонаж; для заклинаний «в точку» — точка под текущей целью (или под персонажем, если цель отсутствует).

::: warning Замечание
Для сторонних аддонов предусмотрены следующие ограничения:
1. Метод недоступен на PvP-активностях.
2. Метод срабатывает исключительно в обработчиках реакций на действия мыши (например, на нажатие кнопки), но не на перемещение курсора.
:::

## Список параметров
- **`spellId`** (`SpellId`)
Идентификатор заклинания, которое необходимо сотворить.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Сотворение первого заклинания из книги
```lua
local spellbook = avatar.GetSpellBook()
avatar.RunSpell( spellbook[1] )
```
::: info Описание примера
Получается список заклинаний из книги персонажа с помощью `avatar.GetSpellBook()`. Затем вызывается `avatar.RunSpell()` для применения первого заклинания из полученного списка.
:::
