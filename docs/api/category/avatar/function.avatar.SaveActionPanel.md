# avatar.SaveActionPanel
Возвращает сериализованное состояние панели действий для указанного класса и индекса билда.
## Описание
```lua
avatar.SaveActionPanel( sysClass: SpellId, buildIndex: number ): table | string | nil
```
Функция возвращает сериализованное состояние панели действий для заданного класса и билда. Номера билдов от 0 до 2 соответствуют первой, второй и третьей граням талантов. Номера билдов от 3 до 9 не имеют прямого соответствия в интерфейсе, однако сохраняются на сервере и могут быть использованы для загрузки или выгрузки конфигураций.
## Список параметров
- **`sysClass`** (`SpellId`)
Системное имя класса.
- **`buildIndex`** (`number`)
Индекс билда от 0 до 9.
## Возвращаемые значения
Возвращает `table`, `string` или `nil` в зависимости от наличия данных и типа класса:
- `nil` — в случае отсутствия данных для выбранного класса или билда.
- `string` — в случае наличия данных, если класс не является WARLOCK.
- `table` — в случае наличия данных, если класс является WARLOCK.
## Примеры
### Получение и вывод состояния панели действий
```lua
LogInfo( avatar.SaveActionPanel( avatar.GetClass(), 0 ) )
```
::: info Описание примера
В примере запрашивается и выводится в лог сериализованное состояние панели действий для текущего класса персонажа и нулевого индекса билда (первая грань талантов).
:::
