# avatar.SetActiveBuild
Задает номер активной специализации персонажа.

## Описание
```lua
avatar.SetActiveBuild( index: number )
```
Функция задает номер активной специализации для текущего аватара. Установка новой специализации инициирует процесс переключения, результаты которого применяются после синхронизации с сервером.

::: warning Замечание
Допустимый диапазон значений параметра `index` — от `0` до значения, на единицу меньшего результата функции `avatar.GetBuildsCount()`. Передача некорректного индекса приводит к ошибке выполнения.
:::

## Список параметров
- **`index`** (`number`)
Номер специализации аватара. Доступный диапазон значений — от `0` до величины, на `1` меньшей результата `avatar.GetBuildsCount()`.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Переключение на первую доступную специализацию
```lua
local buildsCount = avatar.GetBuildsCount()
if buildsCount > 0 then
    avatar.SetActiveBuild( 0 )
end
```
::: info Описание примера
В примере сначала получается общее количество доступных специализаций с помощью `avatar.GetBuildsCount()`. Если хотя бы одна специализация доступна (значение больше `0`), выполняется переключение на первую специализацию (с индексом `0`) через вызов `avatar.SetActiveBuild`.
:::
