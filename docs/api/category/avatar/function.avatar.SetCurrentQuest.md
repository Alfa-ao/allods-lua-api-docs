# avatar.SetCurrentQuest
Назначает квест с заданным идентификатором текущим выполняемым квестом.
## Описание
```lua
avatar.SetCurrentQuest( questId: QuestId|nil )
```
Функция назначает квест с указанным идентификатором текущим выполняемым квестом. Если в качестве параметра передается значение `nil`, статус текущего квеста сбрасывается, и ни одно задание не остается в статусе выполняемого. Событие `EVENT_CURRENT_QUEST_CHANGED` приходит после успешного изменения текущего квеста.
## Список параметров
- **`questId`** (`QuestId` | `nil`)
Идентификатор квеста, задаваемого как текущий. При передаче значения `nil` текущий квест сбрасывается.
## Возвращаемые значения
Возвращаемое значение отсутствует.
## Примеры
### Назначение текущего квеста
```lua
local questId = 12345
avatar.SetCurrentQuest( questId )
```
