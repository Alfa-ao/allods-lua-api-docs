# avatar.SetFatalityPhrase
Устанавливает фразу для добивания.

## Описание
```lua
avatar.SetFatalityPhrase( message: WString )
```
Функция задает текстовую фразу, используемую при выполнении добивания. 

::: warning Замечание
В пользовательских аддонах метод функционирует исключительно внутри обработчиков реакции на действия мыши, например, при нажатии кнопки.
:::

## Список параметров
- **`message`** (`WString`)
Текстовая фраза, устанавливаемая для добивания.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Установка фразы из текстового поля
```lua
avatar.SetFatalityPhrase( wEditLine:GetText() )
```
