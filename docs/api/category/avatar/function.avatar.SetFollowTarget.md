# avatar.SetFollowTarget
Включает или выключает режим автоматического следования за целевым объектом.

## Описание
```lua
avatar.SetFollowTarget( isFollow: boolean )
```
Функция управляет автоматическим движением персонажа вслед за объектом, который является текущей целью на момент вызова. Позволяет начать или прекратить режим следования. При изменении состояния режима следования клиенту передается событие `EVENT_FOLLOW_MODE_CHANGED`.

## Список параметров
- **`isFollow`** (`boolean`)
Флаг, определяющий состояние режима следования. При передаче значения `true` включает автоматическое движение за целью, при передаче `false` — выключает его.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Включение режима следования за целью
```lua
avatar.SetFollowTarget( true )
```
