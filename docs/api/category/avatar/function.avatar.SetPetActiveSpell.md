# avatar.SetPetActiveSpell
Устанавливает активное заклинание для пета аватара.

## Описание
```lua
avatar.SetPetActiveSpell( index: number )
```
Функция назначает активное заклинание для питомца аватара, используя индекс заклинания из списка доступных способностей.

## Список параметров
- **`index`** (`number`)
Индекс заклинания в списке способностей питомца, которое необходимо сделать активным.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Установка активного заклинания для питомца
```lua
local spells = avatar.GetPetSpells()
if spells[ 0 ] then
    avatar.SetPetActiveSpell( 0 )
end
```
::: info Описание примера
В примере сначала извлекается список заклинаний питомца с помощью функции `avatar.GetPetSpells`. Затем проверяется наличие заклинания с индексом `0`. Если заклинание существует, оно назначается как активное посредством вызова функции `avatar.SetPetActiveSpell`.
:::
