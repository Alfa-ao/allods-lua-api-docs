# avatar.SetPetMoveMode

Устанавливает тип движения питомца аватара.

## Описание

```lua
avatar.SetPetMoveMode( mode )
```

Функция задает и применяет указанный режим перемещения для питомца аватара. Выбранный режим определяет поведение питомца в мире: активное следование за хозяином или статичное ожидание на текущем месте.

## Список параметров

- **`mode`** (`number`)
Тип движения питомца. Принимает значения из предопределенных констант:
- `PET_MOVE_FOLLOW` — питомец следует за хозяином.
- `PET_MOVE_STAY` — питомец остается на месте.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка режима следования за хозяином

```lua
avatar.SetPetMoveMode( PET_MOVE_FOLLOW )
```

::: info Описание примера
Питомец получает команду следовать за главным персонажем. Если до этого был активен режим `PET_MOVE_STAY`, питомец начнет движение к хозяину.
:::
