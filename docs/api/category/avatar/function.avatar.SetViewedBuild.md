# avatar.SetViewedBuild
Устанавливает номер специализации, данные о которой доступны для просмотра.

## Описание
```lua
avatar.SetViewedBuild( index: number )
```
Функция задает номер специализации аватара, информацию о которой разрешено просматривать.

## Список параметров
- **`index`** (`number`)
Номер специализации аватара. Допустимый диапазон значений: от `0` до величины, на единицу меньшей результата функции `avatar.GetBuildsCount()`.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Установка первой специализации для просмотра
```lua
avatar.SetViewedBuild( 0 )
```
