# avatar.SetWalkMode
Задает режим движения персонажа (шаг или бег).

## Описание
```lua
avatar.SetWalkMode( walk: boolean )
```
Функция устанавливает состояние движения главного персонажа, принудительно переключая его на ходьбу или бег в зависимости от переданного логического параметра.

## Список параметров
- **`walk`** (`boolean`)
Задает режим ходьбы или бега. Значение `true` включает режим ходьбы (шага), `false` — режим бега.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Переключение персонажа на режим ходьбы
```lua
avatar.SetWalkMode( true )
```
