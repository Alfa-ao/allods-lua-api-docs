# avatar.ShareQuest
Инициирует предложение другим игрокам группы принять указанное разделяемое задание.
## Описание
```lua
avatar.ShareQuest( questId: QuestId )
```
Функция отправляет запрос на предложение другим игрокам текущей группы принять указанное задание. Задание обязательно должно поддерживать механику разделения (быть разделяемым).

## Список параметров
- **`questId`** (`QuestId`)
Уникальный идентификатор задания, которое необходимо предложить другим участникам группы.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Предложение разделяемого задания группе
```lua
local questId = 12345 -- Идентификатор разделяемого задания
avatar.ShareQuest( questId )
```
