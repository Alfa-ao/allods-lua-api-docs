# avatar.SheathNext
Визуально переключает состояние отображения оружия.

## Описание
```lua
avatar.SheathNext()
```
Функция не возвращает значений. Визуально показывает или убирает оружие. Циклически переключаются следующие состояния: показать ближнее оружие, показать дальнее оружие, убрать оружие.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Переключение состояния оружия
```lua
avatar.SheathNext()
```
