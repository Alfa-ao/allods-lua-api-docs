# avatar.ShowMetaUpgradeDialog
Управляет состоянием отображения диалога улучшения мета-предметов.

## Описание
```lua
avatar.ShowMetaUpgradeDialog( show: boolean )
```
Функция инициирует открытие или закрытие диалога улучшения мета-предметов из скрипта. Вызов обязателен для корректного отслеживания закрытия интерфейса по внутренним причинам, что фиксируется передачей события `EVENT_META_UPGRADE_BREAK`.

::: warning Замечание
Событие `EVENT_META_UPGRADE_BREAK` передается при закрытии диалога по внутренним причинам. Для корректной синхронизации состояния интерфейса требуется обязательный вызов данной функции.
:::

## Список параметров
- **`show`** (`boolean`)
Флаг состояния диалога. Значение `true` инициирует показ диалога, `false` — его закрытие.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Открытие диалога улучшения мета-предметов
```lua
avatar.ShowMetaUpgradeDialog( true )
```
