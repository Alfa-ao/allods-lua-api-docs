# avatar.StartInspect
Инициирует процесс инспектирования игрока для получения информации о его экипировке.

## Описание
```lua
avatar.StartInspect( unitId: ObjectId )
```
Функция отправляет запрос на инспектирование указанного игрока. После успешной инициации процесса становится доступным получение детальной информации об экипировке инспектируемой цели.

::: warning Замечание
В некоторых MM-активностях функция недоступна. Для проверки текущей доступности инспектирования используется функция `avatar.IsInspectAllowed()`.
:::

## Список параметров
- **`unitId`** (`ObjectId`)
Уникальный идентификатор игрока, которого необходимо инспектировать.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Инициирование инспектирования игрока
```lua
avatar.StartInspect( unitId )
```
