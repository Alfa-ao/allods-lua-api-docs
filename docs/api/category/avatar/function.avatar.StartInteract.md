# avatar.StartInteract
Стартует взаимодействие с указанным NPC.

## Описание
```lua
avatar.StartInteract( unitId: ObjectId )
```
Функция отправляет запрос на начало взаимодействия с заданным юнитом. При успешной обработке приходит событие `EVENT_TALK_STARTED`.

## Список параметров
- **`unitId`** (`ObjectId`)
Уникальный идентификатор юнита, с которым требуется начать взаимодействие.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Инициация взаимодействия с NPC
```lua
avatar.StartInteract( unitId )
```
