# avatar.StopInteract
Завершает взаимодействие с NPC.

## Описание
```lua
avatar.StopInteract()
```
Функция прекращает активное взаимодействие главного персонажа с неигровым персонажем (NPC). Применяется для принудительного закрытия диалогов, меню торговцев, окон квестов или других форм интерактива, которые были инициированы ранее.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Закрытие окна взаимодействия
```lua
avatar.StopInteract()
```
