# avatar.StopMoveToPoint
Принудительно останавливает движение аватара игрока к заданной точке, если персонаж находится в соответствующем режиме.

## Описание
```lua
avatar.StopMoveToPoint()
```
Функция не принимает параметров и не возвращает значений. Принудительно прерывает процесс движения аватара к указанной точке, если персонаж в данный момент находится в режиме перемещения к цели.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Проверка состояния и остановка движения
```lua
if avatar.GetMoveToPointState().isMoving then
    avatar.StopMoveToPoint()
end
```
::: info Описание примера
В примере сначала запрашивается текущее состояние движения аватара с помощью функции `avatar.GetMoveToPointState()`. Если персонаж находится в процессе перемещения (поле `isMoving` имеет значение `true`), вызывается `avatar.StopMoveToPoint()` для принудительной остановки.
:::
