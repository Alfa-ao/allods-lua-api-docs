# avatar.StoreBaseTalent
Помечает талант из базовой таблицы талантов для последующего изучения.

## Описание
```lua
avatar.StoreBaseTalent( layer: number, line: number )
```
Функция помечает конкретный талант в базовой таблице `Умения` талантов для изучения. Помеченные таланты не применяются автоматически, для их фактической активации требуется дополнительный вызов функции применения.

## Список параметров
- **`layer`** (`number`)
Номер строки в базовой таблице талантов.
- **`line`** (`number`)
Номер столбца в базовой таблице талантов.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Пометка и применение таланта
```lua
local layer = 1
local line = 2

avatar.StoreBaseTalent( layer, line )
avatar.ApplyStoredTalents()
```
::: info Описание примера
В примере помечается талант, расположенный на первой строке и во втором столбце базовой таблицы. Затем вызывается функция `avatar.ApplyStoredTalents` для применения всех накопленных изменений.
:::
