# avatar.StoreFieldTalent
Помечает веху из поля талантов для изучения.
## Описание
```lua
avatar.StoreFieldTalent( field: number, row: number, column: number )
```
Функция помечает указанную веху в заданном поле, строке и столбце дерева талантов для последующего изучения. Помеченные вехи не применяются мгновенно, а сохраняются в буфере до вызова функции применения.
## Список параметров
- **`field`** (`number`)
Номер поля в дереве талантов.
- **`row`** (`number`)
Номер строки в указанном поле.
- **`column`** (`number`)
Номер столбца в указанном поле.
## Возвращаемые значения
Возвращаемое значение отсутствует.
## Примеры
### Пометка вехи и применение талантов
```lua
local field = 1
local row = 2
local column = 3

avatar.StoreFieldTalent( field, row, column )
avatar.ApplyStoredTalents()
```
::: info Описание примера
В примере помечается веха на поле `1`, в строке `2` и столбце `3`. После пометки вызывается функция `avatar.ApplyStoredTalents` для фактического применения и изучения помеченных талантов.
:::
