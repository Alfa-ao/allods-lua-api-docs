# avatar.SwapActionsInActionPanel
Меняет местами действия, назначенные в двух указанных слотах панели действий.

## Описание

```lua
avatar.SwapActionsInActionPanel( slot1, slot2 )
```

Функция выполняет обмен содержимым двух указанных слотов панели действий. После выполнения операции действие из первого слота перемещается во второй, а действие из второго — в первый. Если один из слотов пуст, его содержимое (`nil`) также участвует в обмене, фактически перемещая действие в пустой слот.

::: warning Замечание
Сторонним аддонам запрещено вызывать данную функцию, пока персонаж игрока находится в бою.
:::

## Список параметров

- **`slot1`** (`number`)
Индекс первого слота панели действий. Значение находится в диапазоне от `0` до максимального количества слотов.

- **`slot2`** (`number`)
Индекс второго слота панели действий. Значение находится в диапазоне от `0` до максимального количества слотов.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Обмен действий между первым и пятым слотами панели

```lua
avatar.SwapActionsInActionPanel( 0, 4 )
```

::: info Описание примера
В примере выполняется обмен содержимым слотов с индексами `0` и `4`. Действие, находившееся в первом слоте, перемещается в пятый, а действие из пятого — в первый.
:::
