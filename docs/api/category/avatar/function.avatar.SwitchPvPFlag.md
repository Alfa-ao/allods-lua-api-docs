# avatar.SwitchPvPFlag
Переключает пользовательский флаг PvP главного игрока.

## Описание
```lua
avatar.SwitchPvPFlag( isOn: boolean )
```
Функция переключает пользовательский флаг PvP главного игрока. При передаче значения `true` флаг включается, при `false` — выключается.

## Список параметров
- **`isOn`** (`boolean`)
Значение, определяющее состояние флага. `true` включает PvP флаг, `false` выключает.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Включение флага PvP
```lua
avatar.SwitchPvPFlag( true )
```
