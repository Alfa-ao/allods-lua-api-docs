# avatar.TargetSelf
Выбирает своего персонажа в качестве цели, задействуя дополнительную цель.

## Описание
```lua
avatar.TargetSelf()
```
Функция возвращает nil. Функция выбирает своего персонажа в качестве цели, используя дополнительную цель.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Выбор себя в качестве цели
```lua
avatar.TargetSelf()
```
