# avatar.TeleportOfferReply
Отправляет ответ на предложение о телепортации от члена группы.

## Описание
```lua
avatar.TeleportOfferReply( accept: boolean )
```
Функция отправляет ответ на событие `EVENT_TELEPORT_OFFER_REQUEST`, которое приходит при получении предложения о телепортации от другого игрока в группе. Передача значения `true` в параметре `accept` означает согласие главного персонажа на предложенную телепортацию.

## Список параметров
- **`accept`** (`boolean`)
Флаг согласия на телепортацию. Значение `true` указывает, что главный персонаж принимает предложение о телепортации от члена группы.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Согласие на телепортацию
```lua
avatar.TeleportOfferReply( true )
```
