# avatar.UnBindFromActionPanel

Снимает привязку действия с указанного слота панели действий.

## Описание

```lua
avatar.UnBindFromActionPanel( index )
```

Функция снимает привязку любого действия с заданного слота экшен-панели. После выполнения операции текущее состояние панели действий немедленно сохраняется в файл `Personal/user.cfg`.

::: warning Замечание
Сторонним дополнениям запрещено вызывать данную функцию, пока персонаж игрока находится в бою.
:::

## Список параметров

- **`index`** (`number`)
Индекс слота панели действий, с которого снимается привязка. Значение находится в диапазоне от `0` до максимального количества слотов.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Снятие привязки с нулевого слота панели действий

```lua
avatar.UnBindFromActionPanel( 0 )
```

::: info Описание примера
Выполняется вызов функции для снятия привязки действия из слота с индексом `0`. Слот становится пустым, а обновленное состояние панели действий автоматически сохраняется в конфигурационный файл.
:::
