# avatar.UnequipItem
Снимает указанный предмет с экипированного слота.

## Описание
```lua
avatar.UnequipItem( itemId: ObjectId )
```
Функция снимает предмет с заданным идентификатором из соответствующего слота экипировки главного персонажа. После выполнения операции предмет перемещается из слота экипировки в инвентарь.

## Список параметров
- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета, который необходимо снять с экипированного слота.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Снятие предмета с экипировки
```lua
local itemId = 12345 -- Идентификатор экипированного предмета
avatar.UnequipItem( itemId )
```
