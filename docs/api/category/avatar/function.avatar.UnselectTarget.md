# avatar.UnselectTarget
Сбрасывает выбранную и дополнительную цели у аватара.

## Описание
```lua
avatar.UnselectTarget()
```
Функция сбрасывает текущую выбранную цель, а также дополнительную цель у аватара.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Сброс текущей цели
```lua
avatar.UnselectTarget()
```
