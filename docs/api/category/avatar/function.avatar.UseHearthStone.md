# avatar.UseHearthStone
Активирует камень возвращения (HearthStone).

## Описание
```lua
avatar.UseHearthStone()
```
Функция возвращает управление после инициации использования камня возвращения (HearthStone) главным персонажем, запуская процесс телепортации в точку привязки.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Использование камня возвращения
```lua
avatar.UseHearthStone()
```
