# avatar.UseItem
Инициирует использование предмета из контейнера игрока.

## Описание
```lua
avatar.UseItem( itemId: ObjectId )
```
Функция инициирует попытку использования предмета из контейнера игрока. 

::: warning Замечание
Для сторонних аддонов действуют строгие ограничения. Метод недоступен на PvP-активностях. Вызов функции допускается исключительно в обработчиках реакции на действия мыши, не связанных с перемещением, например, при нажатии кнопки.
:::

## Список параметров
- **`itemId`** (`ObjectId`)
Идентификатор предмета, который планируется использовать.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Использование предмета по идентификатору
```lua
avatar.UseItem( itemId )
```
