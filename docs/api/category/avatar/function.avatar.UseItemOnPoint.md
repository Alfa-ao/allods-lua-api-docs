# avatar.UseItemOnPoint
Использует предмет из контейнера игрока на указанную точку ландшафта.

## Описание
```lua
avatar.UseItemOnPoint( itemId: ObjectId, position: GamePosition )
```
Функция инициирует использование предмета из контейнера игрока на заданную точку ландшафта.

::: warning Замечание
Для сторонних аддонов существуют следующие ограничения при использовании данного метода:
1. Метод недоступен на PvP-активностях.
2. Метод работает исключительно в обработчиках реакции на действия мыши (например, на нажатие кнопки), но не на события перемещения.
:::

## Список параметров
- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета в контейнере игрока.
- **`position`** (`GamePosition`)
Таблица, представляющая точку на ландшафте, на которую необходимо использовать предмет.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Использование предмета на точке ландшафта
```lua
local itemId = 12345
local position = GamePosition.Create( 100.0, 200.0, 300.0 )
avatar.UseItemOnPoint( itemId, position )
```
::: info Описание примера
В примере формируется идентификатор предмета и координаты точки на ландшафте с помощью `GamePosition.Create`. Затем вызывается функция `avatar.UseItemOnPoint` для применения предмета на заданных координатах.
:::
