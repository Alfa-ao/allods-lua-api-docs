# battleground.GetAvatarMark
Возвращает идентификатор команды баттлграунда, к которой принадлежит аватар.

## Описание
```lua
battleground.GetAvatarMark(): BattlegroundMarkId | nil
```
Функция извлекает идентификатор команды, к которой принадлежит текущий аватар в рамках баттлграунда. Если аватар не участвует в сражении или его команда определяется исключительно фракцией, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `BattlegroundMarkId` — идентификатор команды баттлграунда, или `nil`, если аватар не участвует в сражении, либо его команда определяется фракцией.

## Примеры
### Получение и вывод системного имени команды
```lua
local commandMarkId = battleground.GetAvatarMark()
if commandMarkId then
    commandInfo = commandMarkId:GetInfo()
    if commandInfo then
        LogInfo( "Вattleground command sysName: ", commandInfo.sysName  )
    end
end
```
::: info Описание примера
В примере запрашивается идентификатор команды аватара. Если идентификатор успешно получен, извлекается дополнительная информация о команде через метод `:GetInfo()`. При наличии информации в лог выводится системное имя (`sysName`) команды.
:::
