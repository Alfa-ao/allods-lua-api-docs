# battleground.GetCheckpointInfo
Возвращает детальную информацию о конкретной контрольной точке баттлграунда.

## Описание
```lua
battleground.GetCheckpointInfo( checkpointId: ObjectId ): table | nil
```
Функция извлекает и возвращает информацию о состоянии контрольной точки на поле боя по её уникальному идентификатору. Возвращаемые данные включают локализованное имя, идентификаторы текстур и команд, текущее и предыдущее состояние, а также параметры процесса захвата. Если контрольная точка с указанным идентификатором не существует, функция возвращает `nil`.

## Список параметров
- **`checkpointId`** (`ObjectId`)
Уникальный идентификатор контрольной точки, информацию о которой необходимо получить.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если контрольная точка с заданным идентификатором не существует, возвращается `nil`.
В противном случае возвращается таблица со следующими полями:
- **`name`** (`WString`) — локализованное имя контрольной точки.
- **`textureId`** (`TextureId`) — идентификатор текстуры иконки для обозначения контрольной точки.
- **`state`** (`string`) — текущее состояние контрольной точки (свободна, в процессе захвата, захвачена). Значение соответствует одному из перечислений `ENUM_RecapturablePlayerSpawnPlaceState_...`.
- **`ownerMarkId`** (`BattlegroundMarkId` | `nil`) — идентификатор команды, которой в данный момент захвачена контрольная точка.
- **`prevState`** (`string`) — предыдущее состояние контрольной точки (до того, как её начал захватывать нынешний владелец). Значение соответствует перечислению `ENUM_RecapturablePlayerSpawnPlaceState_...`.
- **`prevOwnerMarkId`** (`BattlegroundMarkId` | `nil`) — идентификатор команды, которой раньше была захвачена контрольная точка. Возвращает `nil`, если точка ранее была свободна.
- **`capturedTime`** (`number` | `nil`) — время в миллисекундах (округляется в большую сторону), оставшееся до окончания захвата контрольной точки. Возвращает `nil`, если точка в данный момент не захватывается (свободна или захват уже завершён).
- **`captureDuration`** (`number` | `nil`) — длительность процесса захвата контрольной точки в миллисекундах. Возвращает `nil`, если точка не может захватываться.
- **`gain`** (`number` | `nil`) — текущий прогресс захвата точки.
- **`defendersCount`** (`number` | `nil`) — число защищающих контрольную точку (если поддерживается стеллой).
- **`attackersCount`** (`number` | `nil`) — число атакующих контрольную точку (если поддерживается стеллой).
- **`oldMechanics`** (`boolean`) — `true`, если для данной точки используется старая механика захвата.
- **`radius`** (`number`) — радиус зоны захвата точки.
- **`isEnable`** (`boolean`) — `true`, если контрольная точка активна.

## Примеры
### Получение информации о первой контрольной точке
```lua
local checkpoints = battleground.GetCheckpoints()
if checkpoints[0] ~= nil then
    local info = battleground.GetCheckpointInfo( checkpoints[0] )
    if info then
        LogInfo( "Имя точки: ", info.name, " | Состояние: ", info.state )
    end
end
```
::: info Описание примера
В примере сначала получается список всех контрольных точек баттлграунда с помощью `battleground.GetCheckpoints()`. Затем проверяется существование первой точки в списке. Если точка существует, вызывается `battleground.GetCheckpointInfo()` для получения её детальной информации, после чего в лог выводятся имя и текущее состояние контрольной точки.
:::
