# battleground.GetCheckpoints
Извлекает список идентификаторов контрольных точек баттлграунда при участии аватара в сражении.

## Описание
```lua
battleground.GetCheckpoints(): table
```
Функция формирует таблицу, индексированную целыми числами начиная с нуля, и заполняет её уникальными идентификаторами (`ObjectId`) контрольных точек текущего баттлграунда. Возвращаемые данные актуальны только при условии участия главного персонажа в сражении. Если в структуре карты присутствуют объединенные контрольные точки, в результирующем списке содержится только одна из них. Идентификатор связанной контрольной точки извлекается посредством `battleground.GetLinkedCheckpoint`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, индексированную целыми числами начиная с `0`, содержащую `ObjectId` контрольных точек баттлграунда. 

## Примеры
### Получение информации о первой контрольной точке
```lua
local checkpoints = battleground.GetCheckpoints()
if checkpoints[0] ~= nil then
    local checkpointInfo = battleground.GetCheckpointInfo( checkpoints[0] )
    LogInfo( "Информация о первой контрольной точке: ", checkpointInfo )
end
```
::: info Описание примера
В примере извлекается список всех контрольных точек баттлграунда. Затем проверяется наличие хотя бы одной точки (с индексом `0`). Если точка существует, с помощью `battleground.GetCheckpointInfo` получается подробная информация о ней, которая выводится в лог.
:::
