# battleground.GetHeroesArenaQuitSpell
Возвращает идентификатор заклинания для выхода из боевой арены.

## Описание
```lua
battleground.GetHeroesArenaQuitSpell(): SpellId | nil
```
Функция возвращает идентификатор ресурса заклинания, используемого для выхода из боевой арены. Если заклинание не найдено или недоступно, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `SpellId` — идентификатор ресурса заклинания для выхода из боевой арены, или `nil`, если заклинание не определено.

## Примеры
### Применение заклинания для выхода из арены
```lua
local spellId = battleground.GetHeroesArenaQuitSpell()
if spellId then
    avatar.RunSpell( spellId )
end
```
::: info Описание примера
В примере запрашивается идентификатор заклинания для выхода из арены. Если идентификатор успешно получен (не равен `nil`), вызывается функция `avatar.RunSpell` для применения этого заклинания.
:::
