# battleground.GetHeroesArenaStartStageSpell
Возвращает идентификатор заклинания для запуска следующего этапа сценария боевой арены.

## Описание
```lua
battleground.GetHeroesArenaStartStageSpell(): SpellId | nil
```
Функция возвращает идентификатор ресурса заклинания (спелла), которое используется для инициации перехода к следующему этапу сценария боевой арены. Если идентификатор не найден или запуск следующего этапа в данный момент невозможен, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `SpellId` — идентификатор ресурса заклинания для запуска следующего этапа, или `nil`, если идентификатор не найден.

## Примеры
### Получение и запуск заклинания перехода на следующий этап
```lua
local spellId = battleground.GetHeroesArenaStartStageSpell()
if spellId then
    avatar.RunSpell( spellId )
end
```
::: info Описание примера
В примере сначала извлекается идентификатор заклинания с помощью функции `battleground.GetHeroesArenaStartStageSpell`. Если идентификатор успешно получен (не равен `nil`), вызывается функция `avatar.RunSpell` для фактического запуска заклинания, что инициирует переход к следующему этапу сценария.
:::
