# battleground.GetLinkedCheckpoint
Возвращает связанную контрольную точку баттлграунда для указанного идентификатора.

## Описание
```lua
battleground.GetLinkedCheckpoint( checkpointId: ObjectId ): ObjectId | nil
```
Функция извлекает и возвращает идентификатор контрольной точки, которая связана с указанной контрольной точкой. Возвращает актуальные данные только в случае, если главный персонаж принимает участие в текущем баттлграунде. Если связанная точка отсутствует или персонаж не участвует в баттлграунде, функция возвращает `nil`.

## Список параметров
- **`checkpointId`** (`ObjectId`)
Уникальный идентификатор контрольной точки, для которой необходимо получить связанную с ней точку.

## Возвращаемые значения
Возвращает `ObjectId` — уникальный идентификатор связанной контрольной точки, или `nil`, если связанная точка отсутствует, либо если аватар не участвует в баттлграунде.

## Примеры
### Получение связанной контрольной точки
```lua
local linkedPointId = battleground.GetLinkedCheckpoint( checkpointId )
```
