# battleground.GetPlayerMark
Извлекает идентификатор команды, к которой принадлежит игрок в сражении (баттлграунде).
## Описание
```lua
battleground.GetPlayerMark( playerId: ObjectId ): BattlegroundMarkId | nil
```
Функция возвращает идентификатор команды игрока, принимающего участие в сражении (баттлграунде), если его команда не определяется исключительно фракцией. В случае отсутствия игрока в сражении или определения его команды по фракции, функция возвращает `nil`.
## Список параметров
- **`playerId`** (`ObjectId`)
Уникальный идентификатор игрока, информацию о команде которого необходимо получить.
## Возвращаемые значения
Возвращает `BattlegroundMarkId` — идентификатор команды игрока, если он участвует в сражении и его команда не определяется фракцией. Возвращает `nil`, если игрок не участвует в сражении или его команда определяется фракцией.
## Примеры
### Получение и вывод системного имени команды игрока
```lua
local commandMarkId = battleground.GetPlayerMark( avatar.GetId() )
if commandMarkId then
    local commandInfo = commandMarkId:GetInfo()
    if commandInfo then
        LogInfo( "Вattleground command sysName: ", commandInfo.sysName )
    end
end
```
::: info Описание примера
В примере сначала извлекается идентификатор главного персонажа с помощью `avatar.GetId`, который передается в функцию `battleground.GetPlayerMark` для получения идентификатора команды. Если идентификатор команды успешно получен, вызывается метод `GetInfo` для извлечения подробной информации о команде, после чего в лог выводится её системное имя (`sysName`).
:::
