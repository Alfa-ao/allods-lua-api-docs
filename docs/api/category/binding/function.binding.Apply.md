# binding.Apply
Применяет и фиксирует новые значения клавиатурных привязок.

## Описание
```lua
binding.Apply()
```
Функция применяет новые клавиатурные настройки и фиксирует их. Поскольку механизм привязок работает в режиме автоматического предварительного просмотра (авто-превью), изменения в интерфейсе отображаются мгновенно при их выборе. Вызов данной функции подтверждает эти изменения на постоянной основе, что напрямую влияет на работу функции `binding.Cancel`, предотвращая откат настроек к предыдущим значениям.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Фиксация новых клавиатурных привязок
```lua
binding.Apply()
```
