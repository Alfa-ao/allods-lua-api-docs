# binding.ApplyRebind
Применяет распознанную горячую клавишу к текущей пользовательской конфигурации.

## Описание
```lua
binding.ApplyRebind()
```
Функция применяет распознанную горячую клавишу к текущей пользовательской конфигурации.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Применение распознанной горячей клавиши
```lua
if binding.HasRebind() then
    binding.ApplyRebind()
end
```
::: info Описание примера
В примере выполняется проверка наличия распознанной горячей клавиши с помощью функции `binding.HasRebind`. Если проверка возвращает истинное значение, вызывается функция `binding.ApplyRebind` для применения изменений в конфигурации.
:::
