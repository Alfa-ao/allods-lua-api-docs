# binding.Cancel
Отменяет внесенные изменения в настройки клавиатурных привязок, возвращая их к предыдущему состоянию.
## Описание
```lua
binding.Cancel()
```
Функция отменяет новые клавиатурные настройки, сбрасывая конфигурацию привязок к предыдущему зафиксированному состоянию. Предыдущим состоянием может быть как начальное состояние привязок, так и состояние по умолчанию, либо состояние, которое было успешно сохранено ранее с помощью функции `binding.Apply`.
## Список параметров
Параметры отсутствуют.
## Возвращаемые значения
Возвращаемое значение отсутствует.
## Примеры
### Отмена изменений привязок клавиш
```lua
binding.Cancel()
```
