# binding.CancelRebind
Отменяет режим распознавания горячей клавиши интерфейсом.

## Описание
```lua
binding.CancelRebind()
```
Функция отменяет активный режим распознавания горячей клавиши в пользовательском интерфейсе. При вызове прерывается состояние ожидания ввода, а также сбрасывается уже распознанное значение клавиши, если оно было успешно определено до момента отмены.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Отмена режима распознавания клавиши
```lua
binding.CancelRebind()
```
