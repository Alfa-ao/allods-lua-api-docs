# binding.GetBind
Извлекает информацию о пользовательском бинде.

## Описание
```lua
binding.GetBind( sysSectionName: string, sysBindName: string ): table | nil
```
Функция извлекает и возвращает детальную информацию о пользовательском бинде по заданным идентификаторам секции и самого бинда. Возвращаемые данные включают служебное название, локализованное имя и список назначенных комбинаций клавиш. Если указанный бинд не существует, функция возвращает `nil`.

::: warning Замечание
Таблица `keys` в возвращаемых данных содержит только непустые элементы, то есть может быть произвольного размера (вплоть до максимума), включая нулевой.
:::

## Список параметров
- **`sysSectionName`** (`string`)
Идентификатор секции, к которой принадлежит бинд.
- **`sysBindName`** (`string`)
Идентификатор (служебное название) искомого бинда.

## Возвращаемые значения
Возвращает таблицу (`table`) с информацией о бинде или `nil` в случае ошибки (при отсутствии бинда).
Таблица содержит следующие поля:
- **`sysName`** (`string`)
Служебное название (идентификатор) бинда (то же самое, что и `sysBindName`).
- **`name`** (`WString`)
Локализованное название бинда.
- **`keys`** (`table`)
Список комбинаций клавиш бинда. Индексация начинается с `0`. Может содержать произвольное количество элементов (вплоть до нуля).
  - (`table`) - список клавиш, из которых состоит данная комбинация. Индексация также начинается с `0`.
    - (`WString`) - локализованное представление клавиши.

## Примеры
### Получение и вывод информации о бинде
```lua
local bind = binding.GetBind( sysSectionName, sysBindName )
if bind then
    LogInfo( "sysName: ", bind.sysName )
    LogInfo( "name: ", userMods.FromWString( bind.name ) )
    for i = 0, GetTableSize( bind.keys ) - 1 do
        local keys = bind.keys[i]
        for j = 0, GetTableSize( keys ) - 1 do
            local key = keys[j]
            LogInfo( "    key ", j, ": ", userMods.FromWString( key ) )
        end
    end
end
```
::: info Описание примера
В примере выполняется запрос информации о бинде. При успешном получении данных в лог выводятся служебное и локализованное названия бинда. Далее происходит перебор всех назначенных комбинаций клавиш и вывод локализованных названий каждой клавиши в комбинации. Для преобразования строк широкого формата используется функция `userMods.FromWString`, а для определения размера таблиц — `GetTableSize`.
:::
