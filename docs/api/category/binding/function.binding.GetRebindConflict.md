# binding.GetRebindConflict
Возвращает информацию о первом обнаруженном конфликте пользовательских горячих клавиш.

## Описание
```lua
binding.GetRebindConflict(): table
```
Функция извлекает и возвращает таблицу с данными о первом встреченном конфликте горячих клавиш. Поиск конфликтов осуществляется исключительно среди пользовательских настроек. Информация о конфликте предоставляется только при наличии распознанной, но еще не примененной горячей клавиши, что соответствует условию `binding.HasRebind() == true`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` с информацией о конфликте, содержащую следующие поля:
- **`isConflict`** (`boolean`) — обнаружен ли конфликт.
- **`sysSection`** (`string`) — секция конфликта (присутствует при наличии конфликта).
- **`sysBind`** (`string`) — бинд конфликта (присутствует при наличии конфликта).

## Примеры
### Получение и проверка информации о конфликте
```lua
local conflict = binding.GetRebindConflict()
if conflict.isConflict then
    LogInfo( "sysSection = ", conflict.sysSection, ", sysBind = ", conflict.sysBind )
end
```
