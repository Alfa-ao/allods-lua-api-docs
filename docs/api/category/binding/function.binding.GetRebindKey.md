# binding.GetRebindKey
Возвращает системное имя распознанной горячей клавиши.
## Описание
```lua
binding.GetRebindKey(): string | nil
```
Функция возвращает системное имя горячей клавиши, если таковая была успешно распознана в процессе ожидания ввода.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `string` — системное имя горячей клавиши, или `nil`, если клавиша не была распознана.

## Примеры
### Проверка распознанной клавиши и обработка отмены
```lua
if binding.HasRebind() then
    if binding.GetRebindKey() == "ESC" then
        -- отменить изменение
    else
        -- принять изменение
    end
end
```
::: info Описание примера
В примере сначала проверяется наличие активного процесса переназначения с помощью функции `binding.HasRebind()`. Если процесс активен, извлекается системное имя распознанной клавиши через `binding.GetRebindKey()`. Если возвращенное имя равно `"ESC"`, инициируется отмена изменения, в противном случае изменение принимается.
:::
