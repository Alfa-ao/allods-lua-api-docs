# binding.GetSection
Возвращает информацию о секции с пользовательскими биндами.

## Описание
```lua
binding.GetSection( sysSectionName: string ): table | nil
```
Функция извлекает и возвращает детальную информацию о секции с пользовательскими биндами по её служебному идентификатору. Возвращаемая таблица содержит идентификатор, локализованное название секции и список привязанных биндов. Если секция с указанным идентификатором отсутствует, функция возвращает `nil`.

## Список параметров
- **`sysSectionName`** (`string`)
Служебный идентификатор секции, информацию о которой необходимо получить.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если секция найдена, возвращается таблица со следующими полями:
- **`sysName`** (`string`) — служебное название (идентификатор) секции, идентичное переданному параметру `sysSectionName`.
- **`name`** (`WString`) — локализованное название секции.
- **`bindNames`** (`table`) — таблица, содержащая список идентификаторов биндов данной секции. Индексация элементов в таблице начинается с нуля.

## Примеры
### Получение и вывод информации о секции биндов
```lua
local section = binding.GetSection( sysSectionName )
if section then
    LogInfo( "  sysName: ", section.sysName )
    LogInfo( "  name: ", userMods.FromWString( section.name ) )
    for i = 0, GetTableSize( section.bindNames ) - 1 do
        local sysBindName = section.bindNames[i]
        LogInfo( "   bind ", i, ": ", sysBindName )
    end
end
```
