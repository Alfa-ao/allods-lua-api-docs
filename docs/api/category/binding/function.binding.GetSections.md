# binding.GetSections
Возвращает таблицу с идентификаторами всех секций настроек, предназначенных для изменения игроком.

## Описание
```lua
binding.GetSections(): table
```
Функция возвращает таблицу, содержащую служебные названия (идентификаторы) всех секций настроек, которые могут быть изменены игроком. Индексация элементов в возвращаемой таблице начинается с нуля.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую идентификаторы секций. Индексация элементов начинается с нуля. Значения элементов представляют собой строки (`string`) — служебные названия секций.

## Примеры
### Получение и перебор списка доступных секций
```lua
local sections = binding.GetSections()
for i = 0, GetTableSize( sections ) - 1 do
    local sysSectionName = sections[i]
    LogInfo( " section ", i, ": ", sysSectionName )
end
```
