# binding.HasRebind
Проверяет наличие распознанной горячей клавиши.

## Описание
```lua
binding.HasRebind(): boolean
```
Функция возвращает логическое значение, указывающее на наличие распознанной горячей клавиши, ожидающей применения в системе привязок.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если имеется распознанная горячая клавиша, и `false` в противном случае.

## Примеры
### Проверка и применение распознанной горячей клавиши
```lua
if binding.HasRebind() then
    binding.ApplyRebind()
end
```
::: info Описание примера
В примере выполняется проверка наличия распознанной горячей клавиши с помощью `binding.HasRebind`. Если функция возвращает `true`, инициируется применение новой привязки через вызов `binding.ApplyRebind`.
:::
