# binding.IsWaitingRebind
Определяет, находится ли пользовательский интерфейс в режиме распознавания горячих клавиш.

## Описание
```lua
binding.IsWaitingRebind(): boolean
```
Функция возвращает логическое значение, указывающее на текущий статус интерфейса привязки клавиш. Результат принимает значение `true`, если UI ожидает ввода пользователем новой комбинации клавиш для переназначения, и `false`, если режим распознавания не активен.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если UI находится в режиме распознавания горячих клавиш, иначе `false`.

## Примеры
### Проверка состояния режима распознавания
```lua
if binding.IsWaitingRebind() then
    -- Показать специальную подсветку и комментарий
end
```
