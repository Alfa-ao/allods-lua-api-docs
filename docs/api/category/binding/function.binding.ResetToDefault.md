# binding.ResetToDefault
Сбрасывает клавиатурные настройки к значениям по умолчанию из базы данных.

## Описание
```lua
binding.ResetToDefault()
```
Функция сбрасывает все текущие клавиатурные привязки и настройки управления, восстанавливая их изначальные значения, сохраненные в базе данных игры.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Сброс настроек управления к стандартным значениям
```lua
binding.ResetToDefault()
```
