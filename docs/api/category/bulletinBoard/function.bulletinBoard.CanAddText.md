# bulletinBoard.CanAddText
Проверяет возможность размещения нового объявления на доске в указанном разделе.

## Описание
```lua
bulletinBoard.CanAddText( postTypeId: PostTypeId ): table
```
Функция анализирует текущее состояние и ограничения для главного персонажа, определяя, разрешено ли создание нового текстового объявления в заданном разделе доски. Возвращает таблицу с детальным отчетом о возможности добавления текста и причинах возможных ограничений.

## Список параметров
- **`postTypeId`** (`PostTypeId`)
Уникальный идентификатор раздела доски объявлений, в котором планируется размещение текста.

## Возвращаемые значения
Возвращает `table`, содержащую следующие поля:
- **`isCanAdd`** (`boolean`)
Флаг, указывающий на возможность создания новых объявлений. Если значение равно `false`, причина уточняется в последующих полях возвращаемой таблицы.
- **`isMsgLimitReached`** (`boolean`)
Флаг, принимающий значение `true`, если для данного персонажа достигнут максимальный предел сообщений в разделе.
- **`sysCause`** (`string`)
Строковый идентификатор причины, соответствующий одному из значений перечисления `ENUM_ActionFailCause_...`. Если ошибка вызвана иными причинами или ограничений нет, возвращается `ENUM_ActionFailCause_NoFail`.
- **`isInNotPredicate`** (`boolean`)
Флаг, принимающий значение `true`, если причина невозможности добавления определяется внутренним предикатом с противоположным значением `sysCause`.

## Примеры
### Проверка возможности и добавление объявления
```lua
local postTypeId = 1 -- Идентификатор раздела доски объявлений
local text = userMods.ToWString( "Текст объявления" )

local canAddResult = bulletinBoard.CanAddText( postTypeId )
if canAddResult.isCanAdd then
    bulletinBoard.AddText( postTypeId, text )
else
    LogInfo( "Невозможно добавить объявление. Причина: ", canAddResult.sysCause )
end
```
::: info Описание примера
В примере сначала запрашивается возможность добавления текста в указанный раздел. Если поле `isCanAdd` возвращаемой таблицы имеет значение `true`, инициируется добавление объявления с помощью `bulletinBoard.AddText`. В противном случае в лог выводится причина отказа, содержащаяся в поле `sysCause`.
:::
