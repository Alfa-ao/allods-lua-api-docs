# bulletinBoard.GetPost
Возвращает информацию об указанном объявлении на доске объявлений.

## Описание
```lua
bulletinBoard.GetPost( postId: number ): table | nil
```
Функция извлекает и возвращает детальную информацию о конкретном объявлении на доске объявлений по его уникальному идентификатору. Возвращаемые данные включают идентификаторы, информацию об авторе, текстовое содержимое и временные параметры существования объявления. Если объявление с указанным идентификатором не найдено (например, было удалено или истек срок его жизни), функция возвращает `nil`.

## Список параметров
- **`postId`** (`number`)
Уникальный идентификатор объявления на доске.

## Возвращаемые значения
Возвращает таблицу (`table`) с информацией об объявлении или `nil`, если объявление не найдено.
Поля возвращаемой таблицы:
- **`postId`** (`number`) — уникальный идентификатор объявления.
- **`isPremium`** (`boolean`) — `true`, если это платное (премиальное) сообщение.
- **`name`** (`WString`) — имя персонажа, создавшего объявление.
- **`guild`** (`WString`) — название гильдии персонажа (если персонаж состоит в гильдии).
- **`isOnline`** (`boolean`) — `true`, если персонаж в данный момент находится в игре.
- **`level`** (`number`) — уровень персонажа.
- **`raceClass`** (`table` | `nil`) — информация о расе и классе персонажа (структура `LuaRaceClassInfoPart`).
- **`sex`** (`table` | `nil`) — описание пола персонажа (структура `LuaSexInfoPart`).
- **`zoneName`** (`WString`) — название зоны, где в данный момент находится персонаж.
- **`text`** (`WString`) — текст объявления, если он уже был получен.
- **`lifeTimeMs`** (`number` | `nil`) — общее время существования объявления в миллисекундах или `nil` в случае ошибки.
- **`lifeTime`** (`table` | `nil`) — время существования объявления в виде таблицы или `nil` в случае ошибки. Содержит поля: `d` (дни), `h` (часы), `m` (минуты), `s` (секунды).
- **`remainingTimeMs`** (`number` | `nil`) — оставшееся время существования объявления в миллисекундах или `nil`, если ограничений по времени нет.
- **`remainingTime`** (`table` | `nil`) — оставшееся время существования объявления в виде таблицы или `nil`, если ограничений по времени нет. Содержит поля: `d` (дни), `h` (часы), `m` (минуты), `s` (секунды).

## Примеры
### Получение и проверка информации об объявлении
```lua
local post = bulletinBoard.GetPost( postId )
if post then
    LogInfo( "name: ", post.name )
end
```
