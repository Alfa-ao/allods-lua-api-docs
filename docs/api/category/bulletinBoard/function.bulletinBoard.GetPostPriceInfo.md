# bulletinBoard.GetPostPriceInfo
Возвращает информацию о стоимости размещения платного сообщения.

## Описание
```lua
bulletinBoard.GetPostPriceInfo(): table | nil
```
Функция возвращает таблицу с информацией о стоимости размещения платного сообщения на доске объявлений. В таблице указаны идентификатор требуемой валюты и её количество. Если информация не может быть получена, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`:
- **`currencyId`** (`CurrencyId`)
Идентификатор денежной единицы, используемой для оплаты размещения.
- **`currencyValue`** (`number`)
Стоимость размещения одного объявления в указанных единицах валюты.

Если информация о цене недоступна, возвращается `nil`.

## Примеры
### Получение и вывод информации о валюте размещения объявления
```lua
local postPriceInfo = bulletinBoard.GetPostPriceInfo()
local currency = avatar.GetCurrencyInfo( postPriceInfo.currencyId )
if currency then
    LogInfo( "валюта: ", currency.name )
end
```
::: info Описание примера
В примере запрашивается информация о стоимости объявления. Если данные успешно получены, извлекается идентификатор валюты, который передается в функцию `avatar.GetCurrencyInfo` для получения полного описания денежной единицы. При наличии информации о валюте её название выводится в лог.
:::
