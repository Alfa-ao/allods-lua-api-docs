# bulletinBoard.GetSectionInfos
Возвращает информацию о доступных разделах доски объявлений.

## Описание
```lua
bulletinBoard.GetSectionInfos(): table
```
Функция возвращает таблицу, содержащую информацию о доступных разделах доски объявлений. Элементы результирующей таблицы индексированы начиная с нуля. Значениями элементов выступают вложенные таблицы, содержащие детальные сведения о каждом разделе.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую информацию о доступных разделах. Элементы таблицы индексированы начиная с нуля. Значения представляют собой таблицы со следующими полями:
- **`postTypeId`** (`PostTypeId`) — идентификатор раздела доски объявлений.
- **`sysName`** (`string`) — служебное название раздела, используемое для уникальной обработки.
- **`name`** (`WString`) — локализованное название раздела.
- **`description`** (`WString`) — локализованное описание раздела.
- **`lengthLimit`** (`number`) — максимальный размер одного объявления в символах (целое число).

## Примеры
### Получение и перебор списка доступных разделов
```lua
local sectionInfos = bulletinBoard.GetSectionInfos()
for i = 0, GetTableSize( sectionInfos ) - 1 do
    local sectionInfo = sectionInfos[i]
    LogInfo( "раздел: ", sectionInfo.name )
end
```
