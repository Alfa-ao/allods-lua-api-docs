# bulletinBoard.GetText
Возвращает текст объявления на доске объявлений.

## Описание
```lua
bulletinBoard.GetText( postId: number ): WString | nil
```
Функция извлекает и возвращает текстовое содержимое указанного объявления на доске. Если объявление с заданным идентификатором не найдено или текст еще не загружен, функция возвращает `nil`.

## Список параметров
- **`postId`** (`number`)
Уникальный целочисленный идентификатор объявления на доске, текст которого необходимо получить.

## Возвращаемые значения
Возвращает `WString` с текстом объявления, если оно успешно найдено, или `nil`, если объявление отсутствует или данные еще не получены.

## Примеры
### Получение и проверка текста объявления
```lua
local text = bulletinBoard.GetText( postId )
if text then
    LogInfo( "Текст объявления: ", text )
else
    LogInfo( "Ожидание загрузки текста объявления." )
end
```
