# bulletinBoard.IsOperationInProgress
Определяет, выполняется ли в данный момент операция с доской объявлений.

## Описание
```lua
bulletinBoard.IsOperationInProgress(): boolean
```
Функция возвращает логическое значение, указывающее на текущий статус асинхронной операции с доской объявлений, такой как размещение нового, редактирование или удаление существующего объявления.

::: warning Замечание
Начинать новую операцию до окончания предыдущей нельзя.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если операция с доской объявлений выполняется в данный момент, и `false` в противном случае.

## Примеры
### Проверка статуса перед выполнением действия
```lua
if bulletinBoard.IsOperationInProgress() then
    -- запретить редактирование и удаление объявлений
end
```
