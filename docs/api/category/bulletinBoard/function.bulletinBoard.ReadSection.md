# bulletinBoard.ReadSection
Возвращает информацию о содержимом указанного раздела доски объявлений.

## Описание
```lua
bulletinBoard.ReadSection( postTypeId: PostTypeId ): table | nil
```
Функция извлекает и возвращает информацию о содержимом указанного раздела доски объявлений. Если запрашиваемая информация еще отсутствует на клиенте, инициируется отправка запроса на сервер, а функция возвращает `nil`. После получения данных от сервера передается событие об изменении доски объявлений.

## Список параметров
- **`postTypeId`** (`PostTypeId`)
Уникальный идентификатор раздела доски объявлений.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую список идентификаторов объявлений, или `nil`, если информация на данный момент недоступна.

Таблица индексирована, начиная с `0`. Значения элементов таблицы:
- `number` (`int`) — уникальный идентификатор объявления.

## Примеры
### Получение и перебор списка объявлений в разделе
```lua
local postIds = bulletinBoard.ReadSection( postTypeId )
if postIds then
    for i = 0, GetTableSize( postIds ) - 1 do
        local post = bulletinBoard.GetPost( postIds[i] )
    end
end
```
