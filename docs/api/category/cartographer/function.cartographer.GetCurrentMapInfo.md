# cartographer.GetCurrentMapInfo
Возвращает описание текущей карты персонажа.

## Описание
```lua
cartographer.GetCurrentMapInfo(): table | nil
```
Функция возвращает таблицу с детальным описанием карты, на которой в данный момент находится персонаж, или `nil` в случае возникновения внутренней ошибки. Возвращаемые данные включают базовые характеристики локации: имя, описание, тип поверхности (земля или астрал), признак астрального острова, геометрический размер в метрах и статус стартового инстанса. При использовании отладочного клиента структура дополнительно пополняется системными идентификаторами карты и зоны.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если внутренняя ошибка отсутствует, возвращается таблица со следующими полями:
- **`name`** (`WString`) — имя карты.
- **`description`** (`WString`) — описание карты.
- **`isTerrain`** (`boolean`) — `true`, если это карта земли, иначе — астрал.
- **`isAstralIsland`** (`boolean`) — `true`, если это карта астрального острова, иначе — `false`.
- **`size`** (`number`) — размер стороны карты в метрах. Начало координат находится в левом нижнем углу карты.
- **`isStartInstance`** (`boolean`) — `true`, если это стартовый инстанс. В стартовом инстансе запрещены сохранения серверной информации (не работают магазин, аукцион, почта, обмен и т.п.).

Следующие поля доступны исключительно в отладочном клиенте:
- **`mapSysName`** (`string`) — отладочное системное имя карты.
- **`zoneSysName`** (`string`) — отладочное системное имя зоны.

Возвращает `nil`, если произошла внутренняя ошибка при получении данных.

## Примеры
### Проверка статуса стартового инстанса
```lua
local map = cartographer.GetCurrentMapInfo()
if map then
    if map.isStartInstance then
        LogInfo( "торговля запрещена" )
    end
end
```
