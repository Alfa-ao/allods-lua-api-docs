# cartographer.GetCurrentZoneAttributes
Возвращает таблицу строковых атрибутов зоны, в которой находится главный игрок.

## Описание
```lua
cartographer.GetCurrentZoneAttributes(): table
```
Функция возвращает таблицу, содержащую строковые атрибуты зоны, в которой находится главный игрок. Атрибуты задаются в ресурсах зон и собираются как с самой зоны, так и с её родительских зон. В полученной таблице по индексу строкового атрибута находится значение `true`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, в которой ключами выступают строковые атрибуты, а значениями является `true` для имеющихся атрибутов.

## Примеры
### Проверка наличия специфического атрибута зоны
```lua
if cartographer.GetCurrentZoneAttributes()[ "Goblinoball" ] then
    LogInfo( "Это зона для гоблинобола." )
end
```
