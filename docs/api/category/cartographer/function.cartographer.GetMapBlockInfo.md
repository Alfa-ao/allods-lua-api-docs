# cartographer.GetMapBlockInfo
Возвращает описание блока интерфейсных карт.
## Описание
```lua
cartographer.GetMapBlockInfo( mapBlockId: ObjectId ): table | nil
```
Функция извлекает и возвращает детальную информацию о блоке интерфейсных карт. Блок объединяет одну или несколько интерфейсных карт зон (например, в блок «Кания» входят карты «Новоград», «Светлолесье», «Сиверия» и другие). В итоговом списке блока отображаются только те карты, у которых параметр `isShowInList` имеет значение `true`. Если блок с указанным идентификатором не найден, функция возвращает `nil`.

## Список параметров
- **`mapBlockId`** (`ObjectId`)
Уникальный идентификатор блока интерфейсных карт.

## Возвращаемые значения
Возвращает таблицу (`table`) с информацией о блоке или `nil`, если блок не найден.

Структура возвращаемой таблицы:
- **`id`** (`ObjectId`) — уникальный идентификатор блока.
- **`name`** (`WString`) — имя блока.
- **`description`** (`WString`) — описание блока.
- **`isShowInList`** (`boolean`) — `true`, если блок отображается в главном списке блоков.
- **`zonesMaps`** (`table`) — таблица, содержащая идентификаторы (`ObjectId`) интерфейсных карт зон, входящих в данный блок.

## Примеры
### Получение и перебор информации о блоках карт
```lua
local mapBlocks = cartographer.GetMapBlocks()
for i = 0, GetTableSize( mapBlocks ) - 1 do
    local mapBlockInfo = cartographer.GetMapBlockInfo( mapBlocks[ i ] )
    if mapBlockInfo then
        local name = mapBlockInfo.name
    end
end
```
::: info Описание примера
В примере сначала извлекается список всех доступных блоков карт с помощью `cartographer.GetMapBlocks()`. Затем в цикле перебирается каждый идентификатор блока. Для каждого идентификатора вызывается `cartographer.GetMapBlockInfo()`. Если информация успешно получена, извлекается имя блока (`name`).
:::
