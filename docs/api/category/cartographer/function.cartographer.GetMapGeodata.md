# cartographer.GetMapGeodata
Возвращает геодату для проецирования координат интерактивного объекта реального мира в координаты текстуры интерфейсной карты.

## Описание
```lua
cartographer.GetMapGeodata( zonesMapId: ObjectId ): table
```
Функция извлекает и возвращает геодату, необходимую для проецирования координат интерактивного объекта реального мира в координаты текстуры интерфейсной карты.

## Список параметров
- **`zonesMapId`** (`ObjectId`)
Идентификатор интерфейсной карты зоны, геодата которой запрашивается.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую геодату (Geodata).

## Примеры
### Получение геодаты текущей зоны
```lua
local geodata = cartographer.GetMapGeodata( currentZonesMapId )
LogInfo( geodata )
```
