# cartographer.GetZonesMapInfo
Возвращает интерфейсные данные карты зоны по её идентификатору.

## Описание
```lua
cartographer.GetZonesMapInfo( zonesMapId: ObjectId | nil ): table | nil
```
Функция извлекает и возвращает таблицу с интерфейсными данными карты зоны. Если карта с указанным идентификатором не найдена, возвращается `nil`. В случае передачи `nil` в качестве параметра, используется идентификатор текущей зоны.

## Список параметров
- **`zonesMapId`** (`ObjectId` | `nil`)
Идентификатор интерфейсной карты. Если передано значение `nil`, функция использует идентификатор текущей зоны.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если карта найдена, возвращается таблица со следующими полями:
- **`id`** (`ObjectId`)
Уникальный идентификатор карты.
- **`sysName`** (`string`)
Уникальное внутреннее имя карты.
- **`name`** (`WString`)
Отображаемое имя карты.
- **`description`** (`WString`)
Описание карты.
- **`mapBlockId`** (`ObjectId` | `nil`)
Идентификатор блока карт, в который входит данная карта.
- **`isShowInList`** (`boolean`)
`true`, если карта отображается в списке карт своего блока.
- **`isShowPositions`** (`boolean`)
`true`, если на карте допускается отображение объектов (доступна геодата для них).
- **`isShowQuestProgress`** (`boolean`)
`true`, если для карты отображается количество выполненных и общих важных заданий. Для таких карт список квестов является пустым.
- **`isUnknown`** (`boolean`)
`true`, если карта представляет собой неизвестные места (астрал, закрытые, ненайденные или ошибочные области).
- **`isNeedTeleportUnlock`** (`boolean`)
`true`, если для карты требуется разблокировка портала (требуется, но еще не получена).
- **`minLevel`** (`number` | `nil`)
Минимальный комфортный уровень игрока для данной карты. Может отсутствовать.
- **`maxLevel`** (`number` | `nil`)
Максимальный комфортный уровень игрока для данной карты. Может отсутствовать.
- **`zoneCeil`** (`number` | `nil`)
Верхний вертикальный предел зоны.
- **`zoneFloor`** (`number` | `nil`)
Нижний вертикальный предел зоны.
- **`texture`** (`TextureId`)
Идентификатор текстуры карты (реализовано только для UI).
- **`sysMapResources`** (`table`)
Массив имен отладочных карт, входящих в состав интерфейсной карты (доступно только в отладочном клиенте).

## Примеры
### Установка фоновой текстуры карты
```lua
local mapInfo = cartographer.GetZonesMapInfo( zonesMapId )
if mapInfo then
    wtMap:SetBackgroundTexture( mapInfo.texture )
end
```
