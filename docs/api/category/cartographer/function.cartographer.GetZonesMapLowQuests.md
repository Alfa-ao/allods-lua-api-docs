# cartographer.GetZonesMapLowQuests
Извлекает информацию о неважных заданиях, относящихся к интерфейсной карте и не участвующих в общем прогрессе.

## Описание
```lua
cartographer.GetZonesMapLowQuests( zonesMapId: ObjectId ): table | nil
```
Функция формирует и возвращает структуру данных, содержащую детальную информацию о неважных заданиях для указанной интерфейсной карты. Возвращаемая таблица включает общее количество заданий, количество активных и завершенных заданий, а также списки идентификаторов ресурсов для активных и завершенных заданий. Если указанная карта не существует или данные недоступны, функция возвращает `nil`.

## Список параметров
- **`zonesMapId`** (`ObjectId`)
Уникальный идентификатор интерфейсной карты.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если карта существует, возвращается таблица со следующими полями:
- **`totalCount`** (`number`) — общее количество заданий, относящихся к локации.
- **`currentCount`** (`number`) — количество заданий, активных в данный момент.
- **`completedCount`** (`number`) — количество завершенных заданий.
- **`currentQuests`** (`table`) — список (индексирован с 1) идентификаторов (`QuestId`) ресурсов активных заданий, относящихся к карте.
- **`completedQuests`** (`table`) — список (индексирован с 1) идентификаторов (`QuestId`) ресурсов завершенных заданий, относящихся к карте.
Возвращает `nil`, если указанная карта не существует.

## Примеры
### Получение информации о неважных заданиях и обработка первого активного задания
```lua
local zoneLowQuests = cartographer.GetZonesMapLowQuests( zonesMapId )
if zoneLowQuests then
    local firstCurrentQuest = zoneLowQuests.currentQuests[ 1 ]
    if firstCurrentQuest then
        local questInfo = avatar.GetQuestInfo( firstCurrentQuest )
    end
end
```
::: info Описание примера
В примере извлекается информация о неважных заданиях для указанной интерфейсной карты. Если данные успешно получены, проверяется наличие активных заданий. При наличии хотя бы одного активного задания извлекается идентификатор первого задания, который затем используется для получения полной информации о задании через функцию `avatar.GetQuestInfo`.
:::
