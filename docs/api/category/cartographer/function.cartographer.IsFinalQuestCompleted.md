# cartographer.IsFinalQuestCompleted
Определяет, выполнено ли финализирующее задание интерфейсной карты.

## Описание
```lua
cartographer.IsFinalQuestCompleted( zonesMapId: ObjectId ): boolean
```
Функция проверяет статус выполнения финализирующего задания для указанной интерфейсной карты. Финализирующим заданием считается задание, выполнение которого достаточно для утверждения, что прогресс соответствующей зоны полностью завершен.

## Список параметров
- **`zonesMapId`** (`ObjectId`)
Идентификатор интерфейсной карты.

## Возвращаемые значения
Возвращает `boolean` — `true`, если финализирующее задание данной карты выполнено, и `false` в противном случае.

## Примеры
### Проверка статуса финализирующего задания
```lua
local isFinalQuestCompleted = cartographer.IsFinalQuestCompleted( zonesMapId )
```
