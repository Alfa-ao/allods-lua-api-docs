# cartographer.IsOnCommon
Проверяет, является ли текущая карта межсерверной.

## Описание
```lua
cartographer.IsOnCommon(): boolean
```
Функция возвращает логическое значение, указывающее на то, является ли текущая карта межсерверной. Результат принимает значение `true`, если локация является межсерверной, и `false` в противном случае.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если текущая карта является межсерверной, и `false`, если карта является локальной или внутрисерверной.

## Примеры
### Проверка типа текущей карты
```lua
local isOnCommonMap = cartographer.IsOnCommon()
if isOnCommonMap then
    -- Выполнение действий, специфичных для межсерверной карты
end
```
