# changeRoom.ChargenShowHelm

Устанавливает признак отображения шлемов на персонажах в экране создания персонажей.

## Описание

```lua
changeRoom.ChargenShowHelm( show: boolean )
```

Функция устанавливает признак отображения шлемов на персонажах в экране создания персонажей. По умолчанию персонажи отображаются без шлемов. Функция реализована в UI.

::: warning Замечание
Рекомендуется состояние показа или скрытия шлема всегда задавать явно из скрипта.
:::

## Список параметров

- **`show`** (`boolean`)
Определяет, отображается ли шлем на персонаже в экране создания.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Включение отображения шлемов

```lua
changeRoom.ChargenShowHelm( true )
```
