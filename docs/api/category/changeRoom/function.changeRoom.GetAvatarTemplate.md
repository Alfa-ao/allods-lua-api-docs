# changeRoom.GetAvatarTemplate
Возвращает информацию о шаблоне аватара и доступных вариациях персонажей.

## Описание
```lua
changeRoom.GetAvatarTemplate( isMale: boolean ): table | nil
```
Функция извлекает и возвращает детальную информацию об указанном шаблоне аватара вместе с данными о вариациях персонажей. Реализована в UI.

## Список параметров
- **`isMale`** (`boolean`)
Задает пол персонажа: `true`, если требуются настройки для мужского персонажа, иначе для женского.

## Возвращаемые значения
Возвращает `table` или `nil`. Если шаблон найден, возвращается таблица со следующими полями, иначе `nil`:

- **`hasActualVariationInfo`** (`boolean`)
Определяет, содержит ли таблица актуальную информацию о вариациях персонажей. Информация может быть неактуальной в случае ошибок данных, либо если она не была запрошена, либо еще не получена с сервера.
- **`primaryVariationTemplateInfo`** (`table`)
Содержит таблицу с количеством вариаций для главного персонажа. Структура полей описана ниже.
- **`isTrio`** (`boolean`)
Указывает, представляет ли персонаж собой троицу (например, гибберлинги). При значении `true` для троицы возвращаются дополнительные поля.
- **`maleVariationTemplateInfo`** (`table`)
(Для троицы) Содержит таблицу с количеством вариаций для дополнительных персонажей мужского пола.
- **`femaleVariationTemplateInfo`** (`table`)
(Для троицы) Содержит таблицу с количеством вариаций для дополнительных персонажей женского пола.
- **`pets`** (`table | nil`)
При наличии настраиваемого питомца содержит таблицу с таблицами описаний обликов питомцев, каждая из которых включает количество вариаций для него, иначе `nil`. Для питомца реально используется только поле `face`.

Поля таблиц с количеством вариаций (`primaryVariationTemplateInfo`, `maleVariationTemplateInfo`, `femaleVariationTemplateInfo` и таблиц питомцев):

- **`skins`** (`table`)
Таблица флагов, описывающая доступные вариации тела.
- **`skinColors`** (`table`)
Таблица флагов, описывающая доступные вариации цвета тела.
- **`hairs`** (`table`)
Таблица флагов, описывающая доступные прически.
- **`hairColors`** (`table | nil`)
Таблица флагов, описывающая доступные цвета волос, или `nil`, если они отсутствуют.
- **`faces`** (`table`)
Таблица флагов, описывающая доступные варианты лица.
- **`facials`** (`table`)
Таблица флагов, описывающая доступные типы особенностей (лица или украшений).
- **`additionals`** (`table`)
Таблица флагов, описывающая доступные варианты татуировок.
- **`morphPresets`** (`table`)
Таблица флагов, описывающая доступные варианты пропорций персонажа.

Каждая из перечисленных таблиц флагов представляет собой индексированную `[0..]` таблицу пар `<number, boolean>`, где `true` обозначает платный вариант (серый список).

## Примеры
### Получение информации о шаблоне мужского аватара
```lua
local avatarTemplate = changeRoom.GetAvatarTemplate( true )
if avatarTemplate then
    LogInfo( "primary skin count: ", avatarTemplate.primaryVariationTemplateInfo.skins )
end
```
