# changeRoom.GetSceneAvatarInfo
Извлекает информацию о персонажах игрока, размещенных на сцене.

## Описание
```lua
changeRoom.GetSceneAvatarInfo(): table | nil
```
Функция возвращает таблицу с идентификаторами основного и дополнительных персонажей, а также питомца, находящихся на сцене комнаты смены облика. Если на сцене отсутствует персонаж, возвращается `nil`. Для рас, обладающих дополнительными моделями (например, гибберлинги), предусмотрены отдельные поля для второго и третьего персонажей.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если на сцене присутствует персонаж, возвращается таблица со следующими полями:
- **`primaryId`** (`ObjectId` | `nil`) — идентификатор основного персонажа.
- **`trio2Id`** (`ObjectId` | `nil`) — идентификатор второго (дополнительного) персонажа (актуально для рас, имеющих три модели, например, гибберлингов).
- **`trio3Id`** (`ObjectId` | `nil`) — идентификатор третьего (дополнительного) персонажа (актуально для рас, имеющих три модели).
- **`petId`** (`ObjectId` | `nil`) — идентификатор питомца.

Если персонаж на сцене отсутствует, возвращается `nil`.

## Примеры
### Позиционирование питомца на сцене
```lua
local avatarInfo = changeRoom.GetSceneAvatarInfo()
if avatarInfo and avatarInfo.petId then
    changeRoom.SetSceneCharacterPos( avatarInfo.petId, { posX = 1.0, posY = 2.0, posZ = 0.0 } )
end
```
::: info Описание примера
В примере извлекается информация о персонажах на сцене. При наличии питомца вызывается функция `changeRoom.SetSceneCharacterPos` для изменения его пространственных координат.
:::
