# changeRoom.GetTicketCurrencyId
Возвращает идентификатор валюты, используемой для оплаты услуг комнаты перемен.

## Описание
```lua
changeRoom.GetTicketCurrencyId(): CurrencyId | nil
```
Функция возвращает идентификатор валюты, которая используется для оплаты услуг комнаты перемен. При возникновении ошибки или отсутствии валюты возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
- **`CurrencyId`** — идентификатор ресурса альтернативной валюты.
- **`nil`** — если произошла ошибка или валюта не определена.

## Примеры
### Получение идентификатора валюты
```lua
local ticketId = changeRoom.GetTicketCurrencyId()
```
