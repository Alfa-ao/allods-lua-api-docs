# changeRoom.IsZoomEnabled
Определяет доступность масштабирования лица при выборе или создании персонажа.

## Описание
```lua
changeRoom.IsZoomEnabled(): boolean
```
Функция возвращает логическое значение, указывающее на возможность масштабирования лица при выборе или создании персонажа. При положительном результате также разрешается изменение масштаба с помощью колеса мыши.

::: warning Замечание
При изменении состояния доступности приходит событие `EVENT_REMORT_ZOOM_TRIGGER_CHANGED`.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если масштабирование лица доступно, и `false` в противном случае.

## Примеры
### Проверка доступности и переключение режима масштабирования
```lua
if changeRoom.IsZoomEnabled() and not changeRoom.IsZoomOnFace() then
    changeRoom.ToggleZoom()
end
```
::: info Описание примера
В примере запрашивается текущее состояние доступности масштабирования и проверяется, не активен ли уже режим зума на лице. Если масштабирование разрешено и в данный момент не включено, вызывается функция переключения режима зума.
:::
