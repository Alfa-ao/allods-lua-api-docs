# changeRoom.RemoveScene
Убирает ранее созданную сцену, если она существует.

## Описание
```lua
changeRoom.RemoveScene()
```
Функция убирает ранее созданную сцену. Может использоваться при работе с несколькими элементами `Control3DSafe`, например, при одновременной работе разных аддонов, чтобы избежать конфликтов отображения и корректно освобождать ресурсы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Очистка сцены
```lua
changeRoom.RemoveScene()
```
