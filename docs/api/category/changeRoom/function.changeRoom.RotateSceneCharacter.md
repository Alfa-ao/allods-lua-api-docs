# changeRoom.RotateSceneCharacter
Поворачивает выбранного персонажа в сцене.

## Описание
```lua
changeRoom.RotateSceneCharacter( characterId: ObjectId, angle: number )
```
Функция поворачивает выбранного персонажа (основного или дополнительного из трио, включая питомца) в сцене на заданный угол.

## Список параметров
- **`characterId`** (`ObjectId`)
Идентификатор персонажа в сцене, которого требуется повернуть.
- **`angle`** (`number`)
Угол поворота в градусах.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Поворот основного персонажа на заданный угол
```lua
changeRoom.RotateSceneCharacter( primaryCharacterId, 20 )
```
