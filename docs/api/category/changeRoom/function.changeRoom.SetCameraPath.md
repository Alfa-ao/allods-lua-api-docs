# changeRoom.SetCameraPath

Запускает движение камеры по заданному пути в режиме парикмахерской.

## Описание

```lua
changeRoom.SetCameraPath( cameraPath: string, targetPath: string, delay: number )
```

Функция запускает камеру по указанному маршруту в режиме парикмахерской. Камера перемещается по маршруту `cameraPath`, а точка фокусировки (прицел) движется по маршруту `targetPath`. Маршруты представляют собой файлы, расположенные в директории `Maps/<Map>/GlobalObjects/Routes/...`, и указываются по имени без пути и расширения. Начало движения можно отложить на заданное количество секунд через параметр `delay`. После завершения движения камеры событие `EVENT_CHANGE_ROOM_CHARACTER_SCENE_CAMERA_READY` приходит на клиент.

::: warning Замечание
Функция реализована на стороне UI и доступна только в режиме парикмахерской.
:::

## Список параметров

- **`cameraPath`** (`string`)
Имя маршрута движения камеры. Указывается без пути и расширения файла.

- **`targetPath`** (`string`)
Имя маршрута движения прицела (точки фокусировки) камеры. Указывается без пути и расширения файла.

- **`delay`** (`number`)
Задержка в секундах перед началом движения камеры.

## Возвращаемые значения

Возвращаемое значение отсутствует.
