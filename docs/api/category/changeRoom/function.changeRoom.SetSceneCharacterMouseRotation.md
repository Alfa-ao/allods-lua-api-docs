# changeRoom.SetSceneCharacterMouseRotation
Включает поворот выбранного персонажа в сцене с помощью мыши.

## Описание
```lua
changeRoom.SetSceneCharacterMouseRotation( characterId: ObjectId )
```
Функция разрешает управление ориентацией выбранного персонажа (основного, дополнительного из трио или питомца) в сцене посредством перемещения мыши. Поворот применяется только к указанному идентификатору персонажа.

::: warning Замечание
Функция реализована на стороне пользовательского интерфейса (UI).
:::

## Список параметров
- **`characterId`** (`ObjectId`)
Уникальный идентификатор персонажа в сцене, для которого необходимо активировать поворот мышью. Значение не может быть `nil`.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Включение поворота мышью для основного персонажа
```lua
changeRoom.SetSceneCharacterMouseRotation( primaryCharacterId )
```
