# changeRoom.ToggleZoom
Переключает состояние приближения камеры к лицу персонажа в экранах выбора или создания персонажа.

## Описание
```lua
changeRoom.ToggleZoom()
```
Функция переключает режим масштабирования камеры. Если камера не приближена к лицу персонажа, она перемещается в крайнюю ближнюю точку. Если камера уже находится в этой точке, она отдаляется. Применяется исключительно в интерфейсах выбора или создания нового персонажа.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Безопасное переключение масштаба камеры
```lua
if changeRoom.IsZoomEnabled() and not changeRoom.IsZoomOnFace() then
    changeRoom.ToggleZoom()
end
```
::: info Описание примера
В примере выполняется проверка доступности функции масштабирования и текущего состояния камеры. Если приближение разрешено и камера еще не находится на лице персонажа, вызывается функция переключения масштаба.
:::
