# changeRoom.Use
Применяет изменения внешнего вида, выбранные в парикмахерской, для основного и дополнительных персонажей, а также питомца.

## Описание
```lua
changeRoom.Use( isMale: boolean, primary: table, trio2: table|nil, trio3: table|nil, pet: table|nil, petIndex: number|nil, price: number )
```
Функция применяет изменения внешнего вида, выбранные в парикмахерской. Реализована на стороне пользовательского интерфейса (UI). Передает индексы вариаций внешнего вида для основного персонажа, дополнительных персонажей (в случае создания трио) и питомца. Значение цены обязательно для передачи на сервер с целью проверки стоимости операции.

## Список параметров
- **`isMale`** (`boolean`)
Пол главного персонажа.
- **`primary`** (`table`)
Таблица с информацией, задающей вариацию для основного персонажа игрока.
- `skinTexture` (`number`) - индекс вариации тела [0..].
- `skinColor` (`number`) - индекс вариации цвета тела [0..].
- `hairType` (`number`) - индекс типа волос [0..].
- `hairColor` (`number`) - индекс цвета волос [0..]. Может игнорироваться (например, для гибберлингов).
- `face` (`number`) - индекс типа лица [0..].
- `facialType` (`number`) - индекс типа особенностей лица или украшений [0..].
- `additional` (`number`) - индекс варианта татуировки [0..].
- `morphPresetIndex` (`number`) - индекс варианта пропорций персонажа [0..].
- **`trio2`** (`table` | `nil`)
Таблица с информацией, задающей вариацию для второго персонажа игрока. Если трио не создается, передается `nil`. Содержит поля, аналогичные `primary`, а также дополнительные:
- `name` (`WString`) - имя персонажа.
- `isMale` (`boolean`) - пол персонажа.
- **`trio3`** (`table` | `nil`)
Таблица с информацией, задающей вариацию для третьего персонажа игрока. Если трио не создается, передается `nil`. Содержит поля, аналогичные `primary`, а также дополнительные:
- `name` (`WString`) - имя персонажа.
- `isMale` (`boolean`) - пол персонажа.
- **`pet`** (`table` | `nil`)
Таблица с информацией, задающей вариацию для питомца. Если настраиваемый питомец отсутствует, передается `nil`. Содержит поля, аналогичные `primary`, а также дополнительное:
- `name` (`WString`) - имя питомца.
- **`petIndex`** (`number` | `nil`)
Индекс облика питомца. Если настраиваемый питомец отсутствует, передается `nil`.
- **`price`** (`number`)
Цена изменений. Обязательна для передачи на сервер для проверки стоимости.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Применение изменений для основного персонажа и питомца
```lua
local primary = {
    skinTexture = 0,
    skinColor = 1,
    hairType = 2,
    hairColor = 3,
    face = 0,
    facialType = 1,
    additional = 0,
    morphPresetIndex = 0
}

local pet = {
    skinTexture = 0,
    skinColor = 0,
    hairType = 0,
    hairColor = 0,
    face = 0,
    facialType = 0,
    additional = 0,
    morphPresetIndex = 0,
    name = userMods.ToWString("Питомец")
}

local petIndex = 1
local price = 100

changeRoom.Use( true, primary, nil, nil, pet, petIndex, price )
```
