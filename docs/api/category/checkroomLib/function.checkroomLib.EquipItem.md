# checkroomLib.EquipItem

Надевает гардеробный предмет в соответствующий слот.

## Описание

```lua
checkroomLib.EquipItem( itemId: ObjectId )
```

Функция надевает гардеробный предмет в соответствующий слот экипировки.

## Список параметров

- **`itemId`** (`ObjectId`)
Уникальный идентификатор гардеробного предмета.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Надевание гардеробного предмета

```lua
checkroomLib.EquipItem( itemId )
```
