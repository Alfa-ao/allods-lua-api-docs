# checkroomLib.EquipItems

Надевает гардеробные предметы в соответствующие слоты.

## Описание

```lua
checkroomLib.EquipItems( itemIds: table )
```

Функция надевает гардеробные предметы в соответствующие слоты. За один вызов функция переодевает не более 6 предметов.

## Список параметров

- **`itemIds`** (`table`) - Идентификаторы предметов.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Переодевание предметов гардероба

```lua
checkroomLib.EquipItems( {
    headItemId,
    shoulderItemId
} )
```
