# checkroomLib.FindCheckroomItem

Возвращает идентификатор предмета в гардеробе, соответствующий реальному предмету в другом контейнере.

## Описание

```lua
checkroomLib.FindCheckroomItem( itemId: ObjectId ): ObjectId | nil
```

Функция возвращает идентификатор предмета в гардеробе, который соответствует указанному реальному предмету из другого контейнера. Если предмет не найден, возвращается `nil`.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `ObjectId` - идентификатор предмета в гардеробе, если он найден, или `nil`, если предмет не найден.

## Примеры

### Получение информации о предмете из гардероба

```lua
local checkroomItemId = checkroomLib.FindCheckroomItem( itemId )
local itemInfo = itemLib.GetItemInfo( checkroomItemId )
```

::: info Описание примера
В примере сначала извлекается идентификатор предмета в гардеробе по идентификатору предмета из другого контейнера. Затем по полученному идентификатору запрашивается полная информация о предмете с помощью функции `itemLib.GetItemInfo`.
:::
