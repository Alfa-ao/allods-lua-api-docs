# checkroomLib.GetCategories

Возвращает индексированный с единицы список идентификаторов ресурсов корневых категорий гардероба.

## Описание

```lua
checkroomLib.GetCategories(): table
```

Функция возвращает индексированный с единицы список идентификаторов ресурсов корневых категорий гардероба.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - индексированный с единицы список идентификаторов (`CheckRoomCategoryId`) ресурсов корневых категорий гардероба.

## Примеры

### Получение списка корневых категорий

```lua
local categories = checkroomLib.GetCategories()
LogInfo( categories[1]:GetInfo().name )
```

::: info Описание примера
В примере выполняется запрос списка корневых категорий гардероба. Затем извлекается идентификатор первой категории из полученного списка, и с помощью метода `GetInfo` выводится её название в информационный лог.
:::
