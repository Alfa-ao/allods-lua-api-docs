# checkroomLib.IsItemEquipped

Возвращает признак того, что данный или аналогичный гардеробный предмет надет на персонажа.

## Описание

```lua
checkroomLib.IsItemEquipped( itemId: ObjectId ): boolean
```

Функция возвращает логическое значение, указывающее на наличие данного или аналогичного гардеробного предмета в экипировке персонажа.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

`boolean` - `true`, если гардеробный предмет надет, и `false` в противном случае.

## Примеры

### Проверка наличия надетого предмета

```lua
local isItemEquipped = checkroomLib.IsItemEquipped( itemId )
```
