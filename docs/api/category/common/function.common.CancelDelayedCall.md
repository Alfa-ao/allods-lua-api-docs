# common.CancelDelayedCall
Отменяет запланированный отложенный вызов функции.

## Описание
```lua
common.CancelDelayedCall( functionRef: integer ): boolean
```
Функция отменяет запланированный отложенный вызов функции, ранее созданный с помощью `common.DelayedCall`.

::: warning Замечание
Ссылка на отмену вызова не является уникальной по времени, но уникальна в пределах конкретного аддона (одинаковые идентификаторы в разных аддонах указывают на разные сущности). Ссылка освобождается после успешной отмены или фактического вызова функции и может быть назначена заново при следующем вызове `common.DelayedCall`. Требуется внимательная проверка отменяемой ссылки.
:::

## Список параметров
- **`functionRef`** (`integer`)
Ссылка на функцию, полученная из возвращаемого значения `common.DelayedCall`.

## Возвращаемые значения
Возвращает `boolean` — `true`, если отложенный вызов был успешно отменен, и `false`, если вызов по указанной ссылке не найден или уже был выполнен.

## Примеры
### Отмена отложенного вызова
```lua
local callRef = common.DelayedCall( 5000, function()
    LogInfo( "Отложенное действие выполнено." )
end )

common.DelayedCall( 2000, function()
    local isCanceled = common.CancelDelayedCall( callRef )
    if isCanceled then
        LogInfo( "Отложенный вызов успешно отменен." )
    else
        LogInfo( "Вызов не найден или уже выполнен." )
    end
end )
```
::: info Описание примера
В примере сначала планируется отложенный вызов через 5000 миллисекунд с помощью `common.DelayedCall`, а возвращаемая ссылка сохраняется в переменную `callRef`. Затем планируется второй отложенный вызов через 2000 миллисекунд, внутри которого вызывается `common.CancelDelayedCall` с переданной ссылкой. Функция отменяет первоначальный вызов, а результат операции выводится в лог.
:::
