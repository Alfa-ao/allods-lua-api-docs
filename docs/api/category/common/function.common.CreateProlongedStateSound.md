# common.CreateProlongedStateSound
Создает и возвращает объект для управления двухмерным звуком, привязанным к определенному слоту.

## Описание
```lua
common.CreateProlongedStateSound( soundId: Sound2DId, slot: number ): Sound2DLua | nil
```
Функция создает объект для работы с двухмерным звуком, который привязывается к указанному слоту. Новый звук, назначенный на уже занятый слот, заменяет предыдущий. Звуки, назначенные на разные слоты, взаимодействуют между собой по общим правилам. Воспроизведение звука продолжается некоторое время после завершения текущего состояния. Доступны слоты с индексами от 0 до 20. Функция предназначена в первую очередь для воспроизведения музыки главного меню, чтобы она продолжала играть во время отображения экрана загрузки и плавно затухала после начала игры. По умолчанию созданный звук не воспроизводится — для запуска требуется явный вызов метода `Play()` у возвращаемого объекта. В режиме без звука функция возвращает `nil`.

::: warning Замечание
По умолчанию звук не воспроизводится. Для начала воспроизведения необходимо вызвать метод `sound:Play()` у полученного объекта.
:::

::: warning Замечание
В режиме без звука функция возвращает `nil`.
:::

## Список параметров
- **`soundId`** (`Sound2DId`)
Идентификатор требуемого звука.

- **`slot`** (`number`)
Индекс слота для привязки звука. Допустимые значения находятся в диапазоне от 0 до 20.

## Возвращаемые значения
Возвращает `Sound2DLua` — новый экземпляр объекта для управления двухмерным звуком, или `nil` в случае возникновения ошибки либо при активном режиме без звука.

## Примеры
### Проверка наличия и запуск звука
```lua
local hasSound = group:HasSound( sysName )
if hasSound then
    local soundId = group:GetSound( sysName )
    local sound = common.CreateProlongedStateSound( soundId, 0 )
    if sound then
        sound:Play()
    end
end
```
::: info Описание примера
В примере выполняется проверка наличия звука в группе по системному имени. При наличии звука извлекается его идентификатор, после чего создается двухмерный звук в нулевом слоте. Если объект успешно создан, вызывается метод `Play()` для начала воспроизведения.
:::
