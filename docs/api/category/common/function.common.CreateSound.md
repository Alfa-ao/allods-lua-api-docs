# common.CreateSound
Создает и возвращает объект для работы с двухмерным звуком.

## Описание
```lua
common.CreateSound( soundId: Sound2DId ): Sound2DLua | nil
```
Функция инициализирует и возвращает новый экземпляр объекта `Sound2DLua`, предназначенный для управления воспроизведением двухмерного звука по заданному идентификатору.

::: warning Замечание
По умолчанию созданный звуковой объект не начинает воспроизведение автоматически. Для запуска звука требуется явный вызов метода `sound:Play()`.
:::
::: warning Замечание
При активном режиме без звука или в случае возникновения ошибки функция возвращает `nil`.
:::

## Список параметров
- **`soundId`** (`Sound2DId`)
Идентификатор требуемого звукового эффекта.

## Возвращаемые значения
Возвращает `Sound2DLua` — новый экземпляр объекта для управления двухмерным звуком, или `nil` в случае возникновения ошибки, а также при активном режиме без звука.

## Примеры
### Проверка наличия и воспроизведение звука
```lua
local hasSound = group:HasSound( sysName )
if hasSound then
    local soundId = group:GetSound( sysName )
    local sound = common.CreateSound( soundId )
    if sound then
        sound:Play()
    end
end
```
::: info Описание примера
В примере сначала проверяется наличие звука в указанной группе с помощью `group:HasSound`. При положительном результате извлекается идентификатор звука через `group:GetSound`. Затем функция `common.CreateSound` создает звуковой объект, и после проверки его успешного создания вызывается метод `sound:Play` для запуска воспроизведения.
:::
