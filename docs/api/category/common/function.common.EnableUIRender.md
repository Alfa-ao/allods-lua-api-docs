# common.EnableUIRender
Задаёт отображаемость пользовательского интерфейса.

## Описание
```lua
common.EnableUIRender( enable: boolean )
```
Функция задает отображаемость пользовательского интерфейса. Если отображение интерфейса выключено, интерфейсные виджеты не показываются и не обрабатывают реакции на пользовательский ввод.

## Список параметров
- **`enable`** (`boolean`)
Флаг состояния отображения интерфейса. Значение `true` включает отображение пользовательского интерфейса, значение `false` — выключает.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Отключение отображения интерфейса
```lua
common.EnableUIRender( false )
```
