# common.EscapeWString
Экранирует специальные символы в строке для предотвращения их интерпретации как HTML-кода.

## Описание
```lua
common.EscapeWString( str: WString ): WString
```
Функция выполняет экранирование текста строки от HTML-кода. Применяется для безопасного отображения пользовательского или динамического текста в элементах интерфейса.

::: warning Замечание
Алгоритм экранирования идентичен тому, что используется при включенном режиме `isHtmlEscaping` для виджета `TextView`.
:::

## Список параметров
- **`str`** (`WString`)
Исходная строка, содержащая текст, который необходимо экранировать.

## Возвращаемые значения
- `WString`
Результирующая строка, в которой специальные символы HTML заменены на их экранированные представления.

## Примеры
### Базовое экранирование строки
```lua
local src = "<color=red>Текст с тегами</color>"
local result = common.EscapeWString( src )
```
::: info Описание примера
В примере исходная строка `src`, содержащая теги форматирования, передается в функцию `common.EscapeWString`. Возвращаемая строка `result` содержит экранированные символы `&lt;color=red&gt;Текст с тегами&lt;/color&gt;`, что позволяет отобразить теги как обычный текст в виджете `TextView`, избегая их интерпретации как HTML-разметки. 
:::
