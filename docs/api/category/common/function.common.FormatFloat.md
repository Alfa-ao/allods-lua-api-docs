# common.FormatFloat
Форматирует вещественное значение в заданном формате и возвращает результат в виде локализуемой строки.

## Описание
```lua
FormatFloat( value: number, format: string, separator: ?string ): WString
```
Функция преобразует числовое значение в строку согласно переданной строке формата. Поддерживаются стандартные форматы представления чисел, а также специфичные для игры форматы отображения больших чисел с автоматическим сокращением разрядов. Дополнительно позволяет указать символ-разделитель для группировки разрядов целой части числа, что повышает читаемость значений.

::: warning Замечание
Специфичные игровые форматы, такие как `"%f3K5"`, предназначены для компактного отображения больших числовых значений (например, урона или здоровья) в пользовательском интерфейсе.
:::

## Список параметров
- **`value`** (`number`)
Исходное вещественное значение, подлежащее форматированию.

- **`format`** (`string`)
Строка, определяющая правило форматирования числа. Допустимые значения:
- `"%g"` — вещественное число в автоматически выбираемом (наиболее компактном) формате.
- `"%.1f"` — вещественное число с одним знаком после десятичной точки (остальные обрезаются).
- `"%.2f"` — вещественное число с двумя знаками после десятичной точки (остальные обрезаются).
- `"%.3f"` — вещественное число с тремя знаками после десятичной точки (остальные обрезаются).
- `"%f3"` — цифры после десятичной точки выводятся по правилам `"%f3K5"`, остальные разряды выводятся без изменений (например, `234567` останется `234567`).
- `"%f3K5"` — специфичный игровой формат для отображения больших чисел:
  - если до запятой имеется один разряд, после запятой выводится максимум 2 разряда;
  - если до запятой имеется 2 разряда, после запятой выводится максимум 1 разряд;
  - если до запятой имеется 3 и более разрядов, дробные значения не выводятся;
  - максимальное количество показываемых целых разрядов — 4 (например, `1068` или `8956`);
  - если количество целых разрядов 5 и более, отображение сворачивается к виду `nК` с нормальным округлением (например, `129876` преобразуется в `13К`).
- `"%<100?.2f:d"` — условный формат: если число меньше 100, применяется правило `"%.2f"`, иначе выполняется округление до целого по правилу `"%d"`.

- **`separator`** (`string` | `nil`)
Необязательный параметр. Разделитель разрядов. Если задан, через каждые 3 разряда до десятичной точки вставляются символы указанного разделителя.

## Возвращаемые значения
Возвращает `WString` — текстовое представление исходного числа в указанном формате.

## Примеры
### Форматирование числа с одним знаком после запятой
```lua
local valueText = common.FormatFloat( 2.2, "%.1f" )
```
