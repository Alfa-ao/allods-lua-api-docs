# common.FormatInt
Форматирует целое значение в указанном формате и возвращает результат в виде локализуемой строки.

## Описание
```lua
common.FormatInt( value: number, format: string, separator: ?string ): WString
```
Функция преобразует переданное целое число в строковое представление согласно заданному шаблону форматирования. Поддерживаются стандартные числовые форматы, а также специфический формат для сокращения больших чисел. Дополнительно позволяет указать символ-разделитель для группировки разрядов.

## Список параметров
- **`value`** (`number`)
Исходное целое значение, подлежащее форматированию.
- **`format`** (`string`)
Строка, определяющая формат вывода. Допустимы следующие варианты:
- `"%d"` — обычное целое число (например, `10`).
- `"%+d"` — целое число с явным указанием знака (например, `+10`, `-10`).
- `"%02d"` — целое число с ведущим нулем, если значение состоит из одной цифры (например, `02`, `10`).
- `"%dK5"` — специфический формат для сокращения больших чисел:
  - Если количество целых разрядов не превышает 4 (например, `1068` или `8956`), число отображается полностью.
  - Если количество целых разрядов равно 5 и более, значение сворачивается к виду `nК` с применением нормального округления (например, `129876` преобразуется в `13К`).
- **`separator`** (`string` | `nil`)
Необязательный параметр. Символ-разделитель разрядов. Если задан, через каждые 3 разряда целой части будут вставляться указанные символы.

## Возвращаемые значения
Возвращает `WString` — текстовое представление исходного числа, отформатированное в соответствии с заданными параметрами.

## Примеры
### Форматирование числа с ведущим нулем
```lua
local valueText = common.FormatInt( 2, "%02d" )
```
::: info Описание примера
В примере целое число `2` форматируется с использованием шаблона `"%02d"`. Результатом будет строка `"02"`, где недостающий разряд заполнен ведущим нулем.
:::

### Форматирование числа с разделителем разрядов
```lua
local formattedValue = common.FormatInt( 1234567, "%d", " " )
```
::: info Описание примера
Число `1234567` форматируется как обычное целое (`"%d"`), но с указанием пробела в качестве разделителя разрядов. Результатом будет строка `"1 234 567"`.
:::

### Сокращение большого числа
```lua
local shortValue = common.FormatInt( 129876, "%dK5" )
```
::: info Описание примера
Используется специальный формат `"%dK5"` для числа `129876`. Поскольку количество разрядов больше 4, значение округляется и сворачивается. Результатом будет строка `"13К"`.
:::
