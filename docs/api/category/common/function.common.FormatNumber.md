# common.FormatNumber
Преобразует число в строку в заданном формате, возвращая результат в виде локализуемой строки.

## Описание
```lua
common.FormatNumber( value: number, format: string | nil ): WString
```
Функция преобразует числовое значение в строковое представление согласно указанному формату. Строка формата имеет вид `"+12.3[4]A5"` и состоит из следующих компонентов: `[sign][width][.precision][[total]][abbreviate]`.
- **`sign`** — флаг, задается символом `+`. Отвечает за вывод знака положительного числа. Отрицательные числа всегда выводятся со знаком. По умолчанию не задан.
- **`width`** — минимальная длина целой части числа. Недостающие разряды в целой части числа заменяются символом `0`. По умолчанию `1`.
- **`.precision`** — максимальное число знаков после десятичной точки. Десятичная точка может быть задана символом `.` или `,`. По умолчанию `2`.
- **`[total]`** — максимальное число знаков в итоговом числе, не считая десятичной точки и знаков аббревиации. Не влияет на вывод целой части. По умолчанию не задан.
- **`abbreviate`** — аббревиация числа, задается в виде `An`, где `n` — номер разряда числа, начиная с которого число подвергается аббревиации.

## Список параметров
- **`value`** (`number`)
Исходное число, которое необходимо преобразовать.
- **`format`** (`string` | `nil`)
Строка с форматом, определяющая правила отображения числового значения.

## Возвращаемые значения
Возвращает `WString` — текстовое представление исходного числа в указанном формате.

## Примеры
### Базовое форматирование с использованием переменных
```lua
local value = 12345
local valueText = common.FormatNumber( value, "+.2[3]A5" )
```

### Применение различных форматов аббревиации и точности
```lua
-- Аббревиация с 4-го разряда (тысячи)
common.FormatNumber( 1234, ".2A4" ) == "1.23K"

-- Отсутствие аббревиации (5-й разряд не достигнут)
common.FormatNumber( 1234, ".2A5" ) == "1234"

-- Аббревиация с ограничением общей длины
common.FormatNumber( 12345, "[3]A5" ) == "12.3K"

-- Вывод знака для положительных и отрицательных чисел
common.FormatNumber( 0.123, "+.2" ) == "+0.12"
common.FormatNumber( -1.234, "+.2" ) == "-1.23"

-- Ограничение общей длины числа
common.FormatNumber( 12.345, ".2[3]" ) == "12.3"
common.FormatNumber( -123.45, ".2[3]" ) == "-123"
```
