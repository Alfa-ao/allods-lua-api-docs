# common.GetAbsTimeMs
Возвращает значение основного игрового таймера.

## Описание
```lua
common.GetAbsTimeMs(): number
```
Функция возвращает текущее значение основного игрового таймера, измеряемое в миллисекундах. Обновление значения таймера происходит один раз за кадр. Данное время используется в качестве базовой метки для отсчетов и передается в обработчике кадра `FrameHandler`, а также приходит в событии `EVENT_SECOND_TIMER`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — текущий отсчет таймера в миллисекундах.

## Примеры
### Измерение прошедшего времени в секундах
```lua
local startTime = common.GetAbsTimeMs()

-- Выполнение некоторой логики или ожидание

local endTime = common.GetAbsTimeMs()
local deltaSeconds = math.floor( ( endTime - startTime ) / 1000 )

LogInfo( "Прошло времени (сек): ", deltaSeconds )
```
::: info Описание примера
В примере фиксируется начальное время с помощью `common.GetAbsTimeMs`. После выполнения логики фиксируется конечное время. Разница между значениями переводится в секунды с использованием `math.floor`, что позволяет вычислить точное количество секунд, прошедших между двумя вызовами.
:::
