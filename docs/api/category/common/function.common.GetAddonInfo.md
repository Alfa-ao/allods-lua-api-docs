# common.GetAddonInfo
Возвращает подробную информацию об аддоне по его системному имени.

## Описание
```lua
common.GetAddonInfo( addonName: string ): table | false
```
Извлекает и возвращает детальную информацию об аддоне, используя его системное имя. Возвращаемая таблица содержит как общие параметры состояния, так и специфичные метаданные для пользовательских аддонов. Если аддон с указанным системным именем не найден, функция возвращает `false`.

## Список параметров
- **`addonName`** (`string`)
Системное имя аддона, информацию о котором необходимо получить. Может быть получено с помощью функции `common.GetAddonSysName`.

## Возвращаемые значения
Возвращает `table` или `false`.

Если аддон найден, возвращается таблица (`table`) со следующими полями:

*Общая часть:*
- **`state`** (`number`)
Текущий статус аддона (соответствует перечислению `AddonState`).
- **`rawSysName`** (`string`)
Имя аддона в ресурсе.
- **`sysName`** (`string`)
Системное название аддона.
- **`userAddon`** (`boolean`)
Признак того, является ли аддон пользовательским.
- **`managed`** (`boolean`)
Признак того, является ли аддон управляемым (для пользовательских аддонов всегда `true`).
- **`autoStart`** (`boolean`)
Признак автоматического запуска аддона (для пользовательских аддонов всегда `false`).
- **`useCommonScripts`** (`boolean`)
Признак использования аддоном общих скриптов (`CoreScripts`).

*Только для пользовательских аддонов:*
- **`name`** (`WString`)
Локализованное название аддона.
- **`description`** (`WString`)
Локализованное описание аддона.
- **`license`** (`WString`)
Текст лицензии аддона.
- **`sourceUrl`** (`WString`)
Ссылка на источник происхождения аддона.
- **`supportUrl`** (`WString`)
Ссылка на техническую поддержку аддона.
- **`author`** (`string`)
Автор аддона.
- **`version`** (`string`)
Версия аддона.
- **`apiVersion`** (`number`)
Версия API аддона.
- **`buildTime`** (`number`)
Время сборки аддона в формате Unix Time (миллисекунды).

Если аддон не найден, возвращается `false`.

## Примеры
### Получение и вывод информации о текущем аддоне
```lua
local addonInfo = common.GetAddonInfo( common.GetAddonSysName() )
if addonInfo then
    LogInfo( "Системное имя: ", addonInfo.sysName )
    LogInfo( "Название: ", addonInfo.name )
    LogInfo( "Версия: ", addonInfo.version )
    LogInfo( "Автор: ", addonInfo.author )
else
    LogInfo( "Информация об аддоне не найдена." )
end
```
::: info Описание примера
В примере выполняется запрос информации о текущем аддоне с использованием его системного имени, полученного через `common.GetAddonSysName`. Если функция возвращает таблицу, в лог выводятся его основные метаданные: системное имя, название, версия и автор. В случае возврата `false` выводится сообщение об отсутствии информации.
:::
