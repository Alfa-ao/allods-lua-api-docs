# common.GetAddonSysName
Возвращает системное имя аддона, в котором выполняется скрипт.

## Описание
```lua
common.GetAddonSysName(): string
```
Функция возвращает действительное системное имя аддона, используемое клиентом во всех связанных функциях и событиях. Для системных аддонов имя задается в ресурсе аддона, а для пользовательских вычисляется как конкатенация строки `"UserAddon/"` и значения константы `FOLDER_NAME`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `string` — системное имя аддона, в котором выполняется скрипт.

## Примеры
### Получение системного имени текущего аддона
```lua
local name = common.GetAddonSysName()
```
