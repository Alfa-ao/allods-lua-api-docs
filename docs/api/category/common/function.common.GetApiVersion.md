# common.GetApiVersion
Возвращает текущую версию Lua API.

## Описание
```lua
common.GetApiVersion(): number
```
Функция возвращает текущую версию Lua API в виде целого числа. Формат возвращаемого значения представляет собой число вида `MM0mmfff`, где:
- `MM` — мажорная версия;
- `mm` — минорная версия;
- `fff` — номер изменения API.

Номер изменения начинается с `000` при смене мажорной или минорной версии и инкрементируется при любых последующих изменениях в Lua API.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — целое число, представляющее собой номер текущей версии Lua API.

## Примеры
### Проверка версии API для аддона
```lua
local version = common.GetApiVersion()
if version > addonApiVersion then
    LogInfo( "Check API changes for addon ", ADDON_NAME )
end
```
