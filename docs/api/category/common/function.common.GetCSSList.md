# common.GetCSSList
Извлекает и возвращает список всех доступных CSS-классов.

## Описание
```lua
common.GetCSSList( id: ?string ): table
```
Функция извлекает список названий CSS-классов. Если передан идентификатор пользовательского набора стилей, возвращает классы из заданного набора. В противном случае возвращает полный список всех зарегистрированных CSS-классов.

## Список параметров
- **`id`** (`string` | `nil`)
Идентификатор пользовательского набора стилей. Если параметр не задан, возвращается список всех CSS-классов.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую названия CSS-классов (`WString`). Индексация элементов в таблице начинается с единицы.

## Примеры
### Получение и применение первого CSS-класса
```lua
local listCSS = common.GetCSSList()
wt:SetGlobalClasses( { listCSS[ 1 ] } )
```
::: info Описание примера
В примере извлекается список всех доступных CSS-классов. Затем первый класс из полученного списка применяется к виджету `wt` в качестве глобального стиля с помощью метода `SetGlobalClasses`.
:::
