# common.GetDateTimeFromMs
Возвращает дату и время в формате `LuaFullDateTime`, преобразуя Unix time из количества миллисекунд.

## Описание
```lua
common.GetDateTimeFromMs( timeMs: number ): table | nil
```
Функция преобразует переданное значение Unix time, выраженное в миллисекундах, прошедших с 1 января 1970 года, в структурированную таблицу формата `LuaFullDateTime`. В случае возникновения ошибки или передачи некорректных данных возвращается `nil`.

## Список параметров
- **`timeMs`** (`number`)
Unix time, представляющий собой количество миллисекунд, прошедших с 1 января 1970 года.

## Возвращаемые значения
Возвращает `table` (данные даты и времени в формате `LuaFullDateTime`) или `nil` в случае ошибки.

## Примеры
### Получение и вывод даты из миллисекунд
```lua
local date = common.GetDateTimeFromMs( 123456789012345 )
if date then
    LogInfo( "date: ", date.d, ".", date.m, ".", date.y )
end
```
