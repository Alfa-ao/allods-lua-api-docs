# common.GetEmptyWString
Возвращает пустую локализуемую строку.

## Описание
```lua
common.GetEmptyWString(): WString
```
Функция возвращает пустую локализуемую строку типа `WString`. Данный тип данных используется в API игры для работы с текстом, поддерживающим различные языковые кодировки и локализации. Полученная строка применяется для инициализации переменных, сброса текстовых значений или в качестве базового значения при формировании текстовых конструкций.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `WString` — пустую локализуемую строку.

## Примеры
### Получение пустой строки для инициализации
```lua
local emptyText = common.GetEmptyWString()
```
