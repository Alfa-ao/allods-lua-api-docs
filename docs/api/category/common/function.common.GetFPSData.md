# common.GetFPSData
Возвращает информацию о частоте смены кадров (FPS).

## Описание
```lua
common.GetFPSData(): table
```
Функция возвращает таблицу с данными о частоте смены кадров (FPS), рассчитанными за последние 100 кадров, а также мгновенное значение FPS на основе времени предыдущего кадра.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`) со следующими полями:
- **`min`** (`number`) — минимальный FPS за расчетный период (100 кадров).
- **`max`** (`number`) — максимальный FPS за расчетный период (100 кадров).
- **`avg`** (`number`) — средний FPS за расчетный период (100 кадров).
- **`instant`** (`number`) — мгновенный FPS на основе времени предыдущего кадра.

## Примеры
### Вывод информации о FPS в лог
```lua
LogInfo( common.GetFPSData() )
```
