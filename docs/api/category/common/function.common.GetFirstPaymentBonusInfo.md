# common.GetFirstPaymentBonusInfo
Возвращает список бонусов за платёж и информацию об условиях их получения.

## Описание
```lua
common.GetFirstPaymentBonusInfo(): table
```
Функция возвращает таблицу, содержащую список бонусов за платёж и информацию об условиях их получения. Данные включают минимальные суммы платежей, размеры бонусов в премиальных кристаллах, идентификаторы текстур подложек и списки дополнительных бонусных предметов.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую список бонусов. Индексация элементов в таблице начинается с единицы. Каждый элемент представляет собой вложенную таблицу со следующими полями:
- **`min`** (`number`) — минимальная сумма платежа для получения бонуса.
- **`bonus`** (`number`) — размер бонуса в премиальных кристаллах.
- **`background`** (`TextureId`) — идентификатор текстуры для подложки.
- **`items`** (`table`) — список бонусных предметов (`ResourceId`). Индексация элементов вложенного списка также начинается с единицы.

## Примеры
### Получение информации о бонусах
```lua
local info = common.GetFirstPaymentBonusInfo()
```
