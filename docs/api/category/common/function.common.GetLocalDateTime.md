# common.GetLocalDateTime
Возвращает клиентскую дату и время в формате LuaFullDateTime.

## Описание
```lua
common.GetLocalDateTime(): table
```
Функция извлекает и возвращает текущую клиентскую дату и время, структурированные в формате LuaFullDateTime. Данные формируются на основе локального времени клиента, полученного из системного состояния GameState.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` (LuaFullDateTime) — таблицу, содержащую клиентскую дату и время.

## Примеры
### Получение и вывод текущей клиентской даты
```lua
local date = common.GetLocalDateTime()
LogInfo( "date: ", date.d, ".", date.m, ".", date.y )
```
