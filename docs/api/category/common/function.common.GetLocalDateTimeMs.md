# common.GetLocalDateTimeMs
Возвращает клиентскую дату и время в формате Unix time в миллисекундах.

## Описание
```lua
common.GetLocalDateTimeMs(): number
```
Функция возвращает клиентскую дату и время в формате Unix time, выраженную в миллисекундах, прошедших с 1 января 1970 года. Значение представляет собой целое 64-битное число (`int64`), упакованное в тип `number`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` (`int64`) — клиентскую дату и время в формате Unix time в миллисекундах.

## Примеры
### Получение текущего клиентского времени
```lua
local timeMs = common.GetLocalDateTimeMs()
```
