# common.GetLocale
Возвращает текущую локаль (раскладку клавиатуры) в виде короткого имени.

## Описание
```lua
common.GetLocale(): string | nil
```
Функция возвращает строку с коротким именем текущей локали (например, `'ru'`, `'en'`, `'de'`) или `nil` в случае возникновения ошибки при определении раскладки.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
- `string` — короткое имя текущей локали.
- `nil` — в случае ошибки при определении локали.

## Примеры
### Получение текущей локали
```lua
local currentLocale = common.GetLocale()
if currentLocale then
    LogInfo( "Текущая локаль: ", currentLocale )
else
    LogInfo( "Не удалось определить локаль." )
end
```
