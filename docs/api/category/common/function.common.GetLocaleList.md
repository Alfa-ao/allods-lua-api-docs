# common.GetLocaleList
Возвращает список доступных для выбора локалей (раскладок клавиатуры).

## Описание
```lua
common.GetLocaleList(): table | nil
```
Функция извлекает и возвращает список доступных для выбора локалей (раскладок клавиатуры). Каждая локаль описывается коротким именем, например `"ru"` или `"en"`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.
Таблица содержит строковые (`string`) значения коротких имен локалей, индексация элементов начинается с единицы. В случае возникновения ошибки возвращается `nil`.

## Примеры
### Получение и вывод первой доступной локали
```lua
local localeList = common.GetLocaleList()
if localeList and localeList[1] then
    LogInfo( "First available locale: ", localeList[1] )
end
```
::: info Описание примера
В примере запрашивается список доступных локалей. Если список успешно получен и не пуст, в информационный лог выводится имя первой доступной локали.
:::
