# common.GetLocalization
Возвращает строковый идентификатор текущей языковой локализации.

## Описание
```lua
common.GetLocalization(): string
```
Функция возвращает строковое значение, представляющее собой идентификатор текущей языковой локализации, заданной в конфигурационном файле `global.cfg`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `string` — строковый идентификатор текущей локализации (например, `"rus"`).

## Примеры
### Получение текущей локализации
```lua
local currentLocale = common.GetLocalization()
LogInfo( "Текущая локализация: ", currentLocale )
```
