# common.GetMks
Возвращает количество микросекунд с момента старта системы для точного таймера при профилировании производительности.

## Описание
```lua
common.GetMks(): number
```
Функция возвращает количество микросекунд (1/1000 миллисекунды), прошедших с момента старта системы. Предназначена для создания высокоточного таймера при профилировании производительности кода.

::: warning Замечание
Категорически не рекомендуется использовать функцию для измерения длинных временных интервалов или в целях, отличных от отладки и профилирования.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — целое 64-битное число (`int64`), представляющее количество микросекунд с момента старта системы.

## Примеры
### Вывод количества микросекунд в лог
```lua
LogInfo( "Ticks count: " .. tostring( common.GetMks() ) )
```
