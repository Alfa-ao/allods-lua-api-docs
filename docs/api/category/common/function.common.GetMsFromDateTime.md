# common.GetMsFromDateTime
Возвращает Unix time (количество миллисекунд с 1 января 1970 года) на основе заданной даты и времени.

## Описание
```lua
common.GetMsFromDateTime( timeTable: table ): number
```
Функция преобразует таблицу с полями даты и времени (формат `LuaFullDateTime`) в количество миллисекунд, прошедших с 1 января 1970 года (Unix time). Обязательными для передачи являются только год, месяц и день. Часы, минуты, секунды и миллисекунды являются опциональными и по умолчанию принимаются за `0`.

## Список параметров
- **`timeTable`** (`table`)
Таблица с полями, описывающими дату и время:
  - `y` (`number`) - год.
  - `m` (`number`) - месяц (начиная с 1).
  - `d` (`number`) - день.
  - `h` (`number` | `nil`) - час (необязательный параметр, по умолчанию 0).
  - `min` (`number` | `nil`) - минута (необязательный параметр, по умолчанию 0).
  - `s` (`number` | `nil`) - секунда (необязательный параметр, по умолчанию 0).
  - `ms` (`number` | `nil`) - миллисекунда (необязательный параметр, по умолчанию 0).

## Возвращаемые значения
Возвращает `number` — Unix time (время в миллисекундах, прошедшее с 1-го января 1970 года).

## Примеры
### Получение текущего времени в миллисекундах
```lua
local ms = common.GetMsFromDateTime( common.GetLocalDateTime() )
```
::: info Описание примера
В примере функция `common.GetLocalDateTime()` возвращает таблицу с текущей локальной датой и временем, которая затем передается в `common.GetMsFromDateTime` для преобразования в Unix time в миллисекундах.
:::
