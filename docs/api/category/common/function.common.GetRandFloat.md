# common.GetRandFloat
Генерирует вещественное псевдо-случайное число в заданном диапазоне.

## Описание
```lua
common.GetRandFloat( min: number, max: number ): number
```
Функция возвращает вещественное псевдо-случайное число, расположенное в пределах от минимального (`min`) до максимального (`max`) значения. Результат вычисляется с использованием встроенного генератора псевдослучайных чисел.

## Список параметров
- **`min`** (`number`)
Минимальное значение диапазона генерации.
- **`max`** (`number`)
Максимальное значение диапазона генерации.

## Возвращаемые значения
Возвращает `number` — вещественное псевдо-случайное число в диапазоне `[min..max]`.

## Примеры
### Генерация и вывод случайного вещественного числа
```lua
LogInfo( "rand float: " .. tostring( common.GetRandFloat( 1.0, 2.0 ) ) )
```
