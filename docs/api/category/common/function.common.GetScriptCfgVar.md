# common.GetScriptCfgVar
Возвращает значение скриптовой переменной из конфигурационного файла.

## Описание
```lua
common.GetScriptCfgVar( name: string ): boolean | number | string | nil
```
(HELP) в каком конфигурационном файле ?
Функция извлекает и возвращает значение скриптовой переменной, заданной в конфигурационном файле. Поддерживаются переменные типов `boolean`, `number` и `string`. Значения переменных инициализируются с помощью специальной команды `script_var` в конфигурационном файле.

::: tip Информация
Формат задания переменных в конфигурационном файле:
- `script_var <name> = true/false`
- `script_var <name> = number_value`
- `script_var <name> = "string_value"`

Примеры объявления:
- `script_var show_chat = true`
- `script_var chat_size = 100`
- `script_var chat_style = "Combined"`
:::

## Список параметров
- **`name`** (`string`)
Имя скриптовой переменной, значение которой необходимо получить.

## Возвращаемые значения
Возвращает `boolean`, `number` или `string` — значение переменной в соответствии с её типом. Если переменная с указанным именем не определена в конфигурационном файле, возвращается `nil`.

## Примеры
### Получение значения скриптовой переменной
```lua
local chatSize = common.GetScriptCfgVar( "chat_size" )
```
::: info Описание примера
В примере запрашивается значение скриптовой переменной `chat_size`. Если переменная была предварительно определена в конфигурационном файле (например, `script_var chat_size = 100`), в переменную `chatSize` будет записано соответствующее значение. В противном случае будет возвращено `nil`.
:::
