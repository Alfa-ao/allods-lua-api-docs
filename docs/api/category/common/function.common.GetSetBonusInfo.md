# common.GetSetBonusInfo
Извлекает и возвращает базовую информацию о бонусе комплектной экипировки.

## Описание
```lua
common.GetSetBonusInfo( setBonusId: SetBonusId ): table | nil
```
Функция извлекает и возвращает базовую информацию о бонусе комплектной экипировки (сетовом бонусе) по его уникальному идентификатору. Возвращаемая таблица содержит текстовые и идентифицирующие ресурсы для отображения названия, описания, художественного текста и источника получения бонуса. Если бонус с указанным идентификатором не найден, функция возвращает `nil`.

## Список параметров
- **`setBonusId`** (`SetBonusId` | `ResourceId`)
Уникальный идентификатор бонуса комплектной экипировки.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если бонус найден, возвращается таблица со следующими полями:
- **`name`** (`WString`) — название сетового бонуса.
- **`description`** (`ValuedText`) — основное описание сетового бонуса.
- **`loreDescription`** (`WString`) — художественное описание сетового бонуса.
- **`sourceDescription`** (`GlossaryId`) — идентификатор ресурса для описания источника получения бонуса.

## Примеры
### Получение и вывод информации о сетовом бонусе
```lua
local setBonusInfo = common.GetSetBonusInfo( setBonusId )
if setBonusInfo then
    LogInfo( "Название бонуса: ", setBonusInfo.name )
end
```
::: info Описание примера
В примере извлекается информация о сетовом бонусе по его идентификатору с помощью функции `common.GetSetBonusInfo`. Если данные успешно получены и таблица не равна `nil`, в информационный лог выводится название бонуса.
:::
