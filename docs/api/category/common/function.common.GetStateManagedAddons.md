# common.GetStateManagedAddons
Возвращает информацию об аддонах, доступных для программного управления.

## Описание
```lua
common.GetStateManagedAddons(): table
```
Функция возвращает индексированную таблицу, содержащую сведения об аддонах, которыми можно управлять. Управление главным аддоном или аддонами, имеющими встроенную поддержку в коде игры, невозможно. Текущие состояния аддонов описываются перечислением `ADDON_STATE_*`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — индексированную (начиная с 1) таблицу, содержащую вложенные таблицы с информацией по каждому доступному аддону.
Поля вложенной таблицы:
- **`name`** (`string`) — имя аддона.
- **`state`** (`number`) — текущий статус аддона. Значения соответствуют перечислению `ADDON_STATE_*`.

## Примеры
### Загрузка всех неактивных управляемых аддонов
```lua
local managedAddons = common.GetStateManagedAddons()
for _, addon in ipairs( managedAddons ) do
    if addon.state == ADDON_STATE_NOT_LOADED then
        common.StateLoadManagedAddon( addon.name )
    end
end
```
::: info Описание примера
В примере извлекается список всех управляемых аддонов. Затем с помощью цикла происходит перебор полученной таблицы. Для каждого аддона проверяется его состояние: если аддон не загружен (`ADDON_STATE_NOT_LOADED`), инициируется его загрузка через функцию `common.StateLoadManagedAddon`.
:::
