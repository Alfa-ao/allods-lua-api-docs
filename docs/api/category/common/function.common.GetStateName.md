# common.GetStateName
Возвращает имя состояния (стейта) аддона, в котором выполняется скрипт.

## Описание
```lua
common.GetStateName(): string
```
Функция возвращает строковое представление имени состояния (стейта) аддона, в контексте которого в данный момент выполняется скрипт (например, "SystemState", "MainMenu", "Mission").

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `string` — имя состояния (стейта) аддона, в котором выполняется скрипт.

## Примеры
### Получение имени текущего состояния аддона
```lua
local stateName = common.GetStateName()
LogInfo( "Текущий стейт: ", stateName )
```
