# common.GetTerritoryInfo
Возвращает таблицу с флагами, специфичными для конкретной локализованной версии клиента.

## Описание
```lua
common.GetTerritoryInfo(): table
```
Функция извлекает и возвращает набор логических флагов, определяющих поведенческие особенности и отображаемые элементы интерфейса, специфичные для текущей локализованной версии игрового клиента.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` — таблицу, содержащую следующие поля:
- **`ignoreAutoShard`** (`boolean`)
Флаг, определяющий необходимость игнорирования автоматического выбора шарда. При значении `true` отображение списка шардов является обязательным.
- **`showZoneSelection`** (`boolean`)
Флаг, указывающий на необходимость использования процедуры ручного выбора зоны игроком при входе в игру.
- **`showRaitingInfo`** (`boolean`)
Флаг, определяющий необходимость отображения информации о возрастном рейтинге игры (например, 18+, насилие) в главном меню и на стартовом экране.

## Примеры
### Проверка необходимости отображения рейтинговой информации
```lua
local territoryInfo = common.GetTerritoryInfo()
if territoryInfo.showRaitingInfo then
    LogInfo( "Требуется показать информацию о рейтинге" )
end
```
