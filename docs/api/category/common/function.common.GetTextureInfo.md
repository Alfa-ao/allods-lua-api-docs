# common.GetTextureInfo
Извлекает и возвращает детальную информацию о текстуре по её идентификатору.

## Описание
```lua
common.GetTextureInfo( textureId: TextureId ): ?table
```
Функция извлекает и возвращает детальную информацию о текстуре, включая пути к файлам ресурсов, тип сжатия и реальные размеры. Если текстура с указанным идентификатором не найдена или данные недоступны, функция возвращает `nil`.

::: warning Замечание
Поле `xdbFile` может содержать строку `"(serialized)"`, если прямой путь к xdb файлу недоступен. Поле `binaryFile` доступно всегда. Типы текстур определяются целочисленными значениями: `0` (DXT1), `1` (DXT3), `2` (DXT5).
:::

## Список параметров
- **`textureId`** (`TextureId`)
Уникальный идентификатор ресурса текстуры.

## Возвращаемые значения
Возвращает таблицу (`table`) со следующими полями:
- **`xdbFile`** (`string`) — путь до xdb файла текстуры. Если файл недоступен, возвращается строка `"(serialized)"`.
- **`binaryFile`** (`string`) — путь до bin файла текстуры. Доступен всегда.
- **`type`** (`number`) — тип текстуры. Возможные значения: `0` (DXT1), `1` (DXT3), `2` (DXT5).
- **`realWidth`** (`number`) — реальная ширина текстуры в пикселях.
- **`realHeight`** (`number`) — реальная высота текстуры в пикселях.

или `nil`, если текстура не найдена.

## Примеры
### Получение информации о текстуре заклинания
```lua
local spellInfo = avatar.GetSpellInfo( spellId )
local textureId = spellInfo and spellInfo.texture
local info = textureId and common.GetTextureInfo( textureId )
if info then
    LogInfo( info )
end
```
::: info Описание примера
В примере сначала извлекается информация о заклинании через `avatar.GetSpellInfo`. Затем проверяется наличие идентификатора текстуры. Если идентификатор существует, вызывается `common.GetTextureInfo` для получения детальных данных о текстуре. При успешном получении данных информация выводится в лог.
:::
