# common.IsAvatarNameValid
Проверяет валидность переданного имени аватара.

## Описание
```lua
common.IsAvatarNameValid( name: WString ): boolean
```
Функция анализирует переданную строку и определяет, соответствует ли она правилам формирования имен персонажей в игре. Возвращает логическое значение, указывающее на допустимость использования данной строки в качестве имени аватара.

## Список параметров
- **`name`** (`WString`)
Строка, содержащая проверяемое имя аватара.

## Возвращаемые значения
Возвращает `boolean` — `true`, если переданная строка является валидным именем аватара, и `false` в противном случае.

## Примеры
### Проверка корректности имени
```lua
local rawName = "ТестовоеИмя"
local wName = userMods.ToWString(rawName)
local isValid = common.IsAvatarNameValid(wName)
```
