# common.IsFatalityPhraseFiltered
Проверяет фразу для добивания на наличие запрещенных выражений.

## Описание
```lua
common.IsFatalityPhraseFiltered( text: WString ): table | nil
```
Функция анализирует переданный текст фразы для добивания на предмет наличия запрещенных выражений и возвращает таблицу с результатами фильтрации, содержащую статус проверки и список отклоненных элементов.

## Список параметров
- **`text`** (`WString`)
Текст фразы для добивания, подлежащий проверке на наличие запрещенных выражений.

## Возвращаемые значения
Возвращает `table` с результатами проверки или `nil`, если операция не может быть выполнена. Таблица содержит следующие поля:
- **`filtered`** (`boolean`) — `true`, если переданный текст содержит запрещенные выражения, и `false` в противном случае.
- **`reason`** (`table` | `nil`) — таблица, содержащая список выражений (`WString`), не прошедших проверку. Если запрещенные выражения отсутствуют, поле принимает значение `nil`.

## Примеры
### Проверка фразы на наличие запрещенных выражений
```lua
local text = userMods.ToWString( "Текст фразы для проверки" )
local result = common.IsFatalityPhraseFiltered( text )

if result then
    if result.filtered then
        LogInfo( "Фраза содержит запрещенные выражения." )
        if result.reason then
            for i = 0, GetTableSize( result.reason ) - 1 do
                LogInfo( "Запрещенное выражение: ", result.reason[ i ] )
            end
        end
    else
        LogInfo( "Фраза прошла проверку." )
    end
else
    LogInfo( "Не удалось выполнить проверку фразы." )
end
```
