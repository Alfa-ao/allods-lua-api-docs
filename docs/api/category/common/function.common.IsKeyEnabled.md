# common.IsKeyEnabled
Возвращает текущее состояние указанной виртуальной клавиши.

## Описание
```lua
common.IsKeyEnabled( vkCode: number ): boolean
```
Функция возвращает логическое значение, указывающее на текущее состояние (активна или неактивна) заданной виртуальной клавиши. Числовые коды клавиш соответствуют стандартам Windows Virtual-Key Codes.

::: tip Совет
Для удобства использования в качестве глобальных переменных предопределены следующие константы виртуальных клавиш:
- `VK_PAUSE` — Pause
- `VK_CAPITAL` — CapsLock
- `VK_NUMLOCK` — NumLock
- `VK_SCROLL` — ScrollLock
- `VK_INSERT` — Insert
- `VK_DELETE` — Delete
:::

## Список параметров
- **`vkCode`** (`number`)
Числовое значение виртуальной клавиши, состояние которой необходимо проверить.

## Возвращаемые значения
`boolean` — `true`, если клавиша в данный момент активна (включена), и `false` в противном случае.

## Примеры
### Проверка состояния клавиши CapsLock
```lua
LogInfo( common.IsKeyEnabled( VK_CAPITAL ) )
```
::: info Описание примера
В примере выполняется проверка состояния клавиши CapsLock с использованием предопределенной глобальной константы `VK_CAPITAL`. Результат проверки выводится в информационный лог.
:::
