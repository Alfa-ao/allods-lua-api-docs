# common.IsOnPayToPlayShard
Определяет, находится ли игрок на подписочном сервере.

## Описание
```lua
common.IsOnPayToPlayShard(): boolean
```
Функция возвращает логическое значение, указывающее на тип текущего сервера. Возвращает `true`, если игрок в данный момент находится на подписочном сервере. В остальных случаях (нахождение на бесплатном сервере, отсутствие подключения к серверу, незавершенная авторизация и т.п.) возвращается `false`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если игрок находится на подписочном сервере, и `false` в остальных случаях (на бесплатном сервере, вне сервера, при незавершенной авторизации).

## Примеры
### Получение статуса подписочного сервера
```lua
local isOnPayToPlayShard = common.IsOnPayToPlayShard()
```
