# common.IsSoundEnabled
Проверяет, включен ли звук в клиенте игры.

## Описание
```lua
common.IsSoundEnabled(): boolean
```
Функция возвращает логическое значение, указывающее на текущее состояние звуковой подсистемы. Возвращаемое значение является константным в рамках одного сеанса работы клиента игры.

::: warning Замечание
При отключенном звуке остальные функции API, связанные с воспроизведением аудио, могут демонстрировать неожиданное поведение, например, возвращать `nil` без выполнения каких-либо действий. Вызов таких функций при отключенном звуке не рекомендуется.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если звук в игре включен, и `false` в противном случае.

## Примеры
### Инициализация звуковых функций в зависимости от состояния
```lua
local isSoundEnabled = common.IsSoundEnabled()
if isSoundEnabled then
    --------------------------------------------------------------------------------
    function InitSounds()
        ....
    end
    --------------------------------------------------------------------------------
    function PlaySound( ... )
        ....
    end
    ....
    --------------------------------------------------------------------------------
else
    function PlaySound() end
end
```
::: info Описание примера
В примере запрашивается состояние звуковой подсистемы. Если звук включен, определяются функции `InitSounds` и `PlaySound` с полной реализацией. В противном случае функция `PlaySound` переопределяется как пустая заглушка, что предотвращает возможные ошибки выполнения при отключенном звуке.
:::
