# common.IsTextFiltered
Определяет наличие запрещенных слов в переданном тексте.

## Описание
```lua
common.IsTextFiltered( text: WString ): boolean
```
Функция анализирует переданную строку и проверяет ее на наличие запрещенных (фильтруемых) слов. Возвращает логическое значение, указывающее на результат проверки.

## Список параметров
- **`text`** (`WString`)
Текст, который необходимо проверить на наличие запрещенных слов.

## Возвращаемые значения
Возвращает `boolean` — `true`, если в тексте содержатся запрещенные слова, и `false` в противном случае.

## Примеры
### Проверка текста на наличие запрещенных слов
```lua
local text = userMods.ToWString( "Пример текста для проверки" )
local isFiltered = common.IsTextFiltered( text )

if isFiltered then
    LogInfo( "Текст содержит запрещенные слова." )
else
    LogInfo( "Текст успешно прошел фильтрацию." )
end
```
