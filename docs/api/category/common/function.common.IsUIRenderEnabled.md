# common.IsUIRenderEnabled
Определяет текущее состояние отображения пользовательского интерфейса.

## Описание
```lua
common.IsUIRenderEnabled(): boolean
```
Функция возвращает логическое значение, указывающее на текущее состояние рендеринга пользовательского интерфейса. Значение `true` означает, что отрисовка UI включена, значение `false` — отключена.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` — `true`, если отображение пользовательского интерфейса включено, и `false`, если отключено.

## Примеры
### Получение состояния отображения UI
```lua
local isUIVisible = common.IsUIRenderEnabled()
```
