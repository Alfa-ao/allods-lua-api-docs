# common.IsValidText
Проверяет, является ли переданное значение валидным текстом.

## Описание
```lua
common.IsValidText( text: any ): boolean
```
Функция анализирует переданное значение и определяет, относится ли оно к допустимым текстовым типам данных. Валидными считаются объекты типа `ValuedObject`, а также непустые значения типов `ValuedText` и `WString`.

## Список параметров
- **`text`** (`any`)
Проверяемое значение любого типа.

## Возвращаемые значения
Возвращает `boolean` — `true`, если переданный аргумент является валидным текстом, и `false` в противном случае.

## Примеры
### Базовая проверка значения на валидность
```lua
local isValid = common.IsValidText( text )
```
