# common.IsWidget
Проверяет, является ли переданный аргумент виджетом.

## Описание
```lua
common.IsWidget( any: any ): boolean
```
Функция проверяет, является ли переданное значение виджетом (объектом типа `userdata`, имеющим тип-наследник `WidgetSafe`).

## Список параметров
- **`any`** (`any`)
Проверяемое значение.

## Возвращаемые значения
Возвращает `boolean` — `true`, если переданный аргумент является виджетом, и `false` в противном случае.

## Примеры
### Проверка значения на принадлежность к виджетам
```lua
local isWidget = common.IsWidget( any )
```
