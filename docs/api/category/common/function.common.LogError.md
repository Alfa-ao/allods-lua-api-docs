# common.LogError
Выводит список строк в консоль или чат с уровнем ошибки и заданным фильтром.

## Описание
```lua
common.LogError( filter: string, text01: string|WString, ...: string|WString )
```
Функция выводит список строк в консоль или чат с префиксом `script: ` и уровнем `ERROR`. Переданные строки объединяются в одну. В качестве аргументов допускается передача как нелокализованных, так и локализованных строк. Параметр `filter` определяет фильтр логирования: если фильтр не активирован соответствующими командами, вызов игнорируется. Глобальная переменная `luaLogErrorShowAddonPath` управляет включением и выключением вывода пути до аддона.

## Список параметров
- **`filter`** (`string`)
Фильтр логирования. Если фильтр не включен соответствующими командами, сообщение игнорируется.
- **`text01`** (`string` | `WString`)
Первый обязательный текстовый аргумент.
- **`...`** (`string` | `WString`)
Список необязательных текстовых аргументов, которые объединяются с первым аргументом.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Вывод сообщения с использованием обычной строки
```lua
common.LogError( "my_filter", "Total Crash! (script/data, not code)" )
```

### Вывод сообщения с использованием локализованной строки
```lua
common.LogError( "my_filter", "Total Crash! ", userMods.ToWString( "(script/data, not code)" ) )
```
