# common.LogInfo
Выводит список строк в консоль или чат с префиксом "script: ".

## Описание
```lua
common.LogInfo( filter: string, text01: string|WString, ...: string|WString )
```
Функция объединяет переданные строки в одну и выводит их в консоль или чат. В качестве аргументов допускается передача как нелокализованных, так и локализованных строк. 

Параметр `filter` определяет фильтр логирования. Если фильтр не включен соответствующими командами (см. `DebugGame`), вызов функции игнорируется.

Для включения или выключения вывода пути до аддона используется глобальная переменная `luaLogInfoShowAddonPath` (см. `DebugGame`).

::: warning Замечание
Первый текстовый аргумент (`text01`) является обязательным для передачи.
:::

## Список параметров
- **`filter`** (`string`)
Фильтр логирования.
- **`text01`** (`string` | `WString`)
Первый обязательный текстовый аргумент.
- **`...`** (`string` | `WString`)
Список необязательных текстовых аргументов.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Вывод простой строки
```lua
common.LogInfo( "common", "Hello, world!" )
```

### Вывод строки с использованием локализованного текста
```lua
common.LogInfo( "common", "Hello, ", userMods.ToWString( "another world!" ) )
```
