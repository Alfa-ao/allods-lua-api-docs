# common.LogWarning
Выводит объединенные строки в консоль или чат с уровнем логирования WARNING.

## Описание
```lua
common.LogWarning( filter: string, text01: string|WString, ...: string|WString )
```
Функция выводит список строк в консоль или чат с префиксом "script: " и уровнем логирования WARNING. Переданные строки объединяются в одну итоговую строку. В качестве аргументов допускается передача как нелокализованных, так и локализованных строк. Параметр `filter` определяет фильтр логирования: если фильтр не включен соответствующими командами (см. DebugGame), вызов функции игнорируется. Для включения или выключения вывода пути до аддона в сообщении используется глобальная переменная `luaLogWarningShowAddonPath` (см. DebugGame).

::: warning Замечание
Вывод сообщения зависит от состояния фильтра логирования. Если фильтр, переданный в параметре `filter`, не активирован командами отладки, сообщение не отображается.
:::

## Список параметров
- **`filter`** (`string`)
Фильтр логирования. Определяет необходимость вывода сообщения в консоль на основе настроек отладки.
- **`text01`** (`string` | `WString`)
Первый обязательный текстовый аргумент.
- **`...`** (`string` | `WString`)
Список необязательных текстовых аргументов. Все переданные аргументы объединяются с первым аргументом в единую строку.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Вывод предупреждения с использованием нелокализованной строки
```lua
common.LogWarning( "common", "The operation may be too long!" )
```

### Объединение нелокализованной и локализованной строк
```lua
common.LogWarning( "common", "The operation may be too long! ", userMods.ToWString( "Or very-very-very long!" ) )
```
