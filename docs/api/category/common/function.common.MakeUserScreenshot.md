# common.MakeUserScreenshot
Создает пользовательский снимок экрана.

## Описание
```lua
common.MakeUserScreenshot( postpone: boolean | nil )
```
Функция инициирует создание пользовательского снимка экрана. В ответ на запрос передается событие `EVENT_USER_SCREENSHOT`. Если параметр `postpone` принимает значение `true`, создание снимка откладывается на два кадра, что обеспечивает применение всех изменений интерфейса, инициированных из скриптов. Указанный флаг затем передается в параметрах события. Функция используется для получения снимков экрана без отображения элементов пользовательского интерфейса.

## Список параметров
- **`postpone`** (`boolean` | `nil`)
Флаг откладки создания снимка. При передаче значения `true` снимок откладывается на два кадра, что необходимо для применения изменений интерфейса. Передача `nil` приводит к немедленному созданию снимка.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Немедленное создание снимка
```lua
common.MakeUserScreenshot( nil )
```

### Создание снимка с задержкой для обновления интерфейса
```lua
common.MakeUserScreenshot( true )
```
