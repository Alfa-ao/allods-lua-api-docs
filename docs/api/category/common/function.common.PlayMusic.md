# common.PlayMusic
Запускает воспроизведение музыки по указанному алиасу.

## Описание
```lua
common.PlayMusic( musicAlias: string )
```
Функция инициирует воспроизведение музыки, параметры которой заданы для указанного алиаса в поле `musicParameters` файла `"\Interface\System\UIGameRoot.(UIGameRoot).xdb"`. Музыка запускается с настройками, соответствующими данному алиасу.

## Список параметров
- **`musicAlias`** (`string`)
Алиас музыкальной композиции, заданный в конфигурационном файле игры.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Запуск музыки по алиасу
```lua
common.PlayMusic( musicAlias )
```
