# common.QuitGame
Немедленно завершает работу игрового клиента, эмулируя разрыв соединения.
## Описание
```lua
common.QuitGame()
```
Функция немедленно завершает работу игрового клиента, независимо от текущего состояния. Эмулирует разрыв соединения аналогично выходу через системное меню в оконном режиме.
## Список параметров
Параметры отсутствуют.
## Возвращаемые значения
Возвращаемое значение отсутствует.
## Примеры
### Завершение работы клиента

```lua
common.QuitGame()
```
