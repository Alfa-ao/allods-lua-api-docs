# common.SendEvent
Посылает событие с заданным именем и параметрами в пределах текущего типа аддонов.

## Описание
```lua
common.SendEvent( eventName: string, eventParams: table )
```
Функция инициирует отправку события с указанным идентификатором и таблицей параметров. Область видимости распространяемого события зависит от контекста вызова: при вызове из встроенного аддона событие передается только встроенным аддонам, при вызове из пользовательского аддона — только пользовательским.

::: warning Замечание
Для имени события настоятельно рекомендуется использовать формат `SCRIPT_[A-Z_]*`. Имя события не должно быть пустым.
:::

::: warning Замечание
Таблица параметров `eventParams` имеет строгие ограничения на структуру и содержимое:
- Допустимые типы ключей: `int32_t`, `string` (непустые, длиной не более 512 символов).
- Допустимые типы значений: `bool`, `number`, `string`, `userdata` (любого зарегистрированного типа), `table` (с учетом вложенных ограничений).
- Таблицы не должны содержать метатаблиц (они будут утрачены при передаче).
- Циклические ссылки запрещены.
- Максимальная глубина вложенности таблиц — 8.
- Максимальное суммарное число ключей в таблице и всех вложенных таблицах — 1024.
:::

## Список параметров
- **`eventName`** (`string`)
Идентификатор события. Должен представлять собой непустую строку. Рекомендуется использовать формат `SCRIPT_[A-Z_]*`.
- **`eventParams`** (`table`)
Таблица с параметрами события. Структура произвольная, но должна строго соответствовать ограничениям на типы данных, глубину вложенности и количество ключей, описанным в разделе «Описание».

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Отправка пользовательского события с параметрами
```lua
local value1 = "test_string"
local value2 = 42

common.SendEvent( "SCRIPT_SAMPLE_EVENT", { eventParam1 = value1, eventParam2 = value2 } )
```
