# common.SetCursor
Устанавливает текущий курсор, получая его из ресурсной системы по указанному псевдониму.

## Описание
```lua
common.SetCursor( name: string )
```
Функция устанавливает текущий курсор интерфейса, получая его из ресурсной системы по указанному псевдониму (alias).

## Список параметров
- **`name`** (`string`)
Псевдоним (alias) курсора, используемый для поиска в ресурсной системе.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Установка курсора по умолчанию
```lua
common.SetCursor( "default" )
```
