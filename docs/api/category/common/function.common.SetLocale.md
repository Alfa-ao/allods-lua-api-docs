# common.SetLocale
Устанавливает текущую языковую локаль.

## Описание
```lua
common.SetLocale( locale: string | number )
```
Функция устанавливает текущую языковую локаль. В качестве параметра принимается либо короткое строковое имя локали, либо её числовой индекс. Допустимые значения берутся из списка, возвращаемого функцией `common.GetLocaleList`.

## Список параметров
- **`locale`** (`string` | `number`)
Короткое строковое имя локали или её числовой индекс. Значение должно присутствовать в списке, полученном через функцию `common.GetLocaleList`.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Установка первой доступной локали из списка
```lua
local localeList = common.GetLocaleList()
if localeList and localeList[1] then
    common.SetLocale( localeList[1] )
end
```
::: info Описание примера
В примере сначала запрашивается список доступных локалей с помощью `common.GetLocaleList`. Если список получен и содержит хотя бы один элемент, вызывается `common.SetLocale` для установки первой локали из этого списка.
:::
