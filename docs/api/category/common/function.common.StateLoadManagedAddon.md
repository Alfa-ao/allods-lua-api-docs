# common.StateLoadManagedAddon
Загружает указанный управляемый аддон.

## Описание
```lua
common.StateLoadManagedAddon( name: string )
```
Функция инициирует загрузку указанного управляемого аддона. Загрузка допускается исключительно для аддонов, которые в данный момент не загружены.

::: warning Замечание
Использование комбинации выгрузки и последующей загрузки (`Unload` + `Load`) не допускается. Вместо нее надлежит использовать механизм перезагрузки (Reload).
:::

## Список параметров
- **`name`** (`string`)
Имя аддона, который необходимо загрузить.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Загрузка аддона Cartographer
```lua
common.StateLoadManagedAddon( "Cartographer" )
```
