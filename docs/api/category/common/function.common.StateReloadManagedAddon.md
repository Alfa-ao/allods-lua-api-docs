# common.StateReloadManagedAddon
Перезагружает указанный управляемый аддон.

## Описание
```lua
common.StateReloadManagedAddon( name: string )
```
Функция инициирует процесс перезагрузки аддона с заданным именем. Перезагрузка возможна исключительно для managed аддонов, которые в данный момент находятся в состоянии `LOADED`.

::: warning Замечание
Перезагрузка доступна только для managed аддонов в состоянии `LOADED`. Попытка перезагрузить аддон в ином состоянии не приведет к выполнению операции.
:::

## Список параметров
- **`name`** (`string`)
Имя аддона, который необходимо перезагрузить.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Перезагрузка аддона Cartographer
```lua
common.StateReloadManagedAddon( "Cartographer" )
```
