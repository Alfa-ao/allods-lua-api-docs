# common.StateUnloadManagedAddon
Выгружает указанный пользовательский аддон из памяти игры.

## Описание
```lua
common.StateUnloadManagedAddon( name: string )
```
Функция инициирует процесс выгрузки указанного аддона из памяти игры. Выгрузка главного аддона или аддонов, имеющих жесткую поддержку в коде игры, невозможна.

::: warning Замечание
Нельзя выгружать главный аддон или аддоны, имеющие поддержку в коде игры.
:::

## Список параметров
- **`name`** (`string`)
Имя выгружаемого аддона.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Выгрузка аддона Cartographer
```lua
common.StateUnloadManagedAddon( "Cartographer" )
```
