# common.UnRegisterReactionHandler
Отписывает обработчик аддона от указанной реакции.

## Описание
```lua
common.UnRegisterReactionHandler( reactionFunction: function, sysReactionName: string )
```
Функция удаляет подписку ранее зарегистрированного обработчика на системную реакцию. На одну реакцию может быть одновременно подписано несколько различных обработчиков, поэтому отписка затрагивает только конкретную указанную функцию.

## Список параметров
- **`reactionFunction`** (`function`)
Функция-обработчик, которую необходимо отписать от реакции.
- **`sysReactionName`** (`string`)
Системное название реакции, от которой производится отписка.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Отписка обработчика от реакции
```lua
common.UnRegisterReactionHandler( OnReactionSample, "reaction_sample" )
```
