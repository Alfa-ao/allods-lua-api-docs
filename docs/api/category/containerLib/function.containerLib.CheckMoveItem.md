# containerLib.CheckMoveItem

Проверяет возможность перемещения предмета.

## Описание

```lua
containerLib.CheckMoveItem( itemId: ObjectId, slotType: number, slot: number|nil, count: number|nil ): boolean, boolean|nil
```

Функция проверяет возможность перемещения предмета. Проверка завершается ошибкой, если предмет не найден в контейнере игрока или указаны некорректные slotType и/или slot. Проверка проваливается, если действует кулдаун на операции с предметами, контейнер не поддерживает перемещение предметов, точка назначения совпадает с текущим положением предмета, слот источника и/или слот назначения находятся в процессе транзакции или запрашивается перемещение большего количества предметов, чем доступно.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

- **`slotType`** (`number`) - Тип контейнера места назначения.

- **`slot`** (`number` | `nil`) - Номер слота в контейнере. Если указан nil, слот подбирается автоматически.

- **`count`** (`number` | `nil`) - Количество перемещаемых предметов. Если указан nil, количество не проверяется.

::: details Допустимые значения
ITEM_CONT_...
:::

## Возвращаемые значения

Функция возвращает два значения:

- `boolean` - `true`, если предмет может быть перемещен, иначе `false`.

- `boolean` | `nil` - `true`, если при перемещении изменится время жизни предмета, иначе `false` или `nil`.

## Примеры

### Проверка возможности перемещения предмета

```lua
containerLib.CheckMoveItem( itemId, ITEM_CONT_INVENTORY, 12, nil )
```
