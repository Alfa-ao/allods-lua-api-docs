# containerLib.CheckMoveSlotItem

Проверяет возможность перемещения предмета.

## Описание

```lua
containerLib.CheckMoveSlotItem( slotTypeFrom: number, slotFrom: number, slotTypeTo: number, slotTo: number | nil, count: number | nil ): boolean, boolean
```

Функция проверяет возможность перемещения предмета.
Проверка завершается с ошибкой, если:
- указаны невалидные `slotType` и/или `slot`.
Проверка проваливается, если:
- действует кулдаун на операции с предметами;
- контейнер не поддерживает перемещение предметов;
- нечего перемещать - в слоте источнике пусто;
- нечего перемещать - точка назначения совпадает с текущим положением предмета;
- слот источник и/или слот назначения в процессе транзакции;
- попытка перенести больше предметов, чем есть в наличии.

Типы контейнеров задаются константами `ITEM_CONT_...`.

## Список параметров

- **`slotTypeFrom`** (`number`) - тип контейнера места источника. Значения из перечисления `ITEM_CONT_...`.

- **`slotFrom`** (`number`) - номер слота в контейнере источнике.

- **`slotTypeTo`** (`number`) - тип контейнера места назначения. Значения из перечисления `ITEM_CONT_...`.

- **`slotTo`** (`number` | `nil`) - номер слота в контейнере назначения. Если указан `nil`, то слот подбирается автоматически.

- **`count`** (`number` | `nil`) - количество перемещаемых предметов. Если указан `nil`, то не проверяется.

## Возвращаемые значения

Функция возвращает два значения:

- `boolean` - `true`, если предмет может быть перемещен, иначе `false`.

- `boolean` - `true`, если при перемещении предмета изменится время его жизни, иначе `false` или `nil`.

## Примеры

### Проверка возможности перемещения

```lua
local canMove = containerLib.CheckMoveSlotItem( ITEM_CONT_INVENTORY, 11, ITEM_CONT_INVENTORY, 12 )
```
