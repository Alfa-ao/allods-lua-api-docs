# containerLib.Close

Закрывает контейнер с предметами у главного игрока.

## Описание

```lua
containerLib.Close( slotType: number )
```

У главного игрока имеется несколько контейнеров с предметами. Некоторые из них всегда присутствуют у игрока, а некоторые (`ITEM_CONT_DEPOSITE`, `ITEM_CONT_BESTIARY`) появляются со своим содержимым только в определенные моменты времени. Для вторых метод закрывает контейнер, и его содержимое становится недоступным на клиенте. Для остальных контейнеров метод не выполняет никаких действий.

## Список параметров

- **`slotType`** (`number`) - Тип контейнера с предметами. Значения соответствуют перечислению `ITEM_CONT_...`.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Закрытие контейнера хранилища

```lua
containerLib.Close( ITEM_CONT_DEPOSITE )
```
