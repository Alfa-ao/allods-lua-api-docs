# containerLib.GetAllResourceItemIds

Возвращает список идентификаторов всех предметов в контейнере с тем же ресурсом, что и у заданного, или `nil`, если такие предметы отсутствуют.

## Описание

```lua
containerLib.GetAllResourceItemIds( slotType: number, id: number ): table | nil
```

Функция извлекает и возвращает список идентификаторов всех предметов в указанном контейнере, которые имеют тот же ресурс, что и предмет с заданным идентификатором. Если в контейнере отсутствуют предметы с таким ресурсом, функция возвращает `nil`.

## Список параметров

- **`slotType`** (`number`) - тип контейнера с предметами. Значения соответствуют перечислению `ITEM_CONT_...`.

- **`id`** (`number`) - идентификатор предмета.

## Возвращаемые значения

Возвращает `table` или `nil`. Возвращаемая таблица содержит идентификаторы предметов и индексируется с 1. Если подходящие предметы не найдены, функция возвращает `nil`.

## Примеры

### Получение списка предметов с тем же ресурсом

```lua
local itemIds = containerLib.GetAllResourceItemIds( ITEM_CONT_INVENTORY, itemResourceId )
```
