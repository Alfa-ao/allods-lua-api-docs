# containerLib.GetAllSimilarItemIds

Возвращает список идентификаторов всех похожих предметов в контейнере.

## Описание

```lua
containerLib.GetAllSimilarItemIds( slotType: number, id: ObjectId ): table | nil
```

Функция считает предметы похожими при совпадении ресурса, уровня, качества, статуса проклятости и статуса привязки. Допустимые типы контейнеров задаются константами `ITEM_CONT_...`.

## Список параметров

- **`slotType`** (`number`) - тип контейнера с предметами. Значения соответствуют константам `ITEM_CONT_...`.

- **`id`** (`ObjectId`) - идентификатор предмета.

## Возвращаемые значения

Возвращает таблицу (`table`), содержащую идентификаторы (`ObjectId`) похожих предметов, индексированную с единицы, или `nil`, если подходящие предметы не найдены.

## Примеры

### Базовое использование

```lua
local itemIds = containerLib.GetAllSimilarItemIds( ITEM_CONT_INVENTORY, itemResourceId )
```
