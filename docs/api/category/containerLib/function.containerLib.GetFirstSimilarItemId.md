# containerLib.GetFirstSimilarItemId

Возвращает идентификатор первого похожего предмета в контейнере.

## Описание

```lua
containerLib.GetFirstSimilarItemId( slotType: number, id: ObjectId ): ObjectId|nil
```

Функция возвращает идентификатор первого предмета в контейнере, похожего на заданный. Предметы считаются похожими, если у них совпадают ресурс, уровень, качество, статус проклятости и статус привязки. Типы контейнеров задаются константами `ITEM_CONT_...`.

## Список параметров

- **`slotType`** (`number`) - тип контейнера с предметами.

- **`id`** (`ObjectId`) - идентификатор предмета.

## Возвращаемые значения

Возвращает `ObjectId` - идентификатор первого похожего предмета, или `nil`, если подходящих предметов не найдено.

## Примеры

### Поиск похожего предмета в инвентаре

```lua
local similarItemId = containerLib.GetFirstSimilarItemId( ITEM_CONT_INVENTORY, itemId )
```
