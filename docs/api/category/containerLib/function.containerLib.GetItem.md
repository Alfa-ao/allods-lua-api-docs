# containerLib.GetItem

Возвращает идентификатор предмета по его слоту или nil, если слот пуст.

## Описание

```lua
containerLib.GetItem( slotType: number, slot: number ): ObjectId | nil
```

Функция извлекает идентификатор предмета по номеру его слота в указанном контейнере. Слот должен быть валидным, в противном случае будет выдана ошибка.

::: warning Замечание
Если указанного слота в контейнере не существует, будет выдана ошибка.
:::

## Список параметров

- **`slotType`** (`number`) - тип контейнера с предметами. Значения соответствуют перечислению `ITEM_CONT_...`.

- **`slot`** (`number`) - номер слота в контейнере.

## Возвращаемые значения

Возвращает `ObjectId` - идентификатор предмета, если он найден, или `nil`, если в указанном слоте нет предмета.

## Примеры

### Базовое использование

```lua
local itemId = containerLib.GetItem( ITEM_CONT_INVENTORY, 12 )
```
