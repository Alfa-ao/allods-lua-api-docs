# containerLib.GetItemSlot
Возвращает информацию о слоте и контейнере, содержащем предмет.

## Описание
```lua
containerLib.GetItemSlot( itemId: ObjectId, slotType: number | nil )
```
Функция извлекает и возвращает информацию о слоте и контейнере, содержащем предмет. Если предмет отсутствует в контейнерах игрока или происходит ошибка, функция возвращает `nil`.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета.

- **`slotType`** (`number` | `nil`) - Тип контейнера для поиска. Если параметр не задан, поиск производится во всех открытых контейнерах.

## Возвращаемые значения
В случае успешного поиска функция возвращает два значения:

- `number` - Тип контейнера (перечисление `ITEM_CONT_...`).

- `number` - Номер слота в контейнере.

В случае отсутствия предмета или возникновения ошибки функция возвращает `nil`.

## Примеры
### Поиск предмета во всех открытых контейнерах
```lua
local slotType, slot = containerLib.GetItemSlot( itemId )
if slotType then
    LogInfo( slotType, slot )
else
    LogInfo( "Not found" )
end
```
