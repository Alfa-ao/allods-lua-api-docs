# containerLib.GetItems

Возвращает таблицу, описывающую содержимое контейнера.

## Описание

```lua
containerLib.GetItems( slotType: number ): table
```

Функция возвращает таблицу, описывающую содержимое контейнера. Ключи - номера слотов (от 0 до `containerLib.GetSize( slotType ) - 1`), значения - идентификаторы предметов в соответствующих слотах. Таблица может содержать пропуски, поэтому обход через `ipairs` или `zpairs` гарантированно приводит к ошибкам. Необходимо использовать итератор `pairs` или, если важен порядок, обход по индексам от 0 до `containerLib.GetSize( slotType ) - 1`.

::: warning Замечание
Категорически запрещается вычислять размер контейнера как размер таблицы, возвращаемой данным API. Необходимо использовать только `containerLib.GetSize( slotType )`.
:::

## Список параметров

- **`slotType`** (`number`) - тип контейнера с предметами.

::: details Допустимые значения
Значения перечисления `ITEM_CONT_...`.
:::

## Возвращаемые значения

Возвращает таблицу (`table`), описывающую содержимое контейнера. Ключи - номера слотов (от 0 до `containerLib.GetSize( slotType ) - 1`), значения - идентификаторы предметов в соответствующих слотах.

## Примеры

### Получение содержимого контейнера

```lua
local size = containerLib.GetSize( ITEM_CONT_INVENTORY )
local itemIds = containerLib.GetItems( ITEM_CONT_INVENTORY )
```

::: info Описание примера
Получается размер контейнера инвентаря и извлекается таблица с идентификаторами предметов, находящихся в слотах данного контейнера.
:::
