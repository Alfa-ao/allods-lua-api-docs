# containerLib.GetSize

Возвращает размер контейнера главного игрока.

## Описание

```lua
containerLib.GetSize( slotType: number ): number
```

Функция возвращает точный размер указанного контейнера главного игрока.

::: warning Замечание
Функция `containerLib.GetItems` может вернуть пустой или неполный список, если контейнер не полон. Точный размер контейнера возвращается только функцией `containerLib.GetSize`. Некоторые контейнеры (например, переполненная сумка) имеют плавающий размер по количеству содержащихся предметов, поэтому проверка размера выполняется перед каждой операцией с предметами по номеру слота.
:::

## Список параметров

- **`slotType`** (`number`) - тип контейнера с предметами.

## Возвращаемые значения

number (integer) - размер контейнера.

## Примеры

### Получение размера и списка предметов контейнера

```lua
local size = containerLib.GetSize( ITEM_CONT_INVENTORY )
local itemIds = containerLib.GetItems( ITEM_CONT_INVENTORY )
```

::: info Описание примера
В примере запрашивается точный размер инвентаря с помощью `containerLib.GetSize`, а затем извлекается список идентификаторов предметов через `containerLib.GetItems`.
:::
