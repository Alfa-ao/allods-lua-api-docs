# containerLib.IsInteracting
Определяет возможность открытия контейнера посредством взаимодействия с соответствующим NPC.

## Описание
```lua
containerLib.IsInteracting( slotType: number ): boolean
```
Функция определяет, возможно ли открытие контейнера с помощью NPC, то есть находится ли персонаж в состоянии разговора с соответствующим персонажем. Если контейнер не требует разговора для открытия или не требует открытия (всегда открыт), функция возвращает `true`. Список допустимых контейнеров см. в `ITEM_CONT_...`.

## Список параметров
- **`slotType`** (`number`) - тип контейнера с предметами. Значения соответствуют константам `ITEM_CONT_...`.

## Возвращаемые значения
Возвращает `boolean` - `true`, если персонаж находится в состоянии разговора с подходящим NPC, позволяющим открыть контейнер, или если контейнер не требует взаимодействия. В противном случае возвращает `false`.

## Примеры
### Проверка и открытие контейнера
```lua
if containerLib.IsInteracting( ITEM_CONT_DEPOSITE ) and not containerLib.IsOpen( ITEM_CONT_DEPOSITE ) then
    containerLib.Open( ITEM_CONT_DEPOSITE )
end
```
::: info Описание примера
В примере проверяется, находится ли персонаж в состоянии взаимодействия с NPC для открытия контейнера депозита, и не открыт ли он уже. Если оба условия соблюдены, инициируется открытие контейнера.
:::
