# containerLib.IsItemEnabled
Проверяет наличие активных транзакций, связанных с указанным предметом.

## Описание
```lua
containerLib.IsItemEnabled( itemId: ObjectId ): boolean
```
При наличии таковых выполнение любых действий (исключая получение информации) с предметом запрещено. Предмет должен быть валиден. Если предмет валиден, но не существует в контейнере игрока (например, предмет это ревард в описании какого-либо события, квеста и т.п.), функция возвращает `true`.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор экземпляра предмета.

## Возвращаемые значения
`boolean` - `true`, если активных транзакций с данным предметом нет и можно выполнять операции, в противном случае `false`.

## Примеры
### Проверка доступности действий с предметом
```lua
local actionsAvailable = containerLib.IsItemEnabled( itemId )
```
