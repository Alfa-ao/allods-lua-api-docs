# containerLib.IsItemSlotEnabled
Проверяет наличие активных транзакций, связанных с указанным слотом контейнера.

## Описание
```lua
containerLib.IsItemSlotEnabled( slotType: number, slot: number ): boolean
```
При наличии активных транзакций выполнение любых действий, за исключением получения информации, со слотом запрещается. Слот должен быть валидным. Если указанный слот в контейнере отсутствует, выдается ошибка.

## Список параметров
- **`slotType`** (`number`) - Тип контейнера с предметами. Значения соответствуют перечислению `ITEM_CONT_...`.

- **`slot`** (`number`) - Номер слота в контейнере.

## Возвращаемые значения
Возвращает `boolean` - `true`, если активные транзакции с данным слотом отсутствуют и выполнение операций разрешено, в противном случае - `false`.

## Примеры
### Проверка доступности слота инвентаря
```lua
local actionsAvailable = containerLib.IsItemSlotEnabled( ITEM_CONT_INVENTORY, 12 )
```
