# containerLib.IsOpen

Проверяет, открыт ли контейнер с предметами и доступно ли его содержимое.

## Описание

```lua
containerLib.IsOpen( slotType: number ): boolean
```

Функция проверяет доступность контейнеров с предметами главного игрока. Некоторые из них присутствуют всегда, а некоторые, такие как `ITEM_CONT_DEPOSITE` и `ITEM_CONT_BESTIARY`, появляются со своим содержимым только в определенные моменты времени. Для вторых функция возвращает `true`, если содержимое доступно на клиенте (контейнер открыт), иначе возвращается `false` (контейнер закрыт). Для остальных контейнеров функция всегда возвращает `true` (эти контейнеры всегда открыты).

## Список параметров

- **`slotType`** (`number`) - тип контейнера с предметами. Значения соответствуют перечислению `ITEM_CONT_...`.

## Возвращаемые значения

Возвращает `boolean` - `true`, если контейнер открыт и его содержимое доступно игроку, иначе `false`.

## Примеры

### Проверка доступности контейнера

```lua
if containerLib.IsOpen( ITEM_CONT_DEPOSITE ) then
    local itemIds = containerLib.GetItems( ITEM_CONT_DEPOSITE )
end
```

::: info Описание примера
В примере проверяется доступность содержимого контейнера `ITEM_CONT_DEPOSITE`. Если контейнер открыт, извлекается список идентификаторов предметов с помощью функции `containerLib.GetItems`.
:::
