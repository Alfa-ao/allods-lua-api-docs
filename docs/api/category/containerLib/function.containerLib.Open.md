# containerLib.Open

Открывает контейнер, в том числе через разговор с NPC, и не выполняет действий, если контейнер уже открыт.

## Описание

```lua
containerLib.Open( slotType: number )
```

Функция открывает контейнер, в том числе через разговор с NPC. Если контейнер уже открыт, функция не выполняет действий.

## Список параметров

- **`slotType`** (`number`) - Тип контейнера с предметами.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Открытие контейнера

```lua
containerLib.Open( ITEM_CONT_DEPOSITE )
```
