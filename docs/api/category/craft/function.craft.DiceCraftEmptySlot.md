# craft.DiceCraftEmptySlot

Делает заданный слот пустым.

## Описание

```lua
craft.DiceCraftEmptySlot( slot: number )
```

Функция делает заданный слот пустым.

## Список параметров

- **`slot`** (`number`) - номер слота (отсчитывается от 0).

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Очистка слота

```lua
craft.DiceCraftEmptySlot( 0 )
```
