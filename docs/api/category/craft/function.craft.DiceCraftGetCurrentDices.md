# craft.DiceCraftGetCurrentDices
Возвращает текущие значения костей.

## Описание
```lua
craft.DiceCraftGetCurrentDices()
```
Функция возвращает таблицу с текущими значениями костей или `nil`, если режим крафтинга не включён, мини-игра не начата или список значений костей пуст. Возвращаемая таблица содержит текущие значения костей, где ключом выступает номер слота кости, а значением - выпавшее число.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

Возвращает `nil`, если режим крафтинга не включён, мини-игра не начата или список значений костей пуст. В противном случае возвращается таблица (индексация начинается с 0) текущих значений костей:

- **ключ** (`number`) - номер слота кости.

- **значение** (`number`) - текущее значение, выпавшее для кости в этом слоте.

## Примеры
### Получение текущих значений костей
```lua
local dices = craft.DiceCraftGetCurrentDices()
```
