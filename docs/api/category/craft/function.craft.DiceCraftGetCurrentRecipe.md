# craft.DiceCraftGetCurrentRecipe

Возвращает идентификатор текущего рецепта крафтинга.

## Описание

```lua
craft.DiceCraftGetCurrentRecipe(): RecipeId | nil
```

Функция возвращает идентификатор рецепта, по которому проводится мини-игра крафтинга. Функция доступна только во время мини-игры крафтинга по механике dice craft.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

`RecipeId` - Идентификатор рецепта, по которому проводится мини-игра крафтинга.

`nil` - Если произошла ошибка или функция вызвана вне мини-игры.

## Примеры

### Получение идентификатора текущего рецепта

```lua
local recipeId = craft.DiceCraftGetCurrentRecipe()
```
