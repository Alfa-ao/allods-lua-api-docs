# craft.DiceCraftGetDiceValues

Возвращает список всех возможных значений, которые могут принимать слоты при ролле.

## Описание

```lua
craft.DiceCraftGetDiceValues(): table|nil
```

Функция возвращает список всех возможных значений, которые могут принимать слоты при ролле. Если режим крафтинга не включён, возвращается `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

Возвращается `nil`, если режим крафтинга не включён. Иначе возвращается список, индексированный от 0, содержащий возможные значения.

## Примеры

### Получение возможных значений слотов

```lua
local values = craft.DiceCraftGetDiceValues()
```
