# craft.DiceCraftGetItemSlotsCount

Возвращает число слотов для размещения предметов в режиме крафтинга.

## Описание

```lua
craft.DiceCraftGetItemSlotsCount(): number|nil
```

Функция возвращает число слотов, в которых можно расположить предметы, или `nil`, если режим крафтинга не включен.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - число слотов, в которых можно расположить предметы, или `nil`, если режим крафтинга не включен.

## Примеры

### Получение числа слотов

```lua
local itemSlotsCount = craft.DiceCraftGetItemSlotsCount()
```
