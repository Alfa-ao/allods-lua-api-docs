# craft.DiceCraftGetLockedSlots

Возвращает список слотов, которые заблокированы от изменений при реролле.

## Описание

```lua
craft.DiceCraftGetLockedSlots(): table|nil
```

Функция возвращает список номеров слотов, которые заблокированы от изменений при реролле, или `nil`, если режим крафтинга не включён.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает таблицу (`table`) со списком номеров (`integer`) слотов, ключи которой начинаются с `0`, или `nil`, если режим крафтинга не включён.

## Примеры

### Получение списка заблокированных слотов

```lua
local lockedSlots = craft.DiceCraftGetLockedSlots()
```
