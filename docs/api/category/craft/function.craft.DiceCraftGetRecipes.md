# craft.DiceCraftGetRecipes
Возвращает список рецептов, доступных в текущем сеансе крафтинга.

## Описание
```lua
craft.DiceCraftGetRecipes(): table|nil
```
Функция возвращает список идентификаторов (`RecipeId`) рецептов, доступных в текущем сеансе крафтинга, или `nil` в случае возникновения ошибки.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую список идентификаторов рецептов (`RecipeId`), индексация которой начинается с нуля, или `nil` в случае возникновения ошибки.

## Примеры
### Получение и проверка списка доступных рецептов
```lua
local recipes = craft.DiceCraftGetRecipes()
if recipes and recipes[0] then
    LogInfo( avatar.GetRecipeInfo( recipes[0] ) )
end
```
::: info Описание примера
В примере запрашивается список доступных рецептов. Если таблица успешно получена и содержит хотя бы один элемент, выводится информация о первом рецепте с помощью функции `avatar.GetRecipeInfo`.
:::
