# craft.FinishReforge

Завершает процесс перековки, выбирая из доступных вариантов нужный предмет, или отказывается от результатов.

## Описание

```lua
craft.FinishReforge( choice: number ): boolean
```

Функция завершает процесс перековки. При передаче индекса варианта происходит выбор соответствующего предмета. При передаче нуля происходит отказ от результатов перековки.

## Список параметров

- **`choice`** (`number`) - Желаемый вариант результата перековки. Индексация начинается с 1. Если передано значение 0, происходит отказ от результатов перековки.

## Возвращаемые значения

Возвращает `boolean` - `true` в случае успешного выполнения, иначе `false`.

## Примеры

### Выбор первого варианта

```lua
craft.FinishReforge( 1 )
```
