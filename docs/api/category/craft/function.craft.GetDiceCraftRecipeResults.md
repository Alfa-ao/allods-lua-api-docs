# craft.GetDiceCraftRecipeResults

Возвращает список всех возможных для рецепта результатов изготовления предметов.

## Описание

```lua
craft.GetDiceCraftRecipeResults( recipeId: RecipeId ): table|nil
```

Функция извлекает и возвращает список всех возможных для рецепта результатов изготовления предметов.

## Список параметров

- **`recipeId`** (`RecipeId`) - Идентификатор рецепта.

## Возвращаемые значения

Возвращает `table` или `nil`.

Значение `nil` возвращается при возникновении ошибки (например, если рецепт не относится к DiceCraft).

В противном случае возвращается таблица со следующими полями:

- **`defaultItemId`** (`ObjectId`) - Идентификатор предмета, который будет результатом, если не выпадут одинаковые значения для всех слотов.

- **`results`** (`table`) - Список (индексированный от 0) возможных значений. Каждое значение представляет собой таблицу с полями:
    - **`diceValue`** (`number`) - Значение, которое должно выпасть для всех слотов, чтобы результат был именно этот.
    - **`itemId`** (`ObjectId`) - Идентификатор предмета, который будет результатом крафтинга (в количестве `resultQuantity`).
    - **`resultQuantity`** (`number`) - Количество предметов.
    - **`bonusQuantity`** (`number`) - Количество дополнительных предметов, которое будет дано за наличие непотраченных роллов.

## Примеры

### Получение результатов изготовления

```lua
local results = craft.GetDiceCraftRecipeResults( recipeId )
```
