# craft.GetForgeRecipes
Возвращает список доступных рецептов для крафта в кузнице.

## Описание
```lua
craft.GetForgeRecipes( ignorePredicates: boolean|nil ): table|nil
```
Функция возвращает список идентификаторов рецептов, доступных для выполнения крафта в кузнице. Если параметр `ignorePredicates` установлен в `true`, в результат включаются также недоступные в данный момент рецепты. В случае возникновения ошибки функция возвращает `nil`.

## Список параметров
- **`ignorePredicates`** (`boolean` | `nil`) - Определяет необходимость игнорирования предикатов. Если передано значение `true`, возвращаются даже недоступные рецепты. Необязательный параметр, по умолчанию принимает значение `false`.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую идентификаторы (`ForgeRecipeId`) рецептов для крафта в кузнице. Индексация элементов в таблице начинается с единицы. В случае возникновения ошибки возвращается `nil`.

## Примеры
### Получение списка рецептов
```lua
local recipes = craft.GetForgeRecipes()
```
