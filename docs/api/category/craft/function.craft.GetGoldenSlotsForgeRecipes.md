# craft.GetGoldenSlotsForgeRecipes
Возвращает список доступных рецептов для объединения золотых слотов.

## Описание
```lua
craft.GetGoldenSlotsForgeRecipes( ignorePredicates: boolean | nil ): table | nil
```
Функция возвращает список идентификаторов рецептов для крафта в системе forge. При возникновении ошибки возвращается `nil`. Индексация элементов в списке начинается с 1.

## Список параметров
- **`ignorePredicates`** (`boolean` | `nil`) - Игнорировать предикаты. Если передано `true`, возвращаются даже недоступные рецепты. Необязательный параметр, по умолчанию `false`.

## Возвращаемые значения
Возвращает `table` или `nil`.
Таблица содержит список идентификаторов рецептов (`ForgeRecipeId`) для forge крафта, индексируемый от 1. При ошибке возвращается `nil`.

## Примеры
### Получение списка рецептов
```lua
local recipes = craft.GetGoldenSlotsForgeRecipes()
```
