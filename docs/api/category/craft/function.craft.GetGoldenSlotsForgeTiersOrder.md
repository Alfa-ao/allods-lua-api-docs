# craft.GetGoldenSlotsForgeTiersOrder
Возвращает список ресурсов тира рецептов для объединения золотых слотов.

## Описание
```lua
craft.GetGoldenSlotsForgeTiersOrder(): table|nil
```
Функция возвращает таблицу с идентификаторами ресурсов тира рецептов для объединения золотых слотов, если они доступны, или `nil` в противном случае. Индексация элементов в таблице начинается с нуля.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую идентификаторы (`ForgeResourceId`) ресурсов тира рецептов для объединения золотых слотов. Индексация элементов в таблице начинается с нуля. Если данные недоступны, возвращается `nil`.

## Примеры
### Получение списка ресурсов тира рецептов
```lua
local recipesTiers = craft.GetGoldenSlotsForgeTiersOrder()
```
