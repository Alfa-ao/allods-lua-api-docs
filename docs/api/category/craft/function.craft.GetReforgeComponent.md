# craft.GetReforgeComponent

Возвращает компонент для reforge крафта, установленный в указанный слот.

## Описание

```lua
craft.GetReforgeComponent( slot: number ): ObjectId | nil
```

Функция извлекает идентификатор компонента для reforge крафта, находящегося в заданном слоте. Если компонент в слот не установлен, функция возвращает `nil`.

## Список параметров

- **`slot`** (`number`) - Идентификатор слота для reforge крафта (от 0 до 7).

## Возвращаемые значения

Возвращает идентификатор предмета или `nil`.

- **`itemId`** (`ObjectId` | `nil`) - Идентификатор предмета в указанном слоте или `nil`, если компонент в этот слот не был установлен.

## Примеры

### Получение компонента из нулевого слота

```lua
local itemId = craft.GetReforgeComponent( 0 )
LogInfo( itemId )
```
