# craft.GetReforgeInfo

Возвращает список доступных рефоржей для предмета.

## Описание

```lua
craft.GetReforgeInfo( itemId: ObjectId, offenceMode: boolean ): table|nil
```

Функция возвращает список доступных рефоржей для предмета с учетом выбранного режима атаки или защиты.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.
- **`offenceMode`** (`boolean`) - Если `true`, то искать доступный рефорж на атаку, если `false`, то искать доступный рефорж на защиту.

## Возвращаемые значения

Возвращает таблицу (`table`) с информацией о рефорже или `nil` в случае возникновения ошибки.

- **`price`** (`number`) - Стоимость модификации.
- **`recipe`** (`ReforgeResourceId`) - Идентификатор ресурса для reforge крафта.
- **`resources`** (`table`) - Список (индексируется с 1) идентификаторов предмета-ресурса для рефоржа.
- **`altCurrencies`** (`table`) - Индексированный с 1 список требуемой альтвалюты, каждая ячейка содержит следующие поля:
    - **`currency`** (`CurrencyId`) - Ресурсный идентификатор валюты.
    - **`count`** (`number`) - Количество требуемой валюты.

## Примеры

### Получение информации о рефорже

```lua
local recipe = craft.GetReforgeInfo( itemId )
```
