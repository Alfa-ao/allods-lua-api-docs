# craft.GetReforgeRecipe

Возвращает текущий рецепт и предмет для reforge крафта.

## Описание

```lua
craft.GetReforgeRecipe(): table | nil
```

Функция возвращает текущий рецепт и идентификатор предмета для reforge крафта. Если рецепт не был установлен, функция возвращает `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает таблицу (`table`) или `nil`, если рецепт не был установлен.

- **`recipe`** (`ReforgeResourceId`) - идентификатор ресурса для reforge крафта.
- **`itemId`** (`ObjectId`) - идентификатор предмета для reforge крафта.

## Примеры

### Получение текущего рецепта reforge

```lua
LogInfo( craft.GetReforgeRecipe() )
```
