# craft.GetReforgeVariants

Возвращает список доступных вариантов для reforge крафта.

## Описание

```lua
craft.GetReforgeVariants(): table|nil
```

Функция возвращает список идентификаторов (`ObjectId`) доступных вариантов для reforge крафта. Если результаты ещё не получены, функция возвращает `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если результаты ещё не получены, возвращается `nil`. В противном случае возвращается список, индексируемый от 1, содержащий идентификаторы (`ObjectId`) вариантов нарефорженных предметов.

## Примеры

### Получение списка вариантов

```lua
local variants = craft.GetReforgeVariants()
```
