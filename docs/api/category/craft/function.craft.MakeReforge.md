# craft.MakeReforge

Совершает крафт перековки (reforge) с установленным рецептом и компонентами.

## Описание

```lua
craft.MakeReforge(): boolean
```

Функция возвращает статус успешности выполнения крафта перековки (reforge) с установленным рецептом и компонентами.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean`.

`true` в случае успеха, `false` в случае неудачи.

## Примеры

### Базовый вызов

```lua
local success = craft.MakeReforge()
```
