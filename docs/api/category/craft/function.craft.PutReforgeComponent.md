# craft.PutReforgeComponent

Размещает компонент для крафта перековки.

## Описание

```lua
craft.PutReforgeComponent( slot: number, itemId: ObjectId | nil ): boolean
```

Функция размещает предмет в заданный слот крафта перековки или извлекает его из слота. Если в качестве идентификатора передано значение `nil`, происходит очистка указанного слота.

## Список параметров

- **`slot`** (`number`) - Индекс слота для крафта перековки.

- **`itemId`** (`ObjectId` | `nil`) - Идентификатор предмета, размещаемого в указанный слот. Передача значения `nil` приводит к очистке слота.

## Возвращаемые значения

Возвращает `boolean` - `true`, если компонент успешно установлен, и `false` в противном случае (например, если не установлен рецепт).

## Примеры

### Размещение компонента в слот перековки

```lua
local success = craft.PutReforgeComponent( slotIndex, itemId )
if success then
    LogInfo( "Компонент успешно установлен." )
else
    LogInfo( "Не удалось установить компонент." )
end
```
