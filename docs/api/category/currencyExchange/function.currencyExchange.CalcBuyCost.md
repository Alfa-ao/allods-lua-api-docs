# currencyExchange.CalcBuyCost
Рассчитывает и возвращает количество золота, необходимое для покупки кристаллов в сервисе обмена валют.

## Описание
```lua
currencyExchange.CalcBuyCost( crystalsToBuy: number ): number
```
Функция возвращает количество золота, требуемое для приобретения заданного количества кристаллов. Курс постоянно изменяется в зависимости от резерва кристаллов и золота.

## Список параметров
- **`crystalsToBuy`** (`number`) - Количество кристаллов, которые планируется приобрести.

## Возвращаемые значения
Возвращает `number` - количество золота, необходимое для покупки указанного объема кристаллов.

## Примеры
### Расчет стоимости покупки кристаллов
```lua
local crystalsToBuy = 100
if currencyExchange.IsServiceActive() then
    local gold = currencyExchange.CalcBuyCost( crystalsToBuy )
    LogInfo( "Необходимо золота: " .. gold )
end
```
::: info Описание примера
В примере проверяется активность сервиса обмена валют. Если сервис активен, рассчитывается стоимость покупки ста кристаллов, а полученное значение выводится в лог.
:::
