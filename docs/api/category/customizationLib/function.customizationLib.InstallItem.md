# customizationLib.InstallItem

Помещает предмет из контейнера игрока в кастомизационный слот во время работы с активным слотом.

## Описание

```lua
customizationLib.InstallItem( itemId: ObjectId )
```

Функция помещает предмет из контейнера игрока в кастомизационный слот при работе с активным слотом во время кастомизации.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета, который необходимо поместить в слот.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка предмета в кастомизационный слот

```lua
customizationLib.InstallItem( itemId )
```
