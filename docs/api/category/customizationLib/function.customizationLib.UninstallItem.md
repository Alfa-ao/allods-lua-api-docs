# customizationLib.UninstallItem
Убирает объект из активного кастомизационного слота, перемещая предмет в контейнер игрока.

## Описание
```lua
customizationLib.UninstallItem()
```
Функция убирает объект из активного кастомизационного слота, перемещая предмет из этого слота в контейнер игрока во время кастомизации.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Удаление предмета из слота кастомизации
```lua
customizationLib.UninstallItem()
```
