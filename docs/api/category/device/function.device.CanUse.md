# device.CanUse

Проверяет возможность использования устройства в текущий момент.

## Описание

```lua
device.CanUse( deviceId: ObjectId ): boolean
```

Функция определяет, можно ли попытаться использовать устройство с указанным идентификатором в текущий момент.

## Список параметров

- **`deviceId`** (`ObjectId`) - Уникальный идентификатор устройства.

## Возвращаемые значения

Возвращает `boolean` - `true`, если устройство можно использовать в текущий момент, и `false` в противном случае.

## Примеры

### Проверка возможности использования устройства

```lua
local canUse = device.CanUse( deviceId )
```
