# device.GetItemInstalled

Возвращает идентификатор предмета, вставленного в указанное устройство.

## Описание

```lua
device.GetItemInstalled( usableDeviceId: ObjectId ): ObjectId|nil
```

Функция возвращает идентификатор предмета, который вставлен в указанное устройство. В настоящее время предметы вставляются только в корабельные устройства. Если предмет не найден, функция возвращает `nil`.

## Список параметров

- **`usableDeviceId`** (`ObjectId`) - Идентификатор устройства.

::: info Перечисление
Допустимые значения идентификаторов устройств начинаются с префикса `USDEV_`.
:::

## Возвращаемые значения

Возвращает `ObjectId` или `nil`.

- **`ObjectId`** (`ObjectId`) - Идентификатор вставленного предмета.

- **`nil`** (`nil`) - Предмет не найден в устройстве.

## Примеры

### Получение вставленного предмета

```lua
local usableDeviceId = 12345
local itemId = device.GetItemInstalled( usableDeviceId )
```
