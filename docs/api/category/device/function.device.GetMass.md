# device.GetMass

Возвращает информацию о массе корабельного устройства (например, рули, пушки на корабле).

## Описание

```lua
device.GetMass( usableDeviceId: ObjectId ): number
```

Функция возвращает массу устройства. Если устройство не найдено или не является UsableDevice, функция бросает исключение и возвращает `0`.

## Список параметров

- **`usableDeviceId`** (`ObjectId`) - Идентификатор устройства.

## Возвращаемые значения

Возвращает `number` - массу.

## Примеры

### Получение массы устройства

```lua
local mass = device.GetMass( usableDeviceId )
```
