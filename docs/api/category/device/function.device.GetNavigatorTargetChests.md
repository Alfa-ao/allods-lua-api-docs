# device.GetNavigatorTargetChests
Извлекает список названий сундуков в трюме корабля, осматриваемого через визор навигатора.

## Описание
```lua
device.GetNavigatorTargetChests(): table|nil
```
Функция возвращает таблицу названий увиденных сундуков при их наличии или `nil` при их отсутствии.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

- **`table`** (`WString`) - Таблица, индексированная с `1`, содержащая названия увиденных сундуков.

## Примеры
### Получение списка сундуков
```lua
local chests = device.GetNavigatorTargetChests()
if chests then
    LogInfo( chests[1] )
end
```
