# device.GetShipSlotInfo

Возвращает информацию о слоте используемого устройства, например пушки на корабле.

## Описание

```lua
device.GetShipSlotInfo( deviceId: ObjectId ): table|nil
```

Функция возвращает таблицу с информацией о слоте, в котором расположено используемое устройство, например пушка на корабле. Сторона корабля определяется перечислением `SHIP_SIDE_...`. Если устройство не найдено, не используется или не находится на корабле, функция возвращает `nil`.

## Список параметров

- **`deviceId`** (`ObjectId`) - Идентификатор устройства.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если информация успешно получена, возвращается таблица со следующими полями:

- **`interfaceSlot`** (`number`) - Индекс слота при показе устройства в интерфейсе.

- **`side`** (`number`) - Сторона корабля.

::: details Допустимые значения
Перечисление `SHIP_SIDE_...`
:::

- **`sysSlotType`** (`string`) - Отладочное имя слота (недоступно на `_FINALRELEASE`).

::: details Допустимые значения
Перечисление `SlotName`
:::

## Примеры

### Получение информации о слоте устройства

```lua
local slotInfo = device.GetShipSlotInfo( deviceId )
if slotInfo then
    local slotIndex = slotInfo.interfaceSlot
end
```
