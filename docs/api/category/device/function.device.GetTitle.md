# device.GetTitle

Возвращает титул устройства.

## Описание

```lua
device.GetTitle( deviceId: ObjectId ): WString
```

Функция возвращает титул устройства. Если имя не определено, возвращается пустая строка.

## Список параметров

- **`deviceId`** (`ObjectId`) - Идентификатор юнита.

## Возвращаемые значения

Возвращает `WString` - имя юнита. Если имя не определено, возвращается пустая строка.

## Примеры

### Получение титула устройства

```lua
local title = device.GetTitle( unitId )
```
