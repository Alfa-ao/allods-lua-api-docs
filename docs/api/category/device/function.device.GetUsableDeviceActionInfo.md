# device.GetUsableDeviceActionInfo

Возвращает информацию о действии устройства, например о выстреле из пушки.

## Описание

```lua
device.GetUsableDeviceActionInfo( usableDeviceId: ObjectId, actionIndex: number ): table|nil
```

Функция возвращает таблицу с информацией о конкретном действии доступного устройства или `nil`, если произошла ошибка.

## Список параметров

- **`usableDeviceId`** (`ObjectId`) - Идентификатор устройства.

- **`actionIndex`** (`number`) - Номер действия. Должен попадать в диапазон от `0` до `actionsCount - 1`.

## Возвращаемые значения

Возвращает `table` или `nil`.

При успешном выполнении возвращается таблица со следующими полями:

- **`name`** (`WString`) - Название действия.

- **`description`** (`ValuedText` | `nil`) - Описание с подставленными текущими значениями параметров.

- **`image`** (`TextureId`) - Пиктограмма действия. Доступно только в состоянии UI.

- **`enabled`** (`boolean`) - Флаг доступности действия для использования в текущий момент.

- **`active`** (`boolean`) - Флаг активности действия.

- **`cost`** (`number` | `nil`) - Цена действия в эманациях, если они требуются.

- **`isPointed`** (`boolean`) - Флаг, указывающий на необходимость указания точки на местности.

- **`radius`** (`number`) - Радиус воздействия. Присутствует, если действие требует указания точки.

- **`range`** (`number`) - Максимальная дальность применения. Присутствует, если действие требует указания точки.

- **`minRange`** (`number`) - Минимальная дальность применения. Присутствует, если действие требует указания точки.

- **`cooldown`** (`table` | `nil`) - Информация о времени отката, если действие может иметь кулдаун. Содержит поля:
    - `predictedCooldown` (`number`) - прогнозируемое время отката.
    - `durationMs` (`number`) - полное время текущего отката.
    - `remainingMs` (`number`) - оставшееся время текущего отката.

## Примеры

### Получение информации о первом действии устройства

```lua
local info = device.GetUsableDeviceInfo( usableDeviceId )

if info and info.actionsCount > 0 then
    LogInfo( "Action list: " )
    local action = device.GetUsableDeviceActionInfo( usableDeviceId, 0 )
    LogInfo( action.name )
else
    LogInfo( "No actions" )
end
```

::: info Описание примера
В примере извлекается общая информация об устройстве. Если устройство имеет хотя бы одно действие, запрашивается информация о нулевом действии и выводится его название в лог. В противном случае выводится сообщение об отсутствии действий.
:::
