# device.HasNavigatorFollowedMarker
Возвращает статус преследования корабля главного игрока активным навигационным устройством.

## Описание
```lua
device.HasNavigatorFollowedMarker(): boolean|nil
```
Функция возвращает статус преследования корабля главного игрока. Метод работает исключительно для активного навигационного устройства корабля.

::: warning Замечание
Функция работает только для активного навигационного устройства, используемого главным игроком.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` или `nil`.

- **`boolean`** - `true`, если корабль главного игрока преследуется, `false`, если не преследуется.

- **`nil`** - если отсутствует возможность отслеживать корабли-преследователи.

## Примеры
### Проверка статуса преследования
```lua
local hasNavigatorFollowedMarker = device.HasNavigatorFollowedMarker()
if hasNavigatorFollowedMarker == true then
    -- точно преследуется
elseif hasNavigatorFollowedMarker == false then
    -- точно не преследуется
end
```
