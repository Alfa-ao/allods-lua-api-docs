# device.IsInUse

Проверяет, используется ли устройство кем-либо в текущий момент.

## Описание

```lua
device.IsInUse( deviceId: ObjectId ): boolean
```

Функция возвращает признак использования устройства кем-либо в текущий момент.

## Список параметров

- **`deviceId`** (`ObjectId`) - Идентификатор устройства.

## Возвращаемые значения

Возвращает `boolean` - `true`, если устройство используется кем-либо в текущий момент, и `false` в противном случае.

## Примеры

### Проверка использования устройства

```lua
local isInUse = device.IsInUse( deviceId )
```
