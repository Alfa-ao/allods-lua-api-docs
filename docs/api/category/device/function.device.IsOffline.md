# device.IsOffline

Определяет, выключено ли используемое устройство.

## Описание

```lua
device.IsOffline( usableDeviceId: ObjectId ): boolean
```

Функция возвращает `true`, если указанное используемое устройство (например, прибор на корабле) находится в выключенном состоянии.

## Список параметров

- **`usableDeviceId`** (`ObjectId`) - Идентификатор устройства.

## Возвращаемые значения

Возвращает `boolean` - `true`, если устройство выключено, и `false` в противном случае.

## Примеры

### Проверка состояния устройства

```lua
local deviceCanNotWork = device.IsOffline( deviceId )
```
