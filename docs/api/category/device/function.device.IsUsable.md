# device.IsUsable

Возвращает возможность принципиального использования устройства и признак его фактического применения.

## Описание

```lua
device.IsUsable( deviceId: ObjectId ): boolean, boolean
```

Функция анализирует состояние устройства и возвращает возможность его принципиального использования, а также признак фактического применения. Результаты вычисления формируют две независимые переменные.

::: warning Замечание
Возвращаемые значения являются двумя независимыми переменными.
:::

## Список параметров

- **`deviceId`** (`ObjectId`) - Идентификатор устройства.

## Возвращаемые значения

Возвращает два значения типа `boolean`.

- **`isUsable`** (`boolean`) - Возможность принципиального использования данного устройства.

- **`isUsableDevice`** (`boolean`) - Признак того, является ли устройство фактически применяемым (UsableDevice).

## Примеры

### Получение статуса использования устройства

```lua
local isUsable, isUsableDevice = device.IsUsable( deviceId )
```
