# device.NavigatorGetZoomedObject

Возвращает объект, на который на зумлена камера устройства-навигатора корабля главного игрока.

## Описание

```lua
device.NavigatorGetZoomedObject(): ObjectId | nil
```

Функция возвращает идентификатор транспорта или астрального юнита, на который на зумлена камера устройства-навигатора корабля главного игрока, или `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `ObjectId` или `nil`.

- **`objectId`** (`ObjectId` | `nil`) - Идентификатор транспорта или астрального юнита, или `nil`.

## Примеры

### Получение назумленного объекта

```lua
local objectId = device.NavigatorGetZoomedObject()
```
