# device.NavigatorSetTarget

Выбирает или сбрасывает цель, отслеживаемую навигационным устройством.

## Описание

```lua
device.NavigatorSetTarget( objectId: ObjectId | nil )
```

Функция позволяет выбрать или сбросить цель, отслеживаемую навигатором, если главный игрок использует навигационное устройство (тип устройства - `USDEV_NAVIGATOR`). Информация об этой цели начинает передаваться для всех членов команды игрока.

::: warning Замечание
Для сторонних аддонов действует ограничение: метод работает только в обработчиках реакции на действия мыши (на нажатие кнопки), а не на перемещение.
:::

## Список параметров

- **`objectId`** (`ObjectId` | `nil`) - Идентификатор транспорта или астрального юнита. Передача значения `nil` приводит к сбросу цели.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Сброс цели навигатора

```lua
device.NavigatorSetTarget( nil )
```
