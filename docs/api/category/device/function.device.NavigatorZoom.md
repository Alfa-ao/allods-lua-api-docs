# device.NavigatorZoom

Наводит камеру на выбранный корабль или астральный юнит при использовании навигационного устройства.

## Описание

```lua
device.NavigatorZoom( objectId: ObjectId | nil )
```

Метод наводит камеру на выбранный корабль или астральный юнит и позволяет рассмотреть его ближе, если главный игрок использует навигационное устройство (тип устройства - `USDEV_NAVIGATOR`). Метод влияет только на камеру, не затрагивая цель навигатора. О настройке данных см. в DeviceViewSight.

::: warning Замечание
Для сторонних аддонов действует ограничение: метод работает только в обработчиках реакции на действия (не перемещение) мыши, т.е. на нажатие кнопки.
:::

## Список параметров

- **`objectId`** (`ObjectId` | `nil`) - Идентификатор корабля или астрального юнита, или `nil`, если необходимо сбросить зум.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Наведение камеры на юнит

```lua
device.NavigatorZoom( objectId )
```
