# dungeonChoice.ChooseDungeon
Выбирает личное приключение для атаки из списка и задает индекс награды.

## Описание

```lua
dungeonChoice.ChooseDungeon( dungeonIndex, rewardIndex )
```

Функция инициирует выбор личного приключения для атаки из списка доступных приключений по указанному индексу, а также задает индекс награды, которую выбирает игрок.

## Список параметров

- **`dungeonIndex`** (`number`) - Индекс личного приключения в списке.

- **`rewardIndex`** (`number`) - Индекс награды, которую выбирает игрок.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Выбор приключения и награды по индексам

```lua
local list = dungeonChoice.GetList()
if list[ 0 ] then
    dungeonChoice.ChooseDungeon( 0, 0 )
end
```

::: info Описание примера
В примере извлекается список доступных личных приключений. При наличии нулевого элемента в списке инициируется выбор первого приключения и первой награды путем передачи соответствующих индексов.
:::
