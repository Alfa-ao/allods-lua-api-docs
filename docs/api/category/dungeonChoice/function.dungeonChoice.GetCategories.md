# dungeonChoice.GetCategories

Возвращает список категорий (лиг) личных приключений на выбор для атаки.

## Описание

```lua
dungeonChoice.GetCategories(): table
```

Функция возвращает индексированный с `0` список названий категорий (лиг) личных приключений на выбор для атаки.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table`.

- **`[index]`** (`WString`) - Название категории (лиги). Индексация начинается с `0`.

## Примеры

### Получение и вывод первой категории

```lua
local categories = dungeonChoice.GetCategories()
if categories[ 0 ] then
    LogInfo( categories[ 0 ] )
end
```
