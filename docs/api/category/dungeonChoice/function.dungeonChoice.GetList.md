# dungeonChoice.GetList

Возвращает список личных приключений на выбор для атаки.

## Описание

```lua
dungeonChoice.GetList(): table
```

Функция возвращает индексированный с 0 список таблиц, содержащих информацию о личных приключениях на выбор для атаки.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - индексированный с 0 список таблиц со следующими полями:

- **`index`** (`number`) - Индекс этого приключения в списке.

- **`categoryName`** (`WString`) - Название категории личного приключения.

- **`hasRooms`** (`boolean`) - Флаг необходимости сражения. Значение `true` указывает на то, что потребуется сражение.

- **`level`** (`number` | `nil`) - Уровень сложности категории приключения. Значение `nil` передается, если информация отсутствует.

## Примеры

### Получение списка приключений

```lua
local list = dungeonChoice.GetList()
if list[ 0 ] then
    LogInfo( list[ 0 ].ownerName )
end
```
