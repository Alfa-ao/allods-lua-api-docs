# dungeonChoice.GetTimeout
Возвращает время, отведенное на выбор личного приключения для атаки.

## Описание
```lua
dungeonChoice.GetTimeout(): table
```
Функция возвращает таблицу с информацией о времени, отведенном на выбор личного приключения для атаки.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table`.

- **`durationMs`** (`number`) - количество миллисекунд, которое отводится на выбор личного приключения для атаки.

- **`remainingMs`** (`number`) - количество миллисекунд, которое осталось на выбор личного приключения для атаки.

## Примеры
### Получение времени на выбор
```lua
local durationMs = dungeonChoice.GetTimeout().durationMs
```
