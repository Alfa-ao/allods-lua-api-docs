# dungeonChoice.IsActive
Определяет, находится ли игрок в состоянии выбора личного приключения для атаки.

## Описание
```lua
dungeonChoice.IsActive(): boolean
```
Функция возвращает `true`, если в данный момент необходимо из списка выбрать личное приключение для атаки, и `false` в противном случае.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если в данный момент необходимо из списка выбрать личное приключение для атаки, и `false` в противном случае.

## Примеры
### Проверка состояния выбора приключения
```lua
local isActive = dungeonChoice.IsActive()
```
