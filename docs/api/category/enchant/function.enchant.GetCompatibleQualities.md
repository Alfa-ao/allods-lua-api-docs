# enchant.GetCompatibleQualities
Извлекает информацию о качествах предметов, в которые можно вставить указанный энчант.

## Описание
```lua
enchant.GetCompatibleQualities( itemId: ObjectId ): table|nil
```
Функция анализирует указанный энчант и возвращает массив допустимых для него качеств предметов. В случае отсутствия данных или невалидного идентификатора возвращается `nil`.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор энчанта.

## Возвращаемые значения
Возвращает `table` или `nil`.
- **`table`** - Массив значений перечисления `ITEM_QUALITY`, представляющий допустимые качества предметов для вставки энчанта. Индексация массива начинается с `1`.

## Примеры
### Получение совместимых качеств
```lua
local qualities = enchant.GetCompatibleQualities( itemId )
```
