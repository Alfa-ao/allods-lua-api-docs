# enchant.GetEnchants

Извлекает список идентификаторов энчантов (улучшителей), вставленных в предмет.

## Описание

```lua
enchant.GetEnchants( itemId: ObjectId ): table|nil
```

Функция извлекает и возвращает список идентификаторов энчантов (улучшителей), вставленных в предмет. Идентификатор энчанта представляет собой идентификатор предмета, информацию о котором можно получить посредством вызова `itemLib.GetItemInfo( enchantId )`. Если информация отсутствует, функция возвращает `nil`.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `table` или `nil`.
Возвращает `nil`, если информация отсутствует. В противном случае возвращает список (таблицу, индексированную целыми числами начиная с `0`) идентификаторов энчантов (`ObjectId`).

## Примеры

### Получение списка энчантов

```lua
local enchants = enchant.GetEnchants( itemId )
```
