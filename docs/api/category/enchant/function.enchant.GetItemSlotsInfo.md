# enchant.GetItemSlotsInfo

Извлекает информацию о слотах предмета, доступных для вставки энчантов и соответствующих этим слотам статах.

## Описание

```lua
enchant.GetItemSlotsInfo( itemId: ObjectId ): table|nil
```

Функция возвращает информацию о слотах предмета, доступных для вставки энчантов и соответствующих этим слотам статах. Если информация отсутствует, функция возвращает `nil`. В противном случае возвращается таблица, проиндексированная элементами числового множества `InnateStats`. Значениями являются таблицы или `nil`. Значение равно `nil`, если у предмета нет статов для данного слота. Иначе возвращается список, индексируемый от `0`, содержащий относящиеся к данному слоту статы из числового множества `InnateStats`.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `table` или `nil`.

Возвращает `nil`, если информация отсутствует. Иначе возвращается таблица, проиндексированная элементами числового множества `InnateStats`. Значениями являются таблицы или `nil`. Значение равно `nil`, если у предмета нет статов для данного слота. Иначе возвращается список, индексируемый от `0`, содержащий относящиеся к данному слоту статы из числового множества `InnateStats`.

## Примеры

### Получение информации о слотах

```lua
local slots = enchant.GetItemSlotsInfo( itemId )
if slots then
    local defenceSlotStats = slots[ENUM_EnchantSlot_Defence]
end
```

::: info Описание примера
В примере извлекается информация о слотах предмета. Если данные успешно получены, выполняется обращение к элементу таблицы по ключу `ENUM_EnchantSlot_Defence` для получения статов слота защиты.
:::
