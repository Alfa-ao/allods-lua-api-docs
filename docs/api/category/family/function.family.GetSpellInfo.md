# family.GetSpellInfo

Возвращает информацию о семейном заклинании.

## Описание

```lua
family.GetSpellInfo( spellId: SpellId ): table|nil
```

Функция извлекает и возвращает таблицу с информацией о семейном заклинании по его идентификатору. Если указанное заклинание не является семейным, функция возвращает `nil`.

## Список параметров

- **`spellId`** (`SpellId`) - Идентификатор ресурса заклинания.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если данное заклинание действительно является семейным, возвращается таблица со следующими полями:

- **`experience`** (`number`) - Накопленный опыт.

- **`level`** (`number`) - Уровень заклинания (зависит от накопленного опыта).

- **`levelExperience`** (`number`) - Опыт, необходимый для получения текущего уровня.

- **`nextLevelExperience`** (`number`) - Опыт, необходимый для получения следующего уровня.

## Примеры

### Получение опыта семейного заклинания

```lua
local familySpell = family.GetSpellInfo( spellId )
if familySpell then
    local exp = familySpell.experience
end
```
