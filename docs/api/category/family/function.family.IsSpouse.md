# family.IsSpouse

Проверяет, является ли данный игрок супругом аватара.

## Описание

```lua
family.IsSpouse( id: ObjectId | WString ): boolean
```

Функция определяет, находится ли игрок с указанным идентификатором или именем в браке с главным аватаром.

## Список параметров

- **`id`** (`ObjectId` | `WString`) - Идентификатор игрока или его имя.

## Возвращаемые значения

Возвращает `boolean` - `true`, если игрок является супругом аватара.

## Примеры

### Проверка родственных связей

```lua
if family.IsSpouse( unitId ) then
    local spouseName = object.GetName( unitId )
end
```

::: info Описание примера
В примере проверяется, является ли объект с заданным идентификатором супругом аватара. Если условие выполняется, извлекается имя игрока с помощью функции `object.GetName`.
:::
