# group.Accept
Соглашается на приглашение о вступлении в группу.

## Описание
```lua
group.Accept()
```

Функция принимает активное приглашение на вступление в группу. Вызов функции при отсутствии активного приглашения приводит к ошибке.

::: warning Замечание
Перед вызовом необходимо убедиться в наличии активного приглашения, иначе выполнение функции считается ошибкой.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Принятие приглашения в группу
```lua
group.Accept()
```
