# group.CanInvite
Проверяет возможность приглашения других игроков в группу.

## Описание
```lua
group.CanInvite(): boolean
```
Функция возвращает флаг, указывающий на возможность приглашения других игроков в группу.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если игрок может приглашать в группу других игроков.

## Примеры
### Проверка возможности приглашения
```lua
local canInvite = group.CanInvite()
```
