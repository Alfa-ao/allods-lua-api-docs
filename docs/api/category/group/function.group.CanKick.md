# group.CanKick
Проверяет возможность выгона других игроков из группы.

## Описание
```lua
group.CanKick(): boolean
```
Функция возвращает `boolean` - `true`, если игрок может выгонять из группы других игроков, и `false` в противном случае.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если игрок может выгонять из группы других игроков, и `false` в противном случае.

## Примеры
### Проверка возможности выгона
```lua
local canKick = group.CanKick()
```
