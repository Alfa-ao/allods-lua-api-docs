# group.CanKickMember

Проверяет возможность исключения игрока из группы главным игроком.

## Описание

```lua
group.CanKickMember(): boolean
```

Функция возвращает `boolean` - `true`, если главный игрок может исключать игроков из группы (является лидером, доступно голосование и т.п.).

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если главный игрок может исключать игроков из группы.

## Примеры

### Проверка возможности и исключение участника

```lua
if group.CanKickMember() then
    group.KickMember( memberName )
end
```

::: info Описание примера
В примере проверяется возможность исключения игрока из группы. Если функция возвращает `true`, инициируется исключение участника с указанным именем посредством вызова `group.KickMember`.
:::
