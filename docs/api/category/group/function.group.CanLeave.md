# group.CanLeave

Определяет, может ли игрок покинуть группу.

## Описание

```lua
group.CanLeave(): boolean
```

Функция возвращает `boolean` - `true`, если игрок может покинуть группу других игроков, и `false` в противном случае.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если игрок может покинуть группу, и `false` в противном случае.

## Примеры

### Проверка возможности выхода из группы

```lua
local canLeave = group.CanLeave()
```
