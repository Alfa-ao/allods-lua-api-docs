# group.Decline
Отклоняет приглашение на вступление в группу.
## Описание
```lua
group.Decline()
```
Функция не возвращает значений. Отправляет запрос на отказ от активного приглашения на вступление в группу.
::: warning Замечание
Вызов функции при отсутствии активного приглашения считается ошибкой. Для проверки наличия приглашения используется функция `group.GetInviteInfo()`.
:::
## Список параметров
Параметры отсутствуют.
## Возвращаемые значения
Возвращаемое значение отсутствует.
## Примеры
### Отказ от приглашения в группу
```lua
group.Decline()
```
