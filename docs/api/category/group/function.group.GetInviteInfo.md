# group.GetInviteInfo

Извлекает информацию о текущем приглашении аватара в группу.

## Описание

```lua
group.GetInviteInfo(): table
```

Функция возвращает таблицу с информацией о приглашении в группу. Если активное приглашение отсутствует, поле `invited` принимает значение `false`, а поля с именами содержат пустые строки.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table`.

- **`invited`** (`boolean`) - Флаг наличия активного приглашения в группу.

- **`inviterName`** (`WString`) - Имя аватара, отправившего приглашение. Если приглашение отсутствует, содержит пустую строку.

- **`inviterShardName`** (`WString`) - Имя сервера приглашающего. Если приглашение отсутствует, содержит пустую строку.

## Примеры

### Автоматическое принятие приглашения

```lua
local inviteInfo = group.GetInviteInfo()
if inviteInfo.invited then
    group.Accept()
end
```

::: info Описание примера
В примере запрашивается информация о приглашении в группу. Если поле `invited` возвращаемой таблицы указывает на наличие активного приглашения, вызывается функция автоматического принятия приглашения.
:::
