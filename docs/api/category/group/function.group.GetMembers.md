# group.GetMembers
Возвращает упорядоченный список членов группы, включая главного игрока.

## Описание

```lua
group.GetMembers(): table|nil
```

Функция возвращает упорядоченный список членов группы, включая главного игрока. Главный игрок всегда первый в списке и имеет индекс 1. Если группы нет, функция возвращает `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

- **`key`** (`number`) - индекс персонажа в списке, персонаж главного игрока всегда имеет индекс 1.

- **`value`** (`table`) - таблица с информацией о персонаже из группы:

  - **`name`** (`WString`) - имя персонажа.

  - **`state`** (`number`) - состояние персонажа (см. `GROUP_MEMBER_STATE_...`).

  - **`isInCombat`** (`boolean`) - если `true`, то игрок в бою.

  - **`id`** (`ObjectId` | `nil`) - идентификатор персонажа или `nil`, если его нет поблизости (порвалась связь, в другой зоне и т.д.).

  - **`uniqueId`** (`UniqueId` | `nil`) - постоянный идентификатор персонажа (у аватара есть всегда и не изменяется при релогине, у моба-наёмника отсутствует).

  - **`level`** (`number` | `nil`) - уровень игрока или `nil`, если он не известен в данный момент.

  - **`className`** (`string` | `nil`) - системное название класса игрока (нелокализуемое).

  - **`classLocalName`** (`WString` | `nil`) - лорное название класса игрока (локализуемое).

## Примеры

### Получение списка членов группы

```lua
local members = group.GetMembers()
```
