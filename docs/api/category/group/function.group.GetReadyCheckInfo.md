# group.GetReadyCheckInfo

Возвращает информацию о текущем состоянии проверки готовности группы.

## Описание

```lua
group.GetReadyCheckInfo(): table | nil
```

Функция возвращает таблицу с информацией о текущем состоянии проверки готовности группы. Если активный или завершённый запрос о готовности отсутствует, функция возвращает `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

При наличии активного или завершённого запроса о готовности возвращается таблица со следующими полями:

- **`isInProgress`** (`boolean`) - Флаг активности запроса. Значение `true` указывает на то, что запрос о готовности находится в процессе доклада, `false` - запрос завершён.

- **`remainingTimeMs`** (`number`) - Оставшееся время до окончания активного запроса о готовности в миллисекундах.

- **`initiatorId`** (`UniqueId`) - Уникальный идентификатор инициатора запроса.

- **`members`** (`table`) - Массив, индексированный `[0..]`, содержащий информацию о готовности членов группы. Каждый элемент массива содержит следующие поля:
    - `uniqueId` (`UniqueId`) - Уникальный идентификатор члена группы.
    - `state` (`number` | `READY_CHECK_READY_STATE`) - Состояние готовности члена группы.

## Примеры

### Получение оставшегося времени проверки готовности

```lua
local info = group.GetReadyCheckInfo()
if info then
    if info.isInProgress then
        LogInfo( "Remaining time (msec): ", info.remainingTimeMs )
    end
end
```
