# group.Invite

Приглашает указанного персонажа из области видимости в свою группу.

## Описание

```lua
group.Invite( unitId: ObjectId )
```

Функция отправляет запрос на приглашение указанного персонажа, находящегося в области видимости, в группу игрока.

## Список параметров

- **`unitId`** (`ObjectId`) - Идентификатор персонажа, приглашаемого в группу.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Приглашение персонажа в группу

```lua
group.Invite( unitId )
```
