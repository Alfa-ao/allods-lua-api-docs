# group.InviteByName

Приглашает персонажа с указанным именем в группу.

## Описание

```lua
group.InviteByName( name: WString )
```

Функция отправляет запрос на приглашение персонажа с заданным именем в текущую группу.

## Список параметров

- **`name`** (`WString`) - Имя персонажа, которого необходимо пригласить в группу.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Приглашение персонажа в группу

```lua
group.InviteByName( name )
```
