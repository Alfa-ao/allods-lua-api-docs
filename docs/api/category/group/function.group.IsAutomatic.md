# group.IsAutomatic
Определяет, является ли группа автоматически собранной.

## Описание
```lua
group.IsAutomatic(): boolean
```
Функция возвращает `true`, если группа является автоматически собранной. Такую группу нельзя распускать и из неё нельзя исключать.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если группа собрана автоматически.

## Примеры
### Проверка статуса группы
```lua
local isAutomatic = group.IsAutomatic()
```
