# group.IsExist

Проверяет, находится ли аватар в какой-либо группе.

## Описание

```lua
group.IsExist(): boolean
```

Функция возвращает признак нахождения аватара в какой-либо группе.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - признак нахождения аватара в группе.

## Примеры

### Проверка наличия группы

```lua
local isExist = group.IsExist()
```
