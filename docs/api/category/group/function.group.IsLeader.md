# group.IsLeader

Проверяет, является ли объект лидером группы, в которой состоит аватар.

## Описание

```lua
group.IsLeader( id: ObjectId | UniqueId | WString | nil ): boolean
```

Функция определяет, является ли объект лидером группы. Если аватар не состоит в группе, функция возвращает `false`. Если аватар состоит в группе и аргумент не передан, функция проверяет лидерство аватара. Проверка по нику (`WString`) неустойчива к коллизии ников на межсерверных активностях.

## Список параметров

- **`id`** (`ObjectId` | `UniqueId` | `WString` | `nil`) - Идентификатор проверяемой сущности. Если передано значение `nil`, функция проверяет аватар.

## Возвращаемые значения

Возвращает `boolean` - является ли объект лидером группы.

## Примеры

### Проверка лидерства аватара

```lua
local isMainPlayerLeader = group.IsLeader()
```
