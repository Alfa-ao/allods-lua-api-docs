# group.KickMember

Исключает указанного игрока из группы.

## Описание

```lua
group.KickMember( name: WString )
```

Функция исключает указанного игрока из группы.

::: warning Замечание
Исключать игрока из группы может только её лидер.
:::

## Список параметров

- **`name`** (`WString`) - Имя игрока в группе.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Проверка возможности и исключение игрока

```lua
if group.CanKickMember() then
    group.KickMember( memberName )
end
```

::: info Описание примера
В примере проверяется возможность исключения игрока из группы с помощью функции `group.CanKickMember`. Если проверка успешна, выполняется исключение игрока по имени `memberName` посредством вызова `group.KickMember`.
:::
