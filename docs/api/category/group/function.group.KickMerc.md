# group.KickMerc

Исключает наемника из группы.

## Описание

```lua
group.KickMerc( objectId )
```

Функция инициирует исключение наемника из группы. Наемник представляет собой NPC, взятого лидером в группу. Произвести исключение может только лидер группы.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор наемника.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Исключение наемника из группы

```lua
group.KickMerc( objectId )
```
