# group.ReportReadyCheck

Уведомляет группу о готовности или неготовности к действиям для текущего активного запроса.

## Описание

```lua
group.ReportReadyCheck( ready: boolean )
```

Функция отправляет уведомление другим членам группы о статусе готовности текущего игрока к действиям в рамках активного запроса на проверку готовности.

## Список параметров

- **`ready`** (`boolean`) - Флаг готовности игрока к действиям.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Подтверждение готовности

```lua
group.ReportReadyCheck( true )
```
