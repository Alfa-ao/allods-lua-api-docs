# group.SetLeader

Назначает другого персонажа из группы на роль лидера.

## Описание

```lua
group.SetLeader( uniqueId: UniqueId )
```

Функция назначает другого персонажа из группы на роль лидера, при условии что вызывающий игрок является текущим лидером данной группы.

## Список параметров

- **`uniqueId`** (`UniqueId`) - Уникальный межсерверный идентификатор игрока из группы.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Назначение нового лидера группы

```lua
group.SetLeader( memberId )
```
