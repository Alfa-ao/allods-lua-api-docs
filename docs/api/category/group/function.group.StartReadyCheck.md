# group.StartReadyCheck

Отправляет серверу запрос на начало проверки готовности группы.

## Описание

```lua
group.StartReadyCheck()
```

Функция отправляет серверу запрос на инициацию проверки готовности участников группы.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Запуск проверки готовности

```lua
group.StartReadyCheck()
```
