# guild.Accept

Принимает приглашение в гильдию.

## Описание

```lua
guild.Accept()
```

Функция отправляет согласие на вступление в гильдию по ранее полученному приглашению.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Принятие приглашения в гильдию

```lua
guild.Accept()
```
