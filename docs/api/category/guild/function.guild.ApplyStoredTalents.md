# guild.ApplyStoredTalents
Выучивает помеченные таланты и вехи.

## Описание
```lua
guild.ApplyStoredTalents()
```
Функция применяет ранее помеченные таланты и вехи, инициируя их изучение. Предварительная пометка необходимых элементов осуществляется посредством вызова функции `guild.StoreTalent`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Пометка и применение талантов
```lua
guild.StoreTalent( field, row, column )
guild.ApplyStoredTalents()
```
::: info Описание примера
В примере сначала выполняется пометка конкретного таланта или вехи с помощью функции `guild.StoreTalent` с указанием координат. Затем вызывается функция `guild.ApplyStoredTalents`, которая применяет все ранее помеченные элементы и инициирует их изучение.
:::
