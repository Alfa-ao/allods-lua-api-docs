# guild.CanDropTalent
Проверяет возможность сброса вехи из поля талантов гильдии за альтвалюту.

## Описание
```lua
guild.CanDropTalent( field: number, row: number, column: number ): boolean, table | nil
```
Функция анализирует состояние дерева талантов гильдии и определяет, разрешен ли сброс конкретной вехи по указанным координатам с использованием альтвалюты. В случае положительного решения возвращает признак возможности и перечень талантов, которые подлежат удалению.

## Список параметров
- **`field`** (`number`)
Номер поля талантов.
- **`row`** (`number`)
Номер строки.
- **`column`** (`number`)
Номер столбца.

## Возвращаемые значения
Возвращает несколько значений:
- `boolean` — `true`, если сброс таланта гильдии возможен, и `false` в противном случае.
- `table` | `nil` — таблица, содержащая информацию о талантах, которые будут удалены. Если сброс невозможен, возвращается `nil`.
  - `field` (`number`) — номер поля удаляемого таланта.
  - `row` (`number`) — номер строки удаляемого таланта.
  - `column` (`number`) — номер столбца удаляемого таланта.

## Примеры
### Проверка возможности и сброс таланта
```lua
local field = 1
local row = 2
local column = 3

if guild.CanDropTalent( field, row, column ) then
    guild.DropTalent( field, row, column )
end
```
::: info Описание примера
В примере выполняется проверка возможности сброса таланта по заданным координатам. Если функция возвращает `true`, инициируется процесс сброса через вызов `guild.DropTalent`.
:::
