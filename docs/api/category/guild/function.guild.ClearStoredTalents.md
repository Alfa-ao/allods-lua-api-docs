# guild.ClearStoredTalents
Сбрасывает выбранные на изучение вехи талантов гильдии.

## Описание
```lua
guild.ClearStoredTalents( field: ?number, row: ?number, column: ?number )
```
Функция сбрасывает выбранные на изучение вехи талантов гильдии. При вызове без параметров выполняется полный сброс всех запланированных к изучению вех. При передаче конкретных координат сбрасывается указанная веха и все зависимые от нее вехи.

## Список параметров
- **`field`** (`?number`)
Номер поля талантов. Если параметр опущен, выполняется полный сброс всех вех.
- **`row`** (`?number`)
Номер строки целевой вехи.
- **`column`** (`?number`)
Номер столбца целевой вехи.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Полный сброс всех вех
```lua
guild.ClearStoredTalents()
```

### Сброс конкретной вехи и сопутствующих
```lua
guild.ClearStoredTalents( 0, 4, 5 )
```
