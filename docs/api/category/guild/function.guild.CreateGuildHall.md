# guild.CreateGuildHall

Создает холл гильдии.

## Описание

```lua
guild.CreateGuildHall( guildHallName: WString )
```

Функция инициирует создание холла гильдии с заданным именем.

## Список параметров

- **`guildHallName`** (`WString`) - желаемое имя холла гильдии.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Создание холла гильдии

```lua
guild.CreateGuildHall( guildHallName )
```
