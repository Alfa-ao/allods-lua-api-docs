# guild.Decline

Отклоняет приглашение в гильдию.

## Описание

```lua
guild.Decline()
```

Функция выполняет отклонение приглашения в гильдию.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Отклонение приглашения

```lua
guild.Decline()
```
