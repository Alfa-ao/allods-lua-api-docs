# guild.DropTalent
Сбрасывает веху из поля талантов гильдии за альтвалюту.

## Описание
```lua
guild.DropTalent( field: number, row: number, column: number )
```
Функция инициирует процесс сброса вехи из указанного поля талантов гильдии с использованием альтвалюты. Для успешного выполнения операции необходимо, чтобы у игрока было достаточно альтвалюты, а сама веха была доступна для сброса.

## Список параметров
- **`field`** (`number`)
Номер поля талантов гильдии.

- **`row`** (`number`)
Номер строки, в которой расположена сбрасываемая веха.

- **`column`** (`number`)
Номер столбца, в котором расположена сбрасываемая веха.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Проверка возможности и сброс вехи талантов
```lua
if guild.CanDropTalent( field, row, column ) then
    guild.DropTalent( field, row, column )
end
```
::: info Описание примера
В примере сначала выполняется проверка возможности сброса вехи с помощью функции `guild.CanDropTalent`. Если проверка возвращает положительный результат, инициируется процесс сброса через вызов `guild.DropTalent` с теми же координатами поля, строки и столбца.
:::
