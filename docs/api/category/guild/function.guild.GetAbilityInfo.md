# guild.GetAbilityInfo

Возвращает информацию о гильдейской абилке по идентификатору её ресурса.

## Описание

```lua
guild.GetAbilityInfo( id: GuildAbilityId ): table|nil
```

Функция возвращает информацию о гильдейской абилке по идентификатору её ресурса. Функция имеет основную реализацию в логике UIState/GameState и расширенную реализацию в пользовательском интерфейсе.

## Список параметров

- **`id`** (`GuildAbilityId`) - Идентификатор ресурса умения.

## Возвращаемые значения

Возвращает `table` с информацией об умении или `nil`, если такое умение отсутствует.

- **`id`** (`GuildAbilityId`) - Идентификатор умения.

- **`spells`** (`table` | `nil`) - Список идентификаторов заклинаний, которые выдает абилка, `nil` если не задано ни одного заклинания.
- **`name`** (`WString`) - Название.
- **`description`** (`ValuedText` | `nil`) - Описание с подставленными текущими значениями параметров.
- **`shortDescription`** (`ValuedText` | `nil`) - Краткое описание с подставленными текущими значениями параметров.
- **`rank`** (`number`) - Ранг умения (начиная с 1).
- **`sysInfo`** (`string`) - Системная информация для отладки (путь к файлу).
- **`texture`** (`TextureId`) - Идентификатор текстуры для иконки абилки (реализовано только для UI).

## Примеры

### Получение информации об умении

```lua
local abilityInfo = guild.GetAbilityInfo( guildAbilityId )
if abilityInfo then
    local name = abilityInfo.name
end
```
