# guild.GetAvatarHistory

Инициирует запрос истории участников гильдии.

## Описание

```lua
guild.GetAvatarHistory()
```

Функция отправляет запрос на получение истории участников гильдии. Результат выполнения передается через событие `EVENT_GUILD_AVATAR_HISTORY`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Запрос истории гильдии

```lua
guild.GetAvatarHistory()
```
