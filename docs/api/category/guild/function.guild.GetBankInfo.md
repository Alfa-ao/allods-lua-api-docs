# guild.GetBankInfo

Возвращает описание свойств гильдейского банка.

## Описание

```lua
guild.GetBankInfo(): table|nil
```

Функция возвращает таблицу с описанием свойств гильдейского банка или `nil`, если информация недоступна.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает таблицу (`table`) или `nil`.

- **`realMinimumTransaction`** (`number`) - минимальное реальное ограничение в данный момент по величине транзакции в меди.

- **`minimumTransaction`** (`number`) - ограничение на минимальное реальное ограничение по величине транзакции в меди.

- **`maximumTransaction`** (`number`) - максимальное ограничение по величине транзакции в меди.

- **`maximumGuildMoney`** (`number`) - максимальное количество меди в банке.

- **`operationHistorySize`** (`number`) - глубина истории операций.

- **`moneyHistorySize`** (`number`) - глубина истории оплат.

- **`memberHistorySize`** (`number`) - глубина истории по отдельному члену.

- **`guildHistoryPageSize`** (`number`) - максимальное количество записей на одной странице истории операций в банке.

- **`guildHistorySize`** (`number`) - текущее общее количество записей истории операций в банке.

## Примеры

### Получение реального минимального ограничения транзакции

```lua
local realMinimumTransaction = guild.GetBankInfo().realMinimumTransaction
```
