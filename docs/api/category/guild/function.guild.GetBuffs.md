# guild.GetBuffs
Возвращает информацию о бафах гилдии игрока.

## Описание
```lua
guild.GetBuffs(): table
```
Функция возвращает информацию о бафах гилдии игрока.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` - индексированный с 0 список идентификаторов бафов.

## Примеры
### Получение списка бафов гилдии
```lua
local guildBuffs = guild.GetBuffs()
```
