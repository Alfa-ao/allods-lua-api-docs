# guild.GetCurrencies

Возвращает таблицу идентификаторов валют, имеющихся у гильдии главного игрока.

## Описание

```lua
guild.GetCurrencies(): table
```

Функция возвращает индексированную с 0 таблицу идентификаторов валют, имеющихся у гильдии главного игрока.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - индексированная с 0 таблица идентификаторов валют.

## Примеры

### Получение информации о первой валюте гильдии

```lua
local currencies = guild.GetCurrencies()
if currencies[ 0 ] then
    local currencyInfo = avatar.GetCurrencyInfo( currencies[ 0 ] )
end
```

::: info Описание примера
В примере получается таблица идентификаторов валют гильдии. Если таблица содержит хотя бы один элемент под индексом 0, извлекается подробная информация о первой валюте с помощью функции `avatar.GetCurrencyInfo`.
:::
