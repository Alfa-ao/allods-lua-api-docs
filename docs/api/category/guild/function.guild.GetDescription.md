# guild.GetDescription

Возвращает описание гильдии.

## Описание

```lua
guild.GetDescription(): WString
```

Функция возвращает описание гильдии.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `WString` - описание гильдии.

## Примеры

### Получение описания гильдии

```lua
local description = guild.GetDescription()
```
