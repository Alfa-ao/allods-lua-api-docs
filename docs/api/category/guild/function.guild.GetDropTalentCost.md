# guild.GetDropTalentCost
Возвращает стоимость сброса (забывания) таланта в гильдии.

## Описание
```lua
guild.GetDropTalentCost()
```
Функция возвращает текущую стоимость сброса (забывания) таланта гильдии в единицах внутриигровой валюты. Полученное значение применяется для предварительной проверки наличия достаточного количества ресурсов у игрока перед инициированием процедуры сброса.

::: warning Замечание
Перед выполнением сброса таланта необходимо убедиться, что у игрока имеется достаточное количество валюты гильдии, сравнив её текущий объем с возвращаемым значением.
:::

## Список параметров
Функция не принимает параметров.

## Возвращаемые значения
Возвращает `number` — текущую стоимость сброса таланта в единицах валюты гильдии.

## Примеры
### Проверка наличия средств для сброса таланта
```lua
local dropTalentCurrencyId = 123 -- Идентификатор валюты гильдии
local acInfo = avatar.GetCurrencyInfo( dropTalentCurrencyId )
if acInfo and acInfo.value >= guild.GetDropTalentCost() then
    LogInfo( "Достаточно средств для сброса таланта." )
end
```
::: info Описание примера
В примере запрашивается информация о текущем количестве валюты гильдии с помощью функции `avatar.GetCurrencyInfo`. Затем полученное значение сравнивается со стоимостью сброса таланта, возвращаемой функцией `guild.GetDropTalentCost`. Если средств достаточно, выводится информационное сообщение.
:::
