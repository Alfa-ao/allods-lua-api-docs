# guild.GetItemsInfo

Возвращает общую информацию об истории операций с предметами гильдии.

## Описание

```lua
guild.GetItemsInfo(): table|nil
```

Функция возвращает общую информацию об истории операций с предметами гильдии.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - таблицу с данными или `nil`.

- **`operationHistorySize`** (`number`) - глубина истории операций.

- **`guildHistoryPageSize`** (`number`) - максимальное количество записей на одной странице истории операций.

- **`guildHistorySize`** (`number`) - текущее общее количество записей истории операций.

## Примеры

### Получение глубины истории операций

```lua
local operationHistorySize = guild.GetItemsInfo().operationHistorySize
```
