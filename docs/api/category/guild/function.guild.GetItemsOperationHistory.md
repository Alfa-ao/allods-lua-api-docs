# guild.GetItemsOperationHistory

Возвращает описание истории операций с предметами гильдии.

## Описание

```lua
guild.GetItemsOperationHistory(): table
```

Функция возвращает индексированную с нуля таблицу с описаниями конкретных операций с предметами гильдии.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - индексированную с нуля таблицу с описаниями конкретных операций. Каждая операция представлена таблицей со следующими полями:

- **`time`** (`LuaFullDateTime`) - время транзакции.

- **`actorMemberId`** (`ObjectId` | `nil`) - идентификатор члена гильдии, производившего операцию, если информация доступна.

- **`actorMemberName`** (`WString`) - имя покупателя.

- **`itemId`** (`ObjectId` | `nil`) - идентификатор предмета.

- **`vendorName`** (`WString` | `nil`) - имя продавца.

## Примеры

### Получение идентификатора предмета из истории

```lua
local history = guild.GetItemsOperationHistory()
local itemId = history[ 0 ] and history[ 0 ].itemId
```
