# guild.GetLeader

Возвращает идентификатор лидера гильдии главного игрока.

## Описание

```lua
guild.GetLeader(): ObjectId | nil
```

Функция возвращает идентификатор лидера гильдии главного игрока.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `ObjectId` | `nil` - идентификатор лидера гильдии, если главный игрок находится в гильдии.

## Примеры

### Получение информации о лидере гильдии

```lua
local memberId = guild.GetLeader()
if memberId then
    local memberInfo = guild.GetMemberInfo( memberId )
    if memberInfo then
        local name = memberInfo.name
    end
end
```

::: info Описание примера
В примере сначала запрашивается идентификатор лидера гильдии. Если главный игрок состоит в гильдии и идентификатор успешно получен, извлекается подробная информация о данном члене гильдии с помощью функции `guild.GetMemberInfo`, после чего считывается его имя.
:::
