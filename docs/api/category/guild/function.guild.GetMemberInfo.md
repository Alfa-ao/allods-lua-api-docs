# guild.GetMemberInfo

Возвращает информацию об одном члене гильдии.

## Описание

```lua
guild.GetMemberInfo( memberId: ObjectId ): table|nil
```

Каждый член гильдии обладает уникальным идентификатором, который не совпадает с идентификатором игрока в игре. Типы отличия описаны в перечислении `ENUM_TabardType_...`.

## Список параметров

- **`memberId`** (`ObjectId`) - идентификатор члена гильдии.

## Возвращаемые значения

Возвращает таблицу (`table`) с информацией о персонаже из гильдии или `nil`, если персонаж не найден.

- **`id`** (`ObjectId`) - идентификатор этого члена гильдии.

- **`playerId`** (`ObjectId` | `nil`) - идентификатор персонажа или `nil`, если его нет в игре.

- **`name`** (`WString`) - имя персонажа.

- **`level`** (`number`) - уровень игрока.

- **`onlineStatus`** (`string`) - онлайн-статус аватара. Значения соответствуют перечислению `ENUM_AvatarOnlineStatus_...`. При изменении статуса передается событие `EVENT_GUILD_MEMBER_ONLINE_STATUS_CHANGED`.

- **`authority`** (`number`) - влияние этого члена гильдии.

- **`monthAuthority`** (`number`) - влияние этого члена гильдии, полученное за месяц.

- **`weekAuthority`** (`number`) - влияние этого члена гильдии, полученное за неделю.

- **`dayAuthority`** (`number`) - влияние этого члена гильдии, полученное за день.

- **`fame`** (`number`) - известность этого члена гильдии.

- **`monthFame`** (`number`) - известность этого члена гильдии, полученная за месяц.

- **`weekFame`** (`number`) - известность этого члена гильдии, полученная за неделю.

- **`dayFame`** (`number`) - известность этого члена гильдии, полученная за день.

- **`loyalty`** (`number`) - верность этого члена гильдии (от 1 до 100).

- **`description`** (`WString`) - описание игрока.

- **`zoneName`** (`WString`) - локализованное название зоны, в которой находится игрок.

- **`subZoneName`** (`WString`) - локализованное название подзоны (пустая строка, если аватар не в подзоне).

- **`class`** (`WString`) - локализованное название класса игрока.

- **`sysClassName`** (`string`) - внутриигровое название класса игрока.

- **`tabardType`** (`number`) - тип отличия. Значения соответствуют перечислению `ENUM_TabardType_...`.

- **`sysTabardType`** (`string`) - строковое имя типа отличия. Значения соответствуют перечислению `ENUM_TabardType_...`.

- **`joinTime`** (`table`) - дата вступления в гильдию. Содержит поля: `y` (год), `m` (месяц), `d` (день).

- **`lastOnlineTime`** (`table`) - время последнего входа в игру в формате `LuaFullDateTime`. Содержит поля: `y` (год), `m` (месяц), `d` (день), `h` (час), `min` (минута), `s` (секунда).

- **`profit`** (`table` | `nil`) - данные о заработанных для гильдии ресурсах. Возвращает `nil`, если информация отсутствует. Иначе возвращает список (индексированный начиная с 0) таблиц со следующими полями:
    - **`key`** (`CurrencyId`) - идентификатор валюты.
    - **`value`** (`table`) - таблица с полями:
        - **`profitLongPeriod`** (`number`) - количество валюты за текущий месяц.
        - **`profitMediumPeriod`** (`number`) - количество валюты за текущую неделю (начиная с четверга).
        - **`profitShortPeriod`** (`number`) - количество валюты за текущий день.
        - **`totalProfit`** (`number`) - количество валюты за всё время.

## Примеры

### Получение информации о первом члене гильдии

```lua
local members = guild.GetMembers()
if members[ 0 ] then
    local memberInfo = guild.GetMemberInfo( members[ 0 ] )
    if memberInfo then
        local name = memberInfo.name
    end
end
```

::: info Описание примера
В примере сначала извлекается список всех членов гильдии с помощью функции `guild.GetMembers()`. Затем проверяется наличие первого элемента в списке. Если элемент существует, вызывается функция `guild.GetMemberInfo()` для получения подробной информации о данном члене гильдии. При успешном получении информации извлекается имя персонажа.
:::
