# guild.GetMessage

Возвращает гильдейское сообщение.

## Описание

```lua
guild.GetMessage(): WString
```

Функция возвращает текст гильдейского сообщения.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `WString` - гильдейское сообщение.

## Примеры

### Получение гильдейского сообщения

```lua
local message = guild.GetMessage()
```
