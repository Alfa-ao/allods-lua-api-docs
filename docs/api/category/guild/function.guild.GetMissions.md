# guild.GetMissions
Возвращает список миссий гильдии.

## Описание
```lua
guild.GetMissions(): table|nil
```
Функция возвращает список миссий гильдии. Возвращаемая таблица индексирована с единицы, каждый элемент которой представляет собой таблицу с полями, описывающими миссию.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.
Возвращаемая таблица индексирована с единицы. Каждый элемент представляет собой таблицу со следующими полями:

- **`currency`** (`CurrencyId`)
Альтвалюта, от которой зависит прогресс миссии.

- **`softCap`** (`number`)
Предельное количество альтвалюты, необходимое для завершения миссии.

- **`name`** (`WString`)
Название миссии.

- **`description`** (`WString`)
Описание миссии.

- **`image`** (`TextureId`)
Иллюстрация миссии.

- **`reward`** (`table`)
Награда за завершение миссии. Структура аналогична возвращаемому значению функции `avatar.GetQuestReward`.

## Примеры
### Получение списка миссий гильдии
```lua
local guildMissions = guild.GetMissions()
```
