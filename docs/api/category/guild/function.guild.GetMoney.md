# guild.GetMoney

Возвращает количество имеющихся у гильдии денег.

## Описание

```lua
guild.GetMoney(): number
```

Функция возвращает количество имеющихся у гильдии денег.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - количество денег.

## Примеры

### Получение количества денег гильдии

```lua
local money = guild.GetMoney()
```
