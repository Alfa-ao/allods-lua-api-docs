# guild.GetMounts

Возвращает список маунтов гильдии.

## Описание

```lua
guild.GetMounts(): table | nil
```

Функция возвращает таблицу маунтов, доступных гильдии. Каждая ячейка таблицы содержит информацию об идентификаторе предмета, выдающего маунта, и списке рангов гильдии, которым данный маунт доступен.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`. Таблица индексирована с 1, каждая ячейка содержит таблицу со следующими полями:

- **`id`** (`ObjectId`) - идентификатор предмета, выдающего маунта.

- **`ranks`** (`table`) - список индексов рангов гильдии, которым доступен данный маунт. Таблица индексирована с 1, содержит значения типа `number`.

## Примеры

### Получение списка маунтов гильдии

```lua
local mounts = guild.GetMounts()
if mounts then
    local mountId = item.GetIncludedMountSkin( mounts[ 1 ].id )
end
```

::: info Описание примера

Вызывается функция `guild.GetMounts()` для получения таблицы маунтов гильдии. Если таблица не равна `nil`, из первого элемента извлекается идентификатор предмета (`mounts[ 1 ].id`), который передается в функцию `item.GetIncludedMountSkin()` для получения идентификатора скина маунта.

:::
