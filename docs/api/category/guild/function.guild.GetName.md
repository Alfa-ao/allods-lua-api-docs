# guild.GetName

Возвращает имя гильдии.

## Описание

```lua
guild.GetName(): WString
```

Функция возвращает имя гильдии.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `WString` - имя гильдии.

## Примеры

### Получение имени гильдии

```lua
local name = guild.GetName()
```
