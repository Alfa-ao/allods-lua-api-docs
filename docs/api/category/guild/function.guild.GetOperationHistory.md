# guild.GetOperationHistory

Возвращает описание истории операций в гильдейском банке.

## Описание

```lua
guild.GetOperationHistory(): table
```

Функция возвращает описание истории операций в гильдейском банке.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - индексированная с нуля таблица с описаниями конкретных операций, каждая из которых представлена таблицей со следующими полями:

- **`balanceChange`** (`number`) - величина транзакции в меди.

- **`comment`** (`WString`) - комментарий при проведении операции.

- **`operationType`** (`number`) - тип изменения баланса. Соответствует перечислению `ENUM_MoneyChangeReason`.

- **`sysOperationType`** (`string`) - тип изменения баланса. Соответствует перечислению `ENUM_MoneyChangeReason`.

- **`time`** (`table`) - время транзакции. Структура соответствует `LuaFullDateTime`.

- **`actorMemberId`** (`ObjectId` | `nil`) - идентификатор члена гильдии, производившего операцию, если информация доступна.

- **`actorMemberName`** (`WString` | `nil`) - имя члена гильдии, производившего операцию, если информация доступна.

- **`members`** (`table`) - индексированная с нуля таблица с описаниями членов гильдии, попавших в операцию. Содержит таблицы со следующими полями:

    - **`id`** (`ObjectId` | `nil`) - идентификатор члена гильдии, если информация доступна.

    - **`name`** (`WString` | `nil`) - имя члена гильдии, если информация доступна.

- **`ranks`** (`table`) - индексы рангов гильдии, попавших в транзакцию. Может быть пустой.

- **`tabards`** (`table`) - накидки гильдии, носители которых попали в транзакцию. Значения соответствуют перечислению `ENUM_TabardType_...`. Может быть пустой.

## Примеры

### Получение истории операций

```lua
local history = guild.GetOperationHistory()
local firstTransactionValue = history[ 0 ] and history[ 0 ].balanceChange
```
