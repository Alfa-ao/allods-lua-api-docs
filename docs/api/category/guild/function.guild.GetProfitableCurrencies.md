# guild.GetProfitableCurrencies

Возвращает список альтвалют гильдии, которые могут добывать игроки.

## Описание

```lua
guild.GetProfitableCurrencies(): table|nil
```

Функция возвращает список альтвалют гильдии, которые могут добывать игроки. Возвращаемая таблица индексирована с единицы и содержит идентификаторы валют (`CurrencyId`). В случае отсутствия данных возвращается `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - список альтвалют, таблица индексированная с 1, или `nil`.

## Примеры

### Получение списка альтвалют гильдии

```lua
local currencies = guild.GetProfitableCurrencies()
```
