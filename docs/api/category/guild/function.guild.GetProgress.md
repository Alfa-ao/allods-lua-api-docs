# guild.GetProgress

Возвращает описание развития гильдии главного игрока.

## Описание

```lua
guild.GetProgress(): table|nil
```

Функция возвращает таблицу с описанием развития гильдии главного игрока или `nil`, если гильдия отсутствует.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если гильдия присутствует, возвращается таблица со следующими полями:

- **`authority`** (`number`) - влияние гильдии
- **`monthAuthority`** (`number`) - влияние гильдии, набранное за последний месяц
- **`maxAuthority`** (`number`) - максимально возможное влияние гильдии на текущем уровне
- **`minAuthority`** (`number`) - минимально возможное влияние гильдии на текущем уровне
- **`unlockedLevel`** (`number`) - доступный уровень для гильдии
- **`level`** (`number`) - уровень гильдии
- **`nextLevelPrice`** (`number`) - цена следующего уровня гильдии в деньгах

## Примеры

### Получение влияния гильдии

```lua
local guildAuthority = guild.GetProgress().authority
```
