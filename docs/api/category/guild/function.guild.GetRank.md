# guild.GetRank

Возвращает ранг члена гильдии.

## Описание

```lua
guild.GetRank( memberId: ObjectId ): number
```

Функция извлекает и возвращает ранг члена гильдии по его идентификатору.

## Список параметров

- **`memberId`** (`ObjectId`) - Идентификатор члена гильдии.

## Возвращаемые значения

Возвращает `number` - ранг игрока.

## Примеры

### Получение ранга

```lua
local rank = guild.GetRank( memberId )
```
