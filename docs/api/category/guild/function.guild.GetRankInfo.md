# guild.GetRankInfo
Возвращает информацию о ранге в гильдии.

## Описание
```lua
guild.GetRankInfo( rank: number ): table|nil
```
Функция возвращает таблицу с информацией о ранге в гильдии или `nil` в случае возникновения ошибки.

## Список параметров
- **`rank`** (`number`) - Числовое значение ранга игрока.

## Возвращаемые значения
Возвращает таблицу (`table`) или `nil` в случае ошибки.

- **`rank`** (`number`) - Числовое значение ранга.

- **`sysName`** (`string`) - Служебное название ранга.
- **`name`** (`WString`) - Локализованное название ранга.
- **`description`** (`WString`) - Локализованное описание ранга.

## Примеры
### Получение и вывод названия ранга
```lua
local rank = guild.GetRank( memberName )
local rankInfo = guild.GetRankInfo( rank )
if rankInfo then
    LogInfo( "name: ", rankInfo.name )
end
```

::: info Описание примера
В примере сначала получается числовой ранг игрока по его имени с помощью `guild.GetRank`. Затем запрашивается подробная информация о ранге через `guild.GetRankInfo`. Если информация успешно получена, в лог выводится локализованное название ранга.
:::
