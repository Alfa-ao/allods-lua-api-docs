# guild.GetRewardFactor

Возвращает значение дополнительного бонуса получения реальгара от изученных вех.

## Описание

```lua
guild.GetRewardFactor(): number | nil
```

Функция возвращает дополнительный множитель получения реальгара от изученных вех.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - дополнительный множитель получения реальгара, или `nil`.

## Примеры

### Получение множителя реварда

```lua
local rewardFactor = guild.GetRewardFactor()
```
