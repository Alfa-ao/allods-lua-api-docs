# guild.GetRewards

Возвращает описание гильдейских наград.

## Описание

```lua
guild.GetRewards(): table|nil
```

Функция возвращает индексированную с нуля таблицу, содержащую таблицы с описанием наград, или `nil`, если гильдия отсутствует.

::: warning Замечание
Функция имеет основную реализацию в логике и расширенную реализацию в интерфейсе.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если гильдия присутствует, возвращается таблица, индексированная с нуля, содержащая таблицы со следующими полями:

- **`level`** (`number`) - уровень гильдии, для которой доступна награда.

- **`amount`** (`number`) - количество наград.

- **`name`** (`WString`) - название награды.

- **`description`** (`WString`) - описание награды.

- **`image`** (`TextureId`) - идентификатор текстуры для иконки награды. Поле реализовано только для интерфейса.

## Примеры

### Получение и проверка доступности награды

```lua
local rewards = guild.GetRewards()
if rewards[0] then
    if guild.GetLevel() >= rewards[0].level then
        local count = rewards[0].amount
    end
end
```

::: info Описание примера
Запрашивается таблица гильдейских наград. Проверяется наличие первой награды и соответствие уровня гильдии требуемому уровню для её получения. При успешной проверке извлекается количество наград.
:::
