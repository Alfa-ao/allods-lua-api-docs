# guild.GetRootInfo

Возвращает общую базовую информацию о ресурсах гильдии.

## Описание

```lua
guild.GetRootInfo(): table|nil
```

Функция возвращает таблицу, содержащую идентификаторы чемпионской и обычной накидок, а также информацию о знамёнах гильдии.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

- **`championTabard`** (`ItemId`) - идентификатор чемпионской накидки.

- **`tabard`** (`ItemId`) - идентификатор обычной накидки.

- **`banners`** (`table` | `nil`) - знамёна, каждый элемент представляет собой таблицу со следующими полями:
    - **`previewItem`** (`ItemId` | `nil`) - идентификатор знамени.

## Примеры

### Получение информации о ресурсах гильдии

```lua
local rootInfo = guild.GetRootInfo()
```
