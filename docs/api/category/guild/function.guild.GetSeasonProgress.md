# guild.GetSeasonProgress
Возвращает описание сезонного развития гильдии главного игрока.

## Описание
```lua
guild.GetSeasonProgress(): table|nil
```
Функция возвращает таблицу с описанием сезонного развития гильдии главного игрока или `nil`, если гильдия отсутствует.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`. Если гильдия существует, возвращается таблица со следующими полями:
- **`bonusPool`** (`number`) - бонусный пул авторитета.

- **`authority`** (`number`) - влияние гильдии, накопленное в данном сезоне.

- **`secureAuthority`** (`number`) - несгораемое влияние.

- **`maxAuthority`** (`number`) - максимально возможное влияние гильдии на текущем уровне.

- **`minAuthority`** (`number`) - минимально возможное влияние гильдии на текущем уровне.

- **`unlockedLevel`** (`number`) - максимальный доступный сезонный уровень для гильдии.

- **`level`** (`number`) - сезонный уровень гильдии.

## Примеры
### Получение влияния гильдии
```lua
local guildSeasonAuthority = guild.GetSeasonProgress().authority
```
