# guild.GetStoredTalentsCount
Возвращает количество выбранных на изучение вех.

## Описание
```lua
guild.GetStoredTalentsCount(): number
```
Функция возвращает целое число, обозначающее количество вех, выбранных на изучение в дереве талантов гильдии.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — количество выбранных на изучение вех.

## Примеры
### Получение количества изучаемых вех
```lua
local count = guild.GetStoredTalentsCount()
```
