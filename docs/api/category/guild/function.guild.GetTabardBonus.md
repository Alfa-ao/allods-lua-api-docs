# guild.GetTabardBonus

Возвращает описание бонусов для носителей гильдейских накидок гильдии игрока.

## Описание

```lua
guild.GetTabardBonus( unitId: ObjectId | nil ): table
```

Функция извлекает и возвращает таблицу с описанием бонусов для носителей гильдейских накидок гильдии игрока.

## Список параметров

- **`unitId`** (`ObjectId` | `nil`) - идентификатор персонажа, либо `nil`, тогда по умолчанию вернет информацию для собственного аватара.

## Возвращаемые значения

Возвращает `table` - таблицу с полями:

- **`alchemyDurationPercent`** (`number (integer)`) - дополнительный процент увеличения длительности алхимических зелий.

- **`characteristicBonus`** (`number (float)`) - значение прибавки к Могуществу и Выносливости.

- **`characteristicPercent`** (`number (float)`) - процент увеличения Могущества и Выносливости (значение умноженное на 100).

- **`reputationBonus`** (`number (float)`) - значение модификатора репутации.

- **`reputationPercent`** (`number (float)`) - процент увеличения репутации (значение умноженное на 100).

## Примеры

### Получение процента увеличения характеристик

```lua
local characteristicPercent = guild.GetTabardBonus().characteristicPercent
```
