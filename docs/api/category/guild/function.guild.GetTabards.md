# guild.GetTabards
Возвращает описание имеющихся у гильдии главного игрока наград (знаков отличия).

## Описание
```lua
guild.GetTabards(): table|nil
```
Функция возвращает описание имеющихся у гильдии главного игрока наград (знаков отличия).

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если гильдия существует, возвращается таблица с полями, проиндексированными по типу награды:

- **`key`** (`number`) - тип награды. Значения соответствуют перечислению `ENUM_TabardType_...`.

- **`value`** (`number` | `nil`) - количество наград данного типа. Если наград нет, возвращается `nil`.

Возвращает `nil`, если гильдия отсутствует.

## Примеры
### Получение количества наград определенного типа
```lua
local rewards = guild.GetTabards()
if rewards then
    local championRewardCount = rewards[ ENUM_TabardType_Champion ] or 0
end
```
