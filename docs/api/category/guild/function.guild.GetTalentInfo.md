# guild.GetTalentInfo
Извлекает информацию о таланте гильдии из указанного поля талантов.

## Описание
```lua
guild.GetTalentInfo( field: number, row: number, column: number ): table | nil
```
Функция извлекает и возвращает детальную информацию о конкретном таланте гильдии, расположенном в заданном поле, строке и столбце. Возвращаемая таблица содержит различные флаги состояния таланта, а также идентификаторы и названия связанных сущностей. Если талант в указанной ячейке отсутствует, функция возвращает `nil`.

## Список параметров
- **`field`** (`number`)
Номер поля талантов, в котором осуществляется поиск.
- **`row`** (`number`)
Номер строки внутри указанного поля.
- **`column`** (`number`)
Номер столбца внутри указанной строки.

## Возвращаемые значения
Возвращает `table` или `nil`.

Таблица с информацией о таланте. Возвращает `nil` в случае, если таланта в данной ячейке не существует. Структура возвращаемой таблицы:

- **`field`** (`number`) — номер поля таланта.
- **`row`** (`number`) — номер строки таланта.
- **`column`** (`number`) — номер столбца таланта.
- **`canUpdate`** (`boolean`) — возможность изучить или обновить данный талант.
- **`isStart`** (`boolean`) — является ли талант стартовым.
- **`isStored`** (`boolean`) — поставлен ли талант на изучение.
- **`isLocked`** (`boolean`) — заблокирован ли талант для изучения.
- **`isEmpty`** (`boolean`) — является ли клетка пустой.
- **`isLearned`** (`boolean`) — выучен ли талант.
- **`groupName`** (`string` | `nil`) — название группы талантов.
- **`groupQuality`** (`number`) — качество группы талантов.
- **`abilityId`** (`GuildAbilityId` | `nil`) — идентификатор способности гильдии.

## Примеры
### Получение и проверка информации о таланте
```lua
local talentInfo = guild.GetTalentInfo( field, row, column )

if talentInfo then
    if talentInfo.isLearned then
        LogInfo( "Талант уже выучен" )
    elseif talentInfo.canUpdate then
        LogInfo( "Талант доступен для изучения" )
    end
end
```
