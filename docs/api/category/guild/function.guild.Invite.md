# guild.Invite

Приглашает указанного персонажа в гильдию.

## Описание

```lua
guild.Invite( unitId: ObjectId )
```

Функция отправляет запрос на приглашение указанного персонажа в гильдию.

::: warning Замечание
В пользовательских аддонах метод работает исключительно в обработчиках реакции на действия мыши, исключая перемещение, то есть на нажатие кнопки.
:::

## Список параметров

- **`unitId`** (`ObjectId`) - идентификатор персонажа, приглашаемого в гильдию.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Приглашение персонажа в гильдию

```lua
guild.Invite( unitId )
```
