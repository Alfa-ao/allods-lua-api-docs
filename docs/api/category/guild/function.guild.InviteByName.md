# guild.InviteByName

Инициирует приглашение указанного персонажа в гильдию.

## Описание

```lua
guild.InviteByName( playerName: WString )
```

Функция отправляет запрос на приглашение указанного персонажа в гильдию по его имени.

::: warning Замечание
В пользовательских аддонах присутствует ограничение: метод работает исключительно в обработчиках реакции на действия мыши, исключая перемещение, то есть на нажатие кнопки.
:::

## Список параметров

- **`playerName`** (`WString`) - Имя игрока.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Приглашение игрока по идентификатору

```lua
guild.InviteByName( object.GetName( unitId ) )
```
