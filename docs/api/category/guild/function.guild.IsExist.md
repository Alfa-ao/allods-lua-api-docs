# guild.IsExist

Определяет, находится ли аватар в какой-либо гильдии.

## Описание

```lua
guild.IsExist(): boolean
```

Функция возвращает логическое значение, указывающее на наличие гильдии у аватара.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если аватар состоит в гильдии, и `false` в противном случае.

## Примеры

### Проверка наличия гильдии

```lua
local isExist = guild.IsExist()
```
