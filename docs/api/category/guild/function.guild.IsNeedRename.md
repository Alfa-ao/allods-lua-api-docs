# guild.IsNeedRename
Определяет необходимость переименования гильдии, в которую входит аватар игрока.

## Описание
```lua
guild.IsNeedRename(): boolean
```
Функция возвращает признак необходимости переименования гильдии, в которую входит аватар игрока. Событие `EVENT_GUILD_NEED_RENAME_CHANGED` приходит при изменении данной информации. Необходимость переименования не связана с правом игрока на переименование гильдии. Для определения возможности переименования гильдии аватаром игрока используется функция `guild.CanRename()`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если гильдию требуется переименовать, иначе `false`.

## Примеры
### Проверка необходимости переименования
```lua
if guild.IsNeedRename() then
    -- ...
end
```
