# guild.IsStoredTalent
Проверяет, установлен ли талант на изучение.

## Описание
```lua
guild.IsStoredTalent( field: number, row: number, column: number ): boolean
```
Функция проверяет, выбран ли конкретный талант для изучения в указанной ячейке сетки талантов гильдии. Возвращает логическое значение, указывающее на наличие или отсутствие выбранного таланта для последующего изучения.

## Список параметров
- **`field`** (`number`)
Индекс поля (вкладки или раздела) в сетке талантов гильдии.
- **`row`** (`number`)
Индекс ряда (строки) в указанном поле сетки талантов.
- **`column`** (`number`)
Индекс колонки (столбца) в указанном ряду сетки талантов.

## Возвращаемые значения
`boolean` — `true`, если талант установлен на изучение, и `false` в противном случае.

## Примеры
### Проверка и очистка выбранного таланта
```lua
if guild.IsStoredTalent( field, row, column ) then
  guild.ClearStoredTalents( field, row, column )
end
```
::: info Описание примера
В примере выполняется проверка наличия выбранного таланта для изучения в заданных координатах сетки. Если функция возвращает `true`, вызывается функция `guild.ClearStoredTalents` для сброса (очистки) выбранного таланта в этой же ячейке.
:::
