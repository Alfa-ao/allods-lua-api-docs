# guild.RaiseFunds

Объявляет сбор в гильдейском банке.

## Описание

```lua
guild.RaiseFunds( sum: number, comment: WString, ranks: table, tabards: table )
```

Объявление сбора в гильдейском банке.

## Список параметров

- **`sum`** (`number`) - сумма сбора в меди

- **`comment`** (`WString`) - комментарий при объявлении сбора

- **`ranks`** (`table`) - индексы рангов гильдии, попавших в транзакцию. Может быть пустым

- **`tabards`** (`table`) - `EnumTabardType` накидки гильдии, носители которых попали в транзакцию. Может быть пустым

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Объявление сбора

```lua
guild.RaiseFunds( 10000, comment, {}, {} )
```
