# guild.Rename

Переименовывает гильдию.

## Описание

```lua
guild.Rename( name: WString )
```

Функция отправляет запрос на переименование гильдии. Возможность выполнения операции проверяется функцией `guild.CanRename()`. После успешного переименования приходит событие `EVENT_GUILD_NAME_CHANGED`.

## Список параметров

- **`name`** (`WString`) - Новое имя гильдии (локализованная строка). Должно соответствовать правилам именования гильдий.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Переименование гильдии

```lua
if guild.CanRename() then
    guild.Rename( newName )
end
```

::: info Описание примера
Проверяется возможность переименования гильдии с помощью функции `guild.CanRename()`. При положительном результате выполняется переименование с новым именем.
:::
