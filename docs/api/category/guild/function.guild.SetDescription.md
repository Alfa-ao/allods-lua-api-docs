# guild.SetDescription
Изменяет описание гильдии.

## Описание
```lua
guild.SetDescription( description: WString )
```
Установка нового текстового описания для гильдии.

::: warning Замечание
В пользовательских аддонах присутствует ограничение - метод может не работать на турнирах.
:::

## Список параметров
- **`description`** (`WString`) - Описание гильдии.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Установка описания
```lua
guild.SetDescription( description )
```
