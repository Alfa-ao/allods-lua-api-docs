# guild.SetMessage

Изменяет сообщение гильдии.

## Описание

```lua
guild.SetMessage( message: WString )
```

Функция изменяет сообщение гильдии.

::: warning Замечание
В пользовательских аддонах присутствует ограничение: метод может не работать на турнирах.
:::

## Список параметров

- **`message`** (`WString`) - Сообщение гильдии.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Изменение сообщения гильдии

```lua
local message = userMods.ToWString( "Текст сообщения гильдии" )
guild.SetMessage( message )
```
