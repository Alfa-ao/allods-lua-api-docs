# guild.SetMinimumTransaction

Устанавливает размер минимальной транзакции в гильдейском банке.

## Описание

```lua
guild.SetMinimumTransaction( minimumTransaction: number )
```

Установка размера минимальной транзакции в гильдейском банке.

## Список параметров

- **`minimumTransaction`** (`number`) - Размер минимальной транзакции в меди.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка минимальной транзакции

```lua
guild.SetMinimumTransaction( 1000000 )
```
