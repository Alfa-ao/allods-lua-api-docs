# guild.StoreTalent
Помечает веху для последующего изучения в поле талантов гильдии.

## Описание
```lua
guild.StoreTalent( field: number, row: number, column: number )
```
Функция помечает указанную веху в дереве талантов гильдии для последующего изучения. Координаты вехи задаются номерами поля, строки и столбца. Для фактического применения помеченных изменений и изучения вех требуется дополнительный вызов функции `guild.ApplyStoredTalents()`.

## Список параметров
- **`field`** (`number`)
Номер поля талантов гильдии.
- **`row`** (`number`)
Номер строки в указанном поле.
- **`column`** (`number`)
Номер столбца в указанном поле.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Пометка вехи и применение изменений
```lua
local field = 1
local row = 2
local column = 3

guild.StoreTalent( field, row, column )
guild.ApplyStoredTalents()
```
::: info Описание примера
В примере сначала помечается конкретная веха в поле талантов гильдии по заданным координатам, после чего инициируется применение всех помеченных вех с помощью функции `guild.ApplyStoredTalents()`.
:::
