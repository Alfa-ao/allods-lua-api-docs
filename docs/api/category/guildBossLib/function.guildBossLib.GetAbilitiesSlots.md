# guildBossLib.GetAbilitiesSlots

Возвращает спецификацию слотов способностей босса гильдии.

## Описание

```lua
guildBossLib.GetAbilitiesSlots(): table | nil
```

Функция возвращает таблицу, где ключом является индекс слота способностей босса гильдии (перечисление `ENUM_GuildBossAbilitySlot`). Каждый элемент таблицы содержит списки возможных заклинаний и способностей для данного слота. При возникновении ошибки или отсутствии данных возвращается `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.
Если данные успешно получены, возвращается таблица, где ключом является индекс слота способностей босса (значения перечисления `ENUM_GuildBossAbilitySlot`). Каждый элемент таблицы содержит следующие поля:

- **`spells`** (`table` | `nil`) - список возможных заклинаний босса гильдии для данного слота (содержит элементы типа `SpellId`).

- **`abilities`** (`table` | `nil`) - список возможных способностей босса гильдии для данного слота (содержит элементы типа `AbilityId`).

Если данные отсутствуют или произошла ошибка, возвращается `nil`.

## Примеры

### Получение информации о слотах способностей

```lua
local abilitiesSlotsInfo = guildBossLib.GetAbilitiesSlots()
```
