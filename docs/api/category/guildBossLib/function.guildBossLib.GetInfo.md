# guildBossLib.GetInfo
Возвращает сводную информацию о боссе гильдии.

## Описание
```lua
guildBossLib.GetInfo(): table | nil
```
Функция возвращает таблицу со сводной информацией о боссе гильдии или `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает таблицу (`table`) или `nil`.

- **`experience`** (`number`) - Накопленный опыт босса гильдии.

- **`gearScore`** (`number`) - Текущая сила босса гильдии.

- **`name`** (`WString` | `nil`) - Нарицательное имя босса гильдии.

- **`description`** (`WString`) - Художественное описание босса гильдии.

- **`currentState`** (`number` | `nil`) - Индекс текущей стадии прогресса босса гильдии.

- **`states`** (`table` | `nil`) - Список стадий прогресса босса, индексируется с 1. Каждый элемент представляет собой таблицу со следующими полями:
  - **`experienceLimit`** (`number`) - Предел накапливаемого опыта босса гильдии на данной стадии.
  - **`availableAbilitySlots`** (`table` | `nil`) - Список доступных слотов способностей босса гильдии (строки, соответствующие перечислению `ENUM_GuildBossAbilitySlot`).

## Примеры
### Получение информации о боссе гильдии
```lua
local guildBossInfo = guildBossLib.IsExist() and guildBossLib.GetInfo()
```
