# guildBossLib.GetVariations

Возвращает список возможных вариантов для выбора босса гильдии.

## Описание

```lua
guildBossLib.GetVariations(): table|nil
```

Функция возвращает список возможных вариантов для выбора босса гильдии. Каждый элемент списка представляет собой таблицу с полями, описывающими характеристики и внешние данные босса.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil` - список возможных вариантов для выбора босса гильдии. Индексация списка начинается с 1. Каждый элемент списка представляет собой таблицу со следующими полями:

- **`name`** (`WString` | `nil`) - нарицательное имя.

- **`description`** (`WString`) - художественное описание.

- **`preview`** (`TextureId`) - идентификатор текстуры для предпросмотра.

- **`background`** (`TextureId`) - идентификатор текстуры для подложки.

- **`spells`** (`table`) - список заклинаний (`SpellId`), которыми может владеть босс. Индексация с 1.

- **`abilities`** (`table`) - список способностей (`AbilityId`), которыми может владеть босс. Индексация с 1.

- **`previewStates`** (`table`) - индексы состояния (`number`), для которых следует позволять делать предпросмотр.

## Примеры

### Получение списка вариантов босса

```lua
local guildBossVariations = guildBossLib.GetVariations()
```
