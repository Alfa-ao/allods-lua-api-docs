# hangar.CanCustomize
Определяет возможность кастомизации внешнего вида корабля в ангаре.

## Описание
```lua
hangar.CanCustomize(): boolean
```
Функция возвращает логическое значение, указывающее на доступность кастомизации внешнего вида корабля в ангаре.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если можно кастомизировать внешний вид корабля.

## Примеры
### Проверка и запуск кастомизации
```lua
if hangar.CanCustomize() then
    hangar.ChangeSlotCustomization( slotId, deviceId )
end
```

::: info Описание примера
Проверяется возможность кастомизации внешнего вида корабля. Если кастомизация доступна, инициируется изменение кастомизации для указанного слота с использованием заданного устройства.
:::
