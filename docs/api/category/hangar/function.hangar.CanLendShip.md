# hangar.CanLendShip
Проверяет возможность передачи активного корабля лучшему другу.

## Описание
```lua
hangar.CanLendShip(): boolean
```
Функция возвращает логическое значение, определяющее наличие возможности передачи активного корабля лучшему другу.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если аватар имеет возможность передать активный корабль лучшему другу, и `false` в противном случае.

## Примеры
### Проверка возможности передачи корабля
```lua
local canLend = hangar.CanLendShip()
```
