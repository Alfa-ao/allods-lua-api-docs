# hangar.CanRemoveDevice
Проверяет возможность удаления текущего корабельного устройства в ангаре в режиме замены.

## Описание
```lua
hangar.CanRemoveDevice(): boolean
```
Функция возвращает признак возможности удаления текущего корабельного устройства в ангаре в режиме замены.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если удаление текущего устройства разрешено, и `false` в противном случае.

## Примеры
### Проверка возможности удаления устройства
```lua
if hangar.CanRemoveDevice() then
    hangar.RemoveDevice()
end
```
