# hangar.GetAllowedDeviceTypes

Возвращает список доступных для покупки устройств, имеющихся в ангаре.

## Описание

```lua
hangar.GetAllowedDeviceTypes(): table
```

Функция возвращает индексированный с 1 список устройств для продажи. Получение данных возможно только после разговора с НПС в ангаре.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - индексированный с 1 список устройств для продажи.

Таблица содержит следующие поля:

- **`deviceId`** (`ObjectId`) - Идентификатор устройства.

- **`currencyId`** (`CurrencyId` | `nil`) - Идентификатор альтернативной валюты, если стоимость выражена в ней, иначе стоимость выражена в деньгах.

- **`amount`** (`number`) - Стоимость устройства в валюте, если есть `currencyId`, или в меди, если `currencyId` равен `nil`.

## Примеры

### Перебор доступных устройств

```lua
for i, device in pairs( hangar.GetAllowedDeviceTypes() ) do
    local deviceInfo = hangar.GetDeviceInfo( device.deviceId )
    local deviceMoney = device.currencyId and 0 or device.amount
end
```

::: info Описание примера
В примере выполняется перебор списка доступных для покупки устройств. Для каждого устройства извлекается подробная информация посредством вызова `hangar.GetDeviceInfo`, а также рассчитывается стоимость в деньгах с учетом наличия альтернативной валюты.
:::
