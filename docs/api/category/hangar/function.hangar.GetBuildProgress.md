# hangar.GetBuildProgress

Возвращает информацию о прогрессе корабля, строящегося в ангаре.

## Описание

```lua
hangar.GetBuildProgress(): table|nil
```

Функция возвращает информацию о прогрессе корабля, строящегося в ангаре, доступную только после специального запроса у НПС.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если прогресс был запрошен, возвращается таблица со следующими полями:

- **`percent`** (`number`) - Процент готовности корабля.

- **`timeLeft`** (`table`) - Оставшееся время до готовности. Поля таблицы: 
    - `d` (`number`) - дни.
    - `h` (`number`) - часы.
    - `m` (`number`) - минуты.
    - `s` (`number`) - секунды.

## Примеры

### Получение прогресса строительства

```lua
local progress = hangar.GetBuildProgress()
if progress then
    local percent = progress.percent
end
```
