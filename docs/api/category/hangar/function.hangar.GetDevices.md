# hangar.GetDevices
Извлекает список идентификаторов устройств, находящихся в ангаре главного персонажа.

## Описание
```lua
hangar.GetDevices(): table
```
Функция возвращает список идентификаторов устройств, находящихся в ангаре главного персонажа.

::: warning Замечание
Получение данных возможно только после разговора с НПС в ангаре.
:::

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` - таблицу идентификаторов устройств.

- **`[1]`** (`ObjectId`) - Идентификатор устройства.

## Примеры
### Получение информации об устройствах ангара
```lua
local devices = hangar.GetDevices()
for _, deviceId in ipairs( devices ) do
    local deviceInfo = hangar.GetDeviceInfo( deviceId )
end
```
::: info Описание примера
В примере извлекается список идентификаторов устройств ангара. Далее в цикле перебираются полученные идентификаторы, и для каждого из них запрашивается подробная информация посредством вызова функции `hangar.GetDeviceInfo`.
:::
