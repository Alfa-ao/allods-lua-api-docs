# hangar.GetRandomName
Возвращает случайное имя для транспортного средства.

## Описание
```lua
hangar.GetRandomName(): WString
```
Функция генерирует имя по принципу «имя-число» (например, «Касатка-2351»), где текстовая часть зависит от фракции аватара.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `WString` - локализованное имя транспорта.

## Примеры
### Получение случайного имени
```lua
local randomName = hangar.GetRandomName()
```
