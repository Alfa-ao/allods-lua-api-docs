# hangar.GetRepairProgress
Возвращает информацию о прогрессе ремонта корабля в ангаре.

## Описание
```lua
hangar.GetRepairProgress(): table|nil
```
Функция возвращает информацию о прогрессе ремонта корабля в ангаре. Данные доступны только после выполнения специального запроса у НПС в ангаре.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если прогресс был запрошен, возвращается таблица со следующими полями:

- **`percent`** (`number`) - Процент ремонта. Значение находится в диапазоне от `0` до `100`.

- **`timeLeft`** (`table`) - Информация о времени, оставшемся до окончания ремонта. Содержит следующие поля:

    - **`d`** (`number`) - Количество дней.

    - **`h`** (`number`) - Количество часов.

    - **`m`** (`number`) - Количество минут.

    - **`s`** (`number`) - Количество секунд.

## Примеры
### Получение процента ремонта
```lua
local progress = hangar.GetRepairProgress()
if progress then
    local percent = progress.percent
end
```
