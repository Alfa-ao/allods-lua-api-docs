# hangar.GetShips
Возвращает список идентификаторов кораблей, находящихся в ангаре главного игрока.

## Описание

```lua
hangar.GetShips(): table
```

Функция возвращает индексированную числами от 1 таблицу идентификаторов кораблей, находящихся в ангаре главного игрока.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - индексированная числами от 1 таблица идентификаторов кораблей (`ObjectId`).

## Примеры

### Получение информации о кораблях

```lua
for i, shipId in ipairs( hangar.GetShips() ) do
    local shipInfo = hangar.GetShipInfo( shipId )
end
```

::: info Описание примера
В примере выполняется итерация по списку кораблей в ангаре с использованием цикла `ipairs`. Для каждого идентификатора корабля вызывается функция `hangar.GetShipInfo` для получения дополнительной информации.
:::
