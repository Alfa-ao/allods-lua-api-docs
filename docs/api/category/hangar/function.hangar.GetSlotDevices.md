# hangar.GetSlotDevices

Возвращает список идентификаторов устройств ангара, доступных для установки в указанный слот корабля.

## Описание

```lua
hangar.GetSlotDevices(): table
```

Функция возвращает таблицу идентификаторов устройств, находящихся в ангаре главного игрока, которые доступны для установки в определённый слот корабля. Получение данных возможно только после предварительного запроса у требуемого слота.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - Таблица (индексированная с 1) идентификаторов устройств.

- **`[index]`** (`ObjectId`) - Идентификатор устройства, доступного для установки в слот.

## Примеры

### Получение списка доступных устройств для слота

```lua
local devices = hangar.GetSlotDevices()
for i, deviceId in ipairs( devices ) do
    local deviceInfo = hangar.GetDeviceInfo( deviceId )
end
```
