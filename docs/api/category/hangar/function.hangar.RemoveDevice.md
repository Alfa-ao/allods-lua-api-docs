# hangar.RemoveDevice

Удаляет корабельное устройство, если персонаж находится в ангаре в режиме замены.

## Описание

```lua
hangar.RemoveDevice()
```

Функция удаляет корабельное устройство, если персонаж находится в ангаре в режиме замены.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Удаление устройства с предварительной проверкой

```lua
if hangar.CanRemoveDevice() then
    hangar.RemoveDevice()
end
```

::: info Описание примера
В примере выполняется проверка возможности удаления устройства с помощью функции `hangar.CanRemoveDevice`. Если проверка успешна, инициируется удаление устройства вызовом `hangar.RemoveDevice`.
:::
