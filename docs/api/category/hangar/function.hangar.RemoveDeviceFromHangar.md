# hangar.RemoveDeviceFromHangar
Удаляет указанное количество корабельных устройств из ангара.

## Описание

```lua
hangar.RemoveDeviceFromHangar( deviceId: ObjectId, count: number )
```

Функция инициирует удаление заданного количества корабельных устройств из хранилища ангара.

## Список параметров

- **`deviceId`** (`ObjectId`) - Идентификатор устройства, хранящегося в ангаре.

- **`count`** (`number`) - Количество удаляемых устройств.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Удаление устройств из ангара

```lua
hangar.RemoveDeviceFromHangar( deviceId, 3 )
```
