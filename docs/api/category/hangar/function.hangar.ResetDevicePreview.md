# hangar.ResetDevicePreview

Сбрасывает предпросмотр устройств в ангаре перед покупкой, находясь в ангаре в режиме замены.

## Описание

```lua
hangar.ResetDevicePreview()
```

Функция выполняет сброс предпросмотра устройств в ангаре перед покупкой, находясь в ангаре в режиме замены.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Сброс предпросмотра после установки устройства

```lua
hangar.PreviewPurchasedDevice( allowedDeviceId )
hangar.ResetDevicePreview()
```

::: info Описание примера
В примере вызывается функция предпросмотра купленного устройства, после чего выполняется сброс предпросмотра. Используется две функции для демонстрации последовательности действий.
:::
