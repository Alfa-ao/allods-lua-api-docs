# hangar.RevertCustomization
Откатывает кастомизацию всех слотов для визуальной кастомизации корабля в ангаре главного игрока.

## Описание
```lua
hangar.RevertCustomization()
```
Функция возвращает визуальную кастомизацию всех слотов корабля в ангаре главного игрока к исходному состоянию.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Откат кастомизации корабля
```lua
hangar.RevertCustomization()
```
