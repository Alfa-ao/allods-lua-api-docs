# interaction.CancelExchange
Прерывает активную торговлю с другим игроком.

## Описание

```lua
interaction.CancelExchange()
```

Функция инициирует принудительное прерывание текущего процесса торговли с другим игроком.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Прерывание торговли

```lua
interaction.CancelExchange()
```
