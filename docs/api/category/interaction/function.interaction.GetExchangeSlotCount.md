# interaction.GetExchangeSlotCount

Возвращает количество слотов обменного стола при торговле с игроком.

## Описание

```lua
interaction.GetExchangeSlotCount(): number
```

Функция возвращает количество слотов обменного стола при торговле с игроком.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - количество слотов обменного стола.

## Примеры

### Получение количества слотов обменного стола

```lua
local exchangeTableSize = interaction.GetExchangeSlotCount()
```
