# interaction.HasExchange
Проверяет, находится ли главный игрок в состоянии торговли с другим игроком.

## Описание
```lua
interaction.HasExchange(): boolean
```
Функция возвращает `true`, если главный игрок находится в состоянии обмена с другим игроком, и `false` в противном случае. В этом состоянии он находится сразу после приглашения в группу (сам пригласил или был приглашен) и до окончания или отмены торговли.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если главный игрок находится в состоянии обмена с другим игроком.

## Примеры
### Проверка состояния торговли
```lua
if interaction.HasExchange() then
    -- некий код
end
```
