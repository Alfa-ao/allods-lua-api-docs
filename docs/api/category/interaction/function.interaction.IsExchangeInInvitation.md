# interaction.IsExchangeInInvitation

Определяет, находится ли главный игрок в состоянии ожидания принятия приглашения на торговлю.

## Описание

```lua
interaction.IsExchangeInInvitation(): boolean
```

Функция возвращает `true`, если главный игрок находится в состоянии приглашения торговли с другим игроком (сам пригласил или был приглашен), и в данный момент приглашение на торговлю ещё не принято.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если главный игрок находится в состоянии приглашения для торговли с другим игроком.

## Примеры

### Проверка состояния приглашения

```lua
if interaction.IsExchangeInInvitation() then
    -- некий код
end
```
