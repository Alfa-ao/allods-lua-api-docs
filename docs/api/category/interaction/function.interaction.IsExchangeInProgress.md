# interaction.IsExchangeInProgress
Возвращает статус нахождения главного игрока в активном состоянии торговли с другим игроком.

## Описание
```lua
interaction.IsExchangeInProgress(): boolean
```
Функция возвращает `true`, если главный игрок находится в активном состоянии торговли с другим игроком, и `false` в противном случае.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если главный игрок уже обменивается товарами с другим игроком.

## Примеры
### Проверка состояния торговли
```lua
if interaction.IsExchangeInProgress() then
    -- некий код
end
```
