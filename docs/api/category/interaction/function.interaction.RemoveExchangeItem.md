# interaction.RemoveExchangeItem

Убирает предмет с обменного стола при торговле с другим игроком.

## Описание

```lua
interaction.RemoveExchangeItem( exchangeSlot: number )
```

Функция убирает предмет с указанного слота обменного стола в процессе торговли с другим игроком.

## Список параметров

- **`exchangeSlot`** (`number`) - Номер слота обменного стола.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Удаление предмета из нулевого слота

```lua
interaction.RemoveExchangeItem( 0 )
```
