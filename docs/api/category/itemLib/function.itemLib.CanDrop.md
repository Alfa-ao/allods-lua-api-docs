# itemLib.CanDrop

Определяет возможность выкинуть предмет из инвентаря.

## Описание

```lua
itemLib.CanDrop( itemId: ObjectId ): boolean
```

Функция возвращает статус возможности удаления предмета из инвентаря по его идентификатору.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета-образца.

## Возвращаемые значения

Возвращает `boolean` - `true`, если можно выкинуть предмет из инвентаря, и `false` в противном случае.

## Примеры

### Проверка возможности выкинуть предмет

```lua
local itemCanDrop = itemLib.CanDrop( itemId )
```
