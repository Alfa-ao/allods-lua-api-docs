# itemLib.GetActivators
Извлекает список описаний активаторов, которыми можно активировать указанный предмет.

## Описание
```lua
itemLib.GetActivators( itemId: ObjectId ): table|nil
```
Функция возвращает список описаний активаторов, применимых к указанному предмету. Если информация об активаторах отсутствует, возвращается `nil`. В противном случае возвращается таблица, индексируемая от `0`, содержащая таблицы-описания активаторов.

## Список параметров
- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения
Возвращает `table` | `nil` - `nil`, если информация об активаторах отсутствует; иначе таблица, индексируемая от `0`, содержащая таблицы-описания активаторов со следующими полями:

- **`name`** (`WString`) - Локализованное имя активатора.

- **`description`** (`WString`) - Локализованное описание активатора.

- **`image`** (`TextureId`) - Идентификатор иконки активатора.

## Примеры
### Получение списка активаторов
```lua
local activators = itemLib.GetActivators( itemId )
```
