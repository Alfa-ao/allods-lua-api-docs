# itemLib.GetBonus
Извлекает и возвращает таблицу с информацией о бонусах, предоставляемых предметом.

## Описание

```lua
itemLib.GetBonus( itemId: ObjectId ): table | nil
```

Функция формирует и возвращает структуру данных, содержащую детализированную информацию о бонусах к характеристикам, сопротивляемостям и специальным параметрам, которые дает указанный предмет. В случае возникновения ошибок или при передаче некорректного идентификатора возвращается `nil`. Значения эффективных характеристик в полях `innateStats` и `resistances` рассчитываются как сумма базовых значений и бонусов от чар (энчантов).

## Список параметров

- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета, информацию о бонусах которого необходимо получить.

## Возвращаемые значения
Возвращает таблицу (`table`) с информацией о бонусах предмета или `nil` в случае возникновения ошибок.

Структура возвращаемой таблицы:
- **`innateStats`** (`table`)
Бонус к основным характеристикам персонажа. Поля аналогичны таковым из функции `avatar.GetInnateStats()`, однако поле `effective` содержит сумму только значений `base` и `enchants`.

- **`resistances`** (`table`)
Бонус к сопротивляемостям персонажа. Поля аналогичны таковым из функции `avatar.GetResistances()`, однако поле `effective` содержит сумму только значений `base` и `enchants`.

- **`miscStats`** (`table`)
Различные дополнительные бонусы.

  - `power` (`table`) - Могущество.
  - `stamina` (`table`) - Выносливость.
  - `wisdom` (`table`) - Мудрость.
  - `minDamage` (`table`) - Минимальный урон.
  - `maxDamage` (`table`) - Максимальный урон.
  - `spellPower` (`table`) - Среднее значение силы заклинаний.
  - `maxSpellPower` (`table`) - Максимальное значение силы заклинаний.
  - `minSpellPower` (`table`) - Минимальное значение силы заклинаний.
  - `weaponSpeed` (`float`) - Скорость атаки (применяется для оружия).
  - `summondDps` (`boolean`) - Флаг, указывающий, что урон от данного предмета необходимо суммировать с текущим, а не заменять его.
  - `hideDPS` (`boolean`) - Флаг, скрывающий отображение урона у оружия.
  - **`specStats`** (`table`)
  Массив таблиц с различными специальными характеристиками.
  
    - `id` (`SpecialStatId`) - Идентификатор специальной характеристики.
    - `name` (`WString`) - Полное название специальной характеристики.
    - `tooltipName` (`WString`) - Упрощенное название специальной характеристики.
    - `shortDescription` (`WString`) - Упрощенное описание специальной характеристики.
    - `value` (`number`) - Числовое значение специальной характеристики.
    - `type` (`number`) - Класс характеристики, соответствует значениям из `ENUM_SpecialStatType`.

Формат полей бонусов в `miscStats` (за исключением `weaponSpeed`, `summondDps` и `hideDPS`) представляет собой таблицы со следующими полями:
- `base` (`number`) - Базовое значение характеристики.
- `effective` (`number`) - Эффективное (текущее) значение характеристики.
- `enchants` (`number`) - Величина изменения от чар (энчантов), вставленных в надетые на персонажа предметы.

## Примеры
### Получение эффективного значения могущества от предмета
```lua
local bonus = itemLib.GetBonus( itemId )
if bonus then
  local power = bonus.miscStats.power.effective
end
```
