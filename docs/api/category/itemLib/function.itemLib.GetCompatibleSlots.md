# itemLib.GetCompatibleSlots

Возвращает список слотов, в которые можно экипировать предмет.

## Описание

```lua
itemLib.GetCompatibleSlots( itemId: ObjectId ): table
```

Функция извлекает и возвращает список слотов, в которые можно экипировать указанный предмет. Если предмет не предназначен для экипировки, возвращается пустая таблица.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `table` - таблица, содержащая массив идентификаторов слотов.

- **`[index]`** (`number`(`DRESS_SLOT_*`)) - Идентификатор слота.

## Примеры

### Получение списка доступных слотов

```lua
local slots = itemLib.GetCompatibleSlots( itemId )
for i, cur_slot in pairs( slots ) do
    LogInfo( "Номер слота: ", cur_slot )
end
```
