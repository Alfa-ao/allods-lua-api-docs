# itemLib.GetCraftInfo

Возвращает таблицу с информацией о предмете, связанную с крафтами.

## Описание

```lua
itemLib.GetCraftInfo( itemId: ObjectId ): table|nil
```

Функция возвращает таблицу с информацией о предмете, связанную с крафтами. Если предмет не связан с крафтами или произошла ошибка, возвращается `nil`.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `table` | `nil`.

При успешном получении информации возвращается таблица со следующими полями:

- **`craftingSkillIds`** (`table`) - (УСТАРЕЛО) Индексированный с `0` список идентификаторов ресурсов скиллов крафтинга, для которых используется предмет. Принимает значение `nil`, если предмет не участвует в крафтинге.

- **`craftingSkillsInfo`** (`table`) - Таблица с подробной информацией о скиллах крафтинга. Содержит вложенные таблицы со следующими полями:

    - **`skillId`** (`SkillId`) - Идентификатор ресурса скилла крафтинга, для которого используется предмет.

    - **`isLearned`** (`boolean`) - Флаг, указывающий, выучен ли скилл.

    - **`minLevel`** (`number` | `nil`) - Минимальный уровень рецепта, для которого нужен этот компонент.

    - **`maxLevel`** (`number` | `nil`) - Максимальный уровень рецепта, для которого нужен этот компонент.

- **`craftingComponents`** (`table`) - Таблица (начиная с `0`) идентификаторов (`ComponentPropertyId`) ресурсов компонентов крафтинга, если предмет участвует в крафтинге. Может быть пустой, если крафтинг не использует компоненты.

- **`foragingInfo`** (`table` | `nil`) - Информация об инструменте или предмете фуражирования. Содержит следующие поля:

    - **`isInstrument`** (`boolean`) - Флаг, указывающий, является ли предмет инструментом фуражирования.

    - **`skills`** (`table`) - Индексированный с `0` список идентификаторов ресурсов скиллов, которые использует этот инструмент.

- **`disassemblerInfo`** (`table` | `nil`) - Информация об инструменте для дизассемблинга, если предмет таковым является. Содержит следующие поля:

    - **`slots`** (`table`) - Список слотов одежды, которые можно разломать данным предметом. Индексация с `0`, значения: `number` (`DRESS_SLOT_*`).

    - **`classes`** (`table`) - Список названий классов предметов, которые можно разломать данным предметом. Индексация с `0`, значения: `WString`.

- **`isAlchemyJoker`** (`boolean` | `nil`) - Флаг, принимающий значение `true`, если предмет является джокером для алхимии.

## Примеры

### Проверка предмета на статус джокера алхимии

```lua
local info = itemLib.GetCraftInfo( itemId )
local isJoker = info and info.isAlchemyJoker
```
