# itemLib.GetCraftToolInfo

Возвращает информацию о ремесленном инструменте, если предмет является инструментом.

## Описание

```lua
itemLib.GetCraftToolInfo( itemId: ObjectId ): table | nil
```

Функция возвращает таблицу с информацией о ремесленном инструменте, если указанный предмет является крафтовым инструментом. Если предмет не является инструментом, функция возвращает `nil`.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `table` или `nil`.

Если предмет является крафтовым инструментом, возвращается таблица со следующими полями:

- **`properties`** (`table`) - Индексированная от `0` таблица со списком свойств инструмента, которые у него уже есть. Элементы: `string`(`ENUM_ToolProperty_*`).

- **`toolImprovers`** (`table`) - Индексированная от `0` таблица с описаниями предметов-улучшителей, способных улучшить данный инструмент. Каждый элемент является таблицей со следующими полями:
    - `item` (`ObjectId`) - Идентификатор предмета-улучшителя.
    - `properties` (`table`) - Индексированная от `0` таблица со списком свойств, которые добавляет к инструменту соответствующий улучшитель. Элементы: `string`(`ENUM_ToolProperty_*`).

## Примеры

### Получение информации о ремесленном инструменте

```lua
local info = itemLib.GetCraftToolInfo( itemId )
if info and info.properties[ 0 ] then
    Loginfo( "Первое свойство: ", info.properties[ 0 ] )
end
```
