# itemLib.GetDressSlot
Возвращает номер слота экипировки, к которому относится указанный предмет.

## Описание
```lua
itemLib.GetDressSlot( objectId: int ): int
```
Функция возвращает номер слота экипировки, к которому относится предмет с заданным идентификатором.

## Список параметров
- **`objectId`** (`int`) - Идентификатор предмета.

## Возвращаемые значения
Возвращает `int`(`DRESS_SLOT_*`) - Номер слота экипировки, к которому относится предмет.

## Примеры
### Получение слота экипировки
```lua
LogInfo( "Информация о слоте: ", itemLib.GetDressSlot( itemId ) )
```
