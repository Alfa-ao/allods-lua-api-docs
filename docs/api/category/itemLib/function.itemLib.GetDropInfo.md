# itemLib.GetDropInfo
Возвращает сообщение, отображаемое перед выбрасыванием предмета.
## Описание
```lua
itemLib.GetDropInfo( itemId: ObjectId ): WString | nil
```
Функция возвращает специальный вопрос, задаваемый перед выбрасыванием предмета, если таковой имеется. Если специальный вопрос отсутствует, функция возвращает `nil`.
## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета.
## Возвращаемые значения
Возвращает `WString` | `nil` - Специальный вопрос, задаваемый перед выбрасыванием предмета. Если вопрос отсутствует, возвращается `nil`.
## Примеры
### Получение сообщения перед выбрасыванием
```lua
LogInfo( itemLib.GetDropInfo( itemId ) )
```
