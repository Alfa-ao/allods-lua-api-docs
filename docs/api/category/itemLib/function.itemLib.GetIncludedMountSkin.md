# itemLib.GetIncludedMountSkin
Возвращает идентификатор скина маунта, который предоставляет предмет при использовании.

## Описание
```lua
itemLib.GetIncludedMountSkin( itemId: ObjectId ): ObjectId | nil
```
Функция возвращает идентификатор скина маунта, который предоставляет данный предмет при использовании. Если предмет не предоставляет скин маунта, функция возвращает `nil`.

## Список параметров
- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения
Возвращает `ObjectId` | `nil` - Идентификатор скина маунта, предоставляемого предметом, или `nil`, если предмет не предоставляет скин.

## Примеры
### Получение идентификатора скина маунта
```lua
local mountSkinId = itemLib.GetIncludedMountSkin( itemId )
```
