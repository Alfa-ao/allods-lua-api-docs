# itemLib.GetItemInfo

Извлекает общую информацию о предмете.

## Описание

```lua
itemLib.GetItemInfo( itemId: ObjectId ): table|nil
```

Функция возвращает таблицу с подробными характеристиками и свойствами указанного предмета. Если предмет не найден, возвращается `nil`.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `table` | `nil` - таблица с информацией о предмете или `nil`, если предмет не найден.

- **`id`** (`ObjectId`) - Идентификатор предмета.

- **`name`** (`WString`) - Название предмета.

- **`description`** (`ValuedText` | `nil`) - Описание с подставленными текущими значениями параметров.

- **`sourceDescription`** (`GlossaryId` | `nil`) - Идентификатор ресурса универсального описателя для источника предмета при наличии.

- **`dressSlot`** (`number`(`DRESS_SLOT_*`)) - Слот одежды или оружия, если предмет надевается.

- **`sysName`** (`string`) - Специальные интерфейсные особенности предмета.

- **`level`** (`number`) - Уровень предмета.

- **`forceShowLevel`** (`boolean`) - Флаг необходимости показа уровня на иконке предмета.

- **`requiredLevel`** (`number`) - Уровень персонажа, необходимый для ношения предмета.

- **`requiredReputationLevel`** (`number`(`REPUTATION_LEVEL_*`)) - Уровень репутации, необходимый для покупки предмета.

- **`requiredReputationQuantity`** (`number`) - Минимальное значение репутации, необходимое для покупки предмета.

- **`isRitual`** (`boolean`) - Флаг, указывающий, является ли предмет ритуальным (надевается, если игрок прошел ритуал и лежит в контейнере ITEM_CONT_EQUIPMENT_RITUAL).

- **`debugName`** (`string`) - Путь к файлу описания предмета.

- **`icon`** (`TextureId`) - Текстура с иконкой предмета (доступно только в UI).

- **`isDoubleHands`** (`boolean`) - Флаг, указывающий, является ли предмет двуручным оружием.

- **`isIgnoreDressSlotLevel`** (`boolean`) - Флаг, указывающий, можно ли одеть предмет в заблокированный по уровню слот.

- **`isDressable`** (`boolean`) - Флаг, указывающий, является ли предмет предметом одежды.

- **`isUsable`** (`boolean`) - Флаг, указывающий, можно ли применять предмет.

- **`isWeapon`** (`boolean`) - Флаг, указывающий, является ли предмет оружием.

- **`isGuildItem`** (`boolean`) - Флаг привязки предмета к гильдии.

- **`buyConfirmationRequired`** (`boolean`) - Флаг необходимости показа подтверждения при покупке предмета у вендора.

- **`needCheckPredicates`** (`boolean`) - Флаг необходимости проверки предикатов при отображении.

- **`showOnlyIconInLink`** (`boolean`) - Флаг отображения только иконки данного предмета в итемлинке.

## Примеры

### Получение информации о предмете

```lua
local itemInfo = itemLib.GetItemInfo( itemId )
```
