# itemLib.GetMetaInfo

Возвращает таблицу с информацией о мета-свойствах предмета.

## Описание

```lua
itemLib.GetMetaInfo( itemId: ObjectId ): table | nil
```

Мета-предмет - это предмет, у которого существует 4 градации качества. Мета-предмет можно улучшать из одного качества в другое. При этом на каждом типе качества характеристики предмета соответствуют обычному предмету данного качества.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `table` | `nil` - Если предмет найден, возвращается таблица со следующими полями.

- **`isInactive`** (`boolean`) - `true`, если предмет неактивирован, требует активации, без которой не может подвергаться апгрейду или апгрейдить другие предметы.

- **`hasMetaState`** (`boolean`) - `true`, если это мета-предмет, который и которым можно апгрейдить (если `isInactive` равно `false`).

- **`isMetaEnchancer`** (`boolean`) - `true`, если это мета-предмет, которым можно апгрейдить другие такие же метапредметы (если `isInactive` равно `false`).

- **`isUniversalMetaEnchancer`** (`boolean`) - `true`, если это мета-предмет, которым можно апгрейдить другие метапредметы из некоторого списка.

- **`isUpgradeAgent`** (`boolean`) - `true`, если это агент, необходимый для улучшения мета-предметов.

- **`isSingleUpgradeAgent`** (`boolean`) - `true`, если это агент, необходимый для улучшения мета-предметов, который можно использовать не более, чем одну штуку за раз (см. также поле `maxImprovement` у `avatar.GetMetaUpgradeResults`).

- **`improvement`** (`number`) - Улучшенность состояния предмета на данный момент. От `0.0` до `99.9`. `0.0` - совсем не улучшен. `99.9` - практически достиг следующего уровня.

## Примеры

### Получение информации о мета-предмете

```lua
local meta = itemLib.GetMetaInfo( itemId )
if meta then
    local isMetaItem = meta.isUpgradeAgent
end
```
