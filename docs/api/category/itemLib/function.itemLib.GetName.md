# itemLib.GetName

Возвращает название предмета.

## Описание

```lua
itemLib.GetName( itemId: ObjectId ): WString
```

Функция возвращает название предмета по его идентификатору.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `WString` - название предмета.

## Примеры

### Получение названия предмета

```lua
local itemName = itemLib.GetName( itemId )
```
