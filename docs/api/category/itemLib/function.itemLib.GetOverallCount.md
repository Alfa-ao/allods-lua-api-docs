# itemLib.GetOverallCount
Возвращает суммарное количество зарядов всех подобных предметов, имеющихся у игрока.

## Описание
```lua
itemLib.GetOverallCount( itemId: ObjectId ): number
```
Функция извлекает и возвращает общее количество зарядов предметов, идентифицируемых по ресурсу, находящихся в инвентаре и экипировке главного персонажа.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета-образца.

## Возвращаемые значения
Возвращает `number` - Количество зарядов всех таких предметов, имеющихся у игрока.

## Примеры
### Получение суммарного количества зарядов
```lua
local itemOverallChargesCount = itemLib.GetOverallCount( itemId )
```
