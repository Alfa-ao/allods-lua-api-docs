# itemLib.GetOwnershipLimit

Возвращает максимальное количество предметов указанного типа во владении персонажа.

## Описание

```lua
itemLib.GetOwnershipLimit( itemId: ObjectId | ItemId ): number
```

Функция возвращает значение ограничения на количество предметов заданного типа, которое может одновременно находиться во владении главного персонажа.

## Список параметров

- **`itemId`** (`ObjectId` | `ItemId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `number` - значение предела владения предметом. Значение `0` означает отсутствие ограничения.

## Примеры

### Проверка ограничения владения предметом

```lua
local limit = itemLib.GetOwnershipLimit( itemId )
if limit <= 0 then
    LogInfo( "No limit" )
end
```
