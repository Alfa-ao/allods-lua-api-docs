# itemLib.GetPriceInfo

Извлекает информацию о ценах предмета при покупке и продаже.

## Описание

```lua
itemLib.GetPriceInfo( itemId: ObjectId ): table|nil
```

Функция возвращает таблицу с информацией о ценах предмета при покупке и продаже, либо `nil`, если информация недоступна.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `table` | `nil`.

Если информация успешно получена, возвращается таблица со следующими полями:

- **`sellPrice`** (`number`) - Цена предмета при продаже текущему торговцу или по умолчанию. Значение `-1` указывает на невозможность продажи вендору.

- **`auctionPrice`** (`number`) - Цена предмета, с которой берется налог на аукционе. Значение `-1` указывает на невозможность продажи на аукционе.

- **`buyPrice`** (`number`) - Цена предмета при покупке у текущего торговца или по умолчанию. Равна базовой цене, не из buyback и secondhand.

- **`basePrice`** (`number`) - Базовая цена предмета по умолчанию. Используется, например, для расчета стоимости аукциона.

- **`altPrices`** (`table` | `nil`) - `nil`, если информация отсутствует. Иначе список, индексируемый от 1, цен в альтернативных валютах. Элемент списка представляет собой таблицу со следующими полями:
    - **`currencyId`** (`CurrencyId`) - Идентификатор альтернативной валюты.
    - **`value`** (`number`) - Цена в данной альтернативной валюте.

## Примеры

### Получение цены продажи предмета

```lua
local itemPrice = itemLib.GetPriceInfo( itemId )
local itemSellPrice = itemPrice and itemPrice.sellPrice
```
