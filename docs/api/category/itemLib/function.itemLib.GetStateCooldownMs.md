# itemLib.GetStateCooldownMs
Возвращает длительность специального суточного кулдауна на использование предмета.

## Описание
```lua
itemLib.GetStateCooldownMs( itemId: ObjectId ): number | nil
```
Такой кулдаун назначается для некоторых предметов при их использовании и длится до начала какого-либо дня.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения
Возвращает `number` | `nil` - сколько миллисекунд осталось до окончания кулдауна, если он есть для этого предмета; `nil`, если кулдаун отсутствует.

## Примеры
### Получение оставшегося времени кулдауна
```lua
local remainingMs = itemLib.GetStateCooldownMs( itemId )
if remainingMs then
    LogInfo( "Осталось часов: ", remainingMs / ( 60 * 60 * 1000 ) )
end
```

::: info Описание примера
В примере запрашивается время кулдауна для предмета. Если кулдаун активен, в лог выводится оставшееся время, пересчитанное в часы.
:::
