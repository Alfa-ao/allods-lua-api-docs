# itemLib.GetTemporaryInfo
Возвращает информацию о временном предмете, если предмет таковым является.

## Описание
```lua
itemLib.GetTemporaryInfo( itemId: ObjectId ): table | nil
```
Функция извлекает и возвращает информацию о временном предмете, если предмет таковым является.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения
Возвращает `table` | `nil` - информация о временном предмете, если предмет таковым является.

- **`durationMs`** (`number`) - сколько миллисекунд всего длится жизнь предмета в сумке.

- **`remainingMs`** (`number`) - через сколько миллисекунд предмет исчезнет. Может быть 0, если предмет не в сумке, а, например, у торговца.

- **`expireItem`** (`ObjectId` | `nil`) - идентификатор предмета, в который превратится данный предмет по истечении срока.

## Примеры
### Получение оставшегося времени жизни предмета
```lua
local itemTemporaryInfo = itemLib.GetTemporaryInfo( itemId )
local remainingMs = itemTemporaryInfo and itemTemporaryInfo.remainingMs
```
