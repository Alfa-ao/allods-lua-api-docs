# itemLib.GetUniversalMetaEnchancerItems

Возвращает индексированный с 0 список идентификаторов предметов, которые может улучшить данный улучшитель.

## Описание

```lua
itemLib.GetUniversalMetaEnchancerItems( itemId: ObjectId ): table | nil
```

Функция формирует и возвращает индексированный с 0 список идентификаторов предметов, которые может улучшить данный улучшитель. В случае возникновения ошибки возвращается `nil`.

## Список параметров

- **`itemId`** (`ObjectId`) - идентификатор предмета.

## Возвращаемые значения

Возвращает `table` | `nil` - индексированный с 0 список идентификаторов предметов, которые может улучшить данный улучшитель. В случае возникновения ошибки возвращается `nil`.

## Примеры

### Базовый вызов

```lua
local itemsId = itemLib.GetUniversalMetaEnchancerItems( itemId )
```
