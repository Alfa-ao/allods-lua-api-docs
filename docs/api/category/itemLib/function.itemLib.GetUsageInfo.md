# itemLib.GetUsageInfo

Извлекает информацию, связанную с использованием предмета.

## Описание

```lua
itemLib.GetUsageInfo( itemId: ObjectId ): table
```

Функция возвращает таблицу с информацией, связанной с использованием предмета.

## Список параметров

- **`itemId`** (`ObjectId`)
Уникальный идентификатор предмета.

## Возвращаемые значения

Возвращает `table` - таблицу с информацией об использовании предмета:

- **`isUseItemAndTakeAction`** (`boolean`) - Является ли предметом, который можно использовать с проверкой предикатов и выполнением кастомных действий.

- **`isUseOnItemAndTakeAction`** (`boolean`) - Является ли предметом, который можно использовать на другом предмете.

- **`isUseOnMountAndTakeAction`** (`boolean`) - Является ли предметом, который можно использовать на средстве передвижения (маунте).

- **`warningUseText`** (`WString` | `nil`) - Текст с предупреждением об использовании.

- **`useStack`** (`boolean`) - Можно ли использовать сразу на группе предметов (на стопке).

- **`useAmount`** (`number`) - Сколько предметов использовать за раз. Рекомендуемое значение, по умолчанию `-1` (использовать все).

## Примеры

### Получение информации об использовании предмета

```lua
local itemUsageInfo = itemLib.GetUsageInfo( itemId )
```
