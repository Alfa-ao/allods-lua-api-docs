# itemLib.GetUseOnItemCursor

Возвращает алиас курсора для использования на другой предмет с выполнением кастомных действий.

## Описание

```lua
itemLib.GetUseOnItemCursor( itemId: ObjectId ): string
```

Функция извлекает и возвращает строковый алиас курсора, предназначенный для использования на другом предмете с выполнением кастомных действий.

## Список параметров

- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения

Возвращает `string` - алиас курсора.

## Примеры

### Получение и установка курсора

```lua
local cursor = itemLib.GetUseOnItemCursor( itemId )
common.SetCursor( cursor )
```

::: info Описание примера
В примере сначала извлекается алиас курсора для указанного предмета, а затем он устанавливается с помощью функции `common.SetCursor`.
:::
