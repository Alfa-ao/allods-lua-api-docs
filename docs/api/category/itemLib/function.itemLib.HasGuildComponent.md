# itemLib.HasGuildComponent
Проверяет наличие в предмете компонентов, зависящих от состояния гильдии игрока.

## Описание
```lua
itemLib.HasGuildComponent( itemId: ObjectId ): boolean
```
Функция возвращает признак наличия в предмете компонентов, изменяющихся в зависимости от состояния гильдии игрока (например, гильдейских наплечников).

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения
Возвращает `boolean` - `true`, если в предмете есть компоненты, зависящие от состояния гильдии игрока.

## Примеры
### Проверка наличия гильдейского компонента
```lua
local hasGuildComponent = itemLib.HasGuildComponent( itemId )
```
