# itemLib.HasRelatedQuestObjectives

Проверяет наличие целей заданий игрока, которые связаны с указанным предметом.

## Описание

```lua
itemLib.HasRelatedQuestObjectives( itemId: ObjectId, object: QuestId | ObjectId | nil ): boolean
```

Функция проверяет наличие целей заданий игрока, связанных с указанным предметом. Параметр `object` позволяет уточнить область проверки: при передаче `nil` анализируются все связанные цели, при передаче `QuestId` проверяется связь с конкретным заданием, при передаче `ObjectId` проверяется связь с конкретной целью задания.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

- **`object`** (`QuestId` | `ObjectId` | `nil`) - Уточнение проверки. При значении `nil` проверяется наличие связанных целей заданий в целом. При значении `QuestId` проверяется связь с конкретным заданием. При значении `ObjectId` проверяется связь с конкретной целью задания.

## Возвращаемые значения

Возвращает `boolean` - `true`, если такие цели заданий имеются у игрока.

## Примеры

### Проверка наличия связанных целей заданий

```lua
local hasQuestObjectives = itemLib.HasRelatedQuestObjectives( itemId )
```
