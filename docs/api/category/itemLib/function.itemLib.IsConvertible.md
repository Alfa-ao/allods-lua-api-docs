# itemLib.IsConvertible
Определяет возможность конвертации предмета в подобные предметы.

## Описание
```lua
itemLib.IsConvertible( itemId: ObjectId ): boolean
```
Функция проверяет, разрешена ли конвертация указанного предмета в подобные предметы.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения
Возвращает `boolean` - `true`, если предмет можно конвертировать в подобные предметы.

## Примеры
### Проверка возможности конвертации
```lua
local itemId = 12345
if itemLib.IsConvertible( itemId ) then
    LogInfo( "Предмет можно конвертировать." )
end
```
