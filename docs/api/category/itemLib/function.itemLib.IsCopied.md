# itemLib.IsCopied
Проверяет, был ли предмет скопирован, и определяет возможность его продажи на бирже наград.

## Описание
```lua
itemLib.IsCopied( itemId: ObjectId ): boolean
```
Функция возвращает логическое значение, указывающее, что предмет не был скопирован и его можно продать на биржу наград.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета. Не может быть `nil`.

## Возвращаемые значения
Возвращает `boolean` - `true`, если предмет не был скопирован и его можно продать на биржу.

## Примеры
### Проверка возможности продажи предмета
```lua
local info = itemLib.IsCopied( itemId )
if info then
    LogInfo( "Предмет можно продать на бирже наград" )
end
```
