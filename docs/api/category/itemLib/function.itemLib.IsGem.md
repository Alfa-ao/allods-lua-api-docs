# itemLib.IsGem

Проверяет, предназначен ли предмет для дерева талантов души.

## Описание

```lua
itemLib.IsGem( itemId: ObjectId ): boolean
```

Функция проверяет, предназначен ли указанный предмет для использования в дереве талантов души.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если предмет является гемом для дерева талантов души, и `false` в противном случае.

## Примеры

### Проверка предмета на гем

```lua
local isProbe = itemLib.IsGem( itemId )
```
