# itemLib.IsItem

Проверяет наличие предмета с указанным идентификатором.

## Описание

```lua
itemLib.IsItem( itemId: ObjectId ): boolean
```

Функция возвращает логическое значение, указывающее на существование предмета с заданным идентификатором.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если предмет с заданным идентификатором существует.

## Примеры

### Проверка существования предмета

```lua
if itemLib.IsItem( itemId ) then
    LogInfo( itemId, " is item" )
end
```
