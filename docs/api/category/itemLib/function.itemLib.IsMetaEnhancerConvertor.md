# itemLib.IsMetaEnhancerConvertor

Определяет, является ли предмет инструментом для конвертации метаитема-улучшителя в обычный предмет.

## Описание

```lua
itemLib.IsMetaEnhancerConvertor( itemId: ObjectId ): boolean
```

Функция проверяет идентификатор предмета и возвращает логическое значение, указывающее на то, является ли он инструментом для конвертации метаитема-улучшителя в обычный предмет.

## Список параметров

- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если предмет является инструментом для конвертации метаитема-улучшителя в обычный предмет.

## Примеры

### Проверка предмета

```lua
local isMetaEnhancerConvertor = itemLib.IsMetaEnhancerConvertor( itemId )
```
