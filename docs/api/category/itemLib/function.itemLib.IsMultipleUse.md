# itemLib.IsMultipleUse

Определяет возможность многократного использования предмета без сброса состояния после первого применения.

## Описание

```lua
itemLib.IsMultipleUse( itemId: ObjectId ): boolean
```

Функция проверяет, можно ли многократно использовать предмет (не сбрасывается ли состояние использования после первого применения). Применяется совместно с `itemLib.IsUseItemAndTakeActions`.

## Список параметров

- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если это предмет для использования с выполнением кастомных действий.

## Примеры

### Проверка многократного использования предмета

```lua
if itemLib.IsMultipleUse( itemId ) then
    -- Предмет можно использовать многократно
end
```
