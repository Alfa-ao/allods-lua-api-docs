# itemLib.IsPointed

Возвращает признак того, что предмет необходимо использовать на точку местности.

## Описание

```lua
itemLib.IsPointed( itemId: ObjectId ): boolean
```

Функция возвращает логическое значение, указывающее на необходимость использования предмета на точку местности.

## Список параметров

- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если предмет необходимо использовать на точку местности.

## Примеры

### Проверка признака использования на точку

```lua
local itemIsPointed = itemLib.IsPointed( itemId )
```
