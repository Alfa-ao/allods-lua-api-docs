# itemLib.IsProbe
Проверяет, является ли предмет пробником.

## Описание
```lua
itemLib.IsProbe( itemId: ObjectId ): boolean
```
Определяет, является ли указанный предмет пробником, и возвращает соответствующее логическое значение.

## Список параметров
- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения
Возвращает `boolean` - `true`, если предмет является пробником, и `false` в противном случае.

## Примеры
### Проверка предмета
```lua
local isProbe = itemLib.IsProbe( itemId )
```
