# itemLib.IsQuestOperator

Определяет возможность предмета давать квесты.

## Описание

```lua
itemLib.IsQuestOperator( itemId: ObjectId ): boolean
```

Функция проверяет, может ли предмет давать квесты. Условие действительно только для предметов, имеющихся у игрока.

## Список параметров

- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если предмет может давать квесты, и `false` в противном случае. Условие действительно только для предметов, имеющихся у игрока.

## Примеры

### Проверка возможности предмета давать квесты

```lua
local itemIsQuestOperator = itemLib.IsQuestOperator( itemId )
```
