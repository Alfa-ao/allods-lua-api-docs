# itemLib.IsQuestRelated
Определяет, является ли предмет квестовым.

## Описание

```lua
itemLib.IsQuestRelated( itemId: ObjectId ): boolean
```

Функция выполняет проверку необходимости предмета для завершения или использования в квесте. Условие действительно только для предметов, имеющихся в инвентаре игрока.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если предмет является квестовым, и `false` в противном случае.

## Примеры

### Проверка предмета на квестовость

```lua
local itemIsQuestRelated = itemLib.IsQuestRelated( itemId )
```
