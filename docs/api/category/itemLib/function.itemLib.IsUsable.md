# itemLib.IsUsable

Определяет возможность использования предмета.

## Описание

```lua
itemLib.IsUsable( itemId: ObjectId ): boolean
```

Функция проверяет, возможно ли использование указанного предмета. Для предметов во время лута значение недействительно и всегда возвращает `false`.

## Список параметров

- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения

Возвращает `boolean` - `true`, если использование предмета возможно, `false` в противном случае.

## Примеры

### Проверка возможности использования предмета

```lua
local itemIsUsable = itemLib.IsUsable( itemId )
```
