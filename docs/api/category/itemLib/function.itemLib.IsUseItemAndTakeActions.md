# itemLib.IsUseItemAndTakeActions
Определяет, можно ли использовать предмет с проверкой предикатов и выполнением кастомных действий.

## Описание
```lua
itemLib.IsUseItemAndTakeActions( itemId: ObjectId ): boolean
```
Функция возвращает логическое значение, указывающее на возможность использования предмета с проверкой предикатов и выполнением кастомных действий.

## Список параметров
- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета.

## Возвращаемые значения
Возвращает `boolean` - `true`, если предмет можно использовать с выполнением кастомных действий.
