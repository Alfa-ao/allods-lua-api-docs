# itemLib.IsWeapon
Проверяет, является ли предмет оружием.

## Описание
```lua
itemLib.IsWeapon( itemId: ObjectId ): boolean
```
Функция анализирует указанный идентификатор и возвращает признак того, что предмет относится к классу оружия.

## Список параметров
- **`itemId`** (`ObjectId`) - Идентификатор предмета.

## Возвращаемые значения
Возвращает `boolean` - `true`, если предмет является оружием.

## Примеры
### Проверка типа предмета
```lua
local itemIsWeapon = itemLib.IsWeapon( itemId )
```
