# lfgLib.EnableReplicateDestinations

Включает и выключает репликацию активностей для LFG.

## Описание

```lua
lfgLib.EnableReplicateDestinations( enable: boolean )
```

Функция включает или выключает репликацию активностей для LFG.

## Список параметров

- **`enable`** (`boolean`) - `true` для включения, `false` для выключения репликации.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Включение репликации

```lua
lfgLib.EnableReplicateDestinations( true )
```
