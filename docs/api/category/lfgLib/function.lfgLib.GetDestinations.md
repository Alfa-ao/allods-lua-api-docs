# lfgLib.GetDestinations

Возвращает список доступных целей для поиска группы.

## Описание

```lua
lfgLib.GetDestinations(): table | nil
```

Функция возвращает список доступных целей для поиска группы.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - список доступных целей. Индексация таблицы начинается с 1.

## Примеры

### Получение списка доступных целей

```lua
local availableDestinations = lfgLib.GetDestinations()
```
