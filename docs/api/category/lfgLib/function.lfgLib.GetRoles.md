# lfgLib.GetRoles
Возвращает список доступных ролей для поиска группы.

## Описание
```lua
lfgLib.GetRoles(): table | nil
```
Функция возвращает список ролей в группе, доступных для аватара.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` | `nil` - список ролей в группе, доступных для аватара. Индексация таблицы начинается с 1.
- **`[index]`** (`number`(`ENUM_LFGRole`)) - Идентификатор роли.

## Примеры
### Получение доступных ролей
```lua
local availableRoles = lfgLib.GetRoles()
```
