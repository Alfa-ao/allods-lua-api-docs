# lfgLib.IsAvailable

Проверяет доступность LFG для игрока.

## Описание

```lua
lfgLib.IsAvailable(): boolean
```

Функция возвращает логическое значение, указывающее на доступность LFG для игрока.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если игрок может воспользоваться LFG, и `false` в противном случае.

## Примеры

### Проверка доступности LFG

```lua
if lfgLib.IsAvailable() then
    -- можно работать с LFG
end
```
