# lfgLib.IsFreePass

Проверяет возможность свободного прохода на локацию без затрат ресурсов или альтвалюты.

## Описание

```lua
lfgLib.IsFreePass( destinationId: LfgDestinationId ): boolean
```

Функция проверяет возможность свободного прохода на указанную локацию без затрат ресурсов или альтвалюты.

## Список параметров

- **`destinationId`** (`LfgDestinationId`) - Идентификатор целевой локации.

## Возвращаемые значения

Возвращает `boolean` - `true`, если вход свободный, и `false`, если вход требует затрат заданного ресурса или альтвалюты.

## Примеры

### Проверка свободного прохода

```lua
local freePass = lfgLib.IsFreePass( destinationId )
```
