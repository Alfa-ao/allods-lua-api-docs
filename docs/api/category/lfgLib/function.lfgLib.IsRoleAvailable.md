# lfgLib.IsRoleAvailable

Проверяет доступность выбранной роли в поиске группы.

## Описание

```lua
lfgLib.IsRoleAvailable( role: number ): table | nil
```

Функция возвращает результат проверки доступности выбранной роли в поиске группы.

## Список параметров

- **`role`** (`number`(`ENUM_LFGRole`)) - роль в группе.

## Возвращаемые значения

Возвращает `table` | `nil` - таблицу с полями или `nil` в случае ошибки.

- **`success`** (`boolean`) - `true`, если роль доступна, `false`, если роль недоступна.

- **`requiredSpell`** (`table` | `nil`) - комбинации ресурсов заклинаний, которые необходимо изучить для того, чтобы выбранная роль стала доступна.

    - `[index]` - Ключ таблицы. Вложенные параметры `table`:

        - **`[value]`** (`SpellId`) - Идентификатор заклинания.

## Примеры

### Проверка доступности роли танка

```lua
local availableTank = lfgLib.IsRoleAvailable( ENUM_LFGRole_Tank )
```
