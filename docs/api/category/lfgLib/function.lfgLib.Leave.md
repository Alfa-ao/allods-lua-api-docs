# lfgLib.Leave

Покидает подбор группы.

## Описание

```lua
lfgLib.Leave()
```

Если подбор группы покидает лидер, всей группе предлагается покинуть подбор вместе с ним (оставаясь в составе одной группы). При этом отказавшиеся покинуть подбор остаются в нем в составе новой группы.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Базовый вызов

```lua
lfgLib.Leave()
```
