# lfgLib.VoteNo

Отправляет отрицательный голос по текущему вопросу.

## Описание

```lua
lfgLib.VoteNo()
```

Функция отправляет отрицательный голос по текущему вопросу. При инициации голосования передается событие `EVENT_LFG_VOTE`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Обработка события голосования

```lua
function OnLfgVote( event )
    lfgLib.VoteNo()
end
```

::: info Описание примера
В примере регистрируется обработчик события `EVENT_LFG_VOTE`, в котором вызывается функция для отправки отрицательного голоса.
:::
