# loot.CanSetLootScheme

Проверяет возможность изменения схемы дележа лута в рейде или группе.

## Описание

```lua
loot.CanSetLootScheme(): boolean | nil
```

Функция возвращает результат проверки возможности изменения схемы дележа лута. Схема дележа лута запрещена, например, в автоматически созданных рейдах. Данное свойство не меняется в течение времени существования рейда или группы.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` | `nil` - `true`, если тип рейда или группы позволяет менять схему дележа лута, `false`, если не позволяет; `nil` при возникновении ошибки.

## Примеры

### Проверка и установка схемы дележа лута

```lua
if loot.CanSetLootScheme() then
    loot.SetLootScheme( LOOT_SCHEME_TYPE_MASTER )
end
```
