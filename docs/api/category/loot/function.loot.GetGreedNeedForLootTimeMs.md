# loot.GetGreedNeedForLootTimeMs

Возвращает время в миллисекундах, через которое будет произведен автоматический Greed/Need ролл.

## Описание

```lua
loot.GetGreedNeedForLootTimeMs(): number
```

Функция возвращает время в миллисекундах, через которое будет произведен автоматический Greed/Need ролл.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - время в миллисекундах, через которое будет произведен автоматический Greed/Need ролл.

## Примеры

### Получение времени до автоматического ролла

```lua
local timeMs = loot.GetGreedNeedForLootTimeMs()
```
