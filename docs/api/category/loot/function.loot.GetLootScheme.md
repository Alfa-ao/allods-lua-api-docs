# loot.GetLootScheme

Возвращает текущую схему дележа лута.

## Описание

```lua
loot.GetLootScheme(): number
```

Функция возвращает тип текущей схемы дележа лута.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number`(`LOOT_SCHEME_TYPE_*`) - тип схемы дележа лута.

## Примеры

### Получение текущей схемы дележа лута

```lua
local scheme = loot.GetLootScheme()
```
