# loot.SwitchToGreedNeed

Переключает схему распределения ролла с master на greed-need.

## Описание

```lua
loot.SwitchToGreedNeed( rollId: ObjectId )
```

Функция переключает схему распределения ролла с master на greed-need.

## Список параметров

- **`rollId`** (`ObjectId`) - Уникальный идентификатор ролла.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Базовый вызов

```lua
loot.SwitchToGreedNeed( rollId )
```
