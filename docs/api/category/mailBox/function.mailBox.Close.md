# mailBox.Close

Принудительно закрывает и деактивирует почтовый ящик.

## Описание

```lua
mailBox.Close()
```

Функция инициирует принудительное закрытие и деактивацию почтового ящика.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Закрытие почтового ящика

```lua
mailBox.Close()
```
