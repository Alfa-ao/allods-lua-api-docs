# mailBox.ExtractMailItems
Извлекает предмет или все предметы из указанного письма.

## Описание
```lua
mailBox.ExtractMailItems( mailId: ObjectId, slot: number | nil ): boolean
```
Функция извлекает предмет из заданного слота письма или все предметы сразу, если слот не указан.

::: warning Замечание
Для выполнения операции необходима готовность почтового ящика, проверяемая функцией `IsReady`.
:::

## Список параметров
- **`mailId`** (`ObjectId`) - Уникальный идентификатор письма.

- **`slot`** (`number` | `nil`) - Индекс слота с предметом. Передача значения `nil` или `-1` приводит к извлечению всех предметов сразу.

## Возвращаемые значения
Возвращает `boolean` - `true`, если операция завершилась успешно, и `false` в противном случае. В случае неудачи также выдается скриптовая ошибка.

## Примеры
### Извлечение всех предметов из письма
```lua
mailBox.ExtractMailItems( mailId, nil )
```
