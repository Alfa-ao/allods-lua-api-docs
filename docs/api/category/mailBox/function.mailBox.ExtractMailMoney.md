# mailBox.ExtractMailMoney

Извлекает деньги из указанного письма.

## Описание

```lua
mailBox.ExtractMailMoney( mailId: ObjectId ): boolean
```

Функция извлекает деньги из письма с заданным идентификатором.

::: warning Замечание
Для выполнения операции почтовый ящик должен быть готов (проверка осуществляется функцией `IsReady`). В случае неудачи генерируется скриптовая ошибка.
:::

## Список параметров

- **`mailId`** (`ObjectId`) - Уникальный идентификатор письма.

## Возвращаемые значения

Возвращает `boolean` - `true`, если операция завершилась успешно, и `false` в противном случае.

## Примеры

### Извлечение денег из письма

```lua
mailBox.ExtractMailMoney( mailId )
```
