# mailBox.IsActive

Определяет, открыт ли почтовый ящик.

## Описание

```lua
mailBox.IsActive(): boolean
```

Функция возвращает логическое значение, указывающее, открыт ли почтовый ящик. Запрашивать информацию о письмах можно только для открытого почтового ящика. Для выполнения других операций требуется его готовность: `mailBox.IsReady`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - открыт ли почтовый ящик.

## Примеры

### Проверка состояния почтового ящика

```lua
if avatar.IsExist() and mailBox.IsActiveNG() then
    self:Activate( true )
end
```

::: info Описание примера
Проверяется существование аватара и состояние почтового ящика. Если почтовый ящик активен, вызывается метод активации.
:::
