# mailBox.IsInteracting

Определяет, находится ли главный персонаж в режиме взаимодействия с владельцем почтового ящика.

## Описание

```lua
mailBox.IsInteracting(): boolean
```

Функция возвращает логическое значение, указывающее на нахождение в режиме взаимодействия с владельцем почтового ящика. Открытие почтового ящика возможно исключительно при активном режиме взаимодействия.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если персонаж находится в режиме взаимодействия с владельцем почтового ящика, и `false` в противном случае.

## Примеры

### Проверка режима взаимодействия

```lua
if mailBox.IsInteracting() then
    mailBox.Open()
end
```
