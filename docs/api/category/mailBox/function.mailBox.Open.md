# mailBox.Open

Активирует почтовый ящик.

## Описание

```lua
mailBox.Open()
```

Функция выполняет попытку активации почтового ящика. Операции можно выполнять только с активным почтовым ящиком.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Открытие почтового ящика

```lua
mailBox.Open()
```
