# mailBox.ReturnMailToSender
Инициирует возврат письма отправителю.

## Описание
```lua
mailBox.ReturnMailToSender( mailId: ObjectId ): boolean
```
Функция инициирует возврат указанного письма его отправителю.

::: warning Замечание
Для выполнения операции необходима готовность почтового ящика, проверяемая функцией `IsReady`.
:::

## Список параметров
- **`mailId`** (`ObjectId`) - Уникальный идентификатор письма.

## Возвращаемые значения
Возвращает `boolean` - значение, указывающее на успешность завершения функции. В случае неудачи также выдается скриптовая ошибка.

## Примеры
### Возврат письма отправителю
```lua
mailBox.ReturnMailToSender( mailId )
```
