# matchMaking.CanJoinInstancedEvent
Проверяет возможность принятия аватаром участия в указанном или любом доступном инстанс-ивенте.

## Описание
```lua
matchMaking.CanJoinInstancedEvent( eventId: ObjectId | InstancedEventResourceId | nil ): boolean
```
Функция проверяет наличие возможности принять участие в конкретном инстанс-ивенте или в любом доступном эвенте, если идентификатор не задан.

## Список параметров
- **`eventId`** (`ObjectId` | `InstancedEventResourceId` | `nil`) - Идентификатор инстанс-ивента для проверки возможности вступления на конкретный эвент, или `nil` для общей проверки возможности вступления.

## Возвращаемые значения
Возвращает `boolean` - `true`, если аватар в данный момент может встать в очередь на инстанс-ивент (не приглашён, не участвует в инстанс-ивенте и тому подобное), иначе `false`.

## Примеры
### Проверка возможности вступления
```lua
if matchMaking.CanJoinInstancedEvent( eventId ) then
    -- Действия при успешной проверке
end
```
