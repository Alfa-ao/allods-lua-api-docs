# matchMaking.CanReturnToBattle

Проверяет возможность возвращения в инстанс-ивент, если аватар его покинул, но остался в рейде или группе.

## Описание

```lua
matchMaking.CanReturnToBattle(): boolean
```

Функция возвращает логическое значение, определяющее возможность возвращения в инстанс-ивент, если аватар его покинул, но остался в рейде или группе.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если можно вернуться, иначе `false`.

## Примеры

### Проверка возможности и возврат в бой

```lua
if matchMaking.CanReturnToBattle() then
    matchMaking.ReturnToBattle()
end
```

::: info Описание примера
В примере выполняется проверка возможности возвращения в инстанс-ивент. Если условие истинно, инициируется возврат с помощью функции `matchMaking.ReturnToBattle`.
:::
