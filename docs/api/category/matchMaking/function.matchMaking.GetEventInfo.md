# matchMaking.GetEventInfo

Извлекает информацию об инстанс-ивенте с заданным идентификатором.

## Описание

```lua
matchMaking.GetEventInfo( eventId: ObjectId ): table | nil
```

Функция извлекает и возвращает таблицу с информацией об инстанс-ивенте по его идентификатору. Если информация не получена, возвращается `nil`.

## Список параметров

- **`eventId`** (`ObjectId`) - Идентификатор инстанс-ивента, о котором запрашивается информация.

## Возвращаемые значения

Возвращает `table` | `nil` - таблица с описанием инстанс-ивента, если информация получена.

- **`id`** (`ObjectId`) - Идентификатор инстанс-ивента.

- **`name`** (`ValuedText` | `WString`) - Преобразованное имя инстанс-ивента.

- **`rawName`** (`WString` | `nil`) - Необработанное название инстанс-ивента, взятое из ресурса. `nil`, если совпадает с `name`.

- **`description`** (`ValuedText` | `WString`) - Описание с подставленными текущими значениями параметров.

- **`resourceId`** (`ResourceId`(`InstancedEventResourceId`)) - Идентификатор ресурса инстанс-ивента.

- **`category`** (`ResourceId`(`InstancedEventCategoryId`)) - Идентификатор ресурса категории инстанс-ивента.

- **`isAvailable`** (`boolean`) - Флаг доступности инстанс-ивента аватару.

- **`allowQuickExit`** (`boolean`) - Флаг разрешения быстрого покидания активности (кнопка в углу экрана).

- **`sysCause`** (`string`(`ENUM_ActionFailCause_*`)) - Строковый код ошибки.

- **`hasDoubleBonus`** (`boolean`) - Флаг наличия бонуса за прохождение.

- **`ignoreFactions`** (`boolean`) - Флаг ивента со смешанными фракциями (при формировании группы принадлежность к фракции игнорируется).

- **`leaderDepartOnly`** (`boolean`) - Флаг, указывающий, что инициировать отправку в активность может только лидер группы.

- **`soloMode`** (`boolean`) - Флаг, указывающий, что ивент доступен только для одного игрока.

- **`requiredItem`** (`ItemId` | `nil`) - Ресурс, необходимый для отправки.

- **`requiredCurrency`** (`CurrencyId` | `nil`) - Альтернативная валюта, необходимая для отправки.

- **`isHighPriority`** (`boolean`) - Флаг приоритетного эвента (должен быть выше остальных в списке).

- **`mechanicsType`** (`number`(`ENUM_MatchMakingMechanicsType_*`)) - Тип игровой механики.

- **`maxVisitsPerDay`** (`number`) - Максимальное количество посещений ивента в день (`0` - без ограничений).

- **`maxVisitsPerWeek`** (`number`) - Максимальное количество посещений ивента в неделю (`0` - без ограничений).

- **`exclusive`** (`boolean`) - Флаг, запрещающий становиться одновременно в очередь к этому и любому другому эвенту.

- **`leaderOnly`** (`boolean`) - Флаг, указывающий, что за выход или постановку в очередь отвечает только лидер группы (соло аватар считается сам себе лидером).

- **`ticket`** (`ItemId` | `nil`) - Идентификатор предмета-билета на эвент. `nil`, если нет информации. При постановке в очередь проверяется наличие предмета, он забирается перед телепортацией.

- **`activeEventJoinPeriod`** (`number`) - Возможность присоединения к арене новых игроков в течение определенного периода после старта. Значение меньше `0` - можно присоединяться всегда, равно `0` - нельзя вообще, иначе - период в миллисекундах.

- **`todayVisits`** (`number` | `nil`) - Количество посещений главного игрока сегодня. `nil`, если информация отсутствует.

- **`weeklyVisits`** (`number` | `nil`) - Количество посещений главного игрока за текущую неделю. `nil`, если информация отсутствует.

- **`duration`** (`number`) - Длительность ивента в миллисекундах.

- **`minAvatarLevel`** (`number`) - Минимальный уровень, необходимый для участия в ивенте.

- **`canJoinActiveEvent`** (`boolean`) - Флаг возможности присоединения к ивенту после его начала.

- **`isAvatarJoined`** (`boolean`) - Флаг нахождения аватара в очереди на этот инстанс-ивент.


::: info
Таблица eventType представляет собой описание типа инстанс-ивента. Так как тип может быть только один (либо solo, либо group, но не оба сразу), то в таблице eventType заполнено только одно поле из представленых ниже. В заполненом поле содержится таблица с описанием данных для этого типа инстанс-ивента.
:::
- **`eventType`** (`table`) - Таблица с описанием типа инстанс-ивента. Заполнено только одно поле:
  - **`solo`** (`table` | `nil`) - Одиночный тип:
    - **`minAvatarsCount`** (`number`) - Минимум игроков, после которого запускается таймер самоуничтожения острова.
    - **`capacity`** (`number`) - Максимальное количество игроков.
    - **`minAvatarLevel`** (`number`) - Минимальный уровень, необходимый для участия в ивенте.
  - **`group`** (`table` | `nil`) - Групповой тип (таблица с теми же полями, что и у `solo`).
  - **`premade`** (`table` | `nil`) - Заранее собранная группа (таблица с теми же полями, что и у `solo`).
  - **`mwar`** (`table` | `nil`) - Глобальная война за территорию (таблица с теми же полями, что и у `solo`).
  - **`pillage`** (`table` | `nil`) - Грабительский налет (таблица с теми же полями, что и у `solo`).
  - **`lastStand`** (`table` | `nil`) - Тип lastStand (таблица с теми же полями, что и у `solo`).
  - **`mwarRaid`** (`table` | `nil`) - Рейдовое сражение Доминиона (таблица с теми же полями, что и у `solo`).
  - **`mwarLadder`** (`table` | `nil`) - Групповое сражение Доминиона (таблица с теми же полями, что и у `solo`).
  - **`ratingPvP`** (`table` | `nil`) - Рейтинговая арена.
  - **`randomBG`** (`table` | `nil`) - Случайное бг.
  - **`lfgRaid`** (`table` | `nil`) - Рейдовый PvE эвент.
  - **`lfg`** (`table` | `nil`) - PvE эвент.


::: info
Таблица startTime представляет собой описание времени старта инстанс-ивента. Так как время старта может быть только разного типа (почасовое, ежедневное, еженедельное и т.п.), то в этой таблице заполнено только одно поле из представленых ниже. В заполненом поле содержится таблица с описанием данных для этого типа времени старта.
:::
- **`startTime`** (`table`) - Таблица с описанием типа и времени старта. Заполнено только одно поле:
  - **`hourly`** (`table` | `nil`) - Ежечасно:
    - **`minute`** (`number`) - Минута начала события, начиная с `0`.
  - **`daily`** (`table` | `nil`) - Ежедневно:
    - **`hour`** (`number`) - Час начала события, начиная с `0`.
    - **`minute`** (`number`) - Минута начала события, начиная с `0`.
  - **`weekly`** (`table` | `nil`) - Еженедельно:
    - **`day`** (`number`) - День недели начала события, начиная с `0`.
    - **`hour`** (`number`) - Час начала события, начиная с `0`.
    - **`minute`** (`number`) - Минута начала события, начиная с `0`.
  - **`monthly`** (`table` | `nil`) - Ежемесячно:
    - **`day`** (`number`) - День месяца начала события, начиная с `1`.
    - **`hour`** (`number`) - Час начала события, начиная с `0`.
    - **`minute`** (`number`) - Минута начала события, начиная с `0`.
  - **`yearly`** (`table` | `nil`) - Ежегодно:
    - **`sysMonth`** (`string`(`ENUM_Month_*`)) - Месяц начала события.
    - **`month`** (`number`(`ENUM_Month_*`)) - Месяц начала события.
    - **`day`** (`number`) - День месяца начала события, начиная с `1`.
    - **`hour`** (`number`) - Час начала события, начиная с `0`.
    - **`minute`** (`number`) - Минута начала события, начиная с `0`.
  - **`now`** (`table` | `nil`) - Сейчас (пустая таблица).
  - **`never`** (`table` | `nil`) - Никогда (пустая таблица).
  - **`once`** (`table` | `nil`) - Однократный:
    - **`year`** (`number`) - Год начала события.
    - **`sysMonth`** (`string`(`ENUM_Month_*`)) - Месяц начала события.
    - **`month`** (`number`(`ENUM_Month_*`)) - Месяц начала события.
    - **`day`** (`number`) - День месяца начала события, начиная с `1`.
    - **`hour`** (`number`) - Час начала события, начиная с `0`.
    - **`minute`** (`number`) - Минута начала события, начиная с `0`.

- **`startTimes`** (`table` | `nil`) - Список (индексируется от `0`) таблиц с описанием типа и времени старта (структура (HELP) аналогична `startTime`). `nil`, если ошибка.

- **`inviteTimeout`** (`number`) - Задержка при телепорте на арену в миллисекундах.

- **`difficulty`** (`WString` | `nil`) - Сложность эвента текстом.

- **`difficultyMode`** (`number`(`ENUM_LFGEventDifficulty`)) - Сложность эвента индексом.

- **`sysDifficultyMode`** (`string`(`ENUM_LFGEventDifficulty`)) - Сложность эвента строкой.

## Примеры

### Получение информации о групповом инстанс-ивенте

```lua
local info = matchMaking.GetEventInfo( eventId )
if info then
    if info.group then
        LogInfo( "Group instanced event, capacity: ", info.group.capacity )
    end
end
```
