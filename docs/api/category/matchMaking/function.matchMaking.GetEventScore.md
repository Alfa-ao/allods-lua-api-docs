# matchMaking.GetEventScore

Возвращает значения достижений, отображаемых в качестве счёта, для всех команд.

## Описание

```lua
matchMaking.GetEventScore(): table | nil
```

Функция возвращает индексированную с нуля таблицу с достижениями команд-участников, отображаемыми в качестве счёта. Если информация отсутствует, возвращается `nil`. Каждый элемент списка представляет собой таблицу с данными о конкретной команде и её достижениях. При изменении данных приходит событие `EVENT_MATCH_MAKING_PROGRESS_ACHIEVEMENTS_CHANGED`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - `nil`, если информация отсутствует; иначе индексированная с нуля таблица достижений команд-участников.

Каждый элемент списка содержит следующие поля:

- **`teamMarkId`** (`BattlegroundMarkId` | `nil`) - Идентификатор команды, достижения которой содержатся в данной таблице. Равно `nil`, если информация отсутствует.

- **`teamAchievements`** (`table` | `nil`) - Индексированная с нуля таблица достижений данной команды. Равно `nil`, если информация отсутствует. Каждый элемент содержит поля:
    - `sysName` (`string` | `nil`) - Нелокализуемое системное имя достижения, если доступно.
    - `name` (`WString`) - Локализуемое имя достижения.
    - `description` (`WString`) - Описание достижения.
    - `amount` (`number`) - Количество (целое число).

## Примеры

### Получение счёта команды

```lua
local score = matchMaking.GetEventScore()
local score1 = score and score[0] and score[0].amount
```
