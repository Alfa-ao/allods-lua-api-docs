# matchMaking.GetEvents

Возвращает список идентификаторов существующих инстанс-ивентов.

## Описание

```lua
matchMaking.GetEvents(): table | nil
```

Функция возвращает таблицу с идентификаторами существующих инстанс-ивентов или `nil`, если информация отсутствует.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - таблица, индексируемая от `0`, каждое поле которой содержит идентификатор инстанс-ивента, или `nil`, если информация о существующих инстанс-ивентах отсутствует.

## Примеры

### Получение информации о первом инстанс-ивенте

```lua
local firstEventId = matchMaking.GetEvents()[0]
if firstEventId ~= nil then
    LogInfo( matchMaking.GetEventInfo( firstEventId ) );
end
```

::: info Описание примера
В примере запрашивается список идентификаторов инстанс-ивентов. Если список не пуст, извлекается идентификатор первого события с индексом `0`, после чего выводится подробная информация о нем с помощью функции `matchMaking.GetEventInfo`.
:::
