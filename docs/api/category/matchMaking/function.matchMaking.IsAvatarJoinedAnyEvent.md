# matchMaking.IsAvatarJoinedAnyEvent

Определяет, стоит ли в данный момент аватар в очереди на какие-либо инстанс-ивенты.

## Описание

```lua
matchMaking.IsAvatarJoinedAnyEvent(): boolean
```

Функция возвращает признак нахождения аватара в очереди на инстанс-ивенты.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если аватар присоединён к очереди на хотя бы один инстанс-ивент, и `false`, если нет.

## Примеры

### Проверка статуса очереди

```lua
if matchMaking.IsAvatarJoinedAnyEvent() then
    LogInfo( "Joined" )
end
```
