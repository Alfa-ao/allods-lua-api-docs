# matchMaking.IsEventIdExist
Проверяет существование описания инстанс-ивента с указанным идентификатором.

## Описание
```lua
matchMaking.IsEventIdExist( eventId: ObjectId ): boolean
```
Функция анализирует реестр инстанс-ивентов и определяет наличие записи с заданным идентификатором. Гарантируется, что при положительном результате идентификатор всегда соответствует одному и тому же ивенту.

## Список параметров
- **`eventId`** (`ObjectId`) - Идентификатор инстанс-ивента.

## Возвращаемые значения
Возвращает `boolean` - `true`, если `eventId` является идентификатором существующего ивента, иначе `false`. Гарантируется, что в случае `true` `eventId` всегда соответствует одному и тому же ивенту.

## Примеры
### Проверка существования ивента и вывод информации
```lua
if matchMaking.IsEventIdExist( eventId ) then
    LogInfo( "Info: ", matchMaking.GetEventInfo( eventId ) )
end
```

::: info Описание примера
В примере проверяется существование ивента по идентификатору. При положительном результате в лог выводится информация о ивенте, полученная посредством вызова `matchMaking.GetEventInfo`.
:::
