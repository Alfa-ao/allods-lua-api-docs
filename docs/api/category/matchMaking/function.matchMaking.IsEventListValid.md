# matchMaking.IsEventListValid

Определяет наличие информации о существующих инстанс-ивентах.

## Описание

```lua
matchMaking.IsEventListValid(): boolean
```

Функция возвращает статус наличия информации о существующих инстанс-ивентах. Значение `true` означает, что получение информации включено и данные уже получены, значение `false` - что информация отсутствует или получение не включено.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если информация о существующих инстанс-ивентах успешно получена, и `false` в противном случае.

## Примеры

### Проверка и вывод списка инстанс-ивентов

```lua
if matchMaking.IsEventListValid() then
    LogInfo( "Info: ", matchMaking.GetEvents() )
end
```

::: info Описание примера
В примере проверяется наличие актуальной информации об инстанс-ивентах. Если данные доступны, в лог выводится список событий с использованием функции `matchMaking.GetEvents`.
:::
