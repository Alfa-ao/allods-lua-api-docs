# matchMaking.IsFreePass

Определяет возможность свободного входа в инстанс-ивент без затрат ресурсов или альтвалюты.

## Описание

```lua
matchMaking.IsFreePass( eventId: ObjectId ): boolean
```

Функция проверяет состояние входа в указанный инстанс-ивент и возвращает признак отсутствия затрат ресурсов или альтвалюты.

## Список параметров

- **`eventId`** (`ObjectId`) - Идентификатор инстанс-ивента.

## Возвращаемые значения

Возвращает `boolean` - `true`, если вход свободный, и `false`, если вход требует затрат ресурса или альтвалюты.

## Примеры

### Проверка свободного входа

```lua
local freePass = matchMaking.IsFreePass( eventId )
```
