# matchMaking.IsRaid

Проверяет, что данный инстанс-ивент является рейдовым сражением.

## Описание

```lua
matchMaking.IsRaid( eventId: ObjectId ): boolean
```

Функция определяет, является ли указанный инстанс-ивент рейдовым сражением.

## Список параметров

- **`eventId`** (`ObjectId`) - Идентификатор инстанс-ивента.

## Возвращаемые значения

Возвращает `boolean` - `true`, если данный инстанс-ивент является рейдовым сражением, и `false` в противном случае.

## Примеры

### Проверка инстанс-ивента на рейдовое сражение

```lua
if matchMaking.IsRaid( eventId ) then
    LogInfo( "It is raid event" )
end
```
