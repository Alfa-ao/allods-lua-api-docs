# medalsLib.GetCategories

Возвращает список категорий достижений.

## Описание

```lua
medalsLib.GetCategories(): table | nil
```

Функция возвращает список категорий достижений в виде таблицы с индексацией, начинающейся с `0`. Каждая ячейка таблицы представляет собой вложенную таблицу с полями категории и её подкатегорий. Если категории отсутствуют, функция возвращает `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - список категорий. Индексация начинается с `0`. Каждая ячейка является таблицей со следующими полями:

- **`name`** (`WString`) - название категории.

- **`subCategories`** (`table`) - список подкатегорий. Индексация начинается с `0`. Каждая ячейка является таблицей со следующими полями:

- **`name`** (`WString`) - название подкатегории.

## Примеры

### Получение списка категорий

```lua
local categories = medalsLib.GetCategories()
```
