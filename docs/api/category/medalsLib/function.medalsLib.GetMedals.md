# medalsLib.GetMedals
Возвращает список активных достижений.

## Описание
```lua
medalsLib.GetMedals(): table|nil
```
Функция возвращает таблицу с идентификаторами активных достижений. Если список пуст, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` | `nil` - таблица идентификаторов активных достижений, либо `nil`, если список пуст.

- **`[index]`** (`ObjectId`) - Идентификатор активного достижения. Индексация начинается с `0`.

## Примеры
### Получение информации о достижении
```lua
local medals = medalsLib.GetMedals()
if medals then
    local medalInfo = medalsLib.GetMedalInfo( medals[0] )
end
```

::: info Описание примера
В примере запрашивается список активных достижений. Если таблица не пуста, извлекается идентификатор первого достижения с индексом `0` и передается в функцию `medalsLib.GetMedalInfo` для получения подробной информации.
:::
