# medalsLib.SearchMedals

Осуществляет поиск достижений.

## Описание

```lua
medalsLib.SearchMedals( name: WString, limit: number | nil )
```

Функция осуществляет поиск достижений на основе строки, в которой через пробел перечислены ключевые слова. Слова, длина которых менее трех символов, не учитываются при поиске. Результат поиска передается в событии `EVENT_MEDALS_SEARCH_RESULTS`.

## Список параметров

- **`name`** (`WString`) - Название достижения, представляющее собой строку с перечисленными через пробел ключевыми словами.

- **`limit`** (`number` | `nil`) - Ограничение на количество искомых достижений. Необязательный параметр. При значениях `nil` и менее `1` будут найдены все достижения, соответствующие запросу.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Базовый поиск достижений

```lua
medalsLib.SearchMedals( "первым" )
```
