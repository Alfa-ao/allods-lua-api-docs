# mentor.GetCooldownsInfo
Возвращает информацию о кулдаунах на запрет действий в системе менторства.

## Описание
```lua
mentor.GetCooldownsInfo(): table|nil
```
Функция возвращает таблицу, содержащую информацию о времени до окончания запретов на действия в системе менторства. Для каждого действия указывается количество оставшихся миллисекунд до окончания запрета. При отсутствии запрета для данного действия возвращается значение `0`. Если возникает ошибка, функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` | `nil`. При успешном выполнении возвращается таблица с информацией о кулдаунах, при ошибке - `nil`.

- **`mentorRefuseRemainingMs`** (`number` (`integer`)) - Количество миллисекунд, оставшихся до окончания запрета на отказ от ментора.

- **`apprenticeRefuseRemainingMs`** (`number` (`integer`)) - Количество миллисекунд, оставшихся до окончания запрета на отказ от ученика.

- **`mentorFlagRemainingMs`** (`number` (`integer`)) - Количество миллисекунд, оставшихся до окончания запрета на сброс флага ментора.

- **`apprenticeFlagRemainingMs`** (`number` (`integer`)) - Количество миллисекунд, оставшихся до окончания запрета на сброс флага ученика.

## Примеры
### Получение информации о кулдаунах
```lua
local info = mentor.GetCooldownsInfo()
LogInfo( info.mentorRefuseRemainingMs )
```
