# mentor.GetManagerInfo

Возвращает информацию о статусе главного игрока в системе менторства.

## Описание

```lua
mentor.GetManagerInfo(): table|nil
```

Функция извлекает данные о текущих менторских взаимоотношениях главного игрока, определяя его роль в качестве ментора или ученика по соответствующим идентификаторам субъектов.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil`.

При успешном выполнении возвращается таблица со следующими полями:

- **`apprenticeMentorId`** (`ObjectId`) - Идентификатор субъекта, у которого главный игрок состоит учеником.

- **`mentorId`** (`ObjectId`) - Идентификатор субъекта, которым является главный игрок в роли ментора.
- **`avatarApprenticeId`** (`ObjectId`) - Идентификатор субъекта, которым является главный игрок в роли ученика.
- **`apprenticeFlag`** (`boolean`) - Значение `true`, если главный игрок выставил флаг желания быть учеником.
- **`mentorFlag`** (`boolean`) - Значение `true`, если главный игрок выставил флаг желания быть ментором.
- **`canBeMentor`** (`boolean`) - Значение `true`, если главный игрок может быть ментором.
- **`canBeApprentice`** (`boolean`) - Значение `true`, если главный игрок может быть учеником.

## Примеры

### Проверка роли ментора

```lua
local info = mentor.GetManagerInfo()
if info and info.mentorId then
    LogInfo( "Главный игрок является ментором." )
end
```
