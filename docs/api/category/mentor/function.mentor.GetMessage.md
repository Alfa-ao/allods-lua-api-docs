# mentor.GetMessage

Возвращает сообщение ментора, предназначенное для показа ученикам.

## Описание

```lua
mentor.GetMessage( mentorId: ObjectId ): WString
```

Функция извлекает и возвращает текстовое сообщение ментора, которое отображается ученикам.

## Список параметров

- **`mentorId`** (`ObjectId`) - Идентификатор ментора.

## Возвращаемые значения

Возвращает `WString` - Текстовое сообщение ментора.

## Примеры

### Получение сообщения ментора

```lua
local message = mentor.GetMessage( mentorId )
LogInfo( message )
```
