# mentor.GetPlayerInfo

Извлекает подробную информацию об игроке, являющемся субъектом менторских взаимоотношений.

## Описание

```lua
mentor.GetPlayerInfo( mentorPlayerId: ObjectId ): table | nil
```

Функция извлекает и возвращает таблицу с подробной информацией об игроке, являющемся субъектом менторских взаимоотношений. Если субъект не найден или произошла ошибка, функция возвращает `nil`.

## Список параметров

- **`mentorPlayerId`** (`ObjectId`) - Идентификатор субъекта менторских взаимоотношений.

## Возвращаемые значения

Возвращает `table` | `nil`.

Если информация успешно получена, возвращается таблица со следующими полями:

- **`id`** (`ObjectId`) - Идентификатор субъекта менторских взаимоотношений.

- **`isOnline`** (`boolean`) - `true`, если субъект находится в сети.

- **`playerId`** (`ObjectId` | `nil`) - Идентификатор игрока (интерактивного объекта), если он находится в сети. `nil` - если в оффлайне или вне репликации.

- **`mentorId`** (`ObjectId` | `nil`) - Идентификатор субъекта менторских взаимоотношений ментора, у которого данный субъект является учеником. Если `nil`, то субъект является ментором.

- **`canRefuse`** (`boolean`) - Флаг, указывающий, может ли главный игрок исключить данного субъекта из списка учеников.

- **`offlineInfo`** (`table`) - Оффлайновая информация о субъекте, доступная всегда.

    - **`name`** (`WString`) - Имя персонажа.

    - **`guild`** (`WString` | `nil`) - Название гильдии игрока, если она есть.

    - **`level`** (`number`) - Уровень персонажа.

    - **`zoneId`** (`ObjectId` | `nil`) - Идентификатор интерфейсной карты, если она доступна для этого игрока.

    - **`raceClass`** (`table` | `nil`) - Информация о расе и классе персонажа (см. `LuaRaceClassInfoPart`).

    - **`sex`** (`table` | `nil`) - Информация о поле персонажа (см. `LuaSexInfoPart`).

    - **`startTime`** (`table` | `nil`) - Информация о времени начала учителства или ученичества (см. `LuaFullDateTime`).

    - **`lastOnlineTime`** (`table` | `nil`) - Информация о времени последнего нахождения в сети (см. `LuaFullDateTime`).

## Примеры

### Получение информации о субъекте

```lua
local info = mentor.GetPlayerInfo( mentorPlayerId )
if info then
    if info.mentorId then
        LogInfo( "Это ученик" )
    end
    local name = info.playerId and object.GetName( info.playerId ) or info.offlineInfo.name
end
```

::: info Описание примера
В примере запрашивается информация о субъекте менторских взаимоотношений. Если таблица успешно получена, проверяется наличие идентификатора ментора для определения статуса ученика. Затем извлекается имя игрока: если объект в сети, используется функция `object.GetName`, в противном случае берется имя из блока оффлайновой информации.
:::
