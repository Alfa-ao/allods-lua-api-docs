# mentor.GetPlayerReward

Возвращает подробное описание наград, которые дает или уже дал ученик.

## Описание

```lua
mentor.GetPlayerReward( mentorPlayerId: ObjectId ): table | nil
```

Функция возвращает таблицу с подробным описанием наград, которые ученик передает ментору. Если субъект не является учеником или произошла ошибка, возвращается `nil`.

## Список параметров

- **`mentorPlayerId`** (`ObjectId`) - Идентификатор субъекта менторских взаимоотношений.

## Возвращаемые значения

Возвращает `table` | `nil`.

Если субъект является учеником, возвращается таблица со следующими полями:

- **`moneyPercent`** (`number`) - Процент заработанных денег, который ментор дополнительно получает от ученика.

- **`realMoney`** (`number`) - Процент заработанных денег, который ментор уже получил от ученика.

- **`items`** (`table`) - Индексированная с `0` таблица описаний предметов, определенный процент которых получает ментор.

  - `itemId` (`ObjectId`) - Идентификатор предмета.

  - `percent` (`number`) - Процент таких предметов, который получает ментор при получении предметов учеником.

  - `realCount` (`number`) - Количество таких предметов, которое ментор уже получил.

## Примеры

### Получение информации о наградах ученика

```lua
local info = mentor.GetPlayerReward( mentorPlayerId )
```
