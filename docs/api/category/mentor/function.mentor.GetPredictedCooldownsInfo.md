# mentor.GetPredictedCooldownsInfo

Возвращает предсказанные кулдауны на запрет действий в менторстве.

## Описание

```lua
mentor.GetPredictedCooldownsInfo(): table | nil
```

Функция возвращает информацию о предсказанных кулдаунах на запрет различных действий в системе менторства. Для каждого запрета указывается оставшееся количество миллисекунд от начала действия до окончания запрета. Если запрет отсутствует, для соответствующего действия возвращается значение `0`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil`. Если ошибка не произошла, возвращается таблица со следующими полями:

- **`mentorRefuseRemainingMs`** (`number`) - Количество миллисекунд, оставшихся до окончания запрета на отказ от ментора.

- **`apprenticeRefuseRemainingMs`** (`number`) - Количество миллисекунд, оставшихся до окончания запрета на отказ от ученика.

- **`mentorFlagRemainingMs`** (`number`) - Количество миллисекунд, оставшихся до окончания запрета на сброс флага ментора.

- **`apprenticeFlagRemainingMs`** (`number`) - Количество миллисекунд, оставшихся до окончания запрета на сброс флага ученика.

## Примеры

### Получение информации о кулдаунах менторства

```lua
local info = mentor.GetPredictedCooldownsInfo()
LogInfo( info.mentorRefuseRemainingMs )
```
