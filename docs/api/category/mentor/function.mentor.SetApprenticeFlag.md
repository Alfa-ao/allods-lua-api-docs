# mentor.SetApprenticeFlag

Поднимает у главного игрока флаг желания стать учеником.

## Описание

```lua
mentor.SetApprenticeFlag( flag: boolean )
```

Функция поднимает у главного игрока флаг желания стать учеником.

## Список параметров

- **`flag`** (`boolean`) - Желаемое значение флага.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка флага желания стать учеником

```lua
mentor.SetApprenticeFlag( true )
```
