# mentor.SetMentorFlag

Поднимает у главного игрока флаг желания стать ментором.

## Описание

```lua
mentor.SetMentorFlag( flag: boolean )
```

Функция устанавливает флаг желания стать ментором для главного игрока.

## Список параметров

- **`flag`** (`boolean`) - Желаемое значение флага.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка флага желания стать ментором

```lua
mentor.SetMentorFlag( true )
```
