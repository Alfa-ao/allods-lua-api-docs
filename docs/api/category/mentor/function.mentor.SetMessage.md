# mentor.SetMessage

Изменяет сообщение ментора для отображения ученикам.

## Описание

```lua
mentor.SetMessage( message: WString )
```

Функция обновляет текстовое сообщение ментора. Изменение применяется только при условии, что главный игрок является ментором.

## Список параметров

- **`message`** (`WString`) - Текстовое сообщение, отображаемое ученикам.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка сообщения из поля ввода

```lua
mentor.SetMessage( wtEditLine:GetText() )
```
