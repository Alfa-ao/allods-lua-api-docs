# mission.CancelLogout

Отменяет завершение сессии после вызова `mission.Logout`.

## Описание

```lua
mission.CancelLogout()
```

Функция отменяет завершение сессии, если ранее был инициирован выход посредством `mission.Logout`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Отмена завершения сессии

```lua
mission.CancelLogout()
```
