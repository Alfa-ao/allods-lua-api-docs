# mission.GetCameraDirection

Возвращает величину поворота (yaw) камеры в радианах.

## Описание

```lua
mission.GetCameraDirection(): number
```

Функция возвращает величину поворота (yaw) камеры в радианах.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - величину поворота в радианах.

## Примеры

### Получение направления камеры

```lua
local yaw = mission.GetCameraDirection()
```
