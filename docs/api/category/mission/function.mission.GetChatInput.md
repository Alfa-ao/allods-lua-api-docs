# mission.GetChatInput

Возвращает состояние строки ввода с префиксами и слеш-командами.

## Описание

```lua
mission.GetChatInput(): table
```

Функция возвращает таблицу с состоянием строки ввода, полученной в результате разбора исходного текста на наличие слеш-команд.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - таблица со следующими полями:

- **`sysCmdType`** (`string`) - тип команды или ее идентификатор.

- **`hasPrefix`** (`boolean`) - наличие префикса и слеш-команды.

- **`cmdName`** (`WString`) - имя активной слеш-команды в случае наличия префикса.

- **`cmdArg`** (`WString`) - аргумент активной слеш-команды или пустая строка.

- **`text`** (`WString`) - текст, оставшийся после разбора и выделения слеш-команд.

## Примеры

### Обработка строки ввода

```lua
local input = mission.GetChatInput()

wtEditLine:SetText( input.text )

if input.hasPrefix then
    -- отобразить строку ввода с учетом префиксов слеш-команды
else
    -- отобразить строку ввода без префиксов
end
```
