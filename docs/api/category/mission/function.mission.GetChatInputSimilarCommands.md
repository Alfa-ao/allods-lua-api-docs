# mission.GetChatInputSimilarCommands

Возвращает список команд, являющихся подходящими вариантами продолжения для переданной строки.

## Описание

```lua
mission.GetChatInputSimilarCommands( text: WString ): table
```

Функция возвращает список команд, которые подходят в качестве вариантов продолжения для указанной строки.

## Список параметров

- **`text`** (`WString`) - Строка, для которой необходимо вернуть похожие команды.

## Возвращаемые значения

Возвращает `table` - таблица, индексированная от `0`, содержащая таблицы со следующими полями:

- **`alias`** (`WString`) - Локализованный алиас команды.

- **`sysId`** (`string`) - Системный идентификатор команды.

## Примеры

### Получение и вывод похожих команд

```lua
local cmds = mission.GetChatInputSimilarCommands( inputText )
for i = 0, GetTableSize( cmds ) - 1 do
    local cmd = cmds[i]
    LogInfo( " ", i, ": sysId = ", cmd.sysId, ", alias = ", cmd.alias )
end
```

::: info Описание примера
В примере выполняется запрос списка похожих команд для введенного текста. Далее в цикле происходит итерация по полученной таблице с выводом системного идентификатора и алиаса каждой команды в лог.
:::
