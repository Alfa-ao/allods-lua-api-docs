# mission.GetGlobalDateTime

Возвращает серверное время в формате `LuaFullDateTime` для вычисления дельты глобальных событий.

## Описание

```lua
mission.GetGlobalDateTime(): table
```

Функция возвращает серверное время, пересчитанное в часовой пояс клиента, в формате `LuaFullDateTime`, от которого можно считать дельту для глобальных событий.

::: warning Замечание
Возвращаемое время является серверным, пересчитанным в часовой пояс клиента.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - таблица в формате `LuaFullDateTime`.

## Примеры

### Вывод глобального времени в миллисекундах

```lua
LogInfo( mission.GetGlobalDateTime().overallMs )
```
