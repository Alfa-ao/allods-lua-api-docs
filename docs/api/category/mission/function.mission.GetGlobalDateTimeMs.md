# mission.GetGlobalDateTimeMs

Возвращает текущее глобальное время в формате Unix time для вычисления дельты глобальных событий.

## Описание

```lua
mission.GetGlobalDateTimeMs(): number
```

Функция возвращает общее время в миллисекундах, прошедшее с 1-го января 1970 года (Unix time), от которого производится отсчёт дельты для глобальных событий.

::: warning Замечание
Возвращаемое значение является серверным временем, пересчитанным в часовой пояс клиента.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - общее время в миллисекундах, прошедшее с 1-го января 1970 года (Unix time, int64).

## Примеры

### Получение текущего глобального времени

```lua
LogInfo( mission.GetGlobalDateTimeMs() )
```
