# mission.GetLotteryHistory
Возвращает информацию о победителях в лотерее "Лото".

## Описание
```lua
mission.GetLotteryHistory(): table|nil
```
Функция возвращает информацию о победителях в лотерее "Лото". Если произошла ошибка, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` | `nil`.

Если информация успешно получена, возвращается таблица со следующими полями:

- **`avatar`** (`WString`) - Имя победителя.

- **`lotteryTime`** (`number`) - Начало розыгрыша, в UTC.

- **`prize`** (`ObjectId`) - Полученный приз.

- **`state`** (`number`) - Состояние лотереи.

- **`lotteryResource`** (`table`) - Общая информация о лотерее.

  - **`maxWinners`** (`number`) - Количество разыгрываемых призов.

  - **`prizes`** (`table`) - Список возможных призов, каждый элемент является `ObjectId` предмета-приза.

  - **`ticketCounter`** (`CurrencyId`) - Идентификатор альтернативной валюты-счётчика для числа активированных билетов.

  - **`ticketResource`** (`ObjectId`) - Идентификатор предмета-лотерейного билета.
