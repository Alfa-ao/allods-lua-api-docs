# mission.GetMaxCharacterSceneCount

Возвращает максимально допустимое количество сцен для отображения персонажей.

## Описание

```lua
mission.GetMaxCharacterSceneCount(): number
```

Функция возвращает максимально допустимое количество сцен для отображения персонажей.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - максимально допустимое число сцен.

## Примеры

### Получение максимального количества сцен

```lua
local count = mission.GetMaxCharacterSceneCount()
```
