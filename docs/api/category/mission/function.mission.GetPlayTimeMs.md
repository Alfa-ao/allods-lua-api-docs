# mission.GetPlayTimeMs

Возвращает время в миллисекундах, прошедшее с начала текущей игровой сессии.

## Описание

```lua
mission.GetPlayTimeMs(): number
```

Функция возвращает количество миллисекунд, прошедших с начала текущей игры.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - количество миллисекунд, прошедших с начала игры.

## Примеры

### Получение времени игры

```lua
local timeMs = mission.GetPlayTimeMs()
LogInfo( "playing (sec): ", timeMs / 1000 )
```
