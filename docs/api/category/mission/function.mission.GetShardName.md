# mission.GetShardName
Возвращает имя шарда, на котором происходит игра.

## Описание
```lua
mission.GetShardName(): string
```
Функция возвращает локализованное имя шарда, на котором происходит игра.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `string` - локализованное имя шарда.

## Примеры
### Получение имени текущего шарда
```lua
LogInfo( "shardName: ", mission.GetShardName() )
```
