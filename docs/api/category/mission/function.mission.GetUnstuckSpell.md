# mission.GetUnstuckSpell

Возвращает идентификатор заклинания извлечения главного персонажа, если он застрял и данная возможность доступна.

## Описание

```lua
mission.GetUnstuckSpell(): SpellId | nil
```

Функция возвращает идентификатор ресурса заклинания извлечения главного персонажа, если персонаж застрял и такая возможность доступна.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `SpellId` | `nil` - идентификатор ресурса заклинания, если возможность вытаскивания доступна.

## Примеры

### Извлечение застрявшего персонажа

```lua
local spellId = mission.GetUnstuckSpell()
if spellId then
    avatar.RunSpell( spellId )
end
```

::: info Описание примера
В примере запрашивается идентификатор заклинания извлечения. Если заклинание доступно, оно запускается для главного персонажа посредством вызова `avatar.RunSpell`.
:::
