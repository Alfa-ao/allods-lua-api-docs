# mission.GetWeakFactionBonus

Возвращает информацию о бонусах более слабой фракции на шарде.

## Описание

```lua
mission.GetWeakFactionBonus(): table | nil
```

Функция возвращает информацию о бонусах более слабой фракции на шарде в виде таблицы или `nil` в случае возникновения ошибки.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - таблица с информацией о бонусах более слабой фракции или `nil` при ошибке.

- **`experienceFactor`** (`number`) - множитель, во сколько раз приходит больше опыта.

## Примеры

### Получение бонуса слабой фракции

```lua
local weakFactionBonus = mission.GetWeakFactionBonus()
```
