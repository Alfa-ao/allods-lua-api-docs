# mission.GetWeakFactionId

Возвращает идентификатор более слабой фракции на шарде, получающей дополнительные бонусы.

## Описание

```lua
mission.GetWeakFactionId(): FactionId | nil
```

Функция возвращает идентификатор более слабой фракции на шарде, если на сервере присутствует соответствующая информация.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `FactionId` | `nil` - идентификатор более слабой фракции на шарде, если информация доступна на сервере, или `nil`, если информация отсутствует.

## Примеры

### Получение названия слабой фракции

```lua
local weakFactionId = mission.GetWeakFactionId()
if weakFactionId then
    LogInfo( weakFactionId:GetInfo().name )
end
```
