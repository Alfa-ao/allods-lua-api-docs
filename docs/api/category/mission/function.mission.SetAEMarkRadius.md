# mission.SetAEMarkRadius

Устанавливает радиус AEMark.

## Описание

```lua
mission.SetAEMarkRadius( radius: number )
```

Функция задает радиус AEMark в метрах.

## Список параметров

- **`radius`** (`number`) - Радиус AEMark в метрах.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка радиуса

```lua
mission.SetAEMarkRadius( 10.0 )
```
