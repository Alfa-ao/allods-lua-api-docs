# mission.SetAEMarkRange

Устанавливает радиус области действия спела AEMark.

## Описание

```lua
mission.SetAEMarkRange( range: number )
```

Функция устанавливает радиус области действия спела AEMark в метрах.

## Список параметров

- **`range`** (`number`) - Радиус области действия спела AEMark в метрах.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка радиуса действия

```lua
mission.SetAEMarkRange( 5.0 )
```
