# mission.SetChatInputArg

Задает аргумент слеш-команды для строки ввода.

## Описание

```lua
mission.SetChatInputArg( arg: WString )
```

Функция задает аргумент слеш-команды для строки ввода.

## Список параметров

- **`arg`** (`WString`) - Аргумент слеш-команды.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка аргумента слеш-команды

```lua
mission.SetChatInputArg( object.GetName( playerId ) )
```
