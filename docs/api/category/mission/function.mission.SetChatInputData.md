# mission.SetChatInputData

Меняет состояние строки ввода чата с префиксами и слеш-командами.

## Описание

```lua
mission.SetChatInputData( wtEditLine: EditLineSafe )
```

Функция разбирает исходную строку на предмет наличия слеш-команд, текста и информации о предметах, изменяя состояние строки ввода чата.

## Список параметров

- **`wtEditLine`** (`EditLineSafe`) - Строка ввода чата.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Изменение состояния строки ввода чата

```lua
mission.SetChatInputData( wtEditLine )
```
