# mission.SetChatInputHistoryMaxSize

Устанавливает максимальное количество записей в истории ввода в строку чата.

## Описание

```lua
mission.SetChatInputHistoryMaxSize( limit: number )
```

Функция задает максимальное количество записей в истории ввода в строку чата. Начальное значение равно 10, предельное составляет 100.

## Список параметров

- **`limit`** (`number`) - Максимальное количество записей.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка лимита истории чата

```lua
mission.SetChatInputHistoryMaxSize( 20 )
```
