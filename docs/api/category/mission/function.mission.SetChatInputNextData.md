# mission.SetChatInputNextData
Делает переход на шаг вперед при обзоре истории ввода и возвращает строку ввода чата в предыдущее состояние.

## Описание
```lua
mission.SetChatInputNextData( wtEditLine: EditLineSafe )
```
Функция выполняет переход на шаг вперед при обзоре истории ввода и возвращает строку ввода чата в предыдущее состояние.

## Список параметров
- **`wtEditLine`** (`EditLineSafe`) - Строка ввода.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Переход по истории ввода
```lua
mission.SetChatInputNextData( wtEditLine )
```
