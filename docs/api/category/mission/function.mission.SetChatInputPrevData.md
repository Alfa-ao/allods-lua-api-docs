# mission.SetChatInputPrevData

Выполняет переход на шаг назад по истории ввода и восстанавливает предыдущее состояние строки ввода чата.

## Описание

```lua
mission.SetChatInputPrevData( wtEditLine: EditLineSafe )
```

Функция выполняет переход на шаг назад по истории ввода и восстанавливает предыдущее состояние строки ввода чата.

## Список параметров

- **`wtEditLine`** (`EditLineSafe`) - Строка ввода чата.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Переход по истории ввода

```lua
mission.SetChatInputPrevData( wtEditLine )
```
