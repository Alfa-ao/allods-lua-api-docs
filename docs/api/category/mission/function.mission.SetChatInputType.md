# mission.SetChatInputType

Задает тип слеш-команды для строки ввода.

## Описание

```lua
mission.SetChatInputType( sysCmdType: string )
```

Функция задает тип слеш-команды для строки ввода. Метод работает только для команд с префиксами, при этом текст слеш-команды очищается.

## Список параметров

- **`sysCmdType`** (`string`) - тип команды (ее идентификатор).

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка типа команды

```lua
mission.SetChatInputType( "party" )
```
