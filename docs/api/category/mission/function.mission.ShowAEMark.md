# mission.ShowAEMark
Показывает или скрывает AEMark типа AEMarkDecal.

## Описание

```lua
mission.ShowAEMark( show: boolean )
```

Функция показывает или скрывает AEMark типа AEMarkDecal.

## Список параметров

- **`show`** (`boolean`) - Показывает или скрывает AEMark типа AEMarkDecal.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Показ AEMark

```lua
mission.ShowAEMark( true )
```
