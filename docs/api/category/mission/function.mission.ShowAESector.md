# mission.ShowAESector

Показывает или скрывает маркер зоны типа AEMarkSector.

## Описание

```lua
mission.ShowAESector( show: boolean )
```

Функция переключает видимость маркера зоны (AEMark) типа AEMarkSector в зависимости от переданного логического значения.

## Список параметров

- **`show`** (`boolean`) - Логический флаг, определяющий видимость маркера. Значение `true` отображает AEMark типа AEMarkSector, значение `false` скрывает его.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Отображение маркера зоны

```lua
mission.ShowAESector( true )
```
