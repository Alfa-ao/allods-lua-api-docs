# mount.Dismount
Отзывает вызванное ездовое животное главного игрока в конюшню, заставляя персонажа слезть с него.

## Описание
```lua
mount.Dismount()
```
Функция инициирует процесс отзыва ездового животного главного игрока в конюшню. В результате выполнения персонаж спешивается, а транспортное средство возвращается в конюшню.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Спешивание и отзыв ездового животного
```lua
mount.Dismount()
```
