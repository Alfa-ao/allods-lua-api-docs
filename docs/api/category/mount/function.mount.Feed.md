# mount.Feed
Кормит ездовое животное из конюшни главного игрока указанной порцией корма.

## Описание
```lua
mount.Feed( mountId: ObjectId, amount: number | nil )
```
Функция инициирует процесс кормления ездового животного, находящегося в конюшне главного игрока, передавая заданное количество порций корма. Если параметр количества корма не передан или имеет значение `nil`, используется значение по умолчанию, равное одной порции.

::: warning Замечание
Операция применяется исключительно к животным, зарегистрированным в конюшне главного игрока.
:::

## Список параметров
- **`mountId`** (`ObjectId`)
Уникальный идентификатор ездового животного в конюшне.
- **`amount`** (`number` | `nil`)
Количество порций корма, передаваемых животному. При передаче значения `nil` или отсутствии параметра используется значение по умолчанию, равное 1.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Кормление животного одной порцией по умолчанию
```lua
local mountId = 12345 -- Идентификатор животного в конюшне
mount.Feed( mountId )
```
