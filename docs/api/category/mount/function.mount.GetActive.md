# mount.GetActive
Возвращает идентификатор активного ездового животного из конюшни главного игрока.

## Описание
```lua
mount.GetActive(): ObjectId | nil
```
Функция возвращает идентификатор ездового животного, которое в данный момент вызвано из конюшни главного игрока. Если ни одно животное не вызвано, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `ObjectId` — идентификатор ездового животного, вызванного из конюшни главного игрока, или `nil`, если животное не вызвано.

## Примеры
### Получение информации о вызванном животном
```lua
local activeId = mount.GetActive()
if activeId then
    local mountInfo = mount.GetInfo( activeId )
end
```
::: info Описание примера
В примере запрашивается идентификатор текущего вызванного животного. Если животное активно, для него извлекается полная информация через функцию `mount.GetInfo`.
:::
