# mount.GetActiveContextActions
Возвращает текущий список контекстных действий ездового животного главного игрока.

## Описание
```lua
mount.GetActiveContextActions(): table | nil
```
Функция возвращает таблицу, содержащую идентификаторы экземпляров контекстных действий ездового животного главного игрока, или `nil`, если список действий пуст. Элементы возвращаемой таблицы представляют собой список идентификаторов, индексация которого начинается с единицы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если список контекстных действий не пуст, возвращается таблица (`table`), содержащая идентификаторы (`ObjectId`) экземпляров контекстных действий. Индексация элементов в таблице начинается с единицы.
Если список пуст или контекстные действия отсутствуют, возвращается `nil`.

## Примеры
### Получение списка контекстных действий
```lua
local contextActions = mount.GetActiveContextActions()
```
