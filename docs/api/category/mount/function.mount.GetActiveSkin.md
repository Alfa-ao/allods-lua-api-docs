# mount.GetActiveSkin
Извлекает идентификатор текущего скина ездового животного главного персонажа.

## Описание
```lua
mount.GetActiveSkin(): ObjectId | nil
```
Функция возвращает идентификатор скина, который в данный момент активен на ездовом животном главного персонажа. Если у персонажа нет активного ездового животного или скин не задан, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
`ObjectId | nil` — уникальный идентификатор активного скина ездового животного. Возвращает `nil`, если ездовое животное не призвано или скин отсутствует.

## Примеры
### Получение информации об активном скине
```lua
local skinId = mount.GetActiveSkin()
if skinId then
    local skinInfo = mount.GetSkinInfo( skinId )
end
```
::: info Описание примера
В примере извлекается идентификатор активного скина с помощью `mount.GetActiveSkin`. При условии, что идентификатор успешно получен, вызывается функция `mount.GetSkinInfo` для получения подробной информации о данном скине.
:::
