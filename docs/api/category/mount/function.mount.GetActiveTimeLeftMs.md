# mount.GetActiveTimeLeftMs
Возвращает оставшееся время в миллисекундах до исчезновения вызванного ездового животного главного игрока из конюшни.

## Описание
```lua
mount.GetActiveTimeLeftMs(): number | nil
```
Функция извлекает и возвращает оставшееся время в миллисекундах до момента исчезновения вызванного ездового животного главного игрока из конюшни. Если у животного отсутствует срок жизни или оно не вызвано, функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — время в миллисекундах до исчезновения животного из конюшни, если у него имеется срок жизни.
Возвращает `nil`, если у животного отсутствует срок жизни или оно не вызвано.

## Примеры
### Получение и вывод оставшегося времени жизни животного
```lua
local timeLeftMs = mount.GetActiveTimeLeftMs()
if timeLeftMs then
  LogInfo( "Осталось ", timeLeftMs / 1000, " секунд" )
end
```
