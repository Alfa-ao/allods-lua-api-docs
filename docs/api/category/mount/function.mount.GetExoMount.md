# mount.GetExoMount
Возвращает идентификатор имеющегося экзоскелета.

## Описание
```lua
mount.GetExoMount(): ObjectId | nil
```
Функция возвращает уникальный идентификатор текущего экзоскелета, находящегося в распоряжении персонажа, или `nil`, если экзоскелет отсутствует.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `ObjectId` — уникальный идентификатор экзоскелета, или `nil`, если экзоскелет отсутствует.

## Примеры
### Получение идентификатора и талантов экзоскелета
```lua
local exoMountId = mount.GetExoMount()
local currentTalents = mount.GetSelectedTalents( exoMountId )
```
::: info Описание примера
В примере извлекается идентификатор текущего экзоскелета с помощью `mount.GetExoMount`. Затем полученный идентификатор передается в функцию `mount.GetSelectedTalents` для получения списка выбранных талантов.
:::
