# mount.GetMetamorph
Возвращает идентификатор ездового животного метаморфа из конюшни главного игрока.

## Описание
```lua
mount.GetMetamorph(): ObjectId | nil
```
Функция возвращает идентификатор ездового животного метаморфа из конюшни главного игрока. Если метаморф отсутствует, возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
- **`ObjectId`** — идентификатор ездового животного метаморфа, если он присутствует.
- **`nil`** — если метаморф отсутствует.

## Примеры
### Получение и обработка информации о метаморфе
```lua
local metamorphId = mount.GetMetamorph()

if metamorphId then
    local mountInfo = mount.GetInfo(metamorphId)
    -- Работа с полученной информацией о маунте
end
```
::: info Описание примера
В примере сначала запрашивается идентификатор метаморфа с помощью `mount.GetMetamorph`. Если идентификатор успешно получен, извлекается подробная информация о маунте через `mount.GetInfo`.
:::
