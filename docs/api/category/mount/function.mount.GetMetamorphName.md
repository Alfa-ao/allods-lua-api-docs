# mount.GetMetamorphName
Возвращает основное название ездового животного метаморфа из конюшни главного игрока, без имени его текущего облика.

## Описание
```lua
mount.GetMetamorphName(): WString | nil
```
Функция извлекает и возвращает базовое имя ездового животного метаморфа, находящегося в конюшне главного игрока. Возвращаемое значение представляет собой оригинальное название существа и не включает имя его текущего визуального облика (формы). Если у главного игрока отсутствует ездовое животное метаморф, функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `WString` — основное имя ездового животного метаморфа, если он присутствует в конюшне.
Возвращает `nil` — если метаморф у главного игрока отсутствует.

## Примеры
### Получение названия метаморфа
```lua
local metamorphName = mount.GetMetamorphName()
```
