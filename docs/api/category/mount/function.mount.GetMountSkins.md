# mount.GetMountSkins
Возвращает список идентификаторов скинов ездового животного.

## Описание
```lua
mount.GetMountSkins( mountId: ObjectId ): table
```
Функция извлекает и возвращает индексированный с нуля список идентификаторов скинов для указанного ездового животного.

## Список параметров
- **`mountId`** (`ObjectId`)
Уникальный идентификатор ездового животного, для которого необходимо получить список доступных скинов.

## Возвращаемые значения
Возвращает таблицу (`table`), содержащую идентификаторы (`ObjectId`) скинов. Индексация элементов в таблице начинается с нуля.

## Примеры
### Получение и проверка первого доступного скина
```lua
local skins = mount.GetMountSkins( mountInfo.id )
if skins[ 0 ] then
  local skinInfo = mount.GetSkinInfo( skins[ 0 ] )
end
```
::: info Описание примера
В примере запрашивается список скинов для ездового животного. Если список не пуст и содержит хотя бы один элемент (индекс `0`), извлекается подробная информация о первом скине с помощью функции `mount.GetSkinInfo`.
:::
