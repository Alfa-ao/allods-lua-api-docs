# mount.GetMounts
Извлекает список ездовых животных, имеющихся в конюшне главного игрока.

## Описание
```lua
mount.GetMounts(): table
```
Функция возвращает таблицу, содержащую идентификаторы ездовых животных, которые находятся в конюшне главного игрока. Индексация элементов в возвращаемой таблице начинается с нуля.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table`, содержащую идентификаторы (`ObjectId`) ездовых животных. Индексация элементов в таблице начинается с нуля.

## Примеры
### Проверка заполненности конюшни
```lua
local mounts = mount.GetMounts()
local stableInfo = mount.GetStableInfo()
if stableInfo then
    local isFull = ( stableInfo.slotsCount <= GetTableSize( mounts ) )
end
```
::: info Описание примера
В примере извлекается список доступных ездовых животных и запрашивается информация о конюшне. Если данные о конюшне успешно получены, вычисляется признак заполненности конюшни путем сравнения количества доступных слотов с фактическим количеством животных в списке.
:::
