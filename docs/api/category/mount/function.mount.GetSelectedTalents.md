# mount.GetSelectedTalents
Возвращает текущий список выбранных талантов маунта.

## Описание
```lua
mount.GetSelectedTalents( mountId: ObjectId ): table | nil
```
Функция извлекает и возвращает таблицу с идентификаторами талантов, которые в данный момент выбраны для указанного маунта. Если у маунта нет выбранных талантов или он не найден, возвращается `nil`. Выбор талантов осуществляется с помощью функции `mount.SelectTalents`. Изменение списка выбранных талантов сопровождается передачей события `EVENT_STABLE_MOUNT_TALENTS_CHANGED`.

## Список параметров
- **`mountId`** (`ObjectId`)
Уникальный идентификатор маунта, для которого необходимо получить список талантов.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если таланты найдены, возвращается таблица, содержащая идентификаторы (`MountTalentId`) выбранных талантов маунта. Индексация элементов в таблице начинается с нуля. Если у маунта нет выбранных талантов или он не существует, возвращается `nil`.

## Примеры
### Получение списка талантов экзо-маунта
```lua
local exoMountId = mount.GetExoMount()
local currentTalents = mount.GetSelectedTalents( exoMountId )
```
::: info Описание примера
В примере сначала извлекается идентификатор текущего экзо-маунта с помощью функции `mount.GetExoMount`. Затем для полученного идентификатора вызывается функция `mount.GetSelectedTalents`, которая возвращает таблицу с идентификаторами выбранных талантов.
:::
