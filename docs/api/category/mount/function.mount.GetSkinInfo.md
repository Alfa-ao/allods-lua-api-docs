# mount.GetSkinInfo
Извлекает и возвращает информацию о скине ездового животного по его идентификатору.

## Описание
```lua
mount.GetSkinInfo( skinId: ObjectId ): table | nil
```
Функция формирует и возвращает таблицу с детальными характеристиками указанного скина ездового животного. Если скин с заданным идентификатором не найден или не существует, функция возвращает `nil`.

::: warning Замечание
Расширенная реализация функции, включающая дополнительные поля (например, `image`), доступна только в контексте пользовательского интерфейса (UI). В базовой логике игры (GameState) эти поля могут отсутствовать.
:::

## Список параметров
- **`skinId`** (`ObjectId`)
Уникальный идентификатор скина ездового животного, информацию о котором необходимо получить.

## Возвращаемые значения
Возвращает `table` или `nil`.

Если скин найден, возвращается таблица со следующими полями:
- **`id`** (`ObjectId`) — уникальный идентификатор скина.
- **`mountId`** (`ObjectId`) — идентификатор ездового животного (маунта), которому принадлежит данный скин.
- **`name`** (`WString`) — название скина.
- **`description`** (`WString`) — текстовое описание скина.
- **`sourceDescription`** (`WString`) — описание источника получения скина (информация о том, как разблокировать данный скин).
- **`enabled`** (`boolean`) — флаг, указывающий, доступен ли этот скин для использования в текущий момент.
- **`debugName`** (`string` | `nil`) — путь к файлу описания предмета (отладочная информация).
- **`image`** (`TextureId`) — идентификатор текстуры для отображения иконки скина (поле доступно исключительно в реализации для UI).

Возвращает `nil`, если скин с указанным идентификатором не существует.

## Примеры
### Получение информации о первом доступном скине
```lua
local mountId = 12345 -- Идентификатор ездового животного
local skins = mount.GetMountSkins( mountId )
if skins and skins[ 0 ] then
    local skinInfo = mount.GetSkinInfo( skins[ 0 ] )
    if skinInfo then
        LogInfo( "Название скина: ", skinInfo.name )
    end
end
```
::: info Описание примера
В примере сначала извлекается список всех скинов для конкретного ездового животного с помощью `mount.GetMountSkins`. Если список не пуст и содержит хотя бы один элемент (с индексом `0`), вызывается функция `mount.GetSkinInfo` для получения детальной информации о первом скине. При успешном получении данных в лог выводится название скина.
:::
