# mount.GetStableInfo
Возвращает информацию о конюшне главного игрока.
## Описание
```lua
mount.GetStableInfo(): table | nil
```
Функция извлекает и возвращает информацию о конюшне главного игрока. При наличии доступа к конюшне возвращается таблица с подробными параметрами, такими как вместимость, количество еды, лимиты опыта и характеристики насыщения. В случае отсутствия конюшни или невозможности получения данных возвращается `nil`.

## Список параметров
Функция не принимает параметров.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если конюшня доступна, возвращается таблица со следующими полями:
- **`singleFeedSatiationTime`** (`number`)
Количество миллисекунд, на которое насыщается животное за одно кормление.
- **`maxSatiationTime`** (`number`)
Максимальное количество миллисекунд, на которое может быть насыщено животное.
- **`maxLevel`** (`number`)
Максимальный уровень животного.
- **`maxGrade`** (`number`)
Максимальный грейд (ранг) животного.
- **`slotsCount`** (`number`)
Общее количество слотов в конюшне.
- **`foodCount`** (`number`)
Текущее количество имеющейся еды.
- **`maxExperienceToday`** (`number`)
Максимально возможное количество опыта, которое можно получить за текущий день.

Возвращает `nil`, если информация о конюшне недоступна.

## Примеры
### Проверка заполненности конюшни
```lua
local mounts = mount.GetMounts()
local stableInfo = mount.GetStableInfo()
if stableInfo then
    local isFull = ( stableInfo.slotsCount <= GetTableSize( mounts ) )
end
```
::: info Описание примера
В примере сначала извлекается список текущих верховых животных с помощью `mount.GetMounts()`, а затем запрашивается информация о конюшне через `mount.GetStableInfo()`. При успешном получении данных о конюшне вычисляется логическое значение `isFull`, которое принимает значение `true`, если количество занятых слотов (размер таблицы `mounts`) больше или равно общему количеству доступных слотов (`slotsCount`).
:::
