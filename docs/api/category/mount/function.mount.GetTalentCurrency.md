# mount.GetTalentCurrency
Возвращает идентификатор ресурса альтернативной валюты талантов маунта.

## Описание
```lua
mount.GetTalentCurrency(): CurrencyId | nil
```
Функция возвращает идентификатор ресурса альтернативной валюты, которая используется для изучения талантов маунта. Если для маунта не предусмотрена альтернативная валюта или она не задана, функция возвращает `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `CurrencyId` — идентификатор ресурса альтернативной валюты, или `nil`, если альтернативная валюта не определена.

## Примеры
### Получение идентификатора альтернативной валюты
```lua
local talentCurrency = mount.GetTalentCurrency()
```
