# mount.GetUnitMountHealth
Возвращает информацию о здоровье используемого ездового животного игрока.

## Описание
```lua
mount.GetUnitMountHealth( unitId: ObjectId ): table | nil
```
Функция извлекает и возвращает данные о текущем состоянии здоровья ездового животного, привязанного к указанному юниту. Если у юнита отсутствует ездовое животное или данные недоступны, функция возвращает `nil`.

## Список параметров
- **`unitId`** (`ObjectId`)
Уникальный идентификатор юнита, для которого запрашивается информация о здоровье ездового животного.

## Возвращаемые значения
Возвращает `table` или `nil`.
Если у юнита присутствует ездовое животное, возвращается таблица со следующими полями:
- **`health`** (`number`)
Текущее значение здоровья ездового животного.
- **`healthLimit`** (`number`)
Максимально возможное значение здоровья ездового животного.

## Примеры
### Получение и расчет относительного уровня здоровья ездового животного
```lua
local mountInfo = mount.GetUnitMountHealth( unitId )
if mountInfo then
    local healthLevel = mountInfo.health / mountInfo.healthLimit
end
```
