# mount.GetValuedObject
Возвращает новый экземпляр `ValuedObject` по идентификатору маунта.

## Описание
```lua
mount.GetValuedObject( mountId: ObjectId ): ValuedObject
```
Функция формирует и возвращает новый экземпляр объекта `ValuedObject`, представляющий средство передвижения (маунт). Тип возвращаемого объекта устанавливается как `VAL_OBJ_TYPE_MOUNT`. Текстовое представление содержит имя маунта, а в качестве изображения выступает его иконка. Идентификатор (`Id`) объекта представляет собой строку (`WString`) с именем маунта.

::: warning Замечание
Поскольку `ValuedObject` является `userdata`, среда Lua взаимодействует с ним по ссылке, а не по значению.
:::

## Список параметров
- **`mountId`** (`ObjectId`)
Уникальный идентификатор средства передвижения (маунта).

## Возвращаемые значения
`ValuedObject` — новый экземпляр объекта `ValuedObject`, представляющий маунт.

## Примеры
### Получение объекта маунта
```lua
local valuedObject = mount.GetValuedObject( mountId )
```
