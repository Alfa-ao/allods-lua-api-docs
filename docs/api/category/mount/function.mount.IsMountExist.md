# mount.IsMountExist
Проверяет наличие указанного ездового животного в конюшне главного игрока.

## Описание
```lua
mount.IsMountExist( mountId: ObjectId ): boolean
```
Функция проверяет, присутствует ли ездовое животное с заданным идентификатором в конюшне главного игрока. Возвращает логическое значение, указывающее на наличие или отсутствие указанного маунта.

## Список параметров
- **`mountId`** (`ObjectId`)
Уникальный идентификатор ездового животного, наличие которого необходимо проверить. Значение не может быть `nil`.

## Возвращаемые значения
Возвращает `boolean` — `true`, если указанное ездовое животное присутствует в конюшне главного игрока, и `false` в противном случае.

## Примеры
### Проверка наличия маунта и получение информации
```lua
if mount.IsMountExist( mountId ) then
    local mountInfo = mount.GetInfo( mountId )
end
```
::: info Описание примера
В примере сначала проверяется наличие ездового животного с заданным идентификатором в конюшне. Если животное существует, извлекается полная информация о нем с помощью функции `mount.GetInfo`.
:::
