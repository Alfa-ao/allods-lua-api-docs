# mount.Ride
Вызывает ездовое животное из конюшни главного игрока.

## Описание
```lua
mount.Ride( skinId: ObjectId )
```
Функция инициирует вызов ездового животного из конюшни главного игрока, используя заданный идентификатор скина.

## Список параметров
- **`skinId`** (`ObjectId`)
Идентификатор скина ездового животного, находящегося в конюшне.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Вызов ездового животного из конюшни
```lua
local skinId = 12345 -- Идентификатор скина животного
mount.Ride( skinId )
```
