# mount.SetMetamorphSkin
Изменяет скин ездового животного метаморфа главного игрока.

## Описание
```lua
mount.SetMetamorphSkin( skinId: ObjectId )
```
Функция изменяет скин ездового животного метаморфа главного игрока. Указанный скин должен быть доступен для использования и предварительно разблокирован в конюшне персонажа.

## Список параметров
- **`skinId`** (`ObjectId`)
Идентификатор скина ездового животного из конюшни главного игрока.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Изменение скина метаморфа
```lua
local skinId = 12345
mount.SetMetamorphSkin( skinId )
```
