# mount.SetSkin
Изменяет скин активного ездового животного главного игрока.

## Описание
```lua
mount.SetSkin( skinId: ObjectId )
```
Функция изменяет скин активного ездового животного главного игрока. Если передан идентификатор скина неактивного животного или уже используемого скина, функция вызывает ошибку.

## Список параметров
- **`skinId`** (`ObjectId`)
Идентификатор скина активного животного.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Изменение скина активного животного
```lua
mount.SetSkin( skinId )
```
