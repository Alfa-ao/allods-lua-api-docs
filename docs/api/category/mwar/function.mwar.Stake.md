# mwar.Stake

Делает ставку в аукционе Доминиона.

## Описание

```lua
mwar.Stake( targetGuildId: ObjectId )
```

Функция инициирует процесс размещения ставки в аукционе Доминиона для указанной гильдии.

## Список параметров

- **`targetGuildId`** (`ObjectId`) - Идентификатор гильдии, на которую делается ставка.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Размещение ставки

```lua
mwar.Stake( targetGuildId )
```
