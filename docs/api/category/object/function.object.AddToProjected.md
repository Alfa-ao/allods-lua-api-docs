# object.AddToProjected
Заносит объект в список для проецирования в 2D координаты.

## Описание
```lua
object.AddToProjected( objectId: ObjectId, heightOffset: number )
```
Функция заносит объект в список для проецирования в 2D координаты с возможностью указания дополнительного смещения. Добавление доступно только для объектов, координаты которых могут быть получены (детектируемые объекты, игроки из отряда главного игрока и тому подобное).

## Список параметров
- **`objectId`** (`ObjectId`) - Идентификатор объекта.

- **`heightOffset`** (`number`(`ATTACHED_OBJECT_POS_*`)) - Место дополнительного смещения проецируемой точки около объекта.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Привязка точки проецирования к голове цели
```lua
object.AddToProjected( avatar.GetTarget(), ATTACHED_OBJECT_POS_UP )
```
::: info Описание примера
В примере выполняется получение цели текущего игрока с помощью `avatar.GetTarget()` и привязка к ней точки проецирования со смещением вверх.
:::
