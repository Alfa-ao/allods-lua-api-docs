# object.CanAttack
Определяет возможность атаки указанного объекта.

## Описание
```lua
object.CanAttack( objectId: ObjectId ): boolean
```
Функция возвращает логическое значение, указывающее на возможность атаки объекта. Атака возможна, если объект враждебен, не мертв и доступен для атаки.

## Список параметров
- **`objectId`** (`ObjectId`) - Уникальный идентификатор объекта.

## Возвращаемые значения
Возвращает `boolean` - `true`, если объект можно атаковать (он враждебен, не мертв и доступен для атаки), иначе `false`.

## Примеры
### Проверка возможности атаки
```lua
local canAttack = object.CanAttack( objectId )
```
