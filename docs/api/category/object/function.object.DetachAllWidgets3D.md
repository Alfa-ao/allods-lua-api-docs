# object.DetachAllWidgets3D
Отвязывает все Widget3D от указанного объекта.

## Описание
```lua
object.DetachAllWidgets3D( objectId: ObjectId )
```
Функция выполняет отвязку всех Widget3D от объекта с заданным идентификатором.

## Список параметров
- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Отвязка всех Widget3D от объекта
```lua
object.DetachAllWidgets3D( objectId )
```
