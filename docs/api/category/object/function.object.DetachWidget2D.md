# object.DetachWidget2D

Убирает 2D-виджеты, привязанные к интерактивному объекту.

## Описание

```lua
object.DetachWidget2D( objectId: ObjectId, widget: WidgetSafe | nil )
```

Убирает 2D-виджеты, привязанные к интерактивному объекту.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта. Не может быть `nil`.

- **`widget`** (`WidgetSafe` | `nil`) - Виджет, который надо убрать. Если параметр не указан, то удаляются все виджеты, относящиеся к объекту.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Отвязка всех виджетов от объекта

```lua
object.DetachWidget2D( targetId )
```
