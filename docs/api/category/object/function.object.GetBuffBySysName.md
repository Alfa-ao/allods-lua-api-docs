# object.GetBuffBySysName
Выполняет поиск инстансов баффа с заданным системным именем на указанном объекте.

## Описание
```lua
object.GetBuffBySysName( objectId: ObjectId, sysName: string ): ObjectId | nil
```
Функция выполняет поиск инстансов баффа с заданным системным именем на объекте. Возвращает идентификатор первого найденного инстанса или `nil`, если таковые не найдены. Порядок обхода не определен.

## Список параметров
- **`objectId`** (`ObjectId`) - Идентификатор объекта, на котором выполняется поиск баффа.

- **`sysName`** (`string`) - Внутреннее имя баффа (соответствующее поле в базе для баффа - `sysUIScriptName`).

## Возвращаемые значения
Возвращает `ObjectId` | `nil` - Идентификатор инстанса баффа, если таковой был найден. Если инстансов несколько, возвращается первый найденный (порядок не определен). В противном случае возвращается `nil`.

## Примеры
### Проверка наличия баффа у персонажа
```lua
if object.GetBuffBySysName( avatar.GetId(), FLAG_BUFF_NAME ) then
    -- Действия при наличии баффа
end
```

::: info Описание примера
В примере выполняется проверка наличия баффа с внутренним именем `FLAG_BUFF_NAME` у главного персонажа. Функция `avatar.GetId()` используется для получения идентификатора объекта персонажа.
:::
