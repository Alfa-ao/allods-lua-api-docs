# object.GetBuffCount

Возвращает количество баффов, висящих на данном объекте, включая дебаффы.

## Описание

```lua
object.GetBuffCount( id: ObjectId ): number
```

Функция возвращает целое число, представляющее количество баффов и дебаффов, наложенных на юнит с указанным идентификатором.

## Список параметров

- **`id`** (`ObjectId`) - Уникальный идентификатор объекта.

## Возвращаемые значения

Возвращает `number` - количество баффов на юните.

## Примеры

### Получение количества активных баффов

```lua
local activeBuffs = object.GetBuffCount( id )
```
