# object.GetBuffInstances
Возвращает список идентификаторов экземпляров бафов, находящихся на данном объекте и соответствующих заданному критерию поиска.

## Описание
```lua
object.GetBuffInstances( objectId: ObjectId, key: BuffId | string | number ): table
```
Функция возвращает индексированный с единицы список идентификаторов бафов, удовлетворяющих заданному критерию. Критерий поиска может быть задан в виде идентификатора ресурса, системного имени или интерфейсного приоритета.

## Список параметров
- **`objectId`** (`ObjectId`) - Идентификатор объекта.

- **`key`** (`BuffId` | `string` | `number`(`ENUM_UIBuffPriority_*`)) - Критерий поиска бафов. Поиск осуществляется по идентификатору ресурса, системному имени или интерфейсному приоритету.

## Возвращаемые значения
Возвращает `table` - индексированный с единицы список идентификаторов (`ObjectId`) бафов, удовлетворяющих заданному критерию.

## Примеры
### Получение списка бафов аватара
```lua
local avatarBuffs = object.GetBuffInstances( avatar.GetId(), buffId )
```
