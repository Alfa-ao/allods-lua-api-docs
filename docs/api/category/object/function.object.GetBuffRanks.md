# object.GetBuffRanks
Возвращает таблицу ресурсов бафов по рангам.

## Описание
```lua
object.GetBuffRanks( buffId: BuffId ): table
```
Функция возвращает таблицу рангов бафа, где ключом является номер ранга, а значением выступает ресурс бафа соответствующего ранга. Если у бафа отсутствуют ранги, он считается бафом 1 ранга.

## Список параметров
- **`buffId`** (`BuffId` (`ResourceId`)) - идентификатор ресурса бафа.

## Возвращаемые значения
Возвращает `table` - таблица рангов бафа. Ключом является номер ранга, а значением выступает ресурс бафа соответствующего ранга. Если у бафа отсутствуют ранги, он считается бафом 1 ранга.
