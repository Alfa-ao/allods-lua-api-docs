# object.GetBuffSysName

Возвращает системное имя для указанного баффа.

## Описание

```lua
object.GetBuffSysName( Id: ObjectId | BuffId ): string | nil
```

Функция возвращает внутреннее системное имя баффа, соответствующее полю `sysUIScriptName` в базе данных, которое может быть использовано для специальной обработки.

## Список параметров

- **`Id`** (`ObjectId` | `BuffId`) - Идентификатор объекта или ресурса баффа.

## Возвращаемые значения

Возвращает `string` | `nil` - внутреннее системное имя баффа, соответствующее полю `sysUIScriptName` в базе данных, которое может быть использовано для специальной обработки.

## Примеры

### Получение системного имени баффа

```lua
local sysName = object.GetBuffSysName( Id )
```
