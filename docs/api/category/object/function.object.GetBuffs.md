# object.GetBuffs

Возвращает список валидных экземпляров бафов, висящих на заданном объекте.

## Описание

```lua
object.GetBuffs( objectId: ObjectId, ignoreInvisible: boolean | nil ): table
```

Функция извлекает и возвращает список валидных экземпляров бафов, висящих на заданном объекте. Объект должен существовать и иметь бафф-менеджер.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта, на котором выполняется поиск бафов. Объект должен существовать и иметь бафф-менеджер.

- **`ignoreInvisible`** (`boolean` | `nil`) - Флаг, указывающий на необходимость игнорирования бафов со свойством `needVisualize == false`. По умолчанию принимает значение `false`.

## Возвращаемые значения

Возвращает `table` - индексированный с 1 список идентификаторов (`ObjectId`) бафов на объекте.

## Примеры

### Получение списка активных бафов

```lua
local activeBuffs = object.GetBuffs( avatar.GetId() )
```
