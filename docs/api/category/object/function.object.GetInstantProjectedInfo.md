# object.GetInstantProjectedInfo

Возвращает мгновенную информацию о двумерной проекции объекта на экран без добавления в список проецирования.

## Описание

```lua
object.GetInstantProjectedInfo( objectId: ObjectId, heightOffset: number ): table | nil
```

Дополнительно указывается смещение проецируемой точки. Информация извлекается только для объектов, для которых возможно вычисление координаты (детектируемые объекты, игроки из отряда главного персонажа и т.п.).

Точка (0, 0) представляет собой центр экрана. Координаты (-1, -1) обозначают левый верхний угол, а (1, 1) - правый нижний угол.

Если хотя бы одна возвращаемая координата равна 1 или -1, объект находится за пределами зоны видимости. Если обе координаты находятся в промежутке (-1, 1), проецируемая точка видна на экране.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта.

- **`heightOffset`** (`number`(`ATTACHED_OBJECT_POS_*`)) - Место дополнительного смещения проецируемой точки около объекта.

## Возвращаемые значения

Возвращает `table` | `nil` - информация о 2D проекции объекта, аналогичная возвращаемому значению функции `object.GetProjectedInfo`, или `nil`, если информация недоступна.

- **`posX`** (`number`) - Координата X проекции на экране.

- **`posY`** (`number`) - Координата Y проекции на экране.

- **`isLos`** (`boolean`) - Флаг наличия прямой видимости.

## Примеры

### Вычисление экранных координат и проверка видимости

```lua
local show = true
local projectedInfo = object.GetInstantProjectedInfo( avatar.GetTarget(), heightOffset )

if projectedInfo then
    local position = {} -- экранные координаты в пикселях
    position.x = projectedInfo.posX * width / 2
    position.y = projectedInfo.posY * height / 2

    -- попадание 3D точки в зону видимости
    if ( math.abs( projectedInfo.posX ) ~= 1 ) and ( math.abs( projectedInfo.posY ) ~= 1 ) then
        if projectedInfo.isLos then
            show = false -- точка видна на экране - не показываем
        else
            position.y = -height / 2 -- точка за препятствием - показываем сверху
        end
    end
end
```

::: info Описание примера

В примере запрашивается информация о проекции целевого объекта с заданным смещением. Если данные успешно получены, вычисляются экранные координаты в пикселях. Затем проверяется попадание точки в зону видимости: если координаты не равны 1 или -1 и есть прямая видимость, флаг отображения сбрасывается; если точка находится за препятствием, координата Y корректируется для отображения сверху.

:::
