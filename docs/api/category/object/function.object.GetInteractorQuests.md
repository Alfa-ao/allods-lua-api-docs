# object.GetInteractorQuests
Извлекает списки квестов, имеющих отношение к интерактивному объекту, для вычисления квестовой метки.

## Описание
```lua
object.GetInteractorQuests( objectId: ObjectId ): table | nil
```
Функция извлекает и возвращает таблицу со списками квестов для каждой категории, по которым можно вычислить квестовую метку для указанного интерактивного объекта. Если информация не получена, функция возвращает `nil`.

## Список параметров
- **`objectId`** (`ObjectId`) - идентификатор объекта.

## Возвращаемые значения
Возвращает `table` | `nil` - таблица со списками квестов для каждой категории или `nil`, если информация не получена. Поля таблицы:

- **`readyToGive`** (`table`) - список квестов, которые объект готов выдать аватару прямо сейчас; формат: индексированный от 0 массив `QuestId`.

- **`readyToAccept`** (`table`) - список квестов, которые объект готов принять от аватара прямо сейчас; формат: индексированный от 0 массив `QuestId`.

- **`soonToGive`** (`table`) - список квестов, которые объект будет готов выдать аватару, когда тот освободит место в квест-буке или получит один уровень (может требоваться одно или оба из условий); формат: индексированный от 0 массив `QuestId`.

- **`soonToAccept`** (`table`) - список квестов, которые объект будет готов принять от аватара, когда последний их выполнит; формат: индексированный от 0 массив `QuestId`.

## Примеры
### Получение списка квестов для выдачи
```lua
local quests = object.GetInteractorQuests( objectId )
if quests then
    if quests.readyToGive[0] then
        common.LogInfo( avatar.GetQuestInfo( quests.readyToGive[0] ) )
    end
end
```
::: info Описание примера
В примере извлекается таблица квестов для указанного интерактивного объекта. Если таблица успешно получена и существует хотя бы один квест, готовый к выдаче, в лог выводится информация об этом квесте с помощью функции `avatar.GetQuestInfo`.
:::
