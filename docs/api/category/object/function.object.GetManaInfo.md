# object.GetManaInfo

Возвращает информацию о мане интерактивного объекта при ее наличии.

## Описание

```lua
object.GetManaInfo( objectId: ObjectId ): table | nil
```

Возвращает таблицу с информацией о мане объекта или `nil`, если информация недоступна. Некоторые поля могут быть недоступны, если их значения отсутствуют для данного типа объектов. Например, нельзя узнать точное значение маны или поглощения у противника.

::: warning Замечание
Тип маны объекта может быть `MANA_TYPE_NONE` (нет маны), при этом механически мана есть и данная API вернет нули, а не `nil`. Для проверки типа маны используется функция `object.GetManaType`.
:::

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения

Возвращает `table` | `nil` - таблица с информацией о мане объекта или `nil`, если информация недоступна.

- **`value`** (`number` | `nil`) - Текущее значение маны объекта.

- **`limit`** (`number` | `nil`) - Предельное значение маны объекта.

- **`valuePercents`** (`number` | `nil`) - Процент текущей маны от предельного.

## Примеры

### Получение и анализ информации о мане объекта

```lua
local manaInfo = objectId and object.GetManaInfo( objectId )
if manaInfo then
    local ratio = manaInfo.limit ~= 0 and manaInfo.value / manaInfo.limit or 1
    local state = manaInfo.valuePercents / 100 or 0
end
```

::: info Описание примера
В примере сначала проверяется наличие идентификатора объекта, после чего запрашивается информация о его мане. Если данные успешно получены, вычисляется соотношение текущей маны к предельной с защитой от деления на ноль и определяется процентное состояние маны.
:::
