# object.GetPos

Возвращает координаты указанного объекта, если их получение разрешено.

## Описание

```lua
object.GetPos( objectId: ObjectId ): GamePosition | nil
```

Функция извлекает и возвращает координаты объекта. В текущей реализации получение координат разрешено для транспортов и детектируемых юнитов.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения

Возвращает `GamePosition` | `nil` - Позиция объекта, если он найден и можно получать его позицию, или `nil`, если объект не найден или получение координат запрещено.

## Примеры

### Получение координат объекта

```lua
local pos = object.GetPos( objectId )
if pos then
    local x = pos.posX
end
```
