# object.GetProjectedInfo

Возвращает информацию о 2D проекции объекта на экран.

## Описание

```lua
object.GetProjectedInfo( objectId: ObjectId ): table | nil
```

Функция возвращает для объекта информацию об его 2D проекции на экран, если объект присутствует в списке для проецирования в 2D координаты. Точка (0, 0) соответствует центру экрана, (-1, -1) - левому верхнему углу, (1, 1) - правому нижнему углу. Если какая-либо возвращаемая координата равна 1 или -1, объект находится за зоной видимости. Если обе координаты находятся в промежутке (-1, 1), проецируемая точка видна на экране.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения

Возвращает `table` | `nil` - таблицу с информацией о проекции, если объект найден и присутствует в списке для проецирования, или `nil`, если объект не найден.

- **`posX`** (`number`) - 2D экранная координата по оси X в интервале [-1, 1].
- **`posY`** (`number`) - 2D экранная координата по оси Y в интервале [-1, 1].
- **`playerDistance`** (`number`) - Дистанция от объекта до главного игрока в метрах.
- **`cameraDistance`** (`number`) - Дистанция от объекта до камеры в метрах.
- **`isLos`** (`boolean`) - Принимает значение `false`, если на линии видимости до объекта присутствует препятствие.
- **`canProject`** (`boolean`) - Принимает значение `false`, если объект не может быть спроецирован (например, изменились условия и объект больше не детектируется). В обычной ситуации всегда `true`.

## Примеры

### Определение видимости цели и расчёт экранных координат

```lua
local show = true

local projectedInfo = object.GetProjectedInfo( avatar.GetTarget() )
if projectedInfo then

    local position = {} -- экранные координаты в пикселях
    position.x = projectedInfo.posX * width / 2
    position.y = projectedInfo.posY * height / 2

    -- попадание 3D точки в зону видимости
    if ( math.abs( projectedInfo.posX ) ~= 1 ) and ( math.abs( projectedInfo.posY ) ~= 1 ) then
        if projectedInfo.isLos then
            show = false -- точка видна на экране - не показываем
        else
            position.y = -height / 2 -- точка за препятствием - показываем сверху
        end
    end
end
```

::: info Описание примера
Запрашивается информация о проекции текущей цели игрока на экран. Если данные получены, вычисляются экранные координаты в пикселях на основе размеров экрана (`width`, `height`). Затем проверяется, находится ли точка в зоне видимости (обе координаты не равны 1 или -1). Если точка видна и линия видимости свободна (`isLos` равен `true`), флаг отображения сбрасывается. Если точка находится за препятствием, координата Y смещается в верхнюю часть экрана.
:::
