# object.HasQuest
Проверяет наличие квестов на выдачу или приём у интерактивного объекта.

## Описание
```lua
object.HasQuest( id: ObjectId ): boolean
```
Функция проверяет наличие квестов на выдачу или приём у интерактивного объекта. В некоторых случаях результат может не соответствовать действительности, для уточнения необходимо вызывать `avatar.HasInteractorQuest()`.

## Список параметров
- **`id`** (`ObjectId`) - Уникальный идентификатор объекта. Если передан идентификатор игрока, функция возвращает `false`.

## Возвращаемые значения
Возвращает `boolean` - `true`, если объект имеет квесты, иначе `false`.

## Примеры
### Проверка наличия квеста у цели
```lua
local hasQuest = object.HasQuest( avatar.GetTarget() )
```
