# object.HasWorldSecret
Определяет возможность завершения тайны мира интерактивным объектом.

## Описание
```lua
object.HasWorldSecret( id: ObjectId ): boolean
```
Функция возвращает логическое значение, указывающее на способность интерактивного объекта завершить и выдать награду за тайну мира.

## Список параметров
- **`id`** (`ObjectId`) - Идентификатор объекта. Если передан идентификатор игрока, функция возвращает `false`.

## Возвращаемые значения
Возвращает `boolean` - `true`, если объект может завершить тайну мира, иначе `false`.

## Примеры
### Проверка возможности завершения тайны мира
```lua
local canFinishWorldSecret = object.HasWorldSecret( avatar.GetTarget() )
```
