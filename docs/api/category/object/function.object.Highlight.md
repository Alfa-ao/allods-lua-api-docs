# object.Highlight

Подсвечивает интерактивный объект заданным цветом или изменяет параметры его окружения.

## Описание

```lua
object.Highlight( objectId: ObjectId, type: string, color: Color | nil, color2: Color | nil, period: number )
```

Функция управляет параметрами подсветки интерактивного объекта. Поддерживается изменение цвета подсветки, цвета окружения и модификаторов теней окружения.

::: info Перечисление (HELP)
- **`SELECTION`** - цвет подсветки (a, r, g, b), накладывается на объект.

- **`AMBIENT`** - цвет окружения (r, g, b).

- **`OCCLUSION`** - модификаторы теней окружения (r, g, b).
:::

## Список параметров

- **`objectId`** (`ObjectId`) - идентификатор интерактивного объекта.

- **`type`** (`string`("SELECTION", "AMBIENT", "OCCLUSION")) - тип изменяемого параметра.

- **`color`** (`Color` | `nil`) - начальный цвет изменяемого параметра.

- **`color2`** (`Color` | `nil`) - конечный цвет изменяемого параметра. Поддерживается только для `AMBIENT`.

- **`period`** (`number`) - полный период изменения параметра `color` -> `color2` -> `color`. Поддерживается только для `AMBIENT`.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Подсветка объекта красным цветом

```lua
local redColor = { r = 1, g = 0, b = 0, a = 0.2 }
object.Highlight( objectId, "SELECTION", redColor )
```

### Снятие подсветки

```lua
object.Highlight( objectId, "SELECTION", nil )
```
