# object.IsDead
Определяет статус смерти интерактивного объекта.

## Описание

```lua
object.IsDead( objectId: ObjectId ): boolean
```

Функция возвращает информацию о статусе смерти интерактивного объекта. Для игроков означает, что игрок мертв и лежит на месте смерти, а не находится в чистилище.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения

Возвращает `boolean` - `true`, если объект мертв.

## Примеры

### Проверка статуса смерти объекта

```lua
local isDead = object.IsDead( objectId )
```
