# object.IsEnemy

Возвращает `true`, если объект враждебен главному игроку.

## Описание

```lua
function object.IsEnemy( objectId: ObjectId ): boolean
```

Функция проверяет, является ли объект (юнит, транспорт, астральный юнит) враждебным по отношению к главному игроку.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта. Не может быть `nil`.

## Возвращаемые значения

Возвращает `boolean` - `true`, если объект враждебен игроку.

## Примеры

### Проверка враждебности объекта

```lua
local isEnemy = object.IsEnemy( objectId )
```
