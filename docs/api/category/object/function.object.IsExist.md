# object.IsExist

Проверяет наличие интерактивного объекта (игрока, устройства, моба, транспорта и т.п.) в мире.

## Описание

```lua
object.IsExist( objectId: ObjectId ): boolean
```

Функция возвращает логическое значение, указывающее на существование интерактивного объекта с заданным идентификатором.

## Список параметров

- **`objectId`** (`ObjectId`) - Уникальный идентификатор интерактивного объекта.

## Возвращаемые значения

Возвращает `boolean` - `true`, если интерактивный объект существует, иначе `false`.

## Примеры

### Проверка существования объекта и получение его имени

```lua
local avatarId = avatar.GetId()
if object.IsExist( avatarId ) then
    local name = object.GetName( avatarId )
end
```

::: info Описание примера
В примере сначала извлекается идентификатор главного персонажа с помощью `avatar.GetId()`. Затем проверяется существование объекта с данным идентификатором. Если объект существует, извлекается его имя через `object.GetName()`.
:::
