# object.IsFar

Проверяет, находится ли объект на другой карте.

## Описание

```lua
object.IsFar( objectId: ObjectId ): boolean
```

Функция возвращает логическое значение, указывающее на то, что объект существует в системе, но находится далеко, на другой карте. В этом случае для него можно получить информацию о здоровье, мане, классе, состоянии в группе главного игрока и т.п., но нельзя получить позицию, приаттачить к нему виджет или визуальный объект, добавить его в список проецируемых объектов, выделить, получить портрет и т.п.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор интерактивного объекта.

## Возвращаемые значения

Возвращает `boolean` - `true`, если объект находится далеко, на другой карте.

## Примеры

### Проверка удаленности объекта и установка текстуры

```lua
if object.IsFar( objectId ) then
    self.wtImage:SetBackgroundTexture( emptyTextureId )
else
    mission.SetPortraitBackgroundTexture( 1, true, widget, objectId )
end
```

::: info Описание примера

В примере выполняется проверка удаленности объекта. Если объект находится на другой карте, для виджета устанавливается пустая фоновая текстура. В противном случае устанавливается фоновая текстура портрета через функцию `mission.SetPortraitBackgroundTexture`.

:::
