# object.IsFriend

Определяет, является ли объект (юнит, транспорт, астральный юнит) дружественным главному игроку.

## Описание

```lua
object.IsFriend( objectId: ObjectId ): boolean
```

Функция возвращает логическое значение, указывающее на то, что объект (юнит, транспорт, астральный юнит) является дружественным главному игроку.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения

Возвращает `boolean` - `true`, если объект дружественен игроку, и `false` в противном случае.

## Примеры

### Проверка дружественности объекта

```lua
local isFriend = object.IsFriend( objectId )
```
