# object.IsInCombat

Возвращает состояние боя интерактивного объекта (юниты и корабли).

## Описание

```lua
object.IsInCombat( objectId: ObjectId ): boolean
```

Функция возвращает логическое значение, указывающее на текущее состояние боя интерактивного объекта. Поддерживаемые типы объектов включают юниты и корабли.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения

Возвращает `boolean` - `true`, если объект находится в состоянии боя, `false` в противном случае.

## Примеры

### Проверка состояния боя объекта

```lua
local inCombat = object.IsInCombat( unitId )
```
