# object.IsInUsageRange

Проверяет, находится ли указанный интерактивный объект в радиусе "meleeRange".

## Описание

```lua
object.IsInUsageRange( objectId: ObjectId ): boolean
```

Функция определяет, попадает ли интерактивный объект с заданным идентификатором в радиус взаимодействия "meleeRange".

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор интерактивного объекта.

## Возвращаемые значения

Возвращает `boolean` - `true`, если объект находится в радиусе взаимодействия, иначе `false`.

## Примеры

### Проверка нахождения объекта в радиусе взаимодействия

```lua
local inUsageRange = object.IsInUsageRange( objectId )
```
