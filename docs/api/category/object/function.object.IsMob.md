# object.IsMob

Определяет, является ли указанный объект мобом.

## Описание

```lua
object.IsMob( objectId: ObjectId ): boolean
```

Функция вызывает ошибку, если объект не существует или не является интерактивным объектом (юниты, устройства, транспорты и т.п.).

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения

Возвращает `boolean` - `true`, если объект является мобом, и `false`, если объект не является мобом.

## Примеры

### Проверка объекта

```lua
local isMob = object.IsMob( objectId )
```
