# object.IsTransport
Определяет, является ли интерактивный объект транспортом.

## Описание
```lua
object.IsTransport( id: ObjectId ): boolean
```
Функция проверяет, относится ли интерактивный объект с заданным идентификатором к типу транспорта, и возвращает соответствующее логическое значение.

## Список параметров
- **`id`** (`ObjectId`) - Уникальный идентификатор интерактивного объекта.

## Возвращаемые значения
Возвращает `boolean` - `true`, если интерактивный объект является транспортом, и `false` в противном случае.

## Примеры
### Проверка объекта на принадлежность к транспорту
```lua
local id = avatar.GetTarget()
if object.IsTransport( id ) then
    local name = object.GetName( id )
end
```

::: info Описание примера
В примере сначала получается идентификатор текущей цели с помощью `avatar.GetTarget()`. Затем проверяется, является ли данный объект транспортом. Если проверка успешна, извлекается имя объекта через `object.GetName()`.
:::
