# object.IsUsable
Определяет возможность принципиального использования объекта.

## Описание
```lua
object.IsUsable( objectId: ObjectId ): boolean
```
Функция возвращает логическое значение, указывающее на принципиальную возможность использования объекта.

## Список параметров
- **`objectId`** (`ObjectId`) - Идентификатор объекта.

## Возвращаемые значения
Возвращает `boolean` - `true`, если объект можно использовать в принципе, и `false` в противном случае.

## Примеры
### Проверка возможности использования объекта
```lua
local isUsable = object.IsUsable( objectId )
```
