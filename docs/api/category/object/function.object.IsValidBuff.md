# object.IsValidBuff

Проверяет, является ли переданный идентификатор валидным экземпляром бафа.

## Описание

```lua
object.IsValidBuff( id: any ): boolean
```

Функция возвращает `true`, если аргумент является идентификатором валидного (на момент вызова функции) экземпляра бафа. Гарантируется, что если баф валиден, то он останется валидным до конца обработки текущего события в Lua.

::: warning Замечание
Проверяются экземпляры бафов, но не ресурсы. Если передать `BuffId` (userdata `ResourceId`), то функция вернет `false`.
:::

## Список параметров

- **`id`** (`any`) - Произвольное значение.

## Возвращаемые значения

Возвращает `boolean` - `true`, если аргумент является идентификатором валидного бафа.

## Примеры

### Проверка валидности бафа и получение оставшегося времени

```lua
local remainingMs = object.IsValidBuff( data.buffId ) and object.GetBuffDynamicInfo( data.buffId ).remainingMs
```

::: info Описание примера
Сначала проверяется валидность бафа с помощью `object.IsValidBuff`. Если баф валиден (возвращается `true`), выполняется вызов `object.GetBuffDynamicInfo` для получения оставшегося времени действия бафа в миллисекундах.
:::
