# object.RemoveBuff

Снимает позитивный бафф с объекта.

## Описание

```lua
object.RemoveBuff( buffId: ObjectId )
```

Функция снимает позитивный бафф с объекта, если этот бафф можно снять (прописано в его свойствах). Бафф можно снять только с персонажа главного игрока, его питомца и его корабля.

## Список параметров

- **`buffId`** (`ObjectId`) - Идентификатор баффа.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Снятие баффа с объекта

```lua
object.RemoveBuff( buffId )
```
