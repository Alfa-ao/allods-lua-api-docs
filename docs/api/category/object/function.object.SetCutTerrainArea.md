# object.SetCutTerrainArea

Задает область вокруг объекта, в которой убирается трава и подобные элементы.

## Описание

```lua
object.SetCutTerrainArea( objectId: ObjectId, scale: number, extraRadius: number )
```

Задание области вокруг объекта, где убирается трава и т.п., с возможностью включения и выключения данной области.

## Список параметров

- **`objectId`** (`ObjectId`) - Идентификатор объекта, вокруг которого отключать траву и т.п., либо `nil`, если такая область не нужна.

- **`scale`** (`number`) - Коэффициент, на который умножается размер объекта.

- **`extraRadius`** (`number`) - Дополнительное увеличение области.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка области

```lua
object.SetCutTerrainArea( objectId, 1.0f, CUT_TERRAIN_AREA_EXTRA_RADIUS )
```
