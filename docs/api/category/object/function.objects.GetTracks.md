# objects.GetTracks

Получает список отслеживаемых объектов Track в зоне действия аватара.

## Описание

```lua
objects.GetTracks(): table
```

Функция возвращает индексированный с нуля список идентификаторов отслеживаемых объектов Track в зоне действия аватара.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - индексированный с нуля список идентификаторов (`ObjectId`) отслеживаемых объектов.

## Примеры

### Получение и перебор списка отслеживаемых объектов

```lua
local tracks = objects.GetTracks()
for i, trackId in pairs( tracks ) do
    LogInfo( "trackId: ", trackId )
end
```
