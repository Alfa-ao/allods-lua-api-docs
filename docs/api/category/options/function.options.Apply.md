# options.Apply
Применяет новые значения для опции или коллекции.

## Описание
```lua
options.Apply( optionBaseId: ObjectId )
```
Применяет новые значения для опции или коллекции, после чего они становятся базовыми. Также применяет результаты режима предварительного просмотра и сбрасывает данный режим.

## Список параметров
- **`optionBaseId`** (`ObjectId`) - Идентификатор опции или коллекции.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Применение значений опции
```lua
options.Apply( optionBaseId )
```
