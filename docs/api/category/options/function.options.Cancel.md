# options.Cancel
Сбрасывает указанную опцию или коллекцию в их базовые значения.

## Описание
```lua
options.Cancel( optionBaseId: ObjectId )
```
Производит сброс указанной опции или коллекции в их базовые значения, а также сбрасывает режим предварительного просмотра.

## Список параметров
- **`optionBaseId`** (`ObjectId`) - Идентификатор опции или коллекции.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Сброс опции
```lua
options.Cancel( optionBaseId )
```
