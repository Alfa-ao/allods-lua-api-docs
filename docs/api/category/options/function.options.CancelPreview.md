# options.CancelPreview

Отменяет для опции или коллекции режим предварительного просмотра.

## Описание

```lua
options.CancelPreview( optionBaseId: ObjectId )
```

Функция выполняет отмену режима предварительного просмотра для заданной опции или коллекции.

## Список параметров

- **`optionBaseId`** (`ObjectId`) - Идентификатор опции или коллекции.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Отмена предварительного просмотра

```lua
options.CancelPreview( optionBaseId )
```
