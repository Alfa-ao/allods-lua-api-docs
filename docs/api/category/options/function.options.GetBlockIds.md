# options.GetBlockIds
Возвращает таблицу с идентификаторами всех блоков для заданной группы опций.

## Описание
```lua
options.GetBlockIds( groupId: ObjectId ): table
```
Блок представляет собой наименьшую коллекцию опций, содержащуюся в группе, и включает в себя непосредственно сами опции.

## Список параметров
- **`groupId`** (`ObjectId`) - идентификатор группы опций.

## Возвращаемые значения
Возвращает `table` - таблицу с идентификаторами блоков (`ObjectId`), индексация начинается с 1.

## Примеры
### Получение и перебор идентификаторов блоков
```lua
local blockIds = options.GetBlockIds( groupId )
for _, blockId in ipairs( blockIds ) do
    -- Обработка идентификатора блока
end
```
