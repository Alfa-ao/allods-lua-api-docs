# options.GetCollectionInfo

Возвращает информацию о заданной коллекции опций (странице, группе или блоке).

## Описание

```lua
options.GetCollectionInfo( collectionId: ObjectId ): table
```

Извлекает и возвращает таблицу с детальной информацией о заданной коллекции опций (странице, группе или блоке).

## Список параметров

- **`collectionId`** (`ObjectId`) - Идентификатор коллекции опций.

## Возвращаемые значения

Возвращает `table` - таблица с информацией о коллекции опций со следующими полями:

- **`isEnabled`** (`boolean`) - Можно ли менять коллекцию и ее элементы.

- **`isNeedPreview`** (`boolean`) - Нужно ли использовать режим предварительного просмотра.

- **`isPreview`** (`boolean`) - Находится ли какой-нибудь элемент коллекции в режиме предварительного просмотра.

- **`sysCustomType`** (`string`) - Уникальный идентификатор коллекции для возможной специальной обработки.

- **`name`** (`WString`) - Название коллекции.

- **`description`** (`WString`) - Описание, тултип, комментарий и тому подобное коллекции.

## Примеры

### Получение информации о коллекции

```lua
local collectionInfo = options.GetCollectionInfo( collectionId )
```
