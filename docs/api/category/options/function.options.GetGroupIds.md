# options.GetGroupIds

Возвращает таблицу с идентификаторами всех групп для указанной страницы опций.

## Описание

```lua
options.GetGroupIds( pageId: ObjectId ): table
```

Функция извлекает и возвращает таблицу, содержащую идентификаторы всех групп, принадлежащих указанной странице опций. Группа представляет собой промежуточную коллекцию опций, которая содержится в странице и сама содержит блоки.

## Список параметров

- **`pageId`** (`ObjectId`) - Идентификатор страницы опций.

## Возвращаемые значения

Возвращает `table` - таблицу с идентификаторами групп (`ObjectId`). Индексация элементов начинается с 1.

## Примеры

### Получение идентификаторов групп страницы

```lua
local groupIds = options.GetGroupIds( pageId )
for _, groupId in ipairs( groupIds ) do
    ...
end
```
