# options.GetPageIds

Возвращает таблицу с идентификаторами всех страниц опций.

## Описание

```lua
options.GetPageIds(): table
```

Функция возвращает таблицу с идентификаторами всех страниц опций. Страница является самой крупной структурной единицей из коллекций опций и содержит группы.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - таблица с идентификаторами страниц. Индексация элементов в таблице начинается с 1. Значения представляют собой уникальные идентификаторы (`ObjectId`).

## Примеры

### Получение и перебор идентификаторов страниц

```lua
local pageIds = options.GetPageIds()
for _, pageId in ipairs( pageIds ) do
    ...
end
```
