# options.Preview

Выставляет для опции или коллекции режим предварительного просмотра.

## Описание

```lua
options.Preview( optionBaseId: ObjectId )
```

Функция устанавливает режим предварительного просмотра, при котором настройки не сохраняются в конфигурационных файлах, пока режим действует хотя бы для одной опции.

## Список параметров

- **`optionBaseId`** (`ObjectId`) - Идентификатор опции или коллекции.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Включение режима предварительного просмотра

```lua
options.Preview( optionBaseId )
```
