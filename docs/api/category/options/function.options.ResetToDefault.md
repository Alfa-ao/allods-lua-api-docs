# options.ResetToDefault

Сбрасывает опцию или коллекцию в значения по умолчанию.

## Описание

```lua
options.ResetToDefault( optionBaseId: ObjectId )
```

Функция устанавливает новое и базовое значения в значение по умолчанию, а также сбрасывает режим предварительного просмотра.

## Список параметров

- **`optionBaseId`** (`ObjectId`) - идентификатор опции или коллекции

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Сброс опции в значения по умолчанию

```lua
options.ResetToDefault( optionBaseId )
```
