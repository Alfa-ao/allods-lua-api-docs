# options.Update
Обновляет значения всех опций по актуальной информации из системы.

## Описание
```lua
options.Update()
```
Сбрасывает все текущие изменения. Базовые и новые значения выставляются согласно актуальным системным данным.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Обновление опций
```lua
options.Update()
```
