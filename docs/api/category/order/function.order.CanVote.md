# order.CanVote
Проверяет доступность голосования за бонус для аватара.

## Описание
```lua
order.CanVote(): boolean
```
Функция возвращает логическое значение, указывающее на доступность голосования за бонус для аватара.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если голосование доступно, иначе `false`.

## Примеры
### Проверка доступности голосования
```lua
local canVoteForBonuses = order.CanVote()
```
