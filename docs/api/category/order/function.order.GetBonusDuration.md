# order.GetBonusDuration
Возвращает количество дней до момента, когда бонусы станут доступны для повторного выбора в голосовании.

## Описание
```lua
order.GetBonusDuration(): number
```
Функция возвращает количество дней до отката бонусов в голосовании.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` - количество дней до отката бонусов в голосовании.

## Примеры
### Получение длительности бонуса
```lua
local bonusDuration = order.GetBonusDuration()
```
