# order.GetMainPlayerVoteWeight
Возвращает вес голоса аватара для голосования за бонусы в ордене.

## Описание
```lua
order.GetMainPlayerVoteWeight(): number | nil
```
Функция возвращает вес голоса аватара для голосования за бонусы в ордене.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` | `nil` - вес голоса аватара.
