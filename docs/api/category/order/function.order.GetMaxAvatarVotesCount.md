# order.GetMaxAvatarVotesCount

Возвращает максимальное количество бонусов, за которые можно проголосовать.

## Описание

```lua
order.GetMaxAvatarVotesCount(): number | nil
```

Функция возвращает максимальное количество бонусов, за которые можно проголосовать.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` | `nil` - максимальное количество голосов.

## Примеры

### Получение максимального количества голосов

```lua
local maxVotes = order.GetMaxAvatarVotesCount()
```
