# order.GetOrderInfo

Возвращает информацию об ордене.

## Описание

```lua
order.GetOrderInfo( orderNumber: number ): table | nil
```

Функция извлекает и возвращает таблицу с информацией об ордене по его порядковому номеру. Если орден с указанным номером не найден, функция возвращает `nil`.

## Список параметров

- **`orderNumber`** (`number`) - порядковый номер ордена.

## Возвращаемые значения

Возвращает `table` | `nil` - таблица с информацией об ордене или `nil`, если орден не найден.

- **`image`** (`TextureId`) - текстура с иконкой предмета.

- **`description`** (`WString` | `nil`) - описание ордена.

- **`name`** (`WString`) - имя ордена в игре.

- **`isAvailable`** (`boolean`) - `true`, если орден доступен для вступления.

- **`sysName`** (`string`) - sysname ордена.

## Примеры

### Получение информации об ордене

```lua
local orderInfo = order.GetOrderInfo( 1 )
```
