# order.GetOrderStats

Возвращает динамическую информацию об ордене.

## Описание

```lua
order.GetOrderStats(): table | nil
```

Функция возвращает таблицу с динамической информацией об ордене или `nil`, если информация недоступна.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - таблица с динамической информацией об ордене или `nil`, если данные недоступны.

- **`orderScore`** (`number`) - очки ордена.

- **`orderTopPosition`** (`number`) - позиция в ТОПе.

## Примеры

### Получение информации об ордене

```lua
local orderInfo = order.GetOrderStats()
```
