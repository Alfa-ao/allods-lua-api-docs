# order.GetOrdersList

Возвращает список орденов.

## Описание

```lua
order.GetOrdersList(): table | nil
```

Функция возвращает таблицу с порядковыми номерами и названиями орденов, или `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - таблица, содержащая порядковые номера и названия орденов, или `nil`.

## Примеры

### Получение списка орденов

```lua
local ordersList = order.GetOrdersList()
```
