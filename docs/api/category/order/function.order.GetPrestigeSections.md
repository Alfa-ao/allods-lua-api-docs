# order.GetPrestigeSections
Возвращает список разделов по престижу ордена.

## Описание
```lua
order.GetPrestigeSections(): table | nil
```
Функция извлекает и возвращает список разделов по престижу ордена. В случае отсутствия данных возвращается `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `table` | `nil` - список разделов по престижу ордена или `nil`, если данные отсутствуют.

- **`name`** (`WString`) - название раздела.

- **`raitingEventId`** (`ObjectId`) - идентификатор события престижа.
