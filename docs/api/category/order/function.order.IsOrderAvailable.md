# order.IsOrderAvailable

Проверяет доступность вступления в орден.

## Описание

```lua
order.IsOrderAvailable(): boolean
```

Функция возвращает логическое значение, указывающее на доступность вступления в орден.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если доступно вступление в орден, иначе `false`.

## Примеры

### Проверка доступности вступления

```lua
local isOrderAvailable = order.IsOrderAvailable()
```
