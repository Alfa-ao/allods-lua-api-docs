# order.SetMainPlayerOrder
Устанавливает орден главного игрока.

## Описание
```lua
order.SetMainPlayerOrder( orderNumber: number )
```
Функция устанавливает указанный орден для главного игрока.

## Список параметров
- **`orderNumber`** (`number`) - Порядковый номер игрока.

## Возвращаемые значения
Возвращаемое значение отсутствует.
