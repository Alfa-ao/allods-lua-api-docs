# poweredLSWeaponsLib.EquipItem

Надевает предмет в соответствующий слот.

## Описание

```lua
poweredLSWeaponsLib.EquipItem( itemId: ObjectId )
```

Функция осуществляет надевание предмета в соответствующий слот экипировки.

## Список параметров

- **`itemId`** (`ObjectId`) - Уникальный идентификатор предмета, который необходимо надеть.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Надевание предмета

```lua
poweredLSWeaponsLib.EquipItem( itemId )
```
