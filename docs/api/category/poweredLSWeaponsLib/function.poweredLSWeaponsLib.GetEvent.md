# poweredLSWeaponsLib.GetEvent
Возвращает идентификатор рула (эвента) для прокачанного оружия.

## Описание
```lua
poweredLSWeaponsLib.GetEvent(): RuleId | nil
```
Функция возвращает идентификатор рула (эвента) для прокачанного оружия. В случае отсутствия возможности получить идентификатор (не в миссии) вернётся `nil`.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `RuleId` | `nil` - идентификатор рула (эвента) для прокачанного оружия. В случае отсутствия возможности получить идентификатор (не в миссии) вернётся `nil`.

## Примеры
### Получение идентификатора рула
```lua
local ruleId = poweredLSWeaponsLib.GetEvent()
```
