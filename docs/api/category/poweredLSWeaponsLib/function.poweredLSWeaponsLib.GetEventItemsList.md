# poweredLSWeaponsLib.GetEventItemsList

Возвращает индексированный с единицы список предметов прокачанного оружия для соответствующего события (эвента).

## Описание

```lua
poweredLSWeaponsLib.GetEventItemsList(): table | nil
```

Функция извлекает и возвращает список идентификаторов предметов прокачанного оружия, если для текущего события (эвента) предусмотрена их прокачка.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - индексированный с единицы список идентификаторов (`ObjectId`) предметов прокачанного оружия, если событие активно; `nil`, если событие отсутствует.

## Примеры

### Получение списка предметов

```lua
local itemsIds = poweredLSWeaponsLib.GetEventItemsList()
```
