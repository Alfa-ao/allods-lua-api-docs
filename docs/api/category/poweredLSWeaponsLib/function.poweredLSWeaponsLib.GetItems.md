# poweredLSWeaponsLib.GetItems

Возвращает индексированный с 1 список фейковых предметов, которые могут попасть в контейнер прокачанного оружия.

## Описание

```lua
poweredLSWeaponsLib.GetItems(): table
```

Функция возвращает индексированный с 1 список фейковых предметов, которые могут попасть в контейнер прокачанного оружия.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - список, индексация с 1, каждый элемент которого представляет собой `table` со следующими полями:

- **`item`** (`ObjectId`) - идентификатор предмета.

- **`inactiveItem`** (`ObjectId` | `nil`) - идентификатор деактивированной версии предмета.

## Примеры

### Получение информации о предмете

```lua
local items = poweredLSWeaponsLib.GetItems()
local itemInfo = itemLib.GetItemInfo( items[ 1 ].item )
```

::: info Описание примера

В примере извлекается список фейковых предметов, после чего для первого элемента списка получается полная информация о предмете с помощью функции `itemLib.GetItemInfo`.

:::
