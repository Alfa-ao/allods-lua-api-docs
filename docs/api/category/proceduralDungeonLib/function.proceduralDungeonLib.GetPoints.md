# proceduralDungeonLib.GetPoints

Возвращает список ключевых точек для выбранного задания процедурного данжа или все точки, если идентификатор задания не задан.

## Описание

```lua
proceduralDungeonLib.GetPoints( goalId: ObjectId | nil ): table | nil
```

Функция извлекает и возвращает массив ключевых точек, связанных с конкретным заданием процедурного данжа. Если параметр `goalId` не передан или равен `nil`, возвращается список всех доступных точек.

## Список параметров

- **`goalId`** (`ObjectId` | `nil`)

Идентификатор задания процедурного данжа.

## Возвращаемые значения

Возвращает `table` | `nil` - Массив координат точек в игровом пространстве, или `nil` в случае отсутствия данных. Индексация элементов начинается с 1.

- **`[index]`** (`GamePosition`)

Координаты точки в игровом пространстве.

## Примеры

### Получение всех точек

```lua
local allPoints = proceduralDungeonLib.GetPoints()
```
