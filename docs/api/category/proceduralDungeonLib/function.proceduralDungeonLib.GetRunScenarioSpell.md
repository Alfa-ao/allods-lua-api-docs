# proceduralDungeonLib.GetRunScenarioSpell

Возвращает идентификатор спелла для запуска следующего сценария процедурного данжа.

## Описание

```lua
proceduralDungeonLib.GetRunScenarioSpell(): SpellId | nil
```

Функция возвращает идентификатор ресурса спелла для запуска следующего сценария процедурного данжа.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `SpellId` | `nil` - идентификатор ресурса спелла.

## Примеры

### Получение и запуск спелла сценария

```lua
local spellId = proceduralDungeonLib.GetRunScenarioSpell()
if spellId then
    avatar.RunSpell( spellId )
end
```

::: info Описание примера
В примере сначала извлекается идентификатор спелла для следующего сценария процедурного данжа. Если идентификатор успешно получен, инициируется выполнение данного спелла от имени аватара.
:::
