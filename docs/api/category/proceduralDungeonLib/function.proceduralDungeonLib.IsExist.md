# proceduralDungeonLib.IsExist
Проверяет нахождение аватара в процедурном данже.

## Описание

```lua
proceduralDungeonLib.IsExist(): boolean
```

Функция возвращает логическое значение, указывающее на текущее местоположение аватара. Возвращает `true`, если аватар находится внутри процедурного данжа, и `false` в противном случае.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если аватар находится в процедурном данже, иначе `false`.

## Примеры

### Проверка нахождения в данже

```lua
if proceduralDungeonLib.IsExist() then
    LogInfo( "avatar in procedural dungeon" )
end
```
