# raid.Accept

Принимает приглашение в рейд.

## Описание

```lua
raid.Accept()
```

Функция принимает активное приглашение в рейд.

::: warning Замечание
Вызов функции в отсутствии активного приглашения считается ошибкой.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Согласие на приглашение

```lua
raid.Accept()
```
