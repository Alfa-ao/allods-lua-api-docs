# raid.CanInvite

Проверяет возможность приглашения указанного персонажа в рейд.

## Описание

```lua
raid.CanInvite( unitId: ObjectId | nil ): boolean
```

Функция проверяет, разрешено ли приглашение персонажа в рейд. Если рейд отсутствует, возвращается `false`. Если параметр `unitId` не передан, проверяется общая возможность приглашения кого-либо в рейд.

## Список параметров

- **`unitId`** (`ObjectId` | `nil`) - Идентификатор персонажа, приглашаемого в рейд.

## Возвращаемые значения

Возвращает `boolean` - `true`, если можно пригласить указанного персонажа или если возможно приглашение в принципе, иначе `false`.

## Примеры

### Проверка и приглашение персонажа

```lua
if raid.CanInvite( unitId ) then
    raid.Invite( unitId )
end
```

::: info Описание примера

В примере выполняется проверка возможности приглашения персонажа. Если функция возвращает `true`, инициируется отправка приглашения через `raid.Invite`.

:::
