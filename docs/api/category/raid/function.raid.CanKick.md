# raid.CanKick

Проверяет возможность выгона игроков из рейда.

## Описание

```lua
raid.CanKick(): boolean
```

Функция возвращает статус возможности выгона участников из рейда.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если можно выгонять игроков из рейда, и `false` в противном случае.

## Примеры

### Проверка возможности выгона

```lua
local canKick = raid.CanKick()
```
