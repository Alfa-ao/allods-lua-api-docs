# raid.CanLeave

Проверяет возможность покинуть рейд.

## Описание

```lua
raid.CanLeave(): boolean
```

Функция возвращает логическое значение, указывающее на возможность покинуть рейд.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если можно покинуть рейд.

## Примеры

### Проверка возможности и выход из рейда

```lua
if raid.CanLeave() then
    raid.Leave()
end
```

::: info Описание примера
В примере сначала проверяется возможность покинуть рейд. Если функция возвращает истинное значение, инициируется выход из рейда.
:::
