# raid.CreateSmall()

Создает малый рейд, численностью 12 участников. Рейд создается лидером группы.

## Синтаксис

```lua
raid.CreateSmall()
```

### Параметры

Функция не принимает параметров.

### Возвращаемые значения

Функция не возвращает значений.

## Пример использования

```lua
raid.CreateSmall()
```
