# raid.Decline

Отказывается от приглашения в рейд.

## Описание

```lua
raid.Decline()
```

Функция инициирует отказ от приглашения в рейд.

::: warning Замечание
Должно быть активное приглашение. Вызов функции в отсутствии приглашения считается ошибкой.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Отказ от приглашения

```lua
raid.Decline()
```
