# raid.GetInviteInfo

Извлекает информацию о приглашении аватара в рейд.

## Описание

```lua
raid.GetInviteInfo(): table
```

Функция возвращает таблицу с информацией о текущем приглашении аватара в рейд.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` с информацией о приглашении:

- **`invited`** (`boolean`) - Наличие активного приглашения в рейд.

- **`inviterName`** (`WString`) - Имя приглашающего. Если приглашения нет - пустая строка.

- **`inviterShardName`** (`WString`) - Сервер приглашающего. Если приглашения нет - пустая строка.

## Примеры

### Проверка и принятие приглашения в рейд

```lua
if raid.GetInviteInfo().invited then
    raid.Accept()
end
```

::: info Описание примера
В примере выполняется проверка наличия активного приглашения в рейд. Если приглашение существует, инициируется его принятие.
:::
