# raid.GetLootMaster
Возвращает уникальный идентификатор ответственного за добычу рейда.

## Описание
```lua
raid.GetLootMaster(): UniqueId | nil
```
Функция возвращает уникальный идентификатор ответственного за добычу рейда. Уникальный идентификатор может быть пустым в случае отсутствия ответственного.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `UniqueId` | `nil` - уникальный идентификатор ответственного за добычу рейда, или `nil` в случае его отсутствия.

## Примеры
### Получение идентификатора ответственного
```lua
local lootMaster = raid.GetLootMaster()
```
