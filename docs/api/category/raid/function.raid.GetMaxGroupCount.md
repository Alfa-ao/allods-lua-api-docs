# raid.GetMaxGroupCount
Возвращает максимальное количество групп в рейде.

## Описание
```lua
raid.GetMaxGroupCount(): number | nil
```
Функция возвращает максимальное количество групп в рейде.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` | `nil` - количество групп, или `nil`, если рейд не существует.

## Примеры
### Получение максимального количества групп
```lua
local maxGroupCount = raid.GetMaxGroupCount()
```
