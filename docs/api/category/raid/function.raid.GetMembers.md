# raid.GetMembers

Возвращает упорядоченный список членов рейда.

## Описание

```lua
raid.GetMembers(): table
```

Функция возвращает упорядоченный список групп, входящих в рейд.

::: warning Замечание
Если аватар не состоит в рейде, вызов функции считается ошибкой.
:::

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` - список групп, входящих в рейд.

- **`key`** (`number`) - индекс группы в списке.

- **`value`** (`table`) - список игроков в группе:
  - **`key`** (`number`) - индекс персонажа в списке `[1..]`.
  - **`value`** (`table`) - таблица с информацией о персонаже:
    - **`id`** (`ObjectId` | `nil`) - идентификатор персонажа или `nil`, если его нет в игре (порвалась связь и т.п.).
    - **`uniqueId`** (`UniqueId` | `nil`) - уникальный постоянный идентификатор персонажа или `nil`, если информация отсутствует.
    - **`name`** (`WString`) - имя персонажа.
    - **`className`** (`string`) - системное название класса игрока (нелокализуемое).
    - **`classLocalName`** (`WString`) - лорное название класса игрока (локализуемое).
    - **`level`** (`number` | `nil`) - уровень игрока или `nil`, если он не известен в данный момент.
    - **`state`** (`number`(`RAID_MEMBER_STATE_*`)) - состояние игрока.

## Примеры

### Получение списка членов рейда

```lua
local members = raid.GetMembers()
```
