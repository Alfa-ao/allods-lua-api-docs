# raid.GetReadyCheckInfo

Возвращает информацию о текущем состоянии проверки готовности рейда.

## Описание

```lua
raid.GetReadyCheckInfo(): table|nil
```

Функция извлекает и возвращает информацию о текущем состоянии проверки готовности рейда. Если активный или завершенный запрос о готовности отсутствует, функция возвращает `nil`.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `table` | `nil` - информация о текущем состоянии проверки готовности рейда. Если запрос отсутствует, возвращается `nil`.

При наличии запроса возвращается таблица со следующими полями:

- **`isInProgress`** (`boolean`) - Флаг активности запроса о готовности.

- **`remainingTimeMs`** (`number`) - Оставшееся время до окончания активного запроса о готовности в миллисекундах.

- **`initiatorId`** (`UniqueId`) - Уникальный идентификатор инициатора запроса.

- **`members`** (`table`) - Таблица, индексированная с `0`, содержащая информацию о готовности членов рейда.

Вложенные поля таблицы `members`:

- **`uniqueId`** (`UniqueId`) - Уникальный идентификатор члена рейда.

- **`state`** (`number`(`READY_CHECK_READY_STATE_*`)) - Состояние готовности.

## Примеры

### Получение информации о проверке готовности

```lua
local info = raid.GetReadyCheckInfo()
if info then
    if info.isInProgress then
        LogInfo( "Remaining time (msec): ", info.remainingTimeMs )
    end
end
```
