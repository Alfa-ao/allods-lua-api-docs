# raid.Invite

Приглашает указанного персонажа в рейд.

## Описание

```lua
raid.Invite( unitId: ObjectId )
```

Функция инициирует отправку запроса на приглашение указанного персонажа в рейд.

## Список параметров

- **`unitId`** (`ObjectId`) - Идентификатор персонажа, приглашаемого в рейд.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Приглашение персонажа в рейд

```lua
raid.Invite( targetUnitId )
```
