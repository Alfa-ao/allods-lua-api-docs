# raid.InviteByName

Предпринимает попытку пригласить указанного персонажа в рейд по его имени.

## Описание

```lua
raid.InviteByName( name: WString )
```

Функция предпринимает попытку пригласить указанного персонажа в рейд по его имени.

## Список параметров

- **`name`** (`WString`) - Имя персонажа, приглашаемого в рейд.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Приглашение персонажа в рейд

```lua
raid.InviteByName( playerName )
```
