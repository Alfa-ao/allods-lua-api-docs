# raid.IsAutomatic

Определяет, является ли отряд автоматически собранным.

## Описание

```lua
raid.IsAutomatic(): boolean
```

Функция возвращает признак автоматического сбора отряда. Автоматически собранный рейд невозможно распустить, а исключение участников из него недоступно.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `boolean` - `true`, если рейд собран автоматически.

## Примеры

### Проверка автоматического сбора отряда

```lua
local isAutomatic = raid.IsAutomatic()
```
