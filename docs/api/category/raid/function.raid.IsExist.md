# raid.IsExist
Определяет, находится ли аватар в каком-либо рейде.

## Описание
```lua
raid.IsExist(): boolean
```
Функция возвращает признак нахождения аватара в активном рейде.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `boolean` - `true`, если аватар находится в рейде, и `false` в противном случае.

## Примеры
### Проверка наличия рейда
```lua
local isExist = raid.IsExist()
```
