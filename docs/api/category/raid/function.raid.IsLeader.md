# raid.IsLeader

Проверяет, является ли объект лидером рейда, в котором состоит аватар.

## Описание

```lua
raid.IsLeader( id: ObjectId | UniqueId | WString | nil ): boolean
```

Функция возвращает `false`, если аватар не состоит в рейде. Если аватар состоит в рейде и аргумент не передан, функция проверяет, является ли аватар лидером рейда. 

::: warning Замечание
Проверка по нику (`WString`) неустойчива к коллизии ников на межсерверных активностях.
:::

## Список параметров

- **`id`** (`ObjectId` | `UniqueId` | `WString` | `nil`) - Идентификатор проверяемой сущности. Если передано значение `nil`, проверяется аватар.

## Возвращаемые значения

Возвращает `boolean` - является ли объект лидером рейда.

## Примеры

### Проверка лидерства аватара

```lua
local isLeader = raid.IsLeader()
```
