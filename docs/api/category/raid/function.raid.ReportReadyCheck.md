# raid.ReportReadyCheck

Уведомляет рейд о готовности или неготовности игрока к действиям для текущего активного запроса.

## Описание

```lua
raid.ReportReadyCheck( ready: boolean )
```

Функция отправляет уведомление рейду о готовности или неготовности игрока к действиям для текущего активного запроса.

## Список параметров

- **`ready`** (`boolean`) - Флаг готовности игрока к действиям.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Подтверждение готовности

```lua
raid.ReportReadyCheck( true )
```
