# social.GetBackMount
Забирает ездовое животное главного игрока у побратима.

## Описание
```lua
social.GetBackMount( mountId: ObjectId )
```
Функция забирает ездовое животное главного игрока у побратима, используя заданный идентификатор животного из конюшни.

## Список параметров
- **`mountId`** (`ObjectId`)
Идентификатор ездового животного из конюшни.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Возврат ездового животного от побратима
```lua
local mountId = 12345 -- Идентификатор животного из конюшни
social.GetBackMount( mountId )
```
