# social.GiveBackMount
Возвращает ездовое животное главного игрока обратно побратиму, у которого оно было ранее взято.

## Описание
```lua
social.GiveBackMount( mountId: ObjectId )
```
Функция инициирует процесс возврата ездового животного его владельцу (побратиму). В качестве параметра передается уникальный идентификатор животного из конюшни. Запрос на выполнение операции отправляется на сервер.

## Список параметров
- **`mountId`** (`ObjectId`)
Уникальный идентификатор ездового животного из конюшни, которое необходимо вернуть.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Возврат ездового животного побратиму
```lua
local mountId = 123456 -- Идентификатор ездового животного из конюшни
social.GiveBackMount( mountId )
```
