# userMods.SendSelfChatMessage

Выводит сообщение в чат.

## Описание

```lua
userMods.SendSelfChatMessage( message, messageLevel )
```

Выводит сообщение в чат в каналы группы Системные -> Дополнения. Сообщение не пересылается другим игрокам.

::: warning Замечание
Функцию можно использовать только в пользовательских дополнениях (userMods).
:::

## Список параметров

- **`message`** (`WString`)  
Текст сообщения.

- **`messageLevel`** (`string` | `nil`)  
Имя канала, в который будет выведено сообщение. По умолчанию `"message"`.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Вывод сообщения об ошибке

```lua
userMods.SendSelfChatMessage( addonErrorMessage, "error" )
```

## Поддерживаемые каналы

На данный момент поддержаны следующие каналы (уровни сообщений):

| Канал (`messageLevel`) | Описание | Цвет (LogColor) |
|------------------------|----------|-----------------|
| `message`              | Сообщения | `LogColorWhite` |
| `notice`               | Уведомления | `LogColorYellow` |
| `warning`              | Предупреждения | `LogColorOrange` |
| `error`                | Ошибки | `LogColorRed` |
