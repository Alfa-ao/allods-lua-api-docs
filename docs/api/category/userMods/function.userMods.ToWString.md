# userMods.ToWString

Возвращает локализуемую строку.

## Описание

```lua
userMods.ToWString( sysText )
```

Возвращает локализуемую строку по обычной.

::: warning Замечание
Функцию можно использовать только в пользовательских дополнениях (userMods).
:::

## Список параметров

- **`sysText`** (`string`)  
  Исходная строка.

## Возвращаемые значения

Возвращает `WString` — исходную строку, преобразованную в локализуемый текст.

## Примеры

### Пример #1 Пример использования `ToWString`

```lua
local text = userMods.ToWString( "Debug information" )
```
