# ButtonSafe:ClearValues
Очищает все подставленные значения в ValuedText-е кнопки.

## Описание
```lua
ButtonSafe:ClearValues()
```
Функция очищает все ранее подставленные значения в элементе `ValuedText`, ассоциированном с кнопкой. Метод сбрасывает динамически отображаемые данные, возвращая текстовое содержимое кнопки к исходному состоянию без подстановок.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Очистка значений текстового поля кнопки
```lua
wtButton:ClearValues()
```
