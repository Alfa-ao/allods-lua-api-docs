# ButtonSafe:GetVariant
Возвращает идентификатор активного варианта кнопки.

## Описание
```lua
ButtonSafe:GetVariant(): number
```
Функция извлекает и возвращает числовой идентификатор активного варианта кнопки. Предназначена для работы с элементами управления (Widgets, Buttons), поддерживающими несколько вариантов отображения. Например, позволяет определить текущее состояние CheckBox, реализованного в виде кнопки с двумя вариантами (обычный и с галочкой).

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — числовой идентификатор активного варианта кнопки.

## Примеры
### Получение текущего варианта кнопки
```lua
local varID = wtButton:GetVariant()
```
