# ButtonSafe:GetVariantCount
Возвращает количество доступных вариантов отображения для кнопки.

## Описание
```lua
ButtonSafe:GetVariantCount(): number
```
Функция извлекает и возвращает целочисленное значение, определяющее общее количество вариантов (состояний) визуального отображения для конкретного экземпляра кнопки. Метод применяется для работы с элементами интерфейса, поддерживающими множественные состояния, например, для переключателей (CheckBox), где один вариант соответствует обычному виду, а другой — виду с установленной галочкой.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращает `number` — целое число, представляющее количество вариантов кнопки.

## Примеры
### Получение количества вариантов кнопки
```lua
local variantCount = wtButton:GetVariantCount()
```
