# ButtonSafe:SetVariant
Устанавливает активный вариант кнопки.

## Описание
```lua
ButtonSafe:SetVariant( variant: number )
```
Метод применяется для работы с кнопками, имеющими несколько вариантов. Выполняет переключение активного состояния указанного элемента управления на заданный порядковый номер.

## Список параметров
- **`variant`** (`number`)
Порядковый номер варианта кнопки. Нумерация начинается с нуля.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Установка нулевого варианта кнопки
```lua
wtButton:SetVariant( 0 )
```
