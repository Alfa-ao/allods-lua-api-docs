# Control3DSafe:AddWidget3D

Добавляет 2D-контрол как Widget3D в контрол типа Control3DSafe.

## Описание

```lua
Control3DSafe:AddWidget3D( widget: WidgetSafe, size: table, pos: GamePosition, autoResizeX: boolean, autoResizeY: boolean, cutDistance: number, bindPointY: number, minSizeLimit: number, maxSizeLimit: number )
```

Функция добавляет указанный 2D-контрол как Widget3D в контрол типа Control3DSafe. Добавляемый 2D-контрол или его дочерние контролы не могут сами быть Control3DSafe. Допускается задавать оба 3D-размера вручную, либо один из них рассчитывается автоматически, исходя из размера исходного 2D-контрола. Основной 3D-контрол находится в аддоне Main и имеет имя MainScreenControl3D.

::: warning Замечание
Функция возвращает ошибку, если выполняется любое из следующих условий:

- Добавляемый дочерний виджет является внутренним.
- Добавляемый дочерний виджет является формой.
- Добавляемый дочерний виджет в данный момент находится в контейнере или добавлен в другую сцену как Widget3D.
:::

## Список параметров

- **`widget`** (`WidgetSafe`) - 2D-контрол, который добавляется как Widget3D.

- **`size`** (`table`):
    - `sizeX` (`number`) - Размер контрола-3D по X в метрах.
    - `sizeY` (`number`) - Размер контрола-3D по Y в метрах.

- **`pos`** (`GamePosition`) - Позиция контрола-3D на карте.

- **`autoResizeX`** (`boolean`) - Флаг автоматического расчёта ширины объекта. Нельзя включать одновременно с `autoResizeY`.

- **`autoResizeY`** (`boolean`) - Флаг автоматического расчёта высоты объекта. Нельзя включать одновременно с `autoResizeX`.

- **`cutDistance`** (`number`) - Расстояние в метрах, на котором контрол перестаёт отображаться.

- **`bindPointY`** (`number`) - Точка привязки контрола-3D по оси Y. Значения соответствуют перечислению `WIDGET_3D_BIND_POINT_*`.

::: details Допустимые значения
- `WIDGET_3D_BIND_POINT_LOW` - Привязка наименьшей координаты (верхний край).
- `WIDGET_3D_BIND_POINT_CENTER` - Привязка центра.
- `WIDGET_3D_BIND_POINT_HIGH` - Привязка наибольшей координаты (нижний край).
:::

- **`minSizeLimit`** (`number`) - Коэффициент, задающий минимальный абсолютный размер контрола (его реальный размер в пикселях на экране). При достижении этого размера контрол перестаёт уменьшаться при отдалении объекта. Абсолютный минимальный размер контрола - его виртуальный размер, умноженный на данный коэффициент с учётом отношения реального и виртуального размеров экранов.

- **`maxSizeLimit`** (`number`) - Коэффициент максимального размера контрола. Аналогичен минимальному.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Добавление 2D-контрола как Widget3D

```lua
wtControl3D:AddWidget3D( widget, size, pos, true, false, 100.0, WIDGET_3D_BIND_POINT_HIGH, 0.5, 1.5 )
```
