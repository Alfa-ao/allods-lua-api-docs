# DiscreteSliderSafe:GetPos

Возвращает текущую позицию дискретного слайдера.

## Описание

```lua
DiscreteSliderSafe:GetPos(): number
```

Функция возвращает текущую позицию дискретного слайдера.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - текущую позицию дискретного слайдера в диапазоне от `0` до значения, возвращаемого функцией `GetStepsCount()`.

## Примеры

### Получение текущей позиции слайдера

```lua
local pos = wtDiscreteSlider:GetPos()
```
