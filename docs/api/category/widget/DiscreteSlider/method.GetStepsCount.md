# DiscreteSliderSafe:GetStepsCount

Возвращает количество разбиений диапазона значений дискретного слайдера.

## Описание

```lua
DiscreteSliderSafe:GetStepsCount(): number
```

Функция возвращает количество разбиений диапазона значений дискретного слайдера. Значения слайдера находятся в диапазоне от `0` до возвращаемого значения, то есть число позиций на 1 больше количества разбиений.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - количество разбиений диапазона слайдера.

## Примеры

### Получение количества разбиений слайдера

```lua
local count = wtDiscreteSlider:GetStepsCount()
```
