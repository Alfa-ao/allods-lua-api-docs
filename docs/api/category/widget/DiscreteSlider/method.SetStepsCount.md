# DiscreteSliderSafe:SetStepsCount

Задает диапазон (количество разбиений) дискретного слайдера.

## Описание

```lua
DiscreteSliderSafe:SetStepsCount( count: number )
```

Функция задает диапазон (количество разбиений) дискретного слайдера. Значения слайдера находятся в диапазоне от `0` до `GetStepsCount`, то есть число позиций на 1 больше количества разбиений.

## Список параметров

- **`count`** (`number`) - диапазон слайдера.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка количества разбиений

```lua
wtDiscreteSlider:SetStepsCount( 4 )
```
