# EditBoxSafe:IsOverflow
Определяет, превышает ли объем текста размеры текстового поля EditBox.

## Описание
```lua
EditBoxSafe:IsOverflow()
```
Функция анализирует текстовое поле EditBox и определяет, не выходит ли содержащийся в нем текст за его видимые границы.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения
Возвращаемое значение отсутствует.

## Примеры
### Отображение полосы прокрутки при переполнении
```lua
if wtEditBox:IsOverflow() then
    wtEditBox:ShowScrollbar( true )
end
```

::: info Описание примера
В примере проверяется состояние переполнения текстового поля. Если текст не помещается в видимой области, вызывается метод отображения полосы прокрутки.
:::
