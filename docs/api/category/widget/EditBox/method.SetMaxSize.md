# EditBoxSafe:SetMaxSize

Задает максимальное количество символов для поля ввода.

## Описание

```lua
EditBoxSafe:SetMaxSize( maxSize: number | nil )
```

Функция устанавливает предельное количество символов, которые можно ввести в текстовое поле.

## Список параметров

- **`maxSize`** (`number` | `nil`) - Новое максимальное количество символов. При передаче `nil` устанавливается предельно допустимое значение.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Установка лимита символов

```lua
wtEditBox:SetMaxSize( 40 )
```
