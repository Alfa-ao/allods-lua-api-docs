# EditBoxSafe:ShowScrollbar

Скрывает или отображает полосу прокрутки виджета EditBox.

## Описание

```lua
EditBoxSafe:ShowScrollbar( show: boolean )
```

Функция управляет видимостью полосы прокрутки виджета EditBox. Для корректного отображения полосы прокрутки требуется, чтобы она была определена в ресурсном файле виджета. По умолчанию полоса прокрутки отображается.

## Список параметров

- **`show`** (`boolean`) - Флаг видимости полосы прокрутки. Значение `true` отображает полосу прокрутки, `false` - скрывает.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Скрытие полосы прокрутки

```lua
wtEditBox:ShowScrollbar( false )
```
