# EditLineSafe:BackspaceCharAtCursorPos

Удаляет символ перед курсором или выделенный фрагмент текста.

## Описание

```lua
EditLineSafe:BackspaceCharAtCursorPos()
```

Функция удаляет один символ, расположенный непосредственно перед текущей позицией курсора в строке ввода. Если в поле ввода `EditLine` присутствует выделенный фрагмент текста, функция удаляет весь выделенный диапазон вместо одного символа. Действие эквивалентно нажатию клавиши Backspace.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Удаление символа в поле ввода

```lua
wtEditLine:BackspaceCharAtCursorPos()
```
