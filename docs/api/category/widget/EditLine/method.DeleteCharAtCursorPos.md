# EditLineSafe:DeleteCharAtCursorPos

Удаляет символ после курсора.

## Описание

```lua
EditLineSafe:DeleteCharAtCursorPos()
```

Если в EditLine выделен фрагмент текста, данный фрагмент уничтожается. Действие аналогично нажатию клавиши Delete.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Удаление символа после курсора

```lua
wtEditLine:DeleteCharAtCursorPos()
```
