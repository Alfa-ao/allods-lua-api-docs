# EditLineSafe:GetInitialGlobalClass

Возвращает класс, задающий исходное оформление текста в EditLine из ресурсной системы.

## Описание

```lua
EditLineSafe:GetInitialGlobalClass(): WString
```

Функция возвращает класс, задающий исходное оформление текста в EditLine из ресурсной системы.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `WString` - класс, задающий исходное оформление текста в EditLine из ресурсной системы.

## Примеры

### Получение исходного класса оформления

```lua
local name = wtEditLine:GetInitialGlobalClass()
```
