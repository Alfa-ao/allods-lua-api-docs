# EditLineSafe:GetPosObjectCount

Возвращает допустимое количество объектов для задания позиции курсора в EditLine.

## Описание

```lua
EditLineSafe:GetPosObjectCount(): number
```

Функция возвращает количество объектов, которое является последним допустимым индексом для курсора. Допустимые позиции курсора лежат в диапазоне от 0 до значения, возвращаемого функцией.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `number` - количество объектов, оно же последний допустимый индекс для курсора.

## Примеры

### Установка курсора в конец EditLine

```lua
local posObjectCount = wtEditLine:GetPosObjectCount()
wtEditLine:SetPos( posObjectCount )
```

::: info Описание примера
В примере запрашивается допустимое количество объектов в виджете EditLine. Полученное значение используется для установки позиции курсора в самый конец текстового поля посредством вызова метода SetPos.
:::
