# EditLineSafe:GetString
Функция возвращает текст, содержащийся в строке ввода, в виде обычной строки, отличной от WString.

## Описание
```lua
EditLineSafe:GetString(): string
```
Если строка ввода содержит пароль, функция возвращает визуальный текст в виде звездочек. Для использования оригинального текста следует применять специальные функции для каждого конкретного случая.

## Список параметров
Параметры отсутствуют.

## Возвращаемые значения

Возвращает `string` - Текст, содержащийся в строке ввода.

## Примеры
### Получение текста из строки ввода
```lua
local text = wtEditLine:GetString()
```
