# EditLineSafe:GetText

Возвращает текст, содержащийся в строке ввода.

## Описание

```lua
EditLineSafe:GetText(): WString
```

Функция возвращает текст, содержащийся в строке ввода. Если строка ввода содержит пароль, возвращается визуальный текст в виде звездочек. Для использования оригинального текста применяются специальные функции для каждого конкретного случая.

## Список параметров

Параметры отсутствуют.

## Возвращаемые значения

Возвращает `WString` - Текст, содержащийся в строке ввода.

## Примеры

### Получение текста из строки ввода

```lua
local text = wtEditLine:GetText()
```
