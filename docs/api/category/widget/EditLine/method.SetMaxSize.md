# EditLineSafe:SetMaxSize

Задает максимальное количество символов для строки ввода.

## Описание

```lua
EditLineSafe:SetMaxSize( maxSize: number|nil )
```

Функция задает максимальное количество символов для строки ввода. При передаче `nil` устанавливается предельное значение.

## Список параметров

- **`maxSize`** (`number` | `nil`) - Новое максимальное количество символов или предельное значение, если передано `nil`.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

### Ограничение длины строки ввода

```lua
wtEditLine:SetMaxSize( 40 )
```
