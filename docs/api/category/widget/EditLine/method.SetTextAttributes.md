# EditLineSafe:SetTextAttributes
Задает атрибуты указанного тега в строке редактирования.

## Описание
```lua
EditLineSafe:SetTextAttributes( isRTag: boolean, tagName: WString, tagAttributes: table )
```

Поведение, параметры и возвращаемые значения идентичны [ValuedText:SetTagAttributes](ValuedText:SetTagAttributes)
