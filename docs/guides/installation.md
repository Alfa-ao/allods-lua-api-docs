# SetClassVal

Задает значение для подстановки класса.

## Описание

```lua
TextViewSafe:SetClassVal( tag, value )
```

Задает значение для подстановки класса (из тега "rs") для данного `TextViewSafe`.

::: warning Замечание
В текущей реализации нет проверки, что данное значение не будет использоваться для подстановки значений (в теге "r").
:::

## Список параметров

- **`tag`** (`string` | `WString`)  
  Тег, для которого задается значение.

- **`value`** (`string` | `WString`)  
  Значение тега.

## Возвращаемые значения

Возвращаемое значение отсутствует.

## Примеры

#### Пример #1 Пример использования `SetClassVal`

```lua
wtTextView:SetClassVal( "log_format", "log_red" )
```
